# 伦理与安全边界

Human Infra 的核心对象是人，不是可压榨的产出机器。所有理论、数据、产品和研究域都必须先保护人的生命、尊严、主体性、退出权和长期自由度。

## 总原则

1. 人是目的，不是工具。
2. 基础设施首先保护人的可持续能动性。
3. 任何量化系统都不能替代人的主观体验和尊严。
4. 不能因为能测量，就认为应该测量。
5. 不能因为能优化，就认为应该优化。
6. 不能把系统性问题重新甩锅给个体。
7. 不能用效率牺牲健康、自由、创造力和长期生命质量。
8. 高风险神经与生命支持研究必须以同意、可退出、可验证、可审计和最小伤害为硬边界。

## 医疗与健康边界

- 项目可以整理公开证据、临床试验和安全风险。
- 项目不能给出医疗诊断、个性化用药建议或治疗方案推荐。
- 任何健康干预结论都必须保留来源、适用对象、证据等级、不确定性和风险边界。
- 机制猜想不能写成确定疗效。

## 组织使用边界

Human Infra 不能被设计成更精密的压榨系统。

禁止方向：

- 用状态监控强迫人持续高负荷工作。
- 用效率指标否定人的恢复、疾病、情绪和生活边界。
- 把任务失败全部归因于个人不努力。
- 用低级 KPI 管理科研、创作、战略判断等高复杂定性任务。

允许方向：

- 帮助个体看见自己的运行规律。
- 帮助团队减少无意义打断、返工和认知负担。
- 帮助高风险职业建立疲劳管理、交接流程、错误报告和事故复盘。
- 帮助组织把“人不行”改写为“哪条运行依赖链断了”。

## 去具身中枢生命系统边界

`disembodied-cns` 是概念与证据建模域，不是实验操作手册。

可以做：

- 系统边界建模。
- 生命支持、接口、控制器、传感器反馈和执行器的高层架构描述。
- 主体连续性、可表达性、可退出性和尊严保护原则。
- 风险分类、验证问题和证据目录。

不能做：

- 人体或动物侵入性实验步骤。
- 绕过伦理审查的操作建议。
- 可直接执行的神经组织维持、操控或人体改造流程。
- 宣称当前技术已经能实现脱离身体的长期人格保存。

## 未来等待边界

`future-waiting` 关注“用较少主观时间等待未来”的思想实验和高层路径建模，不是可执行太空工程或人体保存方案。

可以做：

- 时间膨胀、休眠、冷冻、相对论旅行等等待路径的概念对比。
- 公开物理理论、科普材料和论文的非操作性整理。
- 风险、反证点、主观时间成本和主体连续性问题建模。
- 明确哪些条件仍停留在思想实验层。

不能做：

- 黑洞接近、极端轨道控制、太空任务或人体实验的可执行步骤。
- 宣称某种相对论时间膨胀路径已经是现实永生方案。
- 把“物理上存在的时间膨胀”直接写成“工程上可安全使用”。
- 隐去潮汐力、辐射、生命支持、通信、返航和事件视界风险。

## 记忆编辑边界

可以做：

- 记忆编辑的概念边界。

不能做：

- 诱导、删除、写入或重构真实个体记忆的可执行操作。
- 绕过知情同意的操控方案。
- 面向控制、胁迫、洗脑、审讯或剥夺主体性的设计。
- 把记忆干预包装成低风险消费级功能。

- 提供法律意见、合规认证、HIPAA/Common Rule 适用判断、真实 PHI/基因/神经/行为数据处理架构、去标识化保证、数据共享许可、隐私绕过或安全攻击方法。
- 提供个体量表分数解释、诊断、残障资格、保险待遇、生活质量结论、机构排名、治疗选择、投资、报销、覆盖或资源拒绝建议。
- 提供疫苗医学建议、禁忌判断、个体免疫状态解释、疫情预测保证、公共卫生规避、病原体操作、传播策略或政治化动员工具。
- 提供孕产、新生儿、儿童诊断、治疗、喂养、发育评估、药物、育儿处方、家庭排名、福利资格、保险筛选或家庭监控建议。
- 提供医疗事故法律意见、索赔策略、个案责任判断、机构背书、惩罚员工、压制报告、甩锅患者或临床操作替代方案。
- 把数据、结局、成本、公共卫生、早期生命或患者安全的结构性问题改写成个人努力、家庭道德、机构宣传或单一指标排名问题。

## 数据与证据边界

- 原始数据、清洗数据和人工判断必须分层保存。
- 公开展示的每条结论必须能追溯到来源。
- 不把 AI 总结当成事实源。
- 不把引用量、营销热度或机制想象当成证据质量。
- 高风险结论必须保留不确定性，而不是追求叙事完整。

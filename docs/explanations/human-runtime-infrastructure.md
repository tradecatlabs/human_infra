# 人类运行基础设施

人类运行基础设施是一套以人类任务执行为中心的工程体系。它把人类看作受生理、认知、情绪、环境、工具、资源、时间和社会关系共同影响的复杂运行系统。

它的目标是让人在复杂任务中维持更高质量、更稳定、更长周期、更低损耗的运行状态。把人变成更高效机器的管理思路不属于本框架。

## 核心模型

```text
Human Performance = f(T, V, E, R, F)
```

- `T = Task`：任务本身，包括目标、问题定义、难度、风险和反馈周期。
- `V = Variable set`：人的内部变量，包括知识、经验、能力、方法、智力、元认知、注意力、动机、情绪和精力。
- `E = Environment`：环境条件，包括空间、噪音、光照、工具、打断和协作界面。
- `R = Resource`：外部资源，包括时间、金钱、数据、导师、合作者、设备和渠道。
- `F = Feedback`：反馈机制，包括测试、评审、复盘、指标、同行意见和恢复信号。

Human Infra 的基本任务，是让这些变量进入“任务可执行区间”。

## 七层结构

### L0 生存与安全基础设施

安全、食物、水、住所、医疗、疾病预防、职业安全和暴露风险控制。

没有 L0，其他层都不稳定。

### L1 生理基础设施

睡眠、饮食、运动、光照、温度、疼痛管理、药物管理、慢病管理、体能和感官条件。

生理状态是任务执行系统的底层硬件。

### L2 认知基础设施

注意力保护、工作记忆管理、长期记忆系统、知识管理、笔记系统、任务分解、决策框架、checklist、SOP 和 AI 辅助工具。

认知基础设施的核心是降低无效认知负担，把高级认知资源留给真正困难的部分。

### L3 情绪与动机基础设施

压力管理、情绪调节、意义感、目标感、奖励结构、社会支持、安全感、失败缓冲和孤独风险控制。

情绪是人类运行状态的一部分。

### L4 环境与工具基础设施

工作空间、噪音控制、屏幕、输入设备、网络、软件工具、自动化、文件结构、通讯边界、协作工具和版本管理。

好的工具缩短从意图到执行的路径。

### L5 社会与资源基础设施

金钱、时间、数据、信息渠道、导师、同伴、合作者、组织结构、权限、实验条件、传播渠道和反馈网络。

资源是能力能否转化为结果的必要条件。

### L6 观测、反馈与恢复基础设施

任务日志、状态记录、睡眠记录、精力记录、错误记录、周期复盘、预警指标、恢复计划、降级策略和失败复盘。

恢复是长期运行能力的一部分。

## 诊断框架

当一个人无法完成任务时，先做系统诊断，避免人格归因：

1. 目标是否清晰？
2. 任务是否被正确定义？
3. 是否具备必要知识？
4. 是否具备合适方法？
5. 是否有足够连续时间？
6. 是否有足够精力？
7. 注意力是否被保护？
8. 情绪是否可承受？
9. 环境是否支持任务？
10. 工具是否可用？
11. 资源是否齐备？
12. 是否有反馈？
13. 是否有恢复机制？

## Human SRE

Human SRE 是把软件可靠性工程迁移到人类任务系统的类比框架。

| 软件 SRE | Human SRE |
| --- | --- |
| uptime | 可持续工作能力 |
| latency | 启动时间、进入状态时间 |
| throughput | 有效产出量 |
| error rate | 错误率、返工率、判断失误率 |
| observability | 睡眠、精力、情绪、注意力、任务日志 |
| incident | 崩溃、过劳、失眠、项目停摆 |
| postmortem | 复盘哪条依赖链断裂 |
| capacity planning | 时间、精力、注意力和资源规划 |
| graceful degradation | 状态差时降级为低认知负荷任务 |

Human SRE 的目的，是让人看见自己的运行规律，并保护长期创造力。监控人的用法不属于本框架。

## 产品化方向

- `Human Runtime Dashboard`：展示状态、任务适配度、风险预警、恢复建议和长期趋势。
- `Task Readiness System`：在任务开始前评估目标、方法、时间、精力、环境、反馈和资源是否到位。
- `Human Incident Review`：对拖延、崩溃、失误、过劳和项目停摆做系统复盘。
- `Personal Operating System`：把任务、知识、目标、时间、精力、情绪、反馈、复盘、资源和 AI 助手统一建模。

## 核心底线

Human Infra 服务于人的展开，排除组织控制人的用法。

增强人，避免榨干人。
保护人，避免规训人。
支持人，避免监控人。
让人更自由，避免让人更像机器。

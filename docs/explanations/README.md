# Explanations

Explanations are understanding-oriented documents. They answer why the project exists, how ideas connect, and what conceptual model should guide future work.

Current documents:

- `effective-immortality-acceleration-flywheel.md`: working paper draft for the subject-continuity acceleration flywheel created by effective immortality, including a running example, formal problem setup, variables, hypotheses, evidence matrix, toy model, threat model, and beginner research route.
- `human-runtime-infrastructure.md`: Human Infra's core model and diagnostic frame.
- `life-path-prediction-model.md`: conceptual explanation for the quantitative life-path prediction layer.
- `noah-ark-genealogy.md`: genealogy of the Noah's Ark metaphor as Human Infra's top-level continuity narrative, including method, problem, concept, theory, infrastructure, technology, institution, resilience, data/model governance, counter-genealogy, and project translation layers.
- `value-lenses.md`: multiple value lenses for Human Infra, including subject continuity, lifetime resource-budget expansion, and anti-scarcity legitimacy.

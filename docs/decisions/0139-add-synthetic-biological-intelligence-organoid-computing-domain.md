# Add Synthetic Biological Intelligence Organoid Computing Domain

Date: 2026-07-02

Status: Accepted

## Context

The local literature draft `sbi_oi_literature_review_draft.md` synthesizes the Cortical Labs / DishBrain / SBI / OI packet. The material does not fit cleanly into existing organoid disease-modeling, BCI governance, AI agency safety, or disembodied-CNS domains.

Its core object is an embodied in vitro neural system: living neural tissue coupled to sensors, stimulation, software, and real-time feedback so it can be studied as an information-processing and learning substrate.

## Decision

Create `domains/c2-source-maintenance/synthetic-biological-intelligence-organoid-computing-continuity/` with:

- `README.md`
- `AGENTS.md`
- `literature-review.md`

Index the domain through root docs, `domains/`, `docs/reference/`, the Research Domain Atlas, Research Domain Radar, ethics boundaries, applications/literature, and repository structure checks.

## Rationale

This domain passes the Research Domain Atlas contract:

- Independent mechanism: closed-loop wetware/hardware/software systems alter learning, adaptation, feedback, agency-like signals, and moral-risk assessment.
- Independent evidence system: the literature uses DishBrain, CL API, CL1, multi-electrode arrays, organoid intelligence, active inference, sample efficiency, drug-response tasks, and ethics papers.
- Independent failure modes: over-claiming consciousness, confusing sentience with phenomenal consciousness, conflating platform capability with intelligence, and turning wetware systems into unsafe experimental recipes.
- Independent ethical boundary: human neural tissue, donor rights, potential moral status, public communication, commercial platform claims, and experimental exit conditions require domain-specific governance.

## Consequences

Human Infra now treats SBI/OI as a biological-digital cognition infrastructure domain:

```text
living neural substrate
  -> closed-loop digital embodiment
  -> measurable learning/adaptation
  -> agency/sentience terminology boundary
  -> ethical governance
  -> external cognitive infrastructure
```

This does not create a consciousness claim, mind-upload path, subject-transfer claim, clinical recommendation, or experimental protocol.

## 使用规则

- 稳定结论提炼到 `docs/` 或对应 `domains/*/README.md`。
- 本目录文件可以保留原始表达，但文件名必须语义化。
- 不从本目录直接驱动产品、脚本或数据模型。
- 论文原文缓存放在仓库外显式忽略的 `.research/literature/`，本目录只保存可追溯清单和资料卡片。

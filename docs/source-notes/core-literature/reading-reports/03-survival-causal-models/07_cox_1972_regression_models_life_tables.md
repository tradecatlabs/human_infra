Cox 比例风险模型
  -> KM 曲线不能直接建模多协变量影响
  -> 生存数据有删失且个体有解释变量
  -> Cox 将 hazard 分解为任意基线风险与协变量效应
  -> 通过部分似然估计系数
  -> 系数解释为风险比
  -> 因此生存分析从曲线估计扩展到回归建模

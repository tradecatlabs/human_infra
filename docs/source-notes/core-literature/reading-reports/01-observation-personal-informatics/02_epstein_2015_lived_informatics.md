# 三、第二遍阅读：掌握主要内容与支持证据

## 1. 结构化摘要

1. **研究问题**：既有行为改变导向模型能否解释不同目标用户如何在日常生活中使用自我追踪工具
2. **研究动机**：很多人追踪数据并不只是为了行为改变，还可能为了社交、奖励、好奇、习惯或记录生活。
3. **核心方法或观点**：调查体力活动、财务和位置追踪者，并访谈 22 名用户，从 lived informatics 视角建模
4. **实验或论证设计**：调查 physical activity N=105、finances N=99、location N=83，并进行 22 次访谈。
5. **主要结果**：提出 lived informatics 模型，覆盖 deciding、selecting、collection、integration、reflection、lapse 和 resume 等过程。
6. **作者结论**：个人信息学应被理解为嵌入日常生活、动机多样且会中断/恢复的实践，而不是单一行为改变流程。
7. **论文意义**：修正了早期个人信息学模型中过度目标导向的假设。

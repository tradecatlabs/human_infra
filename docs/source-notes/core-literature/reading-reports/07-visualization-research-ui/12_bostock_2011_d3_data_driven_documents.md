```markdown
基于 DOM 的数据驱动可视化框架 D3
  -> Web 可视化需要整合 HTML/CSS/JS/SVG
  -> 隐藏 scenegraph 会降低透明性和生态兼容
  -> D3 直接操作标准 DOM
  -> 数据被绑定到任意文档元素
  -> 动态 transforms 支持生成、更新、动画和交互
  -> 因此 D3 在表达力、调试和性能上有优势
```

# 长期照护保险与福利启动连续性目录说明

## 目录结构

```text
long-term-care-insurance-benefit-activation-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义长期照护保险、ADL/认知触发、等待期、claim 和照护福利支付如何影响失能照护连续性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀长期照护保险机制、触发条件、照护计划、claim 和保单失效边界，不写个案购买或理赔策略。
- 涉及美国制度时优先使用 NAIC、州保险监管机构、ACL、Medicare、Medicaid 和官方老龄照护资料。
- 不把本域写成保险推荐、照护机构选择、Medicaid 规划、遗产规划、保险欺诈、发票造假或法律建议。

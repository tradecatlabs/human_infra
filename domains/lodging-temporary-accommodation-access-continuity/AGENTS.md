# 临时住宿与住宿接入连续性目录说明

## 目录结构

```text
lodging-temporary-accommodation-access-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义酒店、短租、临时安置、无障碍住宿和住宿诈骗如何影响离家后的恢复基地。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀住宿接入机制、无障碍边界、灾害临时安置和消费者风险，不写订房、退款、住宿选择或安全处置建议。
- 涉及美国制度时优先使用 ADA.gov、FTC、USA.gov、FEMA 和公开消费者保护资料。
- 不把本域写成酒店推荐、短租争议、入住规避、占住、逃费、定位或个人安全策略入口。

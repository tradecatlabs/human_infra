# Housing Built Environment Stability

`housing-built-environment-stability/` 研究住房稳定性和建成环境能否支撑主体长期存在、恢复、行动、照护和服务接入。

> 核心问题：主体需要一个不会持续制造风险、排除和损耗的生活空间；住房与建成环境不是背景，而是健康、照护、移动、恢复和尊严的基础接口。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须在具体空间中睡眠、恢复、饮食、行动、照护和连接服务
  -> 住房和建成环境必须稳定、安全、可负担、可达且适配能力变化
  -> 长寿和健康寿命才不会被居住风险、环境暴露和空间排除截断
```

## 关注对象

- 住房稳定性、住房质量、室内空气、温湿度、拥挤、霉菌、铅、噪声和安全。
- 无障碍、可步行性、社区资源、绿地、交通连接、热风险和灾害暴露。
- 住房不安全、无家可归、搬迁、驱逐、住房负担和建成环境健康差异。
- 与交通、照护、医疗可及、气候风险和社会服务的接口。

## Human Infra 模型链路

```text
住房与建成环境稳定
  -> 睡眠、恢复、营养、卫生、照护和社区连接获得稳定空间
  -> 环境暴露、跌倒、热/冷风险、拥挤和服务断点减少
  -> 健康寿命、有效时间、照护可行性和行动半径提高
  -> 主体持续性获得稳定生活底座
```

## 非目标

- 不提供房地产、租房、买房、装修、维修、法律、福利或住房申请建议。
- 不做个体环境医学诊断、检测解释或危险处置指南。
- 不把住房责任个体化，忽视政策、收入、社区和公共基础设施条件。

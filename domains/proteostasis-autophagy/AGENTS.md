# proteostasis-autophagy 目录说明

`domains/proteostasis-autophagy/` 保存蛋白稳态与自噬研究域。它关注蛋白质量控制和细胞清除系统如何支撑长期功能。

## 目录结构

```text
proteostasis-autophagy/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责蛋白稳态、自噬、溶酶体、蛋白聚集和质量控制机制。
- 本域可整理机制文献、疾病交叉、指标边界和安全审查问题。
- 本域不提供药物、禁食、补剂、自噬激活或治疗建议。

## 维护规则

- 任何自噬主张必须区分自噬通量、标志物变化、功能改善和长期终点。
- 与神经、肌肉、免疫、代谢和营养感知交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

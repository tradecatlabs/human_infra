# stalking-harassment-protection-order-continuity

`stalking-harassment-protection-order-continuity` 研究 stalking、harassment、threats、protective orders、restraining orders、no-contact orders、service/enforcement、address confidentiality 和数字/线下交叉风险如何影响主体行动自由、住所安全、通信安全和制度保护。

## Scope

- Stalking、harassment、credible threat、protective order、restraining order、no-contact order、order service、violation reporting 和 enforcement interface。
- 数字跟踪、位置隐私、社交媒体骚扰、设备滥用、地址保密和跨辖区承认的公开资料。
- 与家庭暴力、性暴力、校园安全、工作场所安全和在线安全交叉的制度边界。

## Continuity Logic

```text
持续跟踪、骚扰或威胁
-> 日常移动、通信、住所、工作、学校和心理恢复被持续压缩
-> 保护令、服务送达、执行、地址保密和数字安全成为制度保护接口
-> 保护链断裂会把单次风险变成长期自由和注意力消耗
```

## Source Spine

- DOJ OVW stalking and protection-order related materials.
- CDC stalking and intimate partner violence materials.
- OVC victim assistance and rights resources.
- State court and victim service public information, when used only as comparative system material.
- FTC / CISA digital safety materials where harassment intersects with account and device safety.

## Evidence Boundary

本域只整理公开 stalking、harassment、protective order、地址保密和制度执行资料。

Do not include:

- 个案保护令申请、送达、报警、诉讼、取证、反跟踪、对抗或安全计划建议。
- 跟踪、定位、监控、骚扰、doxxing、规避保护令、规避送达、规避执法或报复方法。
- 可识别个人、地址、学校、工作地点、庇护所、服务点或法院案件的信息。

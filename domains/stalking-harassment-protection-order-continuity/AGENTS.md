# stalking-harassment-protection-order-continuity 目录说明

本目录维护跟踪、骚扰与保护令连续性域。它关注持续威胁如何压缩主体行动自由，以及保护令、执行、地址保密和数字安全资料如何成为制度保护接口。

```text
stalking-harassment-protection-order-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义 stalking、harassment、protective order、address confidentiality、数字/线下交叉风险和禁止用途。
- `AGENTS.md`：说明本目录职责和维护边界。

## 维护规则

- 新增资料必须优先来自 DOJ OVW、CDC、OVC、FTC、CISA、州法院或公开受害者服务机构。
- 必须区分公共制度信息和个案保护令、送达、报警、取证、安全计划。
- 不保存可识别个人、住址、学校、工作地点、服务点、法院案件或实时位置。
- 不输出跟踪、定位、监控、骚扰、doxxing、规避保护令、规避送达、规避执法或报复方法。

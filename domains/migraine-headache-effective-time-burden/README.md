# Migraine Headache Effective Time Burden

`migraine-headache-effective-time-burden/` 研究偏头痛和头痛障碍如何通过疼痛、感官敏感、认知雾、发作不可预测性和社会功能受限，影响 Human Infra 的有效时间、学习和工作连续性。

> 核心问题：主体持续性不只看死亡风险。偏头痛等头痛障碍可能长期吞噬有效时间，使主体在“活着但不可行动”的周期中损失学习、工作、关系和恢复窗口。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 日历寿命必须转化为可行动、可学习、可创造的有效时间
  -> 偏头痛和头痛障碍会反复制造疼痛、光声敏感、恶心、脑雾和不可预测停摆
  -> 若发作频率、触发因素、治疗可及、工作学习调整和污名问题被忽略，健康寿命会被高负担低死亡风险疾病侵蚀
  -> 因此偏头痛和头痛负担是有效时间损耗域
```

## 关注对象

- 偏头痛、紧张型头痛、丛集性头痛、慢性每日头痛、药物过度使用性头痛和头痛相关残疾。
- 发作频率、严重度、触发因素、感官敏感、认知影响、工作/学习缺勤、照护负担和治疗可及性。
- 与 `pain-suffering-control/` 的边界：疼痛域关注痛苦整体；本域关注头痛障碍特有的发作、感官、认知和有效时间损耗机制。
- 与 `occupational-work-design/` 的边界：工作域关注组织设计；本域关注偏头痛如何改变任务窗口和合理调整需求。

## Human Infra 模型链路

```text
偏头痛与头痛负担 T
  -> 改变发作频率、疼痛强度、感官敏感、认知雾、治疗可及和环境触发变量 X
  -> 改变有效时间、工作学习连续性、恢复需求和生活质量状态 S
  -> 改变失能天数、收入损失、社交退缩、心理压力和药物过度使用风险 λ(t)
  -> 影响创造密度、长期能力积累和主体持续性
```

## 非目标

- 不提供头痛诊断、药物建议、急症判断、触发因素个体处方或产品推荐。
- 不替代神经科、疼痛医学、职业健康、学校/工作合理便利或临床指南。
- 不把头痛日记、可穿戴数据或自测量表写成个人诊断。

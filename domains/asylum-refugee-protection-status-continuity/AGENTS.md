# asylum-refugee-protection-status-continuity 目录说明

`asylum-refugee-protection-status-continuity/` 是 Human Infra 的庇护、难民与保护状态连续性域，负责研究人道保护身份如何在原制度失效时维持主体安全和恢复路径。

## 目录结构

```text
asylum-refugee-protection-status-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义庇护、难民与保护状态连续性域的对象、链路、非目标、来源信号和下一步。
- 本域只整理公开保护框架、变量、风险边界和证据入口，不写个案救济路线。

## 维护规则

- 新增资料必须说明它影响保护身份、等待期、工作许可、家庭团聚、安置、医疗教育接入、数据保护还是法律援助。
- 优先使用 UNHCR、USCIS、EOIR、IOM、OCHA、IDMC、WHO 和官方人道机构资料。
- 禁止输出庇护、难民、临时保护、边境、证据、面谈、上诉、旅行、工作许可或法律建议。
- 禁止提供危险迁移、藏匿、伪造材料、规避程序、暴露受保护者身份或削弱保护安全的方法。

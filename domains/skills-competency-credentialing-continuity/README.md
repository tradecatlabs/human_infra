# Skills Competency Credentialing Continuity

Skills Competency Credentialing Continuity 关注技能、能力、胜任力、微证书、开放徽章、技能本体和可迁移能力证明如何被定义、评估、组合和携带，使主体能把实际会做的事情转成跨任务、跨雇主、跨平台的可验证资产。

## 对象

- Skills taxonomies、competency frameworks、micro-credentials、digital badges、assessment records、portfolio evidence 和 skills-based hiring。
- O*NET、ESCO、Credential Transparency Description Language、Open Badges、learning outcomes 和 competency-based education。
- 技能测评、工作样本、项目作品、认证考试、能力档案、AI 辅助技能推断和误分类风险。

## Human Infra 价值

```text
主体不只需要学习机会，还需要能力被准确表达
-> 技能框架把具体任务能力拆成可比较、可组合、可迁移的变量
-> 微证书和能力证明把学习、工作和项目经验转成可携带信号
-> 若技能定义失真、评估偏差或平台锁定，主体能力会被低估或误配
-> 可审查的胜任力凭证让学习轮次更容易沉淀为工作、协作和技术采用能力
```

## 边界

- 本域不提供求职包装、简历优化、测评作弊、证书购买、培训推荐、职业规划、录用概率或个案能力判断。
- 不判断某微证书、徽章、课程、平台或测评是否足以代表某职业能力。
- 不提供绕过考试、伪造作品集、自动生成虚假经历或操控技能评分方法。
- 只整理公开资料、技能分类、胜任力框架、评估可靠性、互操作和治理风险。

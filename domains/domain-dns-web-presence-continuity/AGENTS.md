# domain-dns-web-presence-continuity 目录说明

本目录承载域名、DNS 与 Web 存在连续性研究域，关注公共地址、邮箱域、证书、托管和链接入口如何维持长期可达。

## 目录结构

```text
domain-dns-web-presence-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义域名、DNS、注册商、续费、证书、托管、邮箱域和 Web 可达性如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 ICANN、IANA、CISA、CA/Browser Forum、注册商和托管商公开文档。
- 不写域名抢注、DNS 操作、证书配置、网站迁移、账号恢复、品牌争议、UDRP 或安全事件处置建议。
- 涉及域名安全时只讨论制度变量和断裂模式，不处理控制台、API token、私钥、注册人资料、支付资料或事件细节。
- 新增指标必须注明口径：域名续费、DNS 完整性、证书有效性、托管可用性、邮箱域依赖或公共链接可达。

# sterilization-disinfection-infection-control 目录说明

本目录维护灭菌、消毒与感染控制域。它是 Human Infra 的“照护环境感染屏障层”，负责清洁、消毒、灭菌、再处理和流程执行如何支撑医疗与照护连续性。

## 目录结构

```text
sterilization-disinfection-infection-control/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明清洁、消毒、灭菌、再处理、环境感染控制和 HAI 预防如何影响主体持续性。
- 本域只做公开感染控制原则、组织治理、再处理监管和安全边界资料整理。
- 不提供消毒配方、灭菌参数、器械再处理、隔离决策、暴发处置或规避感染控制建议。

## 维护规则

- 新增材料必须区分清洁、消毒、灭菌、再处理、环境控制、组织执行和不良事件。
- 不把机构指南、消毒剂资料或暴发信息写成现场操作 SOP。
- 涉及可复用器械、污染或消毒剂时，只写治理框架，不写操作步骤。

# research-misconduct-investigation-correction-continuity 目录说明

本目录负责科研不端调查与纠错连续性，关注研究完整性状态如何进入 Human Infra 的证据门槛。

## 结构

```text
research-misconduct-investigation-correction-continuity/
├── AGENTS.md
└── README.md
```

## 边界

- 只维护定义、调查状态、纠错链、证据解释边界和 Source Card 字段建议。
- 上游：`domains/trustworthy-scientific-communication-peer-review/`、`domains/conflict-of-interest-funding-disclosure-continuity/`。
- 下游：系统综述、活证据更新、模型输入审查和科研传播材料。
- 不提供不端指控、举报策略、调查规避、数据伪造、论文洗稿或法律意见。

## 维护规则

使用 ORI、National Academies、COPE 和 ICMJE 作为锚点。任何不端相关表述必须区分“疑点、调查中、机构结论、出版动作和模型使用限制”。

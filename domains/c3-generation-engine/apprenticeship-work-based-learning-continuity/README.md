# Apprenticeship Work Based Learning Continuity

Apprenticeship Work Based Learning Continuity 关注注册学徒制、工作场景学习、导师制、在岗训练、工时记录、技能标准和行业认可证书如何把学习直接嵌入生产现场，并让主体通过工作获得可迁移能力。

## 对象

- Registered apprenticeship、work-based learning、on-the-job training、related instruction、journeyworker status、mentorship 和 competency progression。
- 工时记录、技能清单、行业标准、雇主赞助、学徒保护、工资进阶、结业证书和跨雇主/跨州可携带性。
- 青年、成人、转岗、退伍军人、移民、残障人士和再就业群体的工作中学习路径。

## Human Infra 价值

```text
主体需要把学习轮次嵌入真实任务环境
-> 学徒制把收入、训练、导师反馈和行业标准结合在同一条路径上
-> 工时记录、技能标准和结业证明把在岗经验转成可迁移资格
-> 若项目质量差、记录不可携带或劳动保护不足，学习会退化为廉价劳动
-> 可审查的工作中学习系统把时间、劳动和技能积累转成长期行动能力
```

## 边界

- 本域不提供学徒申请、雇主选择、工资谈判、劳动争议、移民、职业规划、法律或个案建议。
- 不判断某学徒项目、行业、工时记录、证书或导师安排是否适合某个人。
- 不提供伪造工时、绕过项目要求、规避劳动保护或取得证书的操作方法。
- 只整理公开资料、项目结构、质量标准、权益边界、记录连续性和可迁移性。

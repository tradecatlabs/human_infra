生命日志、个人档案与资料连续性域研究：主体如何把经历、健康记录、作品、关系、决策、偏好和证据链长期保存为可检索、可迁移、可解释、可授权的个人资料基础设施。

## 对象

- 生命日志、个人数字档案、照片、视频、文档、笔记、健康记录、通信记录、作品和项目历史。
- 文件格式、元数据、备份、校验、迁移、加密、访问控制、长期可读性和数字保存策略。
- 自我追踪、个人信息学、时间线重建、证据包、生活史、疾病史和主体状态历史。
- 个人资料对记忆外化、身份连续、研究参与、医疗转诊、AI 助手和数字代理的影响。
- 资料丢失、格式腐烂、账号锁定、隐私泄露、过度监控、选择性记录和解释偏差。

## Human Infra 价值

主体持续性不仅依赖身体活着，也依赖其历史可被找回、解释和继续使用。这个域把个人资料从“杂乱文件”提升为主体长期行动的基础设施：保存过去、支撑判断、减少重复解释，并为未来的医学、AI、法律和数字连续性提供可信上下文。

```text
个人经历和资料被长期保存
  -> 记忆、病史、作品、偏好和证据链可被检索与迁移
  -> 决策、照护、学习、AI 辅助和身份连续性的上下文更完整
  -> 主体有效时间、解释能力和未来选择权提高
```

## 边界

本域只整理公开的个人数字归档、个人信息学、数据保存、元数据、备份、访问控制和长期可读性资料。

不提供：

- 个体隐私配置、取证、监控、账号破解、数据恢复、规避平台规则或法律建议。
- 把生命日志用于强制监控、雇佣评估、保险定价、家庭控制或非自愿画像。
- 以“数字资料完整”为由替代主体同意、医疗判断、法律授权或人格连续性评估。
- 对具体存储产品、云平台、加密方案或 AI 记忆系统的背书。

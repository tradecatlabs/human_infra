`disability-access-functional-needs-personal-preparedness-continuity/` 研究残障者、辅助技术用户和有功能需求主体在灾害前如何准备通信、照护、设备、运输和支持网络。

> 核心问题：通用应急准备默认主体能听见、看见、行走、读懂、搬运、开车和自我管理；残障与功能需求会把这些默认能力变成必须显式设计的条件。

## 先验位置

```text
主体要在灾害中持续行动
  -> 必须让准备方案适配真实身体、感官、认知、通信和照护条件
  -> 通用方案若忽略功能需求，会把准备系统变成排除系统
  -> 因而残障与功能需求准备是公平主体持续性的基础约束
```

## 关注对象

- 无障碍通信、辅助技术、药品设备、个人支持网络、照护者、交通、服务动物、备用资料和机构交接。
- 残障者、慢病患者、感官障碍者、认知障碍者、行动受限者、医疗设备用户和照护者。

## Human Infra 模型链路

```text
功能需求个人准备系统 T
  -> 改变通信、移动、设备、药物、照护、交通、服务动物和支持网络变量 X
  -> 改变主体在灾害中的可执行状态 S
  -> 改变被遗漏、无法撤离、设备中断、照护断裂、服务不可达和尊严损失风险 λ(t)
  -> 改变安全退出、服务接入、恢复速度和主体控制权
```

## 非目标

- 不提供个人残障评估、照护计划、设备选择、交通安排、医疗建议、福利申请、法律权利或登记建议。
- 不收集残障状态、诊断、设备型号、照护者、住址、路线、服务动物、医疗记录或身份证明。

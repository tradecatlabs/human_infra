# personal-computing-device-lifecycle-repair-continuity 目录说明

本目录承载个人计算设备生命周期与维修连续性研究域，关注设备可用、可修、可替换和可迁移如何影响主体数字行动。

## 目录结构

```text
personal-computing-device-lifecycle-repair-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义个人设备、维修权、支持窗口、数据迁移、锁定和报废如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 FTC、NIST、EPA、厂商公开维修/支持文档和消费者保护资料。
- 不写绕过锁定、破解加密、规避 MDM、拆机攻击、保修欺诈或数据窃取方法。
- 涉及设备安全时只讨论变量、断裂模式、治理边界和消费者可及性。
- 新增指标必须注明口径：维修可得性、支持窗口、数据迁移成功率、备件可得性、替换成本或资料丢失风险。

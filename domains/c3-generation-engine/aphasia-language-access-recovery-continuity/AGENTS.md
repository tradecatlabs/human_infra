# aphasia-language-access-recovery-continuity AGENTS

## 职责

- 维护失语、语言通道、沟通参与、表达权和恢复连续性的研究边界。
- 将失语相关公开资料、指南、量表边界和沟通伙伴支持整理为 Source Card。
- 与 `speech-language-communication-continuity/`、`cerebrovascular-stroke-brain-perfusion-resilience/` 和 `augmentative-alternative-communication-aac-continuity/` 保持边界清晰。

## 非目标

- 不提供诊断、分型、训练、治疗、预后、急诊判断或法律能力判断。
- 不替代神经、言语语言治疗、康复、心理健康、法律或急诊服务。

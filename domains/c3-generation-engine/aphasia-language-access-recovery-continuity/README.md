Aphasia language access recovery continuity studies acquired language loss as a direct constraint on preference expression, consent, care participation, relationship continuity, and social role recovery.

核心问题：

> 失语不是“说话少了”，而是主体把内部意图转成可被他人理解的语言接口受损；接口断裂会让偏好、同意、求助和关系维护同时变难。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能表达需求、理解信息并参与决策
  -> 失语会扰动说、听、读、写等语言通道
  -> 医疗、法律、家庭和社会互动中的意思表达成本上升
  -> 因而失语是主体表达权与服务参与之间的语言接口失效模式
```

## 关注对象

- 卒中、脑损伤或神经疾病后的失语、语言理解、命名、阅读、书写和会话参与。
- 沟通伙伴支持、环境可达、长期恢复、身份角色和照护沟通。
- 与 `cerebrovascular-stroke-brain-perfusion-resilience/` 的关系：卒中域关注急性脑灌注；本域关注语言接口恢复。
- 与 `augmentative-alternative-communication-aac-continuity/` 的关系：AAC 域关注替代沟通工具；本域关注失语机制和参与恢复。


## 非目标

- 不提供失语诊断、分型、治疗方案、家庭训练计划、预后判断或个体康复建议。
- 不根据个人语言表现判断卒中、痴呆、TBI 或其他病因。
- 不替代神经、言语语言治疗、康复、心理健康、法律或急诊服务。

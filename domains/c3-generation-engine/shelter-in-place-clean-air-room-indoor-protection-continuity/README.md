`shelter-in-place-clean-air-room-indoor-protection-continuity/` 研究就地避险、室内空气保护、清洁空气房间、门窗密封、过滤和污染暴露控制如何影响主体持续性。

> 核心问题：当撤离不可行或短期更危险时，主体持续性取决于室内空间能否临时转化为可呼吸、可等待、可通信的保护环境。

## 先验位置

```text
主体要在外部危险中持续存在
  -> 必须在撤离不可行时降低室内暴露
  -> 烟雾、化学品、极端天气和污染会穿透住宅边界
  -> 因而就地避险与室内空气保护是等待救援或风险消退的空间条件
```

## 关注对象

- 就地避险、清洁空气房间、过滤、门窗密闭、室内污染、野火烟雾、化学释放、极端天气和通风边界。
- 呼吸系统疾病者、老人、儿童、孕产人群、室外工作者、污染/烟雾风险地区居民。

## Human Infra 模型链路

```text
就地避险与室内保护系统 T
  -> 改变室内空气、颗粒物、污染物、通风、过滤、热负荷和通信变量 X
  -> 改变主体在住所或避难空间中的暴露状态 S
  -> 改变呼吸恶化、中毒、热/冷应激、恐慌和撤离失败风险 λ(t)
  -> 改变等待窗口、健康稳定性和外部响应接入
```

## 非目标

- 不提供房屋密封、空气净化器选择、滤材安装、化学事故处置、野火烟雾个人行动或医疗建议。
- 不收集住址、房屋布局、通风系统、健康状态、空气监测数据、儿童/老人信息或实时灾害位置。

# healthcare-language-interpreter-effective-communication-continuity

## 职责

本目录承载医疗语言服务、口译、翻译材料、effective communication、CLAS 和 LEP 医疗场景资料。

## 边界

- 只研究医疗沟通制度和语言可及失败模式。
- 不提供医学翻译、口译、诊断解释、投诉代写或法律意见。
- 不保存 PHI、音视频、联系方式、语言身份、移民状态或医疗资料。

## 结构

```text
healthcare-language-interpreter-effective-communication-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分一般清晰沟通、医疗口译、辅助沟通和民权义务。
- 涉及个体医疗内容、语言服务争议或法律判断的内容必须转回官方渠道和专业人员。

# Portable Air Cleaner Room Filtration Continuity

`portable-air-cleaner-room-filtration-continuity` 研究便携式空气清洁器、房间级过滤、清洁空气递送、滤芯维护、噪声、电源和房间匹配如何影响睡眠、呼吸恢复、学习工作和照护空间。

> 核心问题：当建筑通风不足、烟雾进入、过敏原负荷升高或临时暴露发生时，房间级过滤是家庭端最低摩擦的空气恢复接口；但它只有在匹配、维护、供电和可承受时才转化为有效保护。

## 关注对象

- 便携式空气清洁器、HEPA/高效过滤、滤芯寿命、房间面积、空气混合、噪声、夜间使用、耗电和维护提醒。
- 卧室、儿童房、老人照护房、家庭办公空间、临时清洁空气房和烟雾/污染事件期间的房间级防护。
- 与 HVAC 滤网、野火烟雾、避难式清洁空气房、室内通风暴发控制和能源韧性的接口。
- 过滤设备可及性如何影响睡眠、注意力、哮喘/COPD 负荷、家庭照护和灾害等待能力。

## Human Infra 链路

```text
房间级便携过滤连续
  -> 局部空间颗粒物和空气负荷下降
  -> 睡眠、呼吸恢复、照护和居家工作学习空间更稳定
  -> 建筑通风不足或外部空气事件不直接切断家庭行动
  -> 主体在室内的有效时间和恢复能力提高
```

## 非目标

- 不提供产品推荐、CADR 处方、房间安全认证、设备安装、电气改造、滤芯购买建议、实时灾害安全判断或医疗建议。
- 不替代 EPA、CDC、ASHRAE、地方公共卫生、消防、建筑专业人员、医生或空气质量专业人员。
- 不把空气清洁器写成替代撤离、口罩、检测、治疗、源控制或建筑通风的单一方案。

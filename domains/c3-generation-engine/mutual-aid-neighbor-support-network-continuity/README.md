Mutual Aid Neighbor Support Network Continuity 研究邻里互助、社区互助网络、灾害前后互助、日常小额帮助、信任边界和资源分发如何增强主体在制度服务之外的恢复能力。

> 核心问题：很多风险发生在正式服务到达之前；邻里互助能把距离最近的人、物资、信息和照护动作变成低延迟支持，但也必须防止剥削、隐私暴露和不公平分配。

## 研究对象

- Mutual aid、neighbor support、community support network、check-in、resource sharing 和 informal care。
- 灾害互助、热浪/停电互助、送餐买药、照看、交通协助、信息转发和小额物资分发。
- 信任、资格、隐私、安全、边界、志愿者负荷和资源公平。

## 先验链路

```text
主体遇到小型危机、灾害或服务延迟时，正式机构可能无法立即到达
  -> 邻里和互助网络可提供信息、物资、照看、交通和情绪支持
  -> 若网络不存在、信任不足或边界混乱，近场支持无法启动
  -> 小问题更容易升级为医疗、财务、住房或安全危机
  -> 主体恢复时间变长，公共系统负担上升
  -> 因而互助与邻里支持是正式服务之外的低延迟恢复层
```

## 关键变量

- 互助覆盖、响应时间、资源匹配、信任边界、志愿者容量、信息准确性和安全协议。
- 灾害、停电、热浪、行动受限、低收入、语言障碍、老年/残障和隐私风险。
- 不公平分配、群体压力、诈骗、家庭暴力暴露、政治/宗教排除和志愿者倦怠。

## 证据入口

- FEMA Whole Community / community lifelines materials。
- Ready.gov neighbor and family preparedness materials。
- Red Cross, VOAD, local mutual aid and community resilience public materials。

## 非目标

- 本域不组织互助、募集资金、分配物资、调度志愿者、审核需求或提供灾害现场建议。
- 不提供绕开正式救援、私自进入危险区域、医疗照护、运输、安保或执法替代方案。
- 不收集地址、健康状况、物资需求、联系方式、家庭结构、定位或脆弱群体名单。

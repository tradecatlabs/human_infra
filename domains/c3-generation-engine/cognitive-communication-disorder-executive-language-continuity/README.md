## 研究推进骨架

### 最小问题集

- 界定被放大的通用能力资源：智力、注意力、记忆、学习、判断、执行、反馈或工具使用。
- 说明任务链路：能力变化如何改变问题求解、行动质量、创造密度或学习轮次。
- 识别复利结构：能力增强、工具采用、协作网络、经验积累和自我升级如何互相强化。
- 记录依赖条件：AI、数据、界面、训练、外部工具、社会支持或反馈系统。
- 列出误用、过载、依赖、幻觉、注意力破坏、人格边界和安全风险。

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。

- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

Cognitive communication disorder executive language continuity studies attention, memory, organization, pragmatics, and executive control in communication as constraints on work, care planning, learning, and social participation.

核心问题：

> 有些沟通失败不是词汇或发音单点坏掉，而是注意、记忆、组织、推理和执行控制不能稳定支持会话；这会让主体“能说”但难以持续完成复杂沟通任务。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能在时间展开的对话中维持目标、记忆、顺序和语境
  -> 注意、记忆、执行功能和推理负荷会影响沟通质量
  -> 医疗计划、工作协作、学习和关系协商容易断裂
  -> 因而认知沟通障碍是认知执行资源与语言互动之间的交叉失效模式
```

## 关注对象

- TBI、卒中、痴呆、脑雾或神经疾病相关认知沟通问题、组织表达、话题维持和信息管理。
- 照护计划、工作协作、教育、会议、家庭沟通和外部记忆支持。
- 与 `attention-executive-control/` 的关系：执行控制域关注通用任务资源；本域关注沟通中的执行语言。
- 与 `personal-knowledge-management-cognitive-offloading/` 的关系：知识管理域关注外部记忆；本域关注沟通任务支持。

## Human Infra 模型链路

```text
识别 / 外部记忆 / 结构化沟通 / 环境支持 T
  -> 改变注意、记忆、组织、推理和话题维持变量 X
  -> 改变会议、照护计划、学习、家庭协作和服务导航状态 S
  -> 改变计划误解、执行失败、关系冲突和机会损失风险 λ(t)
  -> 影响主体复杂沟通能力、任务执行和未来选择权
```

## 非目标

- 不提供认知沟通障碍诊断、神经病因判断、认知训练、康复计划、工作许可或能力评估。
- 不根据对话、写作、记忆表现或 AI 记录判断痴呆、TBI、脑雾或法律能力。
- 不替代神经、言语语言治疗、心理、职业治疗、康复、法律或急诊服务。

Social communication pragmatic language continuity studies conversational reciprocity, inference, context, nonliteral language, and social-pragmatic participation as constraints on relationships, school, work, and service navigation.

核心问题：

> 沟通不只是词语是否正确，还包括轮流、语境、暗示、修复、礼貌和共同注意；语用失配会让主体在看似会说话的情况下仍被关系和制度误读。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能在具体社会语境中被理解并理解他人
  -> 语用、推理、会话规则和非字面语言困难会制造隐性摩擦
  -> 学校、工作、服务、关系和冲突处理成本上升
  -> 因而社会沟通是语言形式到关系参与之间的语境接口
```

## 关注对象

- 社会沟通、语用语言、会话轮替、语境理解、非字面语言、修复策略和关系参与。
- 学校、工作、服务窗口、同伴关系、在线沟通和冲突风险。
- 与 `autism-neurodiversity-support-continuity/` 的关系：自闭症域关注神经多样性支持；本域关注语用沟通机制。
- 与 `language-access-plain-communication/` 的关系：plain communication 偏制度文本；本域偏人际语境。

## Human Infra 模型链路

```text
环境适配 / 伙伴训练 / 明确语境 / 支持工具 T
  -> 改变语用理解、会话修复、推理负荷和社会参与变量 X
  -> 改变学校、工作、服务、关系和冲突处理状态 S
  -> 改变误解、排斥、孤立和服务失败风险 λ(t)
  -> 影响关系连续性、制度参与和未来选择权
```

## 非目标

- 不提供社会沟通障碍诊断、自闭症判断、训练方案、心理干预、学校资格或法律便利判断。
- 不根据个人沟通风格判断人格、能力、危险性或工作适合度。
- 不替代言语语言治疗、心理、教育、职业、法律或社会服务。

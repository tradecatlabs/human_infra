# assistive-technology-access 目录说明

`domains/c3-generation-engine/assistive-technology-access/` 保存辅助技术与无障碍研究域。它关注工具、设备和环境适配如何补偿或放大主体功能。

## 目录结构

```text
assistive-technology-access/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责辅助技术、无障碍、义肢、助听、低视力辅助、轮椅和沟通辅助。
- 本域与 `rehabilitation-functioning/`、`sensory-continuity/` 和 `resource-social-infra/` 交叉。
- 本域不提供个体设备处方、购买建议或医疗建议。

## 维护规则

- 任何辅助技术主张必须审查可及性、适配、维护、训练、支付和环境兼容。
- 不把残障仅定义为身体缺陷；必须保留环境和制度维度。
- 新增文件或子目录时，同步更新本文件。

Assistive Technology Access 负责整理辅助技术、无障碍、义肢、助听、低视力辅助、轮椅、通信辅助和环境适配。

核心问题：

> 主体能力不只来自身体本身，也来自能否用工具补偿、替代或放大受损功能。

## 先验位置

```text
主体持续性最大化
  -> 主体可能出现感官、运动、认知或沟通能力下降
  -> 辅助技术可以把损伤从终止条件转化为可补偿条件
  -> 若辅助技术不可及，残障会扩大为参与和行动能力丧失
  -> 因此辅助技术可及是功能持续性的工具基础设施
```

## 关注对象

- 助听器、视觉辅助、轮椅、义肢、矫形器、沟通辅助和认知辅助。
- 无障碍环境、数字可访问性、适配服务、维修、支付和供应链。
- 与 `rehabilitation-functioning/` 的关系：康复恢复功能，辅助技术补偿或增强功能。

## Human Infra 模型链路

```text
辅助技术 T
  -> 改变感知、移动、沟通、自理和环境适配能力 X
  -> 改变主体可行动状态 S
  -> 降低失能、隔离、依赖和服务排除风险
  -> 增加有效时间、社会参与和未来选择权
```

## 非目标

- 不提供个体设备处方、医疗建议或购买建议。
- 不把技术可用等同于技术可及；可负担、适配、维护和训练同样重要。
- 不把残障问题缩减为个人身体缺陷。

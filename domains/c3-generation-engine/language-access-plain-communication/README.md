`language-access-plain-communication/` 研究清晰语言、语言可达、有限英语能力服务、文化与语言适配、认知可访问性和关键服务材料理解。

> 核心问题：制度、医疗、法律、福利和公共服务只有在人能够读懂、听懂、用自己的语言和认知方式完成任务时，才是真正可达。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须理解风险、权利、义务、服务流程和选择后果
  -> 关键材料必须清晰、可翻译、可解释、认知友好且文化适配
  -> 理解可达性决定资源能否转化为行动、保护和恢复
```

## 关注对象

- plain language、语言服务、有限英语能力访问、CLAS 标准、认知可访问性和表单理解。
- 医疗说明、福利通知、法律信息、就业材料、教育服务、数字服务和危机信息。
- 术语过载、机器翻译误差、阅读负担、认知负担、文化不适配和服务歧视。
- 与 `health-literacy-navigation/`、`access-to-justice-legal-aid/`、`social-protection-benefits-delivery/` 和 `digital-inclusion-connectivity/` 的接口。

## Human Infra 模型链路

```text
语言可达与清晰沟通基础设施
  -> 改变材料理解、语言服务、认知可访问性和任务完成能力
  -> 改变主体对权利、风险、服务、医学信息和行政流程的判断状态
  -> 降低误解、延误、放弃申请、错误同意、服务排除和风险沟通失败
  -> 增强行动能力、主体自主性和未来选择权
```

## 非目标

- 不提供法律、医疗、移民、福利或就业材料的个体解释。
- 不替代认证口译、专业翻译、法律顾问、临床沟通或危机服务。
- 不把“简化语言”作为删减风险、权利、限制或不确定性的理由。

# cochlear-implant-communication-continuity AGENTS

## 职责

- 维护人工耳蜗沟通连续性的研究边界、变量表、证据入口和禁止用途。
- 将候选评估、植入设备、映射编程、听觉康复、设备维护和沟通参与整理为 Source Card、机制链和模型变量。
- 与 `hearing-auditory-communication-continuity/`、`assistive-technology-access/`、`medical-device-equipment-safety-maintenance/` 保持边界清晰。

## 非目标

- 不提供候选资格、手术建议、编程参数、康复训练、品牌/设备选择、保险覆盖或个体效果预测。
- 不替代耳鼻喉、听力学、手术团队、言语语言治疗、康复、设备厂商或保险/法律服务。

## 上游

- `hearing-auditory-communication-continuity/`：听觉沟通总框架。
- `assistive-technology-access/`：辅助设备可及性和维护。
- `medical-device-equipment-safety-maintenance/`：植入和外部设备安全维护。

## 下游

- `speech-language-communication-continuity/`：语言参与和沟通训练。
- `education-access-lifelong-learning/`：学校和学习参与。
- `workplace-accommodation-return-to-work-continuity/`：工作沟通和合理便利。

# time-allocation-effective-time 目录说明

`domains/c3-generation-engine/time-allocation-effective-time/` 保存时间分配与有效时间研究域。它关注日历寿命如何转化为可支配、可恢复、可学习和可行动的时间。

## 目录结构

```text
time-allocation-effective-time/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责时间使用、时间贫困、等待成本、行政负担、恢复时间、任务摩擦和有效时间。
- 本域可整理公开时间使用调查、社会决定因素、服务可及和任务摩擦资料。
- 本域不提供个体日程、职业、家庭、法律、福利、保险或财务安排建议。

## 维护规则

- 任何时间收益主张必须说明节省对象、替代用途、分配风险和是否增加恢复/行动能力。
- 涉及工作、照护、福利、通勤或行政系统时只写公共资料和系统变量。
- 新增文件或子目录时，同步更新本文件。

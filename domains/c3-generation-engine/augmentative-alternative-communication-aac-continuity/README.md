# Augmentative Alternative Communication AAC Continuity

Augmentative alternative communication AAC continuity studies non-speech and speech-supporting communication systems as infrastructure for preference expression, consent, education, work, and care participation.

核心问题：

> AAC 不是“说不了话的补丁”，而是当自然言语不足以承载主体意图时，把表达权重新接入世界的沟通基础设施。

## 先验位置

```text
主体持续性最大化
  -> 主体必须有至少一个可靠表达通道
  -> 言语、运动、认知或感官限制可能让自然口语不足
  -> 低技术、高技术、伙伴支持和环境适配可重建表达接口
  -> 因而 AAC 是主体表达权从单一口语通道转向多通道冗余的基础设施
```

## 关注对象

- AAC、沟通板、符号、文本、语音生成设备、眼控/开关访问、伙伴训练和环境可达。
- 医疗、教育、工作、照护、紧急求助、同意表达和身份参与。
- 与 `assistive-technology-access/` 的关系：辅助技术域关注广义可及；本域关注表达通道。
- 与 `aphasia-language-access-recovery-continuity/` 的关系：失语域关注语言恢复；本域关注替代/增强接口。

## Human Infra 模型链路

```text
工具 / 训练 / 伙伴支持 / 环境集成 T
  -> 改变表达通道冗余、速度、可靠性、可携带性和维修变量 X
  -> 改变医疗参与、学习、工作、关系和紧急求助状态 S
  -> 改变无声化、误解、被代理和服务排除风险 λ(t)
  -> 影响主体自主表达、权利参与和未来选择权
```

## 非目标

- 不提供 AAC 设备处方、访问方法选择、个体评估、训练计划、保险资格或学校服务判断。
- 不根据个人障碍、运动能力、语言能力或认知能力推荐具体设备。
- 不替代言语语言治疗、辅助技术、教育、康复、保险或法律服务。

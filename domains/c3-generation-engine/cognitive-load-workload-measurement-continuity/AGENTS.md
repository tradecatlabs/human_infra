# cognitive-load-workload-measurement-continuity 目录说明

本目录承载认知负荷、任务负荷和主观工作负荷测量资料。它关注任务设计如何消耗有限认知资源，而不是个体能力评判。

## 结构

```text
cognitive-load-workload-measurement-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分 cognitive load、mental workload、subjective workload、performance load 和 stress。
- 必须记录量表适用范围、测量边界、群体/个体解释边界、工作场景和反滥用风险。
- 不保存心理诊断、学习成绩、员工监控、生产率、学校个案、医疗个案或绩效数据。

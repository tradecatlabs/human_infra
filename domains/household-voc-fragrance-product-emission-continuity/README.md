# Household VOC Fragrance Product Emission Continuity

`household-voc-fragrance-product-emission-continuity` 研究家庭 VOC、香氛、清洁产品、喷雾、空气清新剂、装修/家具释放、气味敏感和源控制如何影响室内空气、头痛刺激、注意力和居住恢复。

> 核心问题：很多室内空气负担来自日常产品和材料释放；当气味和 VOC 源不可见、不可协商或不可替代时，居住空间会变成持续刺激、冲突和执行摩擦。

## 关注对象

- VOC、甲醛等材料释放、香氛产品、空气清新剂、清洁剂、喷雾、油漆、胶黏剂、家具/地板/装修材料、储存化学品和气味投诉。
- 气味敏感、哮喘/过敏、头痛刺激、儿童、老人、租房、多户住宅、家庭办公和共享空间。
- 与家庭危废处置、清洁消毒、厨房通风、空气过滤、住房可居住性和化学安全的接口。
- 产品释放如何影响睡眠、注意力、家庭成员协作、照护、室内停留时间和恢复空间。

## Human Infra 链路

```text
家庭 VOC 与气味源控制连续
  -> 产品释放、材料释放和香氛刺激更可观测、可替换、可通风或可移出
  -> 头痛、刺激、呼吸负担、家庭冲突和清洁摩擦下降
  -> 工作学习、睡眠、照护、共享空间使用和居住恢复更稳定
  -> 主体有效时间、注意力和家庭协作能力提高
```

## 非目标

- 不提供化学品混合、装修施工、材料检测解释、产品推荐、气味纠纷处理、职业暴露合规、医学诊断、过敏治疗或法律建议。
- 不替代 EPA、CDC/NIOSH、毒物控制、医疗人员、建筑/材料专业人员、物业、住房部门或环境检测机构。
- 不判断某个气味、产品、装修材料或检测数值是否安全、是否违法或是否需要搬离。

# Periodontal Disease Systemic Inflammation Continuity

`periodontal-disease-systemic-inflammation-continuity/` 负责整理牙周病、牙齿支持结构、慢性炎症和全身交叉风险。

核心问题：

> 牙周系统是牙齿稳定、咀嚼功能和慢性炎症暴露的接口；牙周崩塌会把口腔问题转化为缺牙、疼痛、感染和长期维护失败。

## 先验位置

```text
主体持续性最大化
  -> 主体必须维持稳定咀嚼、低感染负担和可维护的身体状态
  -> 牙周组织支撑牙齿并与炎症、代谢、吸烟和医疗可及性交叉
  -> 牙周疾病会增加缺牙、咀嚼下降、疼痛感染和维护成本
  -> 因此牙周连续性是口腔结构持续和全身风险治理的接口条件
```

## 关注对象

- 牙龈炎、牙周炎、牙周支持组织、牙齿松动、缺牙风险和维护治疗。
- 牙周病与糖尿病、吸烟、慢性炎症、心血管风险叙事和证据边界。
- 口腔卫生、牙周维护、服务可及性、费用和长期随访摩擦。
- 与 `dental-caries-prevention-restoration-continuity/` 的关系：龋病关注牙体；本域关注牙周支持结构。

## Human Infra 模型链路

```text
牙周预防 / 维护服务 T
  -> 改变牙周炎症、牙齿稳定性、感染、出血、维护成本和缺牙风险变量 X
  -> 改变口腔结构和慢性风险状态 S
  -> 改变咀嚼丧失、疼痛感染、照护依赖和全身共病管理摩擦
  -> 增加有效时间、身体维护能力和长期行动稳定性
```

## 非目标

- 不提供牙周分期、探诊解释、影像判读、洁治/刮治/手术或抗生素建议。
- 不把牙周病简化为全身疾病的单一原因。
- 不替代牙周专科、牙医、内分泌或戒烟相关专业服务。

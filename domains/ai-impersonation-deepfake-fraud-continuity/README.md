# AI Impersonation Deepfake Fraud Continuity

AI Impersonation Deepfake Fraud Continuity 关注 AI 语音、视频、文本和身份冒充如何影响主体的金钱、声誉、关系、安全、工作和机构信任。

## 对象

- AI impersonation、deepfake scam、voice cloning、video impersonation、business/government impersonation、family-emergency scam 和 verification workflow。
- 冒充亲友、同事、雇主、客服、政府、金融机构、医疗机构、律师、媒体或平台支持的社会工程路径。
- 判断压力、紧急感、权威感、情感勒索、多渠道交叉验证失败和举报恢复链路。

## Human Infra 价值

```text
主体持续性依赖关系和机构信任
-> AI 冒充降低了伪装声音、面孔、语气和身份的成本
-> 冒充会把信任、亲情、工作流程、支付和身份核验转成攻击面
-> 验证流程、延迟决策、官方渠道和举报恢复能降低不可逆损失
-> 主体更能保护金钱、声誉、关系和行动安全
```

## 边界

- 本域不提供 deepfake 制作、声音克隆、提示词、脚本、社工话术、诈骗流程、检测规避或目标选择方法。
- 不判断某通话、视频、账号或消息是否诈骗，不替代执法、金融机构、平台或法律服务。
- 不生成验证脚本、反诈话术、诱捕方案、公开曝光材料或报复方案。
- 只整理公开警示、冒充模式、验证边界、恢复入口和禁止用途。

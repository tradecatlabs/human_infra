# bppv-positional-vertigo-continuity AGENTS

## 职责

- 维护 BPPV、位置性眩晕、头位触发、姿势切换和跌倒暴露的研究边界。
- 将 BPPV 资料整理为位置触发、行动回避、活动半径和有效时间损耗 Source Card。
- 与 `vestibular-balance-spatial-orientation/`、`fall-risk-prevention-home-safety-continuity/` 和 `gait-balance-assessment-fall-screening-continuity/` 保持边界清晰。

## 非目标

- 不提供诊断、体位试验解释、复位手法、康复动作、药物、驾驶许可或急诊建议。
- 不替代耳鼻喉、神经、急诊、前庭康复或物理治疗服务。

## 上游

- `vestibular-balance-spatial-orientation/`：前庭和平衡总框架。
- `sensory-continuity/`：空间输入和感官反馈。

## 下游

- `fall-risk-prevention-home-safety-continuity/`：跌倒风险和家庭安全。
- `time-allocation-effective-time/`：眩晕回避造成的有效时间损耗。

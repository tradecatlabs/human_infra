# 记忆编辑

本子域研究对生物神经系统中记忆痕迹进行读取、写入、删除、重构或调制时涉及的高层模型、证据、验证和伦理边界。

一句话定位：

> 记忆编辑不是控制一个人的工具，而是一个必须以主体连续性、知情同意、可验证性、可退出性和尊严为硬边界的高风险研究域。

## 核心对象

- 记忆痕迹：特定经验、语义、情绪关联或行为倾向在神经系统中的可变表征。
- 读写接口：对记忆相关神经动态进行观察、解释或调制的技术边界。
- 人格连续性：干预前后主体是否仍能保持自我叙事、价值、偏好和权利连续。
- 验证系统：确认干预结果、失败模式、副作用和可逆性的证据框架。
- 治理系统：同意、退出、审计、监督、责任和滥用防护。

## 研究问题

- 记忆、情绪、身份和人格之间的边界如何定义？
- 什么算读取、删除、写入、重构、弱化或再巩固干预？
- 如何证明某次干预没有破坏主体连续性？
- 如何区分治疗性干预、增强性干预和控制性操控？
- 如何建立足够强的同意、撤回、审计和责任机制？

## 安全边界

本子域不提供任何可执行的神经操作、人体实验或记忆操控步骤。

禁止方向：

- 未经同意的记忆读取、写入、删除或重构。
- 面向控制、胁迫、审讯、洗脑或剥夺主体性的设计。
- 把高风险神经干预包装成低风险消费功能。
- 用不完整证据宣称能精确操控真实个体记忆。

允许方向：

- 建立概念边界和证据目录。
- 整理公开文献中的高层技术路线和风险类型。
- 设计非操作性的伦理、审计和验证框架。
- 与 `disembodied-cns` 共同维护主体连续性和尊严原则。

## 与 Human Infra 的关系

记忆编辑属于 Human Infra 的认知基础设施极端边界。普通认知基础设施强调笔记、知识系统、反馈和 AI 工具；记忆编辑则触及生物神经系统本身，因此必须拥有更严格的安全、伦理和证据门槛。

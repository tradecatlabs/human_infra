# Brain Preservation Connectomics Emulation

`brain-preservation-connectomics-emulation/` 研究脑保存、连接组学、神经结构可读性、全脑仿真和 mind uploading 相关假设，如何影响 Human Infra 的人格连续性、信息保存和远期去具身路径。

> 核心问题：如果主体连续性部分依赖大脑结构、连接、动态和记忆痕迹，那么远期“保存或重建主体”的任何说法，都必须先回答脑信息能否被足够完整地保存、读取、解释和重新实现。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体连续性依赖大脑、记忆、人格、行动策略和自我模型
  -> 若生物身体失败，远期路径可能诉诸脑结构保存、连接组读取或仿真实现
  -> 若保存不可逆损坏、读取不充分或身份理论不成立，上传叙事不能代表主体连续
  -> 因此脑保存、连接组和仿真是信息连续性假设的远期审查域
```

## 关注对象

- 脑保存、神经组织超微结构、连接组学、全脑扫描、神经数据压缩和长期可读性。
- 全脑仿真、mind uploading、数字主体、人格连续性和“复制 vs 连续”的哲学边界。
- 与 `biostasis-cryopreservation/` 的边界：生物停滞域关注整体结构保存和未来复温/修复；本域关注大脑信息是否可读、可解释、可仿真。
- 与 `disembodied-cns/` 的边界：去具身中枢域关注维持活中枢生命系统；本域关注保存后读取和仿真假设。
- 与 `digital-legacy-data-succession/` 的边界：数字遗产保存外部资料；本域关注脑内主体信息。

## Human Infra 模型链路

```text
脑保存与连接组仿真 T
  -> 改变神经结构保真度、记忆痕迹可读性、扫描分辨率、模型可解释性和身份连续性变量 X
  -> 改变主体信息是否可保存、重建、验证和治理的状态 S
  -> 改变不可逆信息损失、伪连续、复制误判和治理失控风险 λ(t)
  -> 影响远期去具身路径、人格连续性边界和主体持续性假设
```

## 非目标

- 不声称 mind uploading、全脑仿真或数字永生已经可行。
- 不提供脑保存操作、人体实验、遗体处理、神经数据采集或上传执行方案。
- 不把数字化身、聊天机器人或个人档案写成主体连续性证明。

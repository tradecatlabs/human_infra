```text
研究域对象
  -> 影响变量 / 中间机制
```

## 研究推进骨架

### 最小问题集

- 界定本域直接改写的主体持续性边界：寿命、死亡、身份连续性、时间差分或未来抵达能力。
- 说明主体连续性条件：身体、记忆、人格、行动能力或法律身份中哪些必须保持。
- 列出不可越过的中止门槛：退出失败、连续性失败、尾部风险、不可逆损伤或治理失效。
- 说明它如何改变有效寿命、有效时间、未来选择权或可能性空间。
- 明确本域不能被宣传成现实永生、工程可行性或个体行动方案。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

### 最小产出

- Source Signals：记录候选资料、来源、日期、主张和待核验点。
- Source Cards：提取 claim、变量、机制、证据类型、边界、反证条件和可迁移位置。
- Claim-Evidence Matrix：把每个稳定主张绑定到来源、证据等级、适用范围和不确定性。
- 变量表：列出输入变量、中间变量、状态变量、风险变量、输出指标和可观测代理指标。
- 上下游关系：说明本域依赖哪些更根本域，并向哪些转化、应用或基础设施域输出。

本子域研究一种远期生命形态：自然身体和器官依赖被外部生命支持、传感器、控制器、执行器和神经接口替代，仅保留生物脑或中枢神经系统作为主体连续性的核心载体。

推荐名称：

> 去具身外部维持型中枢生命系统

或更技术化地说：

> 外置生命支持与外部执行器耦合的离体神经中枢系统

## 核心对比

现有人类可以称为“具身半自主开放生命系统”：

```text
生物脑
+ 身体器官
+ 肢体执行系统
+ 感觉系统
+ 自主代谢系统
+ 与外界进行物质 / 能量 / 信息交换
```

目标形态可以称为“去具身外部维持型中枢生命系统”：

```text
生物脑 / 中枢神经系统
+ 人工生命支持系统
+ 电信号输入输出接口
+ 外部控制器
+ 外部执行器
+ 传感器反馈
```

一句话：

> 现有人类是“身体作为主体接口”的生命系统；目标形态是“外部系统作为身体代理”的中枢神经系统。

## 接口架构

### 自然身体接口架构

```text

↓

↓
身体接口：肺、胃肠、皮肤、感官
↓
内部器官系统：循环、代谢、免疫、内分泌
↓
中枢神经系统
↓
神经-肌肉系统
↓
肢体行动 / 语言 / 表情
↓
环境反馈
```

### 人工接口化中枢架构

```text

↓

外部控制器
↓

↓

↓

↓

```

## 研究对象

本子域关注高层系统建模，不提供实验步骤。

核心问题：

- 生命支持系统如何代理自然器官的代谢、供氧、清除、温控和保护功能？
- 神经接口如何建立可解释、可校验、低伤害的输入输出闭环？
- 外部控制器如何避免反馈失真、控制错误和主体表达受损？
- 系统故障时如何降级、恢复和保护主体尊严？
- 如何定义主体连续性、可表达性、可退出性和权利边界？

## 不属于本子域

- 普通长寿干预证据页。
- 医疗建议或手术建议。
- 人体或动物实验操作协议。
- 绕过伦理审查的可执行技术路线。

## 与其他子域的关系

- 与 `longevity-evidence` 的关系：共享证据追溯和风险边界原则，但研究尺度不同。
- 与 `memory-editing` 的关系：共享主体连续性和神经安全问题；本子域关注生命支持与接口，记忆编辑关注记忆痕迹读写。

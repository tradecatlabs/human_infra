# Biostasis And Cryopreservation

Biostasis And Cryopreservation 负责整理冷冻保存、生物停滞、低温医学、休眠和损耗暂停路径。

核心问题：

> 如果当前技术不能修复主体，是否可以降低或暂停损耗，让主体等待未来修复窗口。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 当前医学无法解决所有终止风险
  -> 主体可能需要等待未来技术成熟
  -> 等待期间身体和神经结构不能继续不可逆损坏
  -> 因此生物停滞是“保存到未来”的研究域
```

## 关注对象

- 人体冷冻、器官冷冻、玻璃化保存、低温损伤和复温问题。
- 休眠、冬眠、代谢降低、缺血保护和长期生理稳定。
- 神经结构保存、身份连续性、法律状态和可逆性问题。
- 与 `future-waiting/` 的关系：两者都降低等待成本，但机制不同；本域靠生物停滞，Future Waiting 靠时间差分。

## Human Infra 模型链路

```text
生物停滞技术 T
  -> 降低代谢、损伤积累和结构退化 X
  -> 改变等待期间主体状态 S
  -> 降低不可逆损坏风险或保存未来修复机会
  -> 增加未来选择权，但不直接证明复活或永生
```

## 非目标

- 不提供冷冻保存、复温、灌流、化学保护剂或人体操作步骤。
- 不把冷冻保存写成已验证复活路径。
- 不把法律死亡、医学死亡和主体连续性混为一谈。

## 下一步

1. 区分器官保存、全身保存、神经保存和可逆休眠。
2. 建立保存质量、结构完整性、可逆性、未来修复依赖和法律身份矩阵。

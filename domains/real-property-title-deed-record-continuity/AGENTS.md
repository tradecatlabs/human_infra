# 不动产权属、地契与土地记录连续性目录说明

## 目录结构

```text
real-property-title-deed-record-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义不动产权属、地契、土地记录和产权链如何支撑住房、资产和继承连续性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀产权记录机制、地方记录系统、交易资料边界和主体持续性变量，不写个体房产操作流程。
- 涉及美国制度时优先使用地方 recorder/register、CFPB、HUD、FEMA、州/地方 assessor 和官方地籍资料。
- 不把本域写成买卖房、产权、税务、抵押、继承、诉讼、估价、保险或规避记录建议。

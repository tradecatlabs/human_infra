# Restorative Justice Repair Accountability Continuity

Restorative Justice Repair Accountability Continuity 研究修复性司法、伤害修复、责任承担、被害者需求、社区会议和重返关系如何在安全边界内降低长期报复、排除和再伤害。

> 核心问题：有些伤害不能只靠惩罚或沉默解决；没有修复、责任和安全边界，受害者恢复、加害者重返、社区信任和未来合作都会被长期破坏。

## 研究对象

- Restorative justice、victim-offender dialogue、family/group conferencing、repair agreement、accountability 和 reentry。
- 学校、青少年司法、社区冲突、轻罪、校园伤害、组织伤害和重返社会场景。
- 自愿性、安全筛查、受害者中心、权力不平衡、保密性、强制参与和再伤害风险。

## 先验链路

```text
主体之间会发生伤害、冲突、违法或信任破坏
  -> 仅有惩罚、排除或沉默可能无法恢复受害者需求和社区关系
  -> 安全且自愿的修复机制可让责任、补偿、理解和重返路径被表达
  -> 若修复机制缺失或被滥用，报复、排除、再伤害和长期不信任会持续
  -> 社区合作能力、主体恢复和未来选择权受损
  -> 因而修复性司法是伤害后关系与责任连续性的高风险治理接口
```

## 关键变量

- 自愿性、安全筛查、受害者需求、责任承认、修复协议、执行跟进和再伤害防护。
- 学校、青少年司法、社区、工作场所、重返社会和组织伤害场景。
- 权力不平衡、性暴力/家庭暴力不适用边界、强制和解、替代正当程序、隐私和报复风险。

## 证据入口

- OJJDP / DOJ restorative justice and juvenile justice public materials。
- School restorative practices and community justice public materials。
- Victim services and trauma-informed practice public materials。

## 非目标

- 本域不提供修复性司法流程设计、个案调解、道歉文本、和解策略、法律建议或犯罪处理建议。
- 不建议在家庭暴力、性暴力、胁迫、持续骚扰或安全风险场景中使用修复程序。
- 不收集伤害事实、犯罪记录、受害者身份、加害者身份、证据、未成年人资料或协议文本。

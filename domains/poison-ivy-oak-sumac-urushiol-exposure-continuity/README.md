# Poison Ivy Oak Sumac Urushiol Exposure Continuity

`poison-ivy-oak-sumac-urushiol-exposure-continuity` 研究毒葛、毒橡、毒漆、漆酚暴露、接触性皮炎和户外行动连续性的关系。

> 核心问题：漆酚暴露不是小皮疹；对户外工作、儿童活动、园艺、灾后清理和野外通行来说，它会造成瘙痒、睡眠破坏、继发感染风险、停工和活动回避。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须能安全接触、穿越和维护自然/半自然空间
  -> 漆酚植物会把普通植被转化为皮肤炎症和行动中断源
  -> 识别、暴露控制、清洗、职业防护和就医连接构成行动前提
  -> 若该前提失败，户外劳动、居家维护、儿童活动和灾后恢复都会被压缩
```

## 关注对象

- 毒葛、毒橡、毒漆、漆酚、接触性皮炎、衣物/工具二次污染、户外活动和园林维护。
- 暴露识别、防护、清洗时序、职业培训、儿童/宠物间接带入和医疗连接。
- 与 `skin-wound-barrier-continuity/` 的边界：上位域关注皮肤屏障；本域关注特定植物暴露链路。

## Human Infra 链路

```text
漆酚植物暴露被识别和控制
  -> 接触性皮炎、瘙痒、睡眠破坏和停工下降
  -> 户外行动、园艺维护、灾后清理和儿童活动更稳定
  -> 皮肤屏障、有效时间和恢复能力被保护
```

## 非目标

- 不提供个体皮疹诊断、用药、严重反应分诊、植物清除、除草剂使用、园林施工或职业防护操作。
- 不替代医疗、职业安全、地方农业推广、学校、园林服务或毒物中心。
- 不把植物照片、地图或 AI 识别写成个体安全许可、诊疗结论或现场清理方案。

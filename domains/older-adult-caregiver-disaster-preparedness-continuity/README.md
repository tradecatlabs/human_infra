# Older Adult Caregiver Disaster Preparedness Continuity

`older-adult-caregiver-disaster-preparedness-continuity/` 研究老年人、照护者、慢病管理、跌倒风险、用药、交通和社会连接如何影响灾害中的主体持续性。

> 核心问题：老年灾害风险不是年龄本身，而是慢病、用药、行动能力、认知、热/冷脆弱性、照护依赖和社会隔离同时叠加。

## 先验位置

```text
主体要在晚年持续被保护
  -> 必须让灾害准备适配慢病、行动、认知、药物和照护依赖
  -> 老年主体更容易被停电、热冷暴露、失联、跌倒和服务中断放大风险
  -> 因而老年与照护者准备是生命历程后段的灾害连续性条件
```

## 关注对象

- 老年应急准备、照护者、药物清单、助行设备、交通、备用联系人、慢病资料、独居风险、机构/居家照护和热冷暴露。
- 独居老人、认知障碍者、行动受限者、多病共存者、家庭照护者、长期照护机构和社区服务网络。

## Human Infra 模型链路

```text
老年照护者灾害准备系统 T
  -> 改变慢病资料、药物、联系人、交通、照护者、助行设备、温控和服务接入变量 X
  -> 改变老年主体灾害中的稳定状态 S
  -> 改变跌倒、失联、热冷应激、用药中断、照护断裂、住院和长期失能风险 λ(t)
  -> 改变晚年主体持续性、照护负担、恢复速度和尊严
```

## 非目标

- 不提供个人老年照护计划、撤离、用药、医疗、福利、机构选择、法律代理或家庭安全建议。
- 不收集年龄、诊断、药物、照护者、住址、路线、认知状态、家庭关系或医疗记录。

# medication-list-implant-device-emergency-continuity 目录说明

本目录承载药物清单、植入设备和急救转交连续性资料。它研究身体中的药物和设备依赖如何在急救、住院、转诊和灾害窗口不丢失。

## 结构

```text
medication-list-implant-device-emergency-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分常规 medication reconciliation、急救可携带清单、设备卡、UDI 和召回/维护资料。
- 必须记录药物安全、设备限制、过期清单、隐私和供应中断风险。
- 不保存个人处方、设备型号/序列号、照片、PHI、账号或机构联系资料。

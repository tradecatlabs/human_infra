# public-library-community-anchor-access-continuity 目录说明

本目录承载公共图书馆与社区锚点接入连续性研究域，关注低门槛公共机构如何提供信息、网络、设备、学习和服务导航。

## 目录结构

```text
public-library-community-anchor-access-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义公共图书馆、社区锚点、公共电脑、Wi-Fi 和服务导航如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 IMLS、FCC E-Rate、Digital.gov、地方图书馆、学校和社区服务公开资料。
- 不写系统绕过、账号滥用、公共设备规避、隐私侵犯、未成年人风险内容或个案申请/法律策略。
- 涉及图书馆和社区锚点数据时只讨论接入服务、开放时间、数字包容、学习支持、隐私边界和任务连续性。

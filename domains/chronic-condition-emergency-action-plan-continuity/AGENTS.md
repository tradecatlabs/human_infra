# chronic-condition-emergency-action-plan-continuity 目录说明

本目录承载慢性病急救行动计划连续性资料。它研究长期健康状态在急性发作、断供或灾害窗口中如何通过预置计划进入可执行恢复路径。

## 结构

```text
chronic-condition-emergency-action-plan-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分病种特异 action plan、学校/工作执行、家庭准备和急救升级路径。
- 必须记录计划来源、更新时间、照护者训练、语言/无障碍、隐私和责任边界。
- 不保存个人 action plan、病历、症状、药物、学校、雇主或联系方式资料。

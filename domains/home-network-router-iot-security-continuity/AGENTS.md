# home-network-router-iot-security-continuity 目录说明

本目录承载家庭网络、路由器与 IoT 安全连续性研究域，关注家庭连接底座如何支撑远程服务和数字行动。

## 目录结构

```text
home-network-router-iot-security-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义路由器、Wi-Fi、IoT 设备、固件更新、默认凭证和设备标签如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 NIST、FCC、CISA、ENISA、设备厂商生命周期说明和 ISP 公开资料。
- 不写网络入侵、Wi-Fi 破解、IoT 漏洞利用、固件篡改、监控规避或取证操作。
- 涉及家庭网络时只讨论公开变量、风险模式和治理边界，不处理私人网络配置或日志。
- 新增指标必须注明口径：连接可用性、固件支持、默认安全、设备标签、漏洞披露或家庭服务依赖。

# Thyroid Disease Endocrine Function Continuity

`thyroid-disease-endocrine-function-continuity/` 研究甲状腺功能减退、甲状腺功能亢进、甲状腺结节/癌症通路、妊娠/中年阶段和长期内分泌稳定如何影响主体持续性。

> 核心问题：甲状腺轴把能量、体温、情绪、心率、体重、认知速度和生殖阶段连接起来；失衡会把看似分散的症状转化为长期行动质量损耗。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要稳定能量代谢、体温、心血管节律、情绪和认知速度
  -> 甲状腺疾病会改变疲劳、焦虑、心率、体重、睡眠、生育和骨骼风险
  -> 结果取决于识别、检测、长期用药/监测、阶段性风险和过度诊疗边界
  -> 因此甲状腺疾病是内分泌稳定和有效时间密度的关键维护域
```

## 关注对象

- Hypothyroidism、hyperthyroidism、thyroid nodules、thyroid cancer pathway、pregnancy thyroid issues、cardiac/bone effects, fatigue, mood, cognitive speed, monitoring continuity。
- 与 `endocrine-hormonal-regulation/` 的边界：该域关注内分泌总机制；本域聚焦甲状腺轴对能量、情绪、心血管和长期功能的影响。
- 与 `perimenopause-menopause-midlife-health-continuity/` 的关系：中年阶段症状重叠会造成识别、归因和治疗边界问题。

## Human Infra 模型链路

```text
甲状腺疾病识别与监测系统 T
  -> 改变甲状腺激素状态、心率、体重、情绪、疲劳、骨骼和生育阶段变量 X
  -> 改变能量、注意力、工作效率、睡眠、心血管负担和长期监测状态 S
  -> 改变有效时间损耗、心律/骨折/妊娠风险和过度/不足治疗风险 λ(t)
  -> 影响主体行动质量、恢复能力和未来选择权
```

## 非目标

- 不提供个人甲状腺化验判读、结节分级、用药、剂量、停药、补碘、妊娠、癌症筛查或治疗建议。
- 不生成个人症状归因、检查排序、药物调整、饮食补充、影像/穿刺或保险判断。
- 不收集个案化验、影像、用药、妊娠、生育、心律或保险资料。

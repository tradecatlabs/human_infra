# financial-resilience-access 目录说明

`domains/financial-resilience-access/` 保存财务韧性与资源可及性研究域。它关注财务冲击、支付能力、保险、债务和资产安全如何影响主体持续性。

## 目录结构

```text
financial-resilience-access/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责收入稳定、储蓄缓冲、债务、保险、医疗负担、金融排斥、欺诈风险和资产安全变量。
- 本域可整理公开财务健康、社会决定因素、经济安全和资源可及性资料。
- 本域不提供投资、借贷、税务、法律、福利申请、保险选择或个体财务建议。

## 维护规则

- 任何财务主张必须说明指标定义、适用人群、制度环境、风险边界和非个体建议属性。
- 涉及金融产品、福利、税务、保险、资产或法律时只写系统变量和安全边界。
- 新增文件或子目录时，同步更新本文件。

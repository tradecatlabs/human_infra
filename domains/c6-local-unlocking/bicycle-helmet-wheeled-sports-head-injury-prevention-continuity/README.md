`bicycle-helmet-wheeled-sports-head-injury-prevention-continuity/` 研究自行车、滑板、滑板车、轮滑等轮式活动中的头部伤害预防、头盔标准和环境风险。

> 核心问题：轮式运动把移动自由、休闲恢复和头部创伤风险绑定在一起，头盔只是系统中的一个变量。

## 先验位置

```text
主体要扩大行动半径
  -> 需要步行、骑行和微移动带来的低成本移动能力
  -> 轮式活动增加跌倒、碰撞和头部冲击暴露
  -> 因而头部保护是微移动连续性的风险控制层
```

## 关注对象

- 自行车、滑板、滑板车、轮滑、头盔标准、道路/场地环境、儿童与青少年参与、照护执行和伤害监测。

## Human Infra 模型链路

```text
轮式活动头部保护 T
  -> 改变头盔使用、速度、环境冲突、技能、可见性和车辆互动变量 X
  -> 改变跌倒或碰撞时头部能量吸收状态 S
  -> 改变 TBI、面部伤害、活动回避和长期认知负担风险 λ(t)
  -> 改变移动自由、运动参与、有效时间和未来选择权
```

## 非目标

- 不提供头盔购买、佩戴调整、骑行教学、路线选择、活动许可、交通法规、伤后处理或个体安全建议。
- 不收集儿童影像、路线轨迹、骑行能力、头盔照片、事故记录或健康史。

# retinal-detachment-urgent-referral-continuity AGENTS

## 职责

- 维护视网膜脱离急转连续性的研究边界、变量表、证据入口和禁止用途。
- 将症状信号、急诊转诊、眼科手术窗口、术后随访和长期功能后果整理为 Source Card、机制链和模型变量。
- 与 `vision-eye-health-continuity/`、`emergency-preparedness-response/`、`patient-advocacy-shared-decision-making/` 保持边界清晰。

## 非目标

- 不提供分诊、急救步骤、是否急诊判断、影像解释、手术建议、术后姿势、个体预后或医疗机构选择。
- 不替代眼科急诊、急诊科、验光、手术团队、护理随访或法律/保险建议。

## 上游

- `vision-eye-health-continuity/`：视觉输入总框架。
- `emergency-preparedness-response/`：急性事件响应框架。
- `healthcare-access-continuity/`：眼科急转和手术可及性。

## 下游

- `low-vision-rehabilitation-assistive-technology-continuity/`：不可逆视觉损害后的补偿。
- `patient-advocacy-shared-decision-making/`：急性信息理解与代理沟通。
- `time-allocation-effective-time/`：急性视觉事件造成的长期时间损耗。

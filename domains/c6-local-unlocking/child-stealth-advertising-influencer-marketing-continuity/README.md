`child-stealth-advertising-influencer-marketing-continuity` 研究儿童青少年面对 influencer marketing、原生广告、玩具/食品/游戏营销、广告披露不足、商业内容与社交内容混合时，注意力、判断力、消费压力和家庭资源如何被影响。

> 核心问题：当儿童难以区分内容、同伴影响和广告时，商业说服会提前进入认知发展和家庭资源分配系统。

## 对象

- 儿童广告、influencer marketing、native advertising、ad disclosures、kidfluencers、玩具/食品/游戏营销和商业内容识别能力。
- 注意力、消费欲望、身体形象、饮食偏好、家庭冲突、平台推荐、品牌赞助和广告透明度。
- FTC 广告披露、儿童广告政策、平台商业内容标签和学校/家庭媒体素养接口。

## Human Infra 价值

```text
儿童商业说服边界可识别
  -> 儿童更少把广告、同伴内容和真实信息混为一体
  -> 家庭、学校、平台和监管之间形成更清晰的广告透明度接口
  -> 注意力、判断力、消费资源和发展窗口得到保护
```

## 边界

- 不提供广告规避、儿童营销策略、influencer 增长技巧、商业披露规避、平台投放优化或定向儿童广告方法。
- 不判断具体账号、创作者、广告、家庭、儿童或学校是否违规或受害。
- 不收集、推断或暴露儿童账号、观看记录、消费偏好、家庭收入、学校、位置或心理状态。
- 不把媒体素养写成责备儿童、责备家庭、恐慌营销或平台审查扩张。

# informed-consent-refusal-shared-decision-continuity

## 职责

本目录承载知情同意、知情拒绝、共同决策、风险沟通和决策记录资料。

## 边界

- 只研究主体选择如何在医疗遭遇中被理解、记录和尊重。
- 不提供治疗选择、同意/拒绝建议、DNR/POLST、法律意见或 capacity 判断。
- 不保存同意书、拒绝书、代理授权、病历、诊断、PHI 或签名资料。

## 结构

```text
informed-consent-refusal-shared-decision-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分当前医疗同意、advance care planning、代理决策和研究知情同意。
- 涉及个人治疗、拒绝、能力判断或法律文件的内容必须转回专业人员。

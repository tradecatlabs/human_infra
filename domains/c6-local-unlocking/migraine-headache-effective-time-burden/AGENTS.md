# migraine-headache-effective-time-burden 目录说明

本目录维护偏头痛和头痛障碍对有效时间的影响资料。它是低死亡、高失能、高时间损耗的神经负担域。

## 目录结构

```text
migraine-headache-effective-time-burden/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义偏头痛、头痛障碍、感官敏感和发作负担如何影响有效时间与主体持续性。
- 本目录不保存诊断、用药、急症判断、个体触发因素处方或产品推荐。
- 资料必须区分公共健康信息、临床指南、疾病负担、工作学习调整和个体医疗判断。

## 上下游关系

- 上游来自神经病学、疼痛医学、职业健康、睡眠、心理健康和疾病负担研究。
- 下游服务有效时间、工作学习连续性、生活质量、疼痛控制和社会支持模型。
- 不替代 `pain-suffering-control/`；本目录聚焦头痛障碍特有的发作性有效时间损耗。

## 维护规则

- 不提供个体诊断、用药、产品、急症或治疗建议。
- 不把头痛日记、可穿戴数据或自测量表写成个人诊断。
- 新增资料必须标注头痛类型、负担指标、适用对象、证据阶段和禁止用途。

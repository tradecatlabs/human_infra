# collaborative-document-permission-version-continuity 目录说明

本目录承载协作文档、权限与版本连续性研究域，关注共同资料如何被授权、编辑、导出、迁移和恢复。

## 目录结构

```text
collaborative-document-permission-version-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义共享文档、权限、版本、评论、所有者、导出和归档如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 ODF、OOXML、WebDAV、NARA 电子记录、平台公开导出和权限治理资料。
- 不写绕过权限、偷取资料、未授权批量导出、破解 DRM、恢复他人文件或隐藏访问痕迹方法。
- 涉及协作平台时只讨论权限变量、版本记录、导出、归档和所有权转移边界。
- 新增指标必须注明口径：权限完整性、版本保留、导出损失、所有者风险、误删恢复或长期可读性。

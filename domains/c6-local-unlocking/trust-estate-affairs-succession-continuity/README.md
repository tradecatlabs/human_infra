Trust Estate Affairs Succession Continuity 关注事务整理、受益人信息、信托/遗产安排、账户继承、文件可达和项目交接如何在失能或死亡边界附近保持资源、责任和长期项目连续性。

## 对象

- Getting affairs in order、重要文件、受益人、账户清单、遗产执行、trustee、executor 和 successor roles。
- 失能、死亡、长期住院、迁移、灾害或家庭断联时的资源、文件、账户和责任交接。
- 医疗、财务、数字、法律、家庭、照护和作品项目之间的继承断点。

## Human Infra 价值

```text
主体持续性也包括失能和死亡边界上的资料与责任不断裂
-> 长期项目、资产、照护责任、数字资料和家庭义务跨越个人行动能力
-> 若事务没有整理，资源可能被冻结、丢失、争议化或无法用于照护与继承
-> 事务与继承连续性把文件、角色、账户和责任提前组织
-> 失能或死亡边界附近的资源、资料和项目更可能被正确接续
```

## 边界

- 本域不提供遗嘱、信托、税务、继承、受益人、财产分配、投资、保险、数字账户转移或法律建议。
- 不判断某文件是否有效、某继承人权利、某遗产如何分配或某账户如何处理。
- 不替代律师、税务专业人员、法院、金融机构、保险公司、遗产执行人或平台政策。
- 只整理公开教育资料、角色边界、文件类型、连续性断点和禁止用途。

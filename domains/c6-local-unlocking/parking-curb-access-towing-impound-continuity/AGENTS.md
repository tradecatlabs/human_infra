# parking-curb-access-towing-impound-continuity 目录说明

本目录承载停车、路缘访问、拖车和扣押连续性研究域，关注出行最后一段如何影响任务完成和车辆移动能力保留。

## 目录结构

```text
parking-curb-access-towing-impound-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义停车、路缘、拖车、扣押和取回流程如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 DOT、FHWA、ADA.gov、本地交通部门和公开消费者/法律信息资料。
- 不写规避停车执法、罚单逃避、拖车规避、伪造许可、隐藏车辆或个案法律策略。
- 新增指标必须注明口径：停车可得性、路缘访问时间、罚费负担、扣押取回时间、残障可达性或任务完成率。

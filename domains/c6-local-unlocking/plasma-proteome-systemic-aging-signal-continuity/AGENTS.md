# plasma-proteome-systemic-aging-signal-continuity 目录说明

`domains/c6-local-unlocking/plasma-proteome-systemic-aging-signal-continuity/` 保存血浆蛋白组和系统性衰老信号研究域。

## 目录结构

```text
plasma-proteome-systemic-aging-signal-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责血浆蛋白组、循环因子、系统性衰老信号和血浆干预假设的研究建模。
- 本域可整理蛋白组年龄信号、跨器官通信、异时共生、血浆置换假设和监管边界资料。
- 本域不提供输血、血浆、血液制品、检测解读、抗衰治疗或购买建议。

## 维护规则

- 所有血浆相关主张必须区分动物实验、机制假设、观察 biomarker、临床终点和监管警示。
- 与 `blood-oxygen-hemostasis/`、`biological-age-clocks-biomarker-validation/` 交叉时保留测量与干预边界。
- 新增文件或子目录时同步更新本文件。

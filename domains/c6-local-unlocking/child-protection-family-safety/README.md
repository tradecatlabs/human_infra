儿童保护与家庭安全域研究：当儿童面临虐待、忽视、家庭暴力暴露、剥削、家庭压力或照护失败时，预防、发现、报告、家庭支持、儿童福利和创伤知情服务如何保护未来主体能力形成。

## 对象

- 儿童虐待、忽视、性剥削、家庭暴力暴露、ACEs、家庭压力、儿童福利和家庭支持服务。
- 预防、报告系统、强制报告框架、儿童保护服务、寄养、家庭保全、创伤知情照护和学校/医疗转介。
- 早期逆境对大脑发展、心理健康、教育、关系、慢病风险、暴力循环和未来选择权的影响。
- 儿童保护系统中的权利、隐私、家庭完整性、公平、误报、过度监控和制度伤害风险。

## Human Infra 价值

主体持续性从儿童阶段就开始被塑造。儿童保护与家庭安全域把虐待、忽视和家庭失稳看作未来主体能力形成的早期断点，同时也审查保护系统本身可能造成的误伤、过度监控和家庭断裂。

```text
儿童风险被预防、发现并支持
  -> 虐待、忽视、创伤和家庭失稳下降
  -> 大脑发展、学习能力、关系能力和健康路径更稳定
  -> 未来主体持续性和选择权提高
```

## 边界

本域只整理公开儿童保护、儿童福利、ACEs、家庭支持、创伤知情服务和儿童安全资料。

不提供：

- 个案报告、法律、监护、探视、取证、调查、家庭冲突、学校纪律或儿童福利个案建议。
- 儿童定位、监控、控制、惩罚、规避调查、规避报告、胁迫儿童或隐瞒伤害的方法。
- 性剥削、虐待、诱导、接触、材料获取、材料传播、规避平台/执法或任何危害儿童的内容。
- 把儿童保护资料写成家庭监控、歧视筛选、惩罚穷人、移民执法或制度剥夺的工具。

# medicaid-chip-renewal-redetermination-continuity 目录说明

本目录维护 Medicaid/CHIP 续期、重新判定和覆盖转换连续性域。它是 Human Infra 的公共医保资格维护层，不是参保、申诉、法律或个案导航入口。

## 目录结构

```text
medicaid-chip-renewal-redetermination-continuity/
├── AGENTS.md
└── README.md
```

## 职责

- 记录 renewal、redetermination、ex parte、notice、coverage transition 和 state variation 如何影响主体持续性。
- 区分公共医保续期、一般保险参保续保、理赔账单和治疗执行。
- 保留 CMS/Medicaid.gov 等官方资料的来源、边界和更新日期。

## 维护规则

- 新增资料必须说明计划类型、州级差异、受影响人群和证据边界。
- 不保存个人 Medicaid/CHIP ID、收入、税表、住址、医疗记录、移民身份或门户凭证。
- 不能把资料整理写成个人资格判断、续期建议或申诉策略。

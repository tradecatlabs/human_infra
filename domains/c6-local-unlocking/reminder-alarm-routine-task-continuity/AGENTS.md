# reminder-alarm-routine-task-continuity 目录说明

本目录承载提醒、闹钟与例行任务连续性研究域，关注重复性维护任务如何被触发、确认和恢复。

## 目录结构

```text
reminder-alarm-routine-task-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义提醒、闹钟、重复任务、到期日、确认和补救如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 iCalendar VTODO/VALARM、患者安全、慢病管理和 HCI 提醒研究。
- 不写具体医疗、用药、法律、账单、财务操作建议，也不写监控、强迫执行或行为控制方案。
- 涉及服药或照护提醒时只讨论提醒系统变量和证据边界，不转化为处方或护理计划。
- 新增指标必须注明口径：提醒触达、确认、完成、错过、补救、过载、依赖单点或例程恢复。

Animal Shelter Rescue Rehoming Continuity 研究动物收容、救助、领养、寄养、再安置和社区支持如何在家庭断点中保护动物福利与人类关系连续性。

> 核心问题：当家庭无法继续照护动物时，是否存在有尊严、可追踪、反虐待和反遗弃的再安置系统。

## 研究对象

- 动物收容、救助、寄养、领养、再安置、低成本支持、失主重聚和社区动物服务。
- 住房不稳定、家庭暴力、灾害、疾病、经济压力和死亡事件如何推动宠物照护断裂。
- 收容容量、动物福利、透明度、公共卫生、志愿者安全和反交易边界。

## 先验链路

```text
伴侣动物连续性需要照护责任可在断点中被安全转接
  -> 家庭危机、住房限制、灾害、疾病或死亡会让原照护者无法继续照护
  -> 收容、救助、寄养、重聚和再安置系统可防止遗弃、虐待和关系突然断裂
  -> 若转接系统失效，动物福利、家庭恢复和社区压力都会恶化
  -> 因而收容救助再安置是宠物照护断点中的转接基础设施域
```

## 关键变量

- 收容容量、开放式/预约式入口、寄养网络、领养流程、失主重聚、行为评估、兽医支持和安乐死政策边界。
- 家暴庇护、无家可归、医院住院、老人照护、灾害撤离和死亡后宠物转接。
- 把收容救助资料写成个案领养、动物交易、规避程序、舆论攻击或志愿者现场操作的风险。

## 证据入口

- AVMA animal shelter and pet owner materials。
- USDA APHIS Animal Welfare resources。
- FEMA / Ready.gov pet disaster materials。
- Local animal control and shelter public materials。

## 非目标

- 本域不提供领养推荐、弃养策略、动物转卖、救助行动、收容评价、安乐死判断、现场志愿操作或个案纠纷处理。
- 不指导规避收容程序、伪造记录、非法转运、网络曝光、骚扰机构或动物交易。
- 不收集领养申请、家庭地址、动物照片、行为记录、兽医记录、虐待细节或收容机构内部资料。

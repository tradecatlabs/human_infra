# campus-basic-needs-student-support-continuity 目录说明

本目录承载校园基本需求与学生支持连续性研究域，关注学生生存性资源如何支撑学习时间、健康恢复和留存。

## 目录结构

```text
campus-basic-needs-student-support-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义食物、住房、交通、托育、应急资金、福利导航和学生支持如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料必须优先回到教育部门、Federal Student Aid、NCES/IPEDS、College Scorecard、USDA、USA.gov、州和高校公开资料。
- 不写个人福利申请、住房、食物援助、紧急住宿、托育、财务援助或危机服务建议。
- 涉及基本需求资料时，只讨论汇总变量和服务断点，不处理个人贫困、住房、家庭、身份或心理资料。
- 新增指标必须注明口径：food insecurity、housing insecurity、emergency aid、case management、retention 或 stop-out。

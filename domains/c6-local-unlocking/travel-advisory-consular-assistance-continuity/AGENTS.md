# 旅行警示与领事协助连续性目录说明

## 目录结构

```text
travel-advisory-consular-assistance-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义旅行警示、海外登记、领事协助、紧急证件和海外危机通信如何支撑跨境主体。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀旅行警示、STEP、领事协助能力边界和海外断点，不写个人旅行安全、撤离或法律策略。
- 涉及美国制度时优先使用 State Department、STEP 和使领馆公开资料。
- 不把本域写成海外路线推荐、冲突区行动、个案领事求助、签证移民或法律建议入口。

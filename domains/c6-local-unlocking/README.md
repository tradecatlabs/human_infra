# C6 局部可能性解锁层

`c6-local-unlocking/` 是 `domains/` 下的真实物理分级目录。它收纳所有被 `_possibility-space-control/classification.tsv` 判定为 `C6` 的正式研究域。

## 分级含义

解除具体疾病、服务、流程、家庭、职业、社区和专项风险中的局部阻塞。

## 维护规则

- 本目录只按可能性空间控制力归类，不按近期可行性、证据成熟度或商业化时间排序。
- 研究域正文仍由各自子目录的 `README.md` 和 `AGENTS.md` 维护。
- 新增、删除或重命名研究域时，必须同步更新 `../_possibility-space-control/classification.tsv` 与 `../_possibility-space-control/summary.md`。

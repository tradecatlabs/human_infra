# 旅行健康准备连续性目录说明

## 目录结构

```text
travel-health-preparedness-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义旅行健康风险、目的地健康资料、药品设备、慢病旅行准备和医疗可及性如何影响移动主体。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀旅行健康风险模型、官方资料入口、药品设备衔接和禁止用途，不写个人医学、疫苗、处方或自我治疗建议。
- 涉及美国制度时优先使用 CDC Travelers' Health、CDC Yellow Book 和 State Department 医疗信息边界资料。
- 不把本域写成旅行门诊、疫苗决策、疫情判断、医疗转运、保险购买或目的地安全建议入口。

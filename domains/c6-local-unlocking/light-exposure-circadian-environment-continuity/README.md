# Light Exposure Circadian Environment Continuity

光照暴露与昼夜节律环境连续性域研究：当主体持续性依赖睡眠、激素节律、警觉性、情绪、夜间安全和长期恢复时，白天光照不足、夜间光污染、轮班照明和屏幕/室内照明如何成为时间结构变量。

## 对象

- 自然光、室内照明、夜间光暴露、蓝光谱、城市光污染和屏幕光环境。
- 昼夜节律、睡眠时相、警觉性、情绪、代谢、轮班疲劳和长期恢复能力。
- 医院、学校、工作场所、养老机构、交通系统和居住空间的照明环境。
- 夜班、倒班、跨时区任务、极地/太空/地下环境中的节律管理。
- 照明标准、建筑采光、工作时间制度、公共安全、能源使用和环境生态影响。

## Human Infra 价值

时间不是只有日历时间，也有生物节律时间。光照环境决定身体如何同步外部世界，进而影响睡眠、学习、判断、恢复和长期健康。本域把光照与节律作为主体持续性的时间基础设施，而不是把疲劳和失眠只写成个人自控问题。

```text
光照与节律环境被正确组织
  -> 睡眠时相、警觉性、代谢和情绪恢复更稳定
  -> 夜班、学习、照护和长期任务的生理摩擦下降
  -> 有效时间、恢复能力和未来选择权提高
```

## 边界

本域只整理光照、昼夜节律、睡眠环境、工作时间和建筑/公共空间照明资料。

不提供：

- 个体睡眠诊断、光疗、褪黑素、药物、补剂、眼科或精神健康建议。
- 照明产品购买、灯具参数、建筑照明设计、施工、电气改造、屏幕设置处方或个人作息处方。
- 夜班排班、劳动争议、职业资格、驾驶/值班许可、医疗机构照明合规或法律判断。
- 把可穿戴睡眠数据、照度读数、AI 评估或单一研究写成个人健康风险或寿命结论。

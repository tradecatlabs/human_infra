`head-start-early-head-start-family-support-continuity/` 研究 Head Start 与 Early Head Start 如何把早期学习、健康、营养、家庭支持和父母参与整合为低收入家庭的儿童发展基础设施。

> 核心问题：早期儿童服务不只是看护时间，更是儿童健康、学习、营养、家庭支持和代际能力形成的综合平台。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 儿童能力形成发生在生命早期
  -> 家庭资源不足会压缩健康、学习、营养和亲子支持
  -> Head Start / Early Head Start 把服务整合到儿童与家庭系统
  -> 儿童发展和照护者行动能力才不会被早期稀缺锁死
```

## 关注对象

- Head Start、Early Head Start、早期学习、健康筛查、营养、家庭参与和父母支持。
- 项目绩效标准、综合服务、家庭目标、转介、社区合作和入学过渡。
- 与 `childcare-family-continuity/`、`maternal-newborn-child-development/`、`school-enrollment-attendance-continuity/` 和 `community-resource-navigation/` 的接口。

## Human Infra 模型链路

```text
低收入家庭早期发展需求
  -> Head Start / Early Head Start 综合服务
  -> 儿童健康、学习、营养和家庭支持改善
  -> 学校准备、家庭稳定和照护者能力提高
  -> 儿童未来选择权和家庭持续性增强
```

## 非目标

- 不判断个体项目资格、申请、录取、等待名单、服务适配或家庭目标。
- 不提供儿童诊断、教育处方、医疗建议、项目投诉、法律建议或家庭评估。
- 不收集儿童记录、家庭收入、项目申请、健康记录、教育记录或家庭服务计划。

# surgical-anesthesia-perioperative-safety 目录说明

本目录维护手术、麻醉与围手术期安全域。它是 Human Infra 的“高风险修复通道安全层”，负责手术和麻醉如何把修复机会转化为可承受风险。

## 目录结构

```text
surgical-anesthesia-perioperative-safety/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明手术安全、麻醉安全、围手术期检查、团队沟通和并发症预防如何影响主体持续性。
- 本域只做公开手术安全、麻醉安全、质量改进和系统治理资料整理。
- 不提供手术、麻醉、护理、康复、医院选择、个体风险判断、操作步骤或法律建议。

## 维护规则

- 新增材料必须区分系统安全清单、临床决策、手术技术、麻醉技术和患者安全组织学习。
- 不把手术安全资料写成个人手术准备或围手术期护理建议。
- 涉及操作、器械或并发症时，只写安全治理和证据边界，不写执行步骤。

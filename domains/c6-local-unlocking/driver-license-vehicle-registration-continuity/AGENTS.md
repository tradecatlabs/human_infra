# 驾照、车辆登记与个人移动资格连续性目录说明

## 目录结构

```text
driver-license-vehicle-registration-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义驾驶资格、车辆登记、title 和移动凭证如何支撑个人行动半径。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀移动凭证、车辆记录、资格同步和主体行动变量，不写个体 DMV、驾驶或车辆操作流程。
- 涉及美国制度时优先使用 USA.gov、DHS REAL ID、州 DMV、NHTSA 和官方交通资料。
- 不把本域写成驾驶许可、车辆购买维修、事故责任、保险、罚单、执法规避或个体出行建议。

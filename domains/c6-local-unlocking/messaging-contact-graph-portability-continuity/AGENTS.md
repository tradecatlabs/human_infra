# messaging-contact-graph-portability-continuity 目录说明

本目录承载消息与联系人图谱可携带连续性研究域，关注数字关系网络如何支撑协作、照护、求助和社会支持。

## 目录结构

```text
messaging-contact-graph-portability-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义消息账号、联系人、群组、聊天历史、备份、导出和互操作如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 Data Transfer Initiative、平台公开导出/备份文档、CISA、消费者数字安全和互操作政策资料。
- 不写聊天恢复、加密绕过、联系人抓取、骚扰、监控、定位、群组渗透、账号申诉或个案迁移建议。
- 涉及关系图谱时只讨论变量和边界，不处理聊天内容、联系人、群组名单、账号 ID、密钥、备份文件或定位。
- 新增指标必须注明口径：联系人完整性、群组连续、消息归档、通知可达、导出迁移或平台互操作。

# Tribal Health Sovereignty Service Access Continuity

部落健康、主权与服务可达连续性域研究：当卫生服务、公共卫生、福利、交通、宽带、文化安全、司法辖区和主权治理交织时，部落社区成员如何持续获得照护、身份承认、服务导航和未来选择权。

## 对象

- 部落卫生系统、Indian Health Service、部落运营健康项目、Urban Indian Organizations 和转诊服务。
- 主权治理、联邦信托责任、跨辖区协调、数据主权、公共卫生报告和服务责任边界。
- 农村/边远地理、交通距离、宽带、卫生人力、药品可及、急救窗口和专科转诊。
- 文化安全、语言、传统照护、历史创伤、机构信任和服务可接受性。
- 慢病、心理健康、物质使用、母婴健康、老年照护、公共卫生和灾害响应连续性。

## Human Infra 价值

主体持续性不能只假设国家公共服务会以同一种方式抵达所有人。部落健康域把主权、地理、文化安全、信任和服务责任边界作为独立变量，解释为什么同样的医疗和公共卫生基础设施在部落社区中需要不同的治理接口。

```text
部落主权和服务责任边界被结构化治理
  -> 卫生、公共健康、转诊、数据和文化安全接口更清晰
  -> 服务缺口不再被误写成个人不配合或单纯地理问题
  -> 主体持续性、社区韧性和未来选择权提高
```

## 边界

本域只整理公开部落健康、主权治理、服务可达、公共卫生、数据治理和文化安全资料。

不提供：

- 个体医疗、福利、法律、部落身份、登记、血统、服务资格或转诊建议。
- 部落政府治理、土地、司法辖区、执法、博彩、税务或政治策略建议。
- 对部落社区的污名化画像、风险定位、商业开发、监控或数据抽取方案。
- 把文化安全简化为表层翻译、宣传材料或外部机构单向教育。

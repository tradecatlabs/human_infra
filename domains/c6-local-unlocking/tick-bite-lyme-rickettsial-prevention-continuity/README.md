`tick-bite-lyme-rickettsial-prevention-continuity` 研究蜱虫叮咬、莱姆病及其他蜱媒疾病预防如何影响户外行动、居住环境和长期功能连续性。

> 核心问题：蜱媒风险会把徒步、园艺、林地工作、儿童活动和宠物接触变成需要持续检查、预防和早期识别的执行问题。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要可进入自然、庭院、农林和户外劳动空间
  -> 蜱虫叮咬会引入感染、延误识别、疲劳、神经/关节后果和活动回避
  -> 身体检查、防护、宠物/庭院接口和早期就医路径构成户外行动前提
  -> 若该前提失败，健康寿命、有效时间和自然空间使用权下降
```

## 关注对象

- 蜱虫叮咬、莱姆病、立克次体病、巴贝虫病等蜱媒疾病风险。
- 户外活动、林地/庭院暴露、儿童、宠物、职业暴露、旅行和延误识别。
- 防护、检查、移除教育、症状识别、公共卫生监测和风险沟通。

## Human Infra 链路

```text
蜱媒暴露被预防和早识别
  -> 感染、延误就医和长期功能损失风险下降
  -> 户外活动、自然接触、园艺、农林工作和宠物互动更可持续
  -> 行动自由、有效时间和健康寿命被保护
```

## 非目标

- 不提供诊断、抗生素、蜱虫移除操作、产品推荐、户外路线建议或旅行医学建议。
- 不替代 CDC、地方公共卫生、医疗服务、兽医服务或职业安全机构。
- 不把蜱媒地图、病例或气候数据写成个体风险、保险、旅行或资产判断。

# Governance And Rights

Governance And Rights 负责整理主体权利、法律身份、同意、退出、数据权、医疗权、AI 治理和研究伦理。

核心问题：

> 有效永生如果没有权利和治理保护，可能变成被监控、被控制、被剥夺退出权的持续存在。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体不仅要存在，还要能选择、拒绝、退出和被承认为权利主体
  -> 医学、AI、神经技术和数据系统会放大控制能力
  -> 没有治理边界，基础设施可能从支撑主体变成支配主体
  -> 因此治理与权利是主体持续性的保护条件
```

## 关注对象

- 法律身份、医疗同意、退出权、数据访问权、隐私、审计和申诉。
- AI 治理、算法决策、神经权利、脑机接口伦理和人格权。
- 临床试验伦理、人体增强边界、商业宣传边界和脆弱群体保护。
- 与所有高风险域的交叉：重编程、记忆编辑、去具身中枢、认知增强和未来等待。

## Human Infra 模型链路

```text
治理 / 权利机制 T
  -> 改变同意、控制权、退出权、透明度和救济能力 X
  -> 改变主体制度状态 S
  -> 降低被剥夺、被操控、被误用和被系统性排除的风险
  -> 保护主体持续性不被基础设施反向吞噬
```

## 非目标

- 不提供法律意见或规避监管建议。
- 不把治理写成阻碍技术，而是把它作为主体持续性的必要保护条件。
- 不把“继续存在”压过同意、退出、人格和权利边界。

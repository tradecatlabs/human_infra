`pool-spa-barrier-drain-entrapment-safety-continuity/` 研究泳池、水疗池、围栏、门闩、排水口吸附夹陷和产品标准如何影响儿童与家庭水域安全。

> 核心问题：泳池安全不是“看住孩子”的单点责任，而是屏障、门禁、排水结构、维护责任和标准执行共同构成的防线。

## 先验位置

```text
主体要避免水域急性中止
  -> 必须降低无监督入水和排水夹陷暴露
  -> 泳池/水疗池把家庭空间转化为高风险水域
  -> 因而屏障和排水安全是水域风险工程的执行层
```

## 关注对象

- 泳池、水疗池、围栏、门、锁扣、警示、排水口、吸附夹陷、公共/私人设施和维护责任。
- 儿童、不会游泳者、公共泳池使用者、家庭访客和设施管理者。

## Human Infra 模型链路

```text
泳池屏障与排水安全 T
  -> 改变无监督进入、物理屏障、排水吸附、维护、召回和警示变量 X
  -> 改变儿童和使用者接触水域危险部位的状态 S
  -> 改变溺水、夹陷、创伤和监护失败风险 λ(t)
  -> 改变家庭安全、主体持续性和公共设施可信度
```

## 非目标

- 不提供泳池设计、围栏安装、排水改造、设备维修、救援动作、儿童监控、产品推荐、法规合规或个体设施安全建议。
- 不收集家庭泳池照片、地址、儿童活动记录、设施缺陷照片或设备型号。

# Backup Power Battery Generator Safety Continuity

`backup-power-battery-generator-safety-continuity/` 研究停电中的备用电源、电池、发电机安全、充电能力和电力依赖服务如何影响主体持续行动。

> 核心问题：备用电源既能维持通信、照明、医疗设备和冷藏，也可能通过一氧化碳、火灾、触电、燃料和误用变成新的风险源。

## 先验位置

```text
主体要在停电中持续行动
  -> 必须维持最低照明、通信、医疗设备和信息接入
  -> 备用电源会同时引入供能收益与 CO/火灾/触电风险
  -> 因而备用电源是灾害连续性的双刃基础设施
```

## 关注对象

- 电池、移动电源、发电机、充电、燃料、通风、CO 风险、医疗设备、冷藏、通信和关键设备负载。
- 电力依赖医疗设备用户、居家照护家庭、停电高风险地区、老年人、慢病患者和远程工作/学习者。

## Human Infra 模型链路

```text
备用电源与安全治理 T
  -> 改变供电冗余、照明、通信、医疗设备、冷藏、CO、火灾和燃料变量 X
  -> 改变停电期间主体自持与暴露状态 S
  -> 改变设备中断、失联、药物失效、CO 中毒、火灾和错误操作风险 λ(t)
  -> 改变恢复窗口、医疗连续性、信息接入和灾后选择权
```

## 非目标

- 不提供发电机操作、接线、燃料、设备购买、负载计算、医疗设备备用方案或电气维修建议。
- 不收集住址、设备型号、医疗诊断、用电负载、发电机照片、燃料储量或家庭布局。

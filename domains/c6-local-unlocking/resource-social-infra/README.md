Resource And Social Infrastructure 负责整理食物、水、能源、住房、医疗可及、金融、照护、教育、就业、社区支持和社会服务。

核心问题：

> 主体不是在真空中存在；没有资源、环境、照护和社会支持，寿命和能力无法转化为持续行动。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须获得生活、医疗、恢复和行动资源
  -> 资源和社会支持决定主体能否从损伤、疾病和危机中恢复
  -> 资源断裂会把可修复问题变成不可逆终止
  -> 因此资源与社会基础设施是主体持续性的外部底座
```

## 关注对象

- 食物、水、能源、住房、交通、金融、保险和基本生活稳定性。
- 医疗可及、药品可及、远程照护、长期照护、康复和护理。
- 学习、就业、职业转换、收入、社区资源和社会服务导航。
- 家庭照护、托育、老年照护、残障支持和危机支持。

## Human Infra 模型链路

```text
资源 / 社会基础设施 T
  -> 改变可及性、负担、连续性、恢复支持和协作网络 X
  -> 改变主体外部支撑状态 S
  -> 降低贫困、断供、医疗中断、照护缺失和社会孤立风险 λ(t)
  -> 增加有效时间、恢复能力和长期行动稳定性
```

## 非目标

- 不把公共服务、福利或社会支持简化成个人效率工具。
- 不提供规避制度、欺诈、虚假申请或违法建议。
- 不把资源基础设施混成单一产品路线。

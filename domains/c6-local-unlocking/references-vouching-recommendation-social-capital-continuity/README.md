References Vouching Recommendation Social Capital Continuity 研究推荐人、背书、职业/住房/教育 reference、信用性证明和社会资本信号如何帮助主体进入工作、住房、教育、志愿和协作机会。

> 核心问题：许多机会并不只看资格文件，还看谁愿意为你作证；如果背书网络断裂，能力和信誉就难以转化为下一次机会。

## 研究对象

- References、recommendation letters、vouching、professional reference、landlord reference、character reference 和 social proof。
- 工作招聘、租房、学校申请、志愿服务、专业社群、项目合作和照护/监护场景。
- 推荐质量、可验证性、偏见、隐私、关系不平等、推荐疲劳和背书责任。

## 先验链路

```text
主体进入新机会时需要向陌生机构证明可靠性、能力和关系可信度
  -> 推荐、背书和 reference 将过去关系转化为未来准入信号
  -> 若主体缺少可联系推荐人或背书被偏见/关系垄断控制，机会入口变窄
  -> 工作、住房、教育、志愿和合作机会减少
  -> 社会资本复利和未来选择权下降
  -> 因而推荐背书是关系网络转化为机会资本的接口
```

## 关键变量

- 推荐人可达性、关系强度、证明内容、可验证性、时效、偏见、责任边界和隐私。
- 工作、租房、教育、志愿、专业社群、移民、重返社会和无关系网络人群。
- 关系垄断、裙带、报复性差评、歧视、虚假推荐、隐私泄露和过度依赖人脉风险。

## 证据入口

- EEOC / employment reference and fair hiring public materials。
- FTC / consumer reporting and tenant background materials。
- Education recommendation, workforce and credentialing public materials。

## 非目标

- 本域不代写推荐信、伪造背书、联系推荐人、提供招聘/租房/入学策略或关系经营建议。
- 不判断推荐内容、背景调查、雇主行为、租房筛查、录取或法律责任。
- 不收集推荐人信息、联系方式、推荐信、申请材料、工作/租房/学校记录或敏感身份。

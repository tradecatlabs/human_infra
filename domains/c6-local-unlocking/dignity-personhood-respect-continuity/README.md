Dignity Personhood Respect Continuity 研究尊严、人格承认、被尊重、被倾听、非物化、非羞辱和 personhood 如何影响照护、临终、残障、老年、认知下降和制度接触中的主体持续性。

> 核心问题：主体持续性不能只以生物存活为准；如果系统把人降格为病例、负担、号码或风险标签，主体连续性会被侵蚀。

## 研究对象

- Human dignity、personhood、respect、recognition、autonomy、non-humiliation、person-centered care 和 dignity-conserving care。
- 医疗、长期照护、痴呆、残障、羁押、福利、临终、灾害和公共服务中的去人格化风险。
- 尊严与同意、共同决策、文化安全、照护质量、生活质量、投诉机制和权利保护之间的关系。

## 先验链路

```text
主体要持续存在，必须被系统承认为仍是一个人而非对象
  -> 尊严和人格承认让偏好、同意、隐私、关系和故事继续具有权重
  -> 去人格化会使照护、制度服务和风险治理只优化对象管理
  -> 主体的声音、价值、边界和未来选择权被边缘化
  -> 生存可能保留，但主体性、尊严和行动资格受损
  -> 因而尊严与人格承认是有效永生的不可化约约束
```

## 关键变量

- 尊重、羞辱、参与决策、被倾听、隐私、偏好、名字/称谓、文化安全、照护关系和权利救济。
- 认知下降、失能、疼痛、临终、贫困、羁押、福利申请、无家可归和医疗依赖。
- 伪尊重、家长主义、代替决策滥用、监控、羞辱式服务、风险标签化和机构便利优先风险。

## 证据入口

- Person-centered care、dignity-conserving care 和 dignity therapy 文献。
- WHO palliative care / quality-of-life / person-centered care 相关资料。
- Dementia personhood、disability rights 和 patient rights 相关资料。

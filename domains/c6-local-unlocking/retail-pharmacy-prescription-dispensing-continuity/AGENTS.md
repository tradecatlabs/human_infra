# retail-pharmacy-prescription-dispensing-continuity

## 职责

本目录承载零售/邮寄/专科药房配药连续性资料，关注处方从临床系统进入药房、支付、配发、取药、配送和 refill 的执行链路。

## 边界

- 只整理公开资料、机制、变量、证据边界和研究问题。
- 不提供处方、换药、剂量、停药、替代药、药房规避、保险规避或个案申诉建议。
- 不保存个人处方、保险、药房账号、诊断、药品照片或身份资料。

## 结构

```text
retail-pharmacy-prescription-dispensing-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须标明来源、地区、适用对象和是否涉及药房网络、支付、库存或处方传递。
- 医疗、药事和保险个案必须回到专业人员、药房、保险方或官方申诉渠道。

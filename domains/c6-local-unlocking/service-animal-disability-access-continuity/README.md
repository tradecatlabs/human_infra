# Service Animal Disability Access Continuity

Service Animal Disability Access Continuity 研究服务动物如何在残障可达、公共场所进入、交通、工作学习参与和主体独立生活中维持功能连续性。

> 核心问题：服务动物不是普通宠物标签，而是部分残障主体行动、警示、导航、稳定和参与的辅助接口。

## 研究对象

- ADA 服务动物定义、公共场所进入、任务训练、处理者权利、机构责任和排除边界。
- 服务动物如何影响移动、感知、医学警示、心理安全、工作学习参与和独立生活。
- 服务动物、情绪支持动物、治疗动物、宠物和辅助技术之间的概念边界。

## 先验链路

```text
主体持续性需要功能缺口可被环境和工具补偿
  -> 部分主体通过服务动物获得导航、警示、稳定、辅助任务和公共参与能力
  -> 如果公共场所、交通、住房或工作学习场景无法正确承接服务动物
  -> 主体行动半径、服务接入、学习工作参与和尊严会被压缩
  -> 因而服务动物可达连续性是辅助技术与残障权利交界的独立研究域
```

## 关键变量

- 服务动物任务、处理者控制、公共场所规则、合理进入、排除条件、过敏/恐惧冲突和误用风险。
- 工作、学校、医疗、交通、住房和灾害场景中服务动物与主体行动的耦合。
- 将服务动物概念滥用于宠物准入、平台规避、法律争议或残障筛选的风险。

## 证据入口

- ADA.gov service animal requirements。
- U.S. Department of Justice ADA guidance。
- ADA National Network service animal resources。
- CDC / FEMA emergency planning materials that mention service animals。

## 非目标

- 本域不提供服务动物认证、训练、注册、法律意见、投诉代写、场所进入策略或个案权利判断。
- 不判断某只动物是否合格、不推荐训练机构、不提供任务训练、行为矫正或公共场所冲突处理。
- 不收集残障诊断、服务动物证明、训练记录、行程、学校/雇主材料或场所纠纷证据。

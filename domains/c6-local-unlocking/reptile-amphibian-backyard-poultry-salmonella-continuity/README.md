# Reptile Amphibian Backyard Poultry Salmonella Continuity

`reptile-amphibian-backyard-poultry-salmonella-continuity` 研究爬行动物、两栖动物、后院家禽等动物接触中的沙门氏菌风险如何影响家庭、儿童、学校和食物卫生连续性。

> 核心问题：某些动物看似是低风险宠物或教育对象，但会把沙门氏菌等风险带入厨房、儿童、免疫低下者和学校环境。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体与动物共处、学习和生产依赖可控的卫生边界
  -> 爬行/两栖动物和后院家禽可把病原带入家庭与教育空间
  -> 接触边界、洗手、年龄/免疫风险识别和环境分隔构成共处前提
  -> 若该前提失败，动物接触会转化为感染、照护负担和学校/家庭中断
```

## 关注对象

- 爬行动物、两栖动物、后院家禽、小型宠物和沙门氏菌相关风险。
- 儿童、孕产、老人、免疫低下者、学校、农场参观、厨房交叉污染和宠物商店。
- 健康宠物资料、One Health、家庭卫生和食品安全接口。

## Human Infra 链路

```text
动物接触卫生边界清楚
  -> 沙门氏菌和相关感染风险下降
  -> 儿童学习、家庭宠物、后院养殖和食物准备更可持续
  -> 照护中断、医疗成本和家庭冲突下降
```

## 非目标

- 不提供动物诊断、治疗、养殖、饲养、清洁操作、个体医疗、食品处理步骤或宠物选择建议。
- 不替代 CDC、兽医、食品安全、学校、公共卫生或农业机构。
- 不鼓励隐瞒动物源感染、违规售卖、危险接触或将动物健康信号直接写成人体诊断。

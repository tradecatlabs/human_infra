Third Place Belonging Social Participation Continuity 研究第三空间、社区活动、公共生活、归属感、非消费性停留空间和社会参与如何把陌生人环境转化为可恢复、可连接、可学习的日常场所。

> 核心问题：人需要家和工作之外的低门槛公共/社区空间；没有第三空间，弱连接、归属感、公共信任、休息恢复和非正式学习会萎缩。

## 研究对象

- Third places、belonging、social participation、community centers、libraries、parks、faith/cultural spaces 和 public life。
- 低成本停留、非消费性空间、包容性活动、跨代接触、语言文化适配、无障碍和安全感。
- 门槛、开放时段、交通、费用、规则、治理、排斥、监控和空间商业化。

## 先验链路

```text
主体需要家和工作之外的安全、可达、低门槛关系空间
  -> 第三空间提供弱连接、归属感、信息交换、休息恢复和社区参与入口
  -> 若空间缺失、商业化、排斥或不可达，关系形成和公共信任会减少
  -> 孤独、信息贫困、恢复不足和社区协作弱化
  -> 主体有效时间、学习机会和未来选择权下降
  -> 因而第三空间与归属感是社会连接空间化的基础设施
```

## 关键变量

- 可达性、费用、开放时段、停留权、无障碍、安全感、活动密度、弱连接和归属感。
- 图书馆、社区中心、公园、宗教/文化空间、青少年中心、老年中心和 maker / learning spaces。
- 排斥、监控、过度商业化、治安化、身份暴露、宗教/政治压力和空间争夺风险。

## 证据入口

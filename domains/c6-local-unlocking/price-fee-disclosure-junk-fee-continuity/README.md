# 价格、费用披露与垃圾费用连续性

本域研究标价、强制费用、drip pricing、junk fees、总价披露、费用通知和账单透明度如何影响主体预算、比较能力和交易选择权。它关注价格可见性作为资源决策基础设施，而不是省钱攻略、谈判脚本或个案费用争议建议。

## 研究对象

- 标价、总价披露、强制费用、可选费用、服务费、平台费、便利费、终止费和隐藏费用。
- Drip pricing、junk fees、广告价格、最终结账价格、账单项目、费用变更和通知。
- 航空、住宿、票务、金融、公用事业、订阅、租赁和平台经济中的费用结构。
- 与合同订阅账单、航空旅客权利、临时住宿、支付争议和消费者评论信任的边界。

## 价值链路

```text
主体需要比较价格并做资源分配
  -> 总价、强制费用和账单项目决定真实成本是否可见
  -> 透明费用降低搜索成本、比较成本和预算误差
  -> 隐藏费用、分段披露和结账后加价会侵蚀现金流与选择权
  -> 价格费用披露连续性把交易成本从事后冲击转化为事前可判断变量
```

## 关键问题

- 隐藏费用和 drip pricing 如何改变消费者的比较、退出和预算能力？
- 哪些费用是可选服务价格，哪些费用实质上属于强制交易成本？
- 价格披露如何与订阅、航空、住宿、票务、金融和平台治理连接？
- 费用透明度如何影响低收入、时间贫困和高行政负担人群的主体持续性？

## 非目标

- 不提供省钱攻略、价格比较、费用谈判、拒付、投诉脚本、套利、个案账单争议或法律建议。
- 不判断某项费用是否违法、是否应退、是否必须支付或某个合同条款是否有效。
- 不协助规避费用、虚假申诉、账单操控、平台套利、逃避债务或欺骗商家/平台。

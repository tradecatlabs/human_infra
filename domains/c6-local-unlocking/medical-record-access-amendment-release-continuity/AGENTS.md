# medical-record-access-amendment-release-continuity

## 职责

本目录承载医疗记录访问、修正、释放、携带、information blocking 和记录错误治理资料。

## 边界

- 只研究记录访问制度、变量、失败模式和安全边界。
- 不代写 HIPAA、records request、amendment、release、complaint 或法律文件。
- 不保存 PHI、病历副本、授权表、签名、身份证件、MRN 或 portal 资料。

## 结构

```text
medical-record-access-amendment-release-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分访问权、修正权、授权释放、互操作和信息阻断。
- 涉及个人记录、隐私争议或法律判断的内容必须转回官方渠道和专业人员。

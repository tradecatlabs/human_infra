# tax-administration-filing-record-continuity

本目录承载税务行政、申报记录和纳税人账户连续性资料。它研究税务记录如何成为收入、家庭、资格、资源和公共服务入口的行政事实底座。

## 目录结构

```text
tax-administration-filing-record-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 说明本域对象、Human Infra 价值、边界、上下游和初始资料线索。
- 后续资料应区分申报记录、账户记录、工资收入记录、纳税人身份、税务通知、纠错、身份盗用和跨系统数据匹配。
- 本域不保存个人税务资料，不提供税务申报、避税、退税、抵免、身份认证绕过、会计、法律或财务建议。

## 维护规则

- 引用资料优先来自 IRS、Taxpayer Advocate Service、官方税务机构、公共财政研究和行政负担研究。
- 必须保留身份保护、记录纠错、数据匹配错误、欺诈防控、隐私和非个人建议边界。
- 不把税务记录写成可随意优化的个人策略工具，也不输出任何规避、伪造或欺诈路径。

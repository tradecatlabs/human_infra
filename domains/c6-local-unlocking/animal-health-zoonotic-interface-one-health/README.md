`animal-health-zoonotic-interface-one-health/` 研究动物健康、人兽共患病、兽医系统、野生动物/家畜界面和 One Health 如何影响主体持续性。

> 核心问题：人类持续性不仅取决于人体健康，也取决于动物健康、生态界面、食品系统和人兽共患风险是否被监测与治理。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体存在于人类、动物和环境相互连接的生态系统中
  -> 病原体、抗微生物耐药、食品动物、野生动物、畜牧系统和生态变化会改变群体风险
  -> 动物健康监测、兽医能力和 One Health 协作构成人类公共卫生屏障
  -> 如果动物-人类-环境界面失控，新的感染风险、食品系统冲击和 AMR 风险会穿透个体防线
```

## 关注对象

- 动物健康、兽医公共卫生、人兽共患病、家畜、野生动物、食品动物、病媒和动物源性食品系统。
- One Health 协作、动物疾病监测、跨境动物疫病、AMR、生态变化、畜牧密度和食品安全接口。
- 动物健康事件如何进入传染病监测、供应链、食品安全、公共卫生应急和风险治理。
- 与 `immunization-public-health-surveillance/` 的边界：公共卫生监测域关注人体和群体监测；本域关注动物-人类-环境界面的上游风险。
- 与 `synthetic-biology-biosecurity/` 的边界：生物安全域关注双重用途和实验治理；本域关注自然界、畜牧、野生动物和兽医系统风险。

## Human Infra 模型链路

```text
动物健康与 One Health 界面可观测
  -> 动物疾病、病媒、食品系统和生态风险更早进入监测
  -> 人兽共患溢出、AMR、食品安全和供应冲击风险更可治理
  -> 公共卫生、兽医、农业、环境和食品系统协作增强
  -> 新发感染、医疗系统冲击和食物系统中断概率下降
  -> 主体持续性和未来选择权被上游风险屏障保护
```

## 非目标

- 不提供动物诊断、治疗、用药、疫苗、养殖、防疫操作、捕捉野生动物或处置病死动物建议。
- 不提供病原体采样、培养、传播、规避监测、走私动物、非法交易或实验操作。
- 不替代兽医、公共卫生、农业、食品安全、野生动物或官方疫情机构判断。
- 不把动物疾病信号写成确定人体疫情预测或个体感染风险判断。

## Source Signals

- WHO One Health：人、动物和环境健康协作框架。
- WOAH：动物健康、动物疫病和全球监测资料。
- FAO One Health：粮农系统、动物健康和 One Health 资料。
- CDC One Health：人兽共患病和跨部门协作资料。
- WOAH WAHIS：全球动物卫生信息系统。

# family-caregiver-respite-work-continuity 目录说明

本目录维护家庭照护者、喘息服务和工作连续性的资料、机制链路和伦理边界。

## 文件结构

```text
family-caregiver-respite-work-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 描述家庭照护负担如何影响照护者的时间、健康、收入、工作角色和长期主体持续性。
- 本目录只做公开资料、制度机制和研究框架，不做个人照护、医疗、福利、法律、保险或雇佣建议。
- 与长期照护、残障服务、痴呆、居家医疗、家庭医疗假和带薪假域保持交叉引用。

## 维护规则

- 新增资料必须区分家庭照护者支持、respite、长期照护、HCBS、雇主灵活性和医疗照护。
- 不把家庭照护者当作无限、免费、可隐形消耗的资源。
- 不输出个人照护计划、申请策略、家庭法律安排或医疗决策建议。

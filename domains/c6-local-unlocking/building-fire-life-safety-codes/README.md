`building-fire-life-safety-codes/` 研究建筑规范、消防生命安全、疏散、烟雾报警、无障碍和韧性改造如何成为主体持续性的栖居安全层。

> 核心问题：主体所在的建筑和室内环境是否能在火灾、结构失效、烟雾、拥挤、停电、灾害和行动限制中保护生命与行动能力。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体多数时间存在于建筑和室内空间中
  -> 建筑结构、防火、疏散、无障碍、烟雾报警和维护质量决定风险暴露
  -> 住房可得不等于建筑安全，居住空间本身可能成为伤害源
  -> 如果生命安全规范、检查、维护和改造失败，一次火灾或结构事故即可清零长期积累
```

## 关注对象

- 建筑规范、消防规范、生命安全规范、疏散、烟雾报警、喷淋、结构韧性和改造。
- 住宅、学校、医院、长期照护、工作场所和公共建筑中的生命安全条件。
- 火灾、烟雾、结构失效、拥挤踩踏、无障碍失败、停电、洪水、风灾和地震对主体持续性的影响。
- 与 `housing-built-environment-stability/` 的边界：住房域关注居住稳定、质量和环境；本域关注建筑/消防生命安全和规范执行。
- 与 `emergency-preparedness-response/` 的边界：应急响应关注事件后的救援；本域关注建筑是否在事件发生时降低伤亡和失能。

## Human Infra 模型链路

```text
建筑与消防生命安全规范可靠
  -> 结构、火灾、烟雾、疏散和无障碍风险下降
  -> 居住、照护、学习、工作和公共服务空间更安全
  -> 火灾、事故、灾害和日常障碍导致的死亡、伤害和失能下降
  -> 家庭、照护者、患者、儿童、老人和行动受限者的持续性增强
  -> 主体有效时间、恢复能力和未来选择权被保护
```

## 非目标

- 不提供建筑施工、结构加固、电气改造、消防系统安装、灭火、救援或危险现场操作建议。
- 不提供规避建筑规范、消防检查、许可、验收、 occupancy 限制或保险要求的方法。
- 不替代工程师、建筑师、消防专业人员、代码官员或当地主管部门判断。
- 不承诺任何建筑、材料、设备、疏散路径或改造方案的安全性。

## Source Signals

- FEMA Building Science 和 Building Codes Save：建筑规范、减灾和韧性证据。
- U.S. Fire Administration：火灾预防、烟雾报警和消防安全资料。
- NIST Fire Research：火灾研究、建筑环境和生命安全科学。
- International Code Council：建筑规范、消防规范和 code development。
- NFPA：消防和生命安全标准资料。

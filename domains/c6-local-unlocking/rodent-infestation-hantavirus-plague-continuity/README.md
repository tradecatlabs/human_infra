`rodent-infestation-hantavirus-plague-continuity` 研究啮齿动物侵入、粪尿暴露、汉坦病毒、鼠疫等风险如何影响居住、清理、睡眠和灾后恢复连续性。

> 核心问题：鼠害不是单纯“卫生差”；它会把住房、储物、清理、灾后返家、露营和粮食储存变成感染、过敏、咬伤和恐惧的复合风险场景。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要可睡眠、可储物、可清理和可恢复的居住空间
  -> 啮齿动物会引入病原、污染、结构破坏和心理压力
  -> 防侵入、清理边界、公共卫生和专业治理构成住房可用性前提
  -> 若该前提失败，居住稳定、灾后恢复和健康时间被持续消耗
```

## 关注对象

- 家鼠、野鼠、粪尿污染、咬伤、食物污染、储物空间、露营和灾后返家。
- 汉坦病毒、鼠疫、钩端螺旋体等啮齿动物相关风险。
- 房屋封堵、清理边界、害虫治理、公共卫生沟通和租住/公共住房接口。

## Human Infra 链路

```text
啮齿动物侵入和污染被控制
  -> 感染、过敏、咬伤、食物污染和住房恐惧下降
  -> 睡眠、储物、清理、灾后返回和居住稳定性提高
  -> 健康寿命、有效时间和家庭恢复能力增强
```

## 非目标

- 不提供捕杀、毒饵、清理步骤、病原采样、动物处理、个体诊疗或房屋施工方案。
- 不替代 CDC、EPA、地方公共卫生、害虫治理、住房维修、医疗或兽医服务。
- 不鼓励危险清理、野生动物接触、非法投药或规避房屋安全/公共卫生报告。

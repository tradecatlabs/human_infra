Social Connection Relational Infra 负责整理社会连接、孤独、社会隔离、家庭、照护网络、信任、社区关系和关系连续性。

核心问题：

> 主体不是孤立机器；长期行动依赖关系、照护、信任、协作、求助和被制度接住的能力。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能被他人识别、支持、协作、照护和接住
  -> 社会连接影响心理、行为、慢病、死亡风险和服务可及性
  -> 孤独和社会隔离会削弱行动、恢复、求助和风险应对
  -> 因此关系基础设施是主体持续性的外部支撑条件
```

## 关注对象

- 孤独、社会隔离、社会支持、家庭照护、朋友网络、社区信任和协作网络。
- 社会连接与心理健康、慢病风险、照护连续性、危机应对和资源获取的关系。
- 弱连接、服务导航、社区组织和数字关系系统。
- 与 `resource-social-infra/` 的关系：本域关注关系网络，资源域关注物资、服务和制度供给。

## Human Infra 模型链路

```text
关系 / 社会连接干预 T
  -> 改变支持网络、求助能力、信任、照护和归属感 X
  -> 改变主体社会状态 S
  -> 改变心理危机、慢病、服务中断、照护失败和孤立风险
  -> 增强恢复能力、未来选择权和长期项目执行能力
```

## 非目标

- 不提供心理治疗、危机服务替代、社交操控或关系控制策略。
- 不把社交数量等同于关系质量。
- 不把组织监控、服从管理或情绪压榨写成关系基础设施。

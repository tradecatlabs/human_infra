# Street Tree Branch Hazard Shade Maintenance Continuity

`street-tree-branch-hazard-shade-maintenance-continuity/` 研究街道树、断枝风险、树荫、根系与人行道冲突、维护请求和城市热缓冲如何影响公共空间行动连续性。

## 先验链路

```text
主体离家后需要可走、可停留、可恢复的公共环境
  -> 街道树同时提供树荫、热缓冲、心理恢复和生态服务
  -> 断枝、倒树、根系隆起、病虫害或维护失败会制造通行和伤害风险
  -> 树荫缺失会放大热暴露，树木失管会放大物理障碍
  -> 公共空间从恢复资产变成通行风险或热负荷
```

## 研究对象

- street tree、urban forestry、branch hazard、tree canopy、shade access、root sidewalk conflict、tree maintenance request、storm damage 和热暴露缓冲。

## 不能做

- 不提供树木修剪、砍伐、病虫害诊断、施工、承包商推荐、责任判断、保险理赔、法律意见或现场风险评估。
- 不收集住址、树木照片元数据、物业信息、事故照片、伤害记录、保险文件或邻里纠纷材料。

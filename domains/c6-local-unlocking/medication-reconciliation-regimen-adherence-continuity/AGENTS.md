# medication-reconciliation-regimen-adherence-continuity

## 职责

本目录承载用药核对、药历一致性、治疗负担、依从性和转诊/出院后用药连续性资料。

## 边界

- 只整理公开研究和系统变量。
- 不提供个体用药建议、相互作用判断、剂量或漏服处理。
- 不保存个人药历、诊断、处方、药名剂量或照护者资料。

## 结构

```text
medication-reconciliation-regimen-adherence-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分系统级 medication reconciliation 和个体临床药事服务。
- 所有个案用药问题必须转回医生、药师、毒物中心或官方安全渠道。

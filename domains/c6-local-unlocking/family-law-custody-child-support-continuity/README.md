本域研究父母身份、监护权、探视/养育时间、子女抚养、家庭法院命令和执行机制如何影响儿童与照护者的资源、身份、教育、医疗和安全连续性。它关注制度接口和主体持续性，不提供个人家庭法建议。

## 研究对象

- 父母身份、亲子关系、监护权、养育时间、子女抚养和家庭法院命令。
- 子女抚养服务、收入扣缴、跨州执行、修改、欠款、健康保险和支付记录。
- 家庭冲突、搬迁、保护令、安全计划、学校医疗授权和照护责任分配。
- 家庭法记录与税务、福利、学校、医疗、住房和身份系统的下游关系。

## 价值链路

```text
儿童和照护者的持续性依赖清晰的照护责任与资源流
  -> 家庭法系统把亲子关系、监护、养育时间和抚养责任制度化
  -> 这些安排决定谁能做决定、谁承担费用、谁接收通知、谁能访问服务
  -> 命令缺失、执行失败、跨州断点或冲突升级会让儿童资源、安全和照护连续性断裂
  -> 家庭法连续性把家庭关系冲突转化为可执行、可审查、可恢复的照护结构
```

## 关键问题

- 监护权、养育时间和子女抚养如何影响儿童医疗、教育、住房和福利连续性？
- 子女抚养记录和支付流如何成为家庭资源连续性基础设施？
- 家庭法院命令、保护令和安全需求如何与服务访问、地址隐私和学校医疗决策交织？
- 如何防止家庭法资料被误用为个案策略、胁迫、规避责任或监控工具？

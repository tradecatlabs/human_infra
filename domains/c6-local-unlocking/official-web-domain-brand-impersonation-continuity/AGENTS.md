# Official Web Domain Brand Impersonation Continuity 架构说明

本目录承载官方域名、品牌冒充、相似域名和真实入口识别连续性资料。它只整理公开治理和风险边界，不提供域名抢注、仿冒、SEO 操控或举报动员。

## 目录结构

```text
official-web-domain-brand-impersonation-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须回到 CISA、FTC、ICANN、WIPO、NIST 或官方域名治理资料。
- 区分官方入口、品牌身份、证书信号、搜索排序和法律争议，不给个案真假结论。
- 不收集真实账号、支付链接、争议域名所有者资料、商标材料或待举报对象。

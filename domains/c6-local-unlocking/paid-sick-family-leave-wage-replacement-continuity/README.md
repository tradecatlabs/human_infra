## 研究推进骨架

### 最小问题集

- 界定局部阻塞：某一疾病、流程、服务、职业、家庭、社区或专项风险中的具体失败点。
- 说明受影响任务：它阻碍了主体完成什么行动、服务、照护、转换或恢复过程。
- 记录可测改善：等待时间、错误率、成本、可及性、完成率、伤害率或恢复速度。
- 明确升级边界：何时应转交 C2-C5 的上层生命、能力、转化或基础设施域。
- 避免把局部改良包装成根本性解决方案，必须标注适用范围和外部依赖。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

本域整理疾病、家庭照护、育儿或医疗事件期间的工资替代机制。它关注主体在暂时无法工作时，是否仍有现金流、账单支付能力、保险连接和复工余地，而不是把每次健康事件都转化为收入断裂。

## 研究对象

- 带薪病假、带薪家庭假、带薪医疗假和临时工资替代。
- 替代率、等待期、缴费来源、雇主计划、公共计划和私营保险之间的差异。
- 病假/家庭假与 FMLA、残障收入、工伤、失业保险和托育支持的接口。
- 申请摩擦、证明负担、隐私、反报复和低收入劳动者覆盖缺口。

## 价值链路

```text
疾病、照护和家庭事件会减少当期劳动能力
  -> 若没有工资替代，短期停工会变成现金流、住房、医疗和债务风险
  -> 带薪病假和家庭/医疗假把无法工作的时间转化为可承受恢复窗口
  -> 现金流稳定降低过早返工、治疗中断和照护崩溃风险
  -> 主体更可能保留健康、工作、家庭和未来任务能力
```

## 关键问题

- 工资替代如何改变休假可用性，而不是只保留名义权利。
- 哪些群体被排除在带薪假、短期残障或雇主福利之外。
- 证明、等待期、替代率和申诉成本如何影响真实可及性。
- 如何区分病假、家庭假、医疗假、残障保险、失业保险和工伤给付。

## 上游与下游

- 上游：工资记录、雇佣身份、劳动权利、健康事件和家庭照护事件。
- 下游：财务韧性、治疗连续性、家庭稳定、工作保留和复工路径。
- 相关域：`income-payroll-employment-record-continuity/`、`family-medical-leave-job-protection-continuity/`、`pregnancy-parental-work-continuity/`、`disability-income-work-capacity-benefit-continuity/`、`workers-compensation-occupational-injury-benefit-continuity/`、`financial-resilience-access/`。

## 非目标

- 不提供个人带薪假、州福利、保险理赔、税务或工资计算建议。
- 不判断资格、替代率、等待期、雇主义务或争议处理。
- 不生成申请表、雇主政策、法律通知或医疗证明。
- 不帮助虚假报工时、虚假病假、欺诈性申领或规避雇主/公共计划规则。

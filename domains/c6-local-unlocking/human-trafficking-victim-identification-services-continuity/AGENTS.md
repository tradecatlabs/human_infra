# human-trafficking-victim-identification-services-continuity 目录说明

本目录维护人口贩运受害者识别与服务连续性域。它关注 trafficking identification、screening、referral、victim services、immigration interface 和长期恢复如何影响主体自由和未来选择权。

```text
human-trafficking-victim-identification-services-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义 trafficking、forced labor、sex trafficking、force/fraud/coercion、识别、服务转介和禁止用途。
- `AGENTS.md`：说明本目录职责、隐私边界和资料要求。

## 维护规则

- 新增资料必须优先来自 HHS OTIP、DOJ、DHS Blue Campaign、OVC、DOL 或同等公共机构。
- 必须区分公开指标、筛查资料、个案识别、救援计划、执法策略和移民法律建议。
- 不保存受害者身份、儿童资料、服务点、庇护所、调查人员、位置或案件信息。
- 不输出招募、控制、运输、剥削、规避执法、隐藏受害者、报复、卧底或追踪方法。

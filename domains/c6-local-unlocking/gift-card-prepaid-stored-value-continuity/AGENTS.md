# 礼品卡、预付卡与储值余额连续性目录说明

## 目录结构

```text
gift-card-prepaid-stored-value-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义礼品卡、预付卡、store credit、平台余额和忠诚度余额如何影响主体资源可用性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀储值余额、费用、到期、错误解决、诈骗风险和资源流动性变量，不写个案找回或交易操作。
- 涉及美国制度时优先使用 FTC、CFPB 和官方消费者保护资料。
- 不把本域写成礼品卡买卖、卡密交易、诈骗追回、套利、提现、洗钱或平台规则规避指南。

## 核心问题

近地天体风险决定小概率高后果天体撞击是否会成为文明、基础设施、生态系统和主体持续性的极端尾部风险。

Human Infra 关注的问题不是天文学科普，而是：

```text
近地天体监测、防御和响应是否能把行星级冲击从不可见风险变成可治理风险？
```

## 先验位置

主体持续性最大化不是只处理个人疾病，还要承认超长时间尺度上存在行星级外部失效模式。近地天体是典型尾部风险：概率低，但一旦失败，可能同时破坏环境、食物、基础设施和公共秩序。

## 关注对象

- 近地小行星、彗星、撞击概率、轨道测定和风险分级。
- 天体发现、跟踪、预警、偏转技术、应急响应和国际协调。
- 撞击后的火灾、海啸、尘埃、气候扰动、供应链和公共服务中断。
- 风险沟通、误报治理和长期监测基础设施。

## Human Infra 模型链路

```text
近地天体风险
  -> 发现 / 跟踪 / 预警 / 偏转能力不足
  -> 撞击或误响应造成基础设施、生态和社会系统冲击
  -> 食物、水、能源、医疗、通信和秩序连续性下降
  -> 主体持续性进入行星级尾部风险状态
```

## 非目标

- 不预测具体天体撞击。
- 不提供恐慌传播、末日准备、军事化或规避公共治理建议。
- 不把行星防御写成已解决问题或无风险问题。

Space Extreme Habitation 负责整理空间飞行、极端环境、闭环生命支持、隔离、辐射和远程医疗如何扩展主体可达未来。

核心问题：

> 如果主体持续性要跨越更长时间和更大空间，必须理解人在极端环境中如何维持身体、认知、协作和资源闭环。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体未来可能需要进入地外、深海、极地、灾后或其他极端环境
  -> 极端环境会同时压迫身体、认知、情绪、资源、协作和医疗可及性
  -> 栖居系统失败会把环境风险转化为主体终止风险
  -> 因此极端栖居是未来可达性和环境适应的综合研究域
```

## 关注对象

- 空间飞行、辐射、微重力、隔离、密闭环境、睡眠、认知和行为健康。
- 闭环生命支持、食物、水、能源、废物处理、医疗支持和系统冗余。
- 极地、沙漠、深海、地下、灾后环境和远程医疗类比场景。
- 人因工程、团队协作、任务风险、恢复路径和长期栖居边界。

## Human Infra 模型链路

```text
极端栖居技术 T
  -> 改变环境防护、资源闭环、远程医疗、团队协作和恢复能力 X
  -> 改变主体在极端环境中的生存与行动状态 S
  -> 降低环境、系统、认知和医疗不可达导致的风险 λ(t)
  -> 扩大主体可进入的未来场景和可能性空间
```

## 非目标

- 不提供太空任务、危险环境进入、生命支持建造或生存操作指南。
- 不把空间栖居写成短期可用的个人永生方案。
- 不提供规避安全程序、极端环境冒险或危险实验建议。

# Animal Welfare Cruelty Reporting Continuity

Animal Welfare Cruelty Reporting Continuity 研究动物福利、虐待忽视报告、执法转介和家庭风险信号如何保护动物、人类关系和社区安全。

> 核心问题：动物虐待和忽视不只是动物问题；它也常暴露家庭压力、暴力、照护失败和社区风险。

## 研究对象

- 动物福利、虐待、忽视、囤积、遗弃、报告入口、动物控制、执法转介和受害者安全。
- 动物虐待与家庭暴力、儿童安全、老人照护、心理压力、住房风险和社区健康之间的关系。
- 报告边界、隐私、证据、误报、报复风险、反私刑和反骚扰边界。

## 先验链路

```text
主体持续性需要照护关系不被暴力和忽视腐蚀
  -> 动物虐待、忽视、遗弃和囤积会伤害动物福利并暴露家庭系统压力
  -> 合法报告、转介、调查和支持系统可降低持续伤害和关系破裂
  -> 若资料被用于网暴、跟踪或私刑，保护系统会反向制造人身风险
  -> 因而动物福利与虐待报告需要独立的安全和伦理边界
```

## 关键变量

- 动物福利标准、照护忽视、暴力、囤积、遗弃、报告入口、跨机构转介和受害者安全。
- 家庭暴力、儿童保护、老年/成人保护、住房检查、心理健康、贫困和社区服务链接。
- 把动物福利资料用于公开羞辱、骚扰、非法调查、执法规避或家庭控制的风险。

## 证据入口

- USDA APHIS Animal Welfare Act / Animal Care materials。
- AVMA animal welfare resources。
- CDC Healthy Pets, Healthy People。
- Local animal control / humane law enforcement public materials。

## 非目标

- 本域不提供个案报案策略、证据收集、执法判断、动物没收、家庭冲突处理、公开曝光或法律建议。
- 不指导虐待、遗弃、囤积、非法繁殖、动物交易、规避检查或报复行为。
- 不收集地址、照片、视频、个人身份、儿童信息、家庭暴力细节、动物伤情或举报线索。

Platform Content Moderation Notice Appeal Continuity 关注平台内容删除、降权、标注、账号限制、通知、原因说明、申诉、透明度报告和外部监督如何影响主体表达、声誉、收入和关系连续性。

## 对象

- Content moderation、notice、statement of reasons、appeal、human review、account restriction、reach reduction、transparency report 和 procedural fairness。
- 创作者、研究者、商家、社群组织、受害者、安全举报者和普通用户在平台裁决中的可理解性和可申诉性。
- 模糊规则、误删、恶意举报、算法降权、通知缺失、申诉不可达、跨平台传播和证据不足。

## Human Infra 价值

```text
数字平台已经成为表达、收入、关系和服务入口
-> 内容裁决会改变主体能否被看见、被相信、被联系和被支付
-> 通知、原因说明、申诉和透明度让平台权力变成可审查程序
-> 主体更能恢复错误裁决、保护声誉和维护表达/服务连续性
-> 数字公共空间更少用黑箱裁决切断行动能力
```

## 边界

- 本域不提供平台申诉代写、规避审核、操控推荐、刷量、恶意举报、规避封禁、内容农场或舆论战术。
- 不判断某条内容是否违规，不替代平台规则、法律意见、内容安全团队或人权评估。
- 不收集真实账号、被删内容、私信、申诉材料、用户身份或争议证据。
- 只整理公开透明度原则、通知/申诉机制、程序风险、边界和禁止用途。

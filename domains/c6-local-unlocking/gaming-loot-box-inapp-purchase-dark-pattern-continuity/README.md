`gaming-loot-box-inapp-purchase-dark-pattern-continuity` 研究儿童青少年在游戏、应用内购买、虚拟货币、loot box、随机奖励、订阅、暗黑模式和支付摩擦中，如何被转化为注意力、金钱、家庭冲突和行为负担。

> 核心问题：游戏和应用内商业机制如果不透明，会把儿童注意力、家庭资源和自我控制窗口转化为持续消费压力。

## 对象

- in-app purchases、loot boxes、虚拟货币、随机奖励、订阅续费、dark patterns、退款摩擦和家长同意边界。
- 游戏时间、注意力、睡眠、家庭冲突、消费压力、广告/推荐和平台支付接口。
- FTC 消费者保护、退款/披露案例、平台商店规则和儿童数据/广告边界。

## Human Infra 价值

```text
儿童游戏商业机制可披露、可限制、可恢复
  -> 注意力、睡眠、家庭资源和消费决策更少被不透明机制消耗
  -> 家庭、平台、商店、监管和支付系统之间形成更清晰责任接口
  -> 儿童自我控制发展和家庭执行能力得到保护
```

## 边界

- 不提供个人消费退款、法律申诉、游戏成瘾判断、家长控制设置、支付争议、平台规避、漏洞利用或绕过内购限制方法。
- 不推荐或评估具体游戏、loot box、抽卡、账号交易、充值策略、刷量、外挂或未成年人消费方案。
- 不收集、推断或暴露儿童账号、消费记录、设备、位置、游戏行为、支付方式或家庭资料。
- 不把消费者保护写成扩大儿童监控、羞辱游戏玩家或制造恐慌营销。

`infant-safe-sleep-suid-risk-reduction-continuity/` 研究婴儿安全睡眠、睡眠环境、照护者信息理解、SUID/SIDS 风险降低和家庭支持如何影响早期生命连续性。

> 核心问题：婴儿睡眠环境是生命最早期的基础设施，错误信息、疲劳照护和不安全环境会把恢复空间变成死亡边界。

## 先验位置

```text
儿童主体要进入未来
  -> 婴儿阶段必须安全通过睡眠和照护恢复窗口
  -> 婴儿无法主动调整环境或表达风险
  -> 因而安全睡眠是早期生命持续性的代理执行条件
```

## 关注对象

- 睡眠表面、共睡/共享房间边界、柔软物品、照护者疲劳、信息传播、产后支持、儿科和公共卫生教育。

## Human Infra 模型链路

```text
婴儿安全睡眠系统 T
  -> 改变睡眠环境、照护理解、疲劳、家庭资源、产品风险和信息一致性变量 X
  -> 改变婴儿睡眠中的呼吸、体位和照护可见状态 S
  -> 改变 SUID、窒息、照护者创伤和家庭持续性风险 λ(t)
  -> 改变儿童生命路径、家庭系统和未来选择权
```

## 非目标

- 不提供婴儿睡眠安排、产品选择、共睡判断、医疗诊断、急救、个案安全评估、法律或育儿处方建议。
- 不收集婴儿照片、睡眠环境照片、家庭布置、健康史、照护者行为或死亡事件资料。

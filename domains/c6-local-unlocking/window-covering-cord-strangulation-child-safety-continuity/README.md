`window-covering-cord-strangulation-child-safety-continuity/` 研究窗帘、百叶窗、拉绳、环形绳、儿童睡眠/玩耍区域、产品标准和召回如何影响儿童窒息与缠绕风险。

> 核心问题：窗帘拉绳是低可见度高后果风险。它通常不被当成“基础设施”，但会在儿童探索和无人看护瞬间形成不可逆伤害窗口。

## 先验位置

```text
儿童需要在家庭空间中探索和休息
  -> 家庭物件必须避免形成缠绕、窒息和不可逆伤害窗口
  -> 窗帘拉绳风险隐蔽、低频但高后果
  -> 因而窗饰绳安全是儿童家庭安全执行层
```

## 关注对象

- 窗帘、百叶窗、拉绳、环形绳、儿童床/玩耍区邻近、产品标准、召回、租赁住房和二手产品。

## Human Infra 模型链路

```text
窗饰拉绳安全 T
  -> 改变缠绕暴露、产品设计、召回、空间布局和照护可见性 X
  -> 改变儿童睡眠、玩耍、探索和照护状态 S
  -> 改变勒颈、窒息、死亡、家庭创伤和产品召回成本 λ(t)
  -> 改变儿童安全、家庭恢复和未来选择权
```

## 非目标

- 不提供安装、产品选择、改装步骤、品牌推荐、召回索赔、租赁争议、儿童监控或个体安全建议。
- 不收集儿童信息、家庭照片、窗户位置、住址、产品型号或事故证据。

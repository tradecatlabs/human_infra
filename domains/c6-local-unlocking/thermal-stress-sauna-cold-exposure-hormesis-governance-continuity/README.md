`thermal-stress-sauna-cold-exposure-hormesis-governance-continuity/` 研究 sauna、passive heat therapy、cold exposure、cold water immersion、热/冷应激和 hormesis 叙事如何进入证据、安全和风险治理。

> 核心问题：冷热应激是强生理扰动，不是低成本长寿捷径。它可能影响血管、炎症、压力恢复和主观状态，也可能触发心血管、热伤害、冷休克和低体温风险。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要热稳态、循环稳定、恢复能力和压力适应
  -> 热暴露和冷暴露可能作为环境应激改变血管、炎症、代谢和心理状态
  -> 观察性关联、短期 RCT 和机制推断不能自动推出寿命收益
  -> 因此冷热 hormesis 必须作为高扰动生活方式干预治理域
```

## 关注对象

- sauna bathing、dry sauna、passive heat therapy、hot bath、cold water immersion、cold plunge、cold shower、brown adipose tissue、heat shock response 和 cold shock response。
- 心血管风险、血压、脱水、晕厥、热应激、冷休克、呼吸反应、低体温、酒精叠加、运动恢复和证据偏倚。
- 与 `thermal-homeostasis-resilience/` 的边界：热稳态域关注体温调节和环境暴露；本域关注主动冷热应激作为干预的证据和安全治理。
- 与 `cardiovascular-resilience/` 的边界：心血管韧性域关注疾病和循环风险；本域关注冷热暴露对血管和心血管风险的干预语境。

## Human Infra 模型链路

```text
冷热应激 T
  -> 影响血管功能、交感反应、炎症、热休克蛋白、棕色脂肪、睡眠和压力恢复 X
  -> 改变身体状态 S、恢复体验和急性风险暴露
  -> 改变风险函数 lambda(t) 的可能方向和不确定性
  -> 若心血管、热伤害、冷休克或低体温风险不可控，则阻塞为安全风险
```

## 非目标

- 不提供 sauna、热水浴、cold plunge、冷水游泳、时长、温度、频率或个体操作建议。
- 不把观察性 sauna 研究、短期 cold exposure 指标或 hormesis 叙事写成延寿证明。
- 不替代医生、心血管评估、运动医学、热病/低体温风险判断或场地安全管理。
- 不鼓励酒后、高温、独自冷水、开放水域或高风险人群实践。

# thermal-stress-sauna-cold-exposure-hormesis-governance-continuity 目录说明

`domains/c6-local-unlocking/thermal-stress-sauna-cold-exposure-hormesis-governance-continuity/` 保存 sauna、热疗、冷暴露、冷水浸泡和 hormesis 证据治理资料。

## 目录结构

```text
thermal-stress-sauna-cold-exposure-hormesis-governance-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义冷热应激作为高扰动生活方式干预的证据、安全和主体持续性边界。
- `AGENTS.md`：记录本目录职责、允许新增资料和禁止用途。

## 职责边界

- 本域只整理公开研究、机制假设、证据等级、风险门槛和误用边界。
- 可以新增 sauna/cold exposure Source Card、risk gate、adverse event map 和 evidence uncertainty matrix。
- 不提供温度、时长、频率、场地、安全操作、训练安排或个体建议。

## 维护规则

- 新增资料必须区分观察性关联、短期生理指标、RCT、长期终点和急性风险。
- 不把 hormesis 叙事、主观体验或相关性写成寿命收益。
- 新增文件或子目录时同步更新本文件。

# school-discipline-exclusion-restraint-continuity 目录说明

本目录承载学校纪律、排除、约束隔离与复学连续性研究域，关注纪律系统如何影响学习日、民权、安全和长期路径。

## 目录结构

```text
school-discipline-exclusion-restraint-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义停学、开除、移除、约束隔离、复学和纪律民权数据如何进入主体持续性模型。
- `AGENTS.md`：说明本目录职责、证据边界和维护规则。

## 维护规则

- 资料必须优先回到 CRDC、OCR、教育部门、州教育机构和公开纪律/民权资料。
- 不写个人纪律案件、申诉、民权投诉、行为危机、法律策略或医疗建议。
- 不处理学生个人纪律、残障、行为、心理、家庭或法院记录。
- 必须区分 exclusionary discipline、supportive intervention、restraint/seclusion、re-entry 和 civil-rights disparity。

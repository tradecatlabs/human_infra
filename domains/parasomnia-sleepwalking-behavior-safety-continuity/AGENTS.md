# parasomnia-sleepwalking-behavior-safety-continuity 目录说明

本目录维护异态睡眠、梦游、睡中行为和夜间安全边界资料。它关注睡眠中异常行为如何影响恢复、家庭支持和伤害风险。

## 目录结构

```text
parasomnia-sleepwalking-behavior-safety-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义异态睡眠如何通过睡中行为、伤害暴露、家庭负担和隐私污名影响主体持续性。
- 本目录只维护系统级变量、来源和边界，不保存个案睡眠视频、家庭布局、儿童、伴侣、用药、伤害或医疗资料。
- 资料进入本域时必须保留行为类型、安全语境、家庭负担、隐私边界和禁止用途。

## 上下游关系

- 上游来自 MedlinePlus 和睡眠医学资料。
- 下游服务 `sleep-circadian-recovery/`、家庭安全、照护和心理压力模型。
- 不替代睡眠医学、儿科、神经科、精神健康、家庭安全、法律、保险或个体医疗建议。

## 维护规则

- 不写个人诊断、睡眠检查解释、药物、安全处置、家庭改造、儿童/成人个案判断或监控建议。
- 不把家庭安全资料写成个案操作清单。
- 新增资料必须区分睡中行为类型、伤害暴露、家庭环境、隐私污名和转诊语境。

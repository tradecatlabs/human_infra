# Dementia Cognitive Decline Care Continuity

痴呆、认知衰退与照护连续性域研究：当记忆、判断、执行功能和日常能力逐步下降时，识别、照护、沟通、安全、权利和家庭支持如何保护主体身份、尊严和可承受持续性。

## 对象

- 痴呆、阿尔茨海默病、轻度认知障碍、认知退行、神经心理评估和早期识别系统。
- 照护计划、家庭照护、长期照护、居家社区支持、记忆辅助、沟通辅助和日常风险管理。
- 用药、财务、居住、驾驶、走失、跌倒、营养、孤立、照护者负担和危机迁移。
- 代理决策、同意能力、尊严、虐待防护、服务导航和照护转介。
- 认知衰退对主体连续性、未来选择权、有效时间和关系网络的影响。

## Human Infra 价值

主体持续性不仅是身体活着，还包括记忆、判断、身份、沟通和参与能力尽可能被保护。痴呆照护连续性域把认知退行作为独立边界条件：当主体逐步失去自我管理能力时，支持系统必须承接风险，而不是让主体被动失权。

```text
认知衰退被早期识别并纳入照护连续性
  -> 记忆、判断、用药、财务、居住和安全风险被支持系统承接
  -> 照护者负担、危机迁移和主体失权下降
  -> 主体身份、尊严、参与和可承受持续性提高
```

## 边界

本域只整理公开痴呆、认知衰退、照护连续性、照护者支持、安全治理和服务导航资料。

不提供：

- 个体诊断、筛查解释、治疗、用药、认知训练、饮食、补剂或医疗决策。
- 法律、财务、代理决策、监护、遗嘱、保险、驾驶资格或机构选择建议。
- 走失防控、监控、定位、门禁、约束、行为管理或家庭冲突的可操作方案。
- 把认知衰退主体视为无权利对象、监控对象或家庭负担的表述。

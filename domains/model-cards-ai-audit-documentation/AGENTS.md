# Model Cards AI Audit Documentation 目录说明

## 目录结构

```text
model-cards-ai-audit-documentation/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义模型卡、数据说明、AI 审计文档和模型治理说明在 Human Infra 中的职责边界。
- `AGENTS.md`：说明本目录的职责边界和维护规则。

## 边界

本目录只承载模型文档契约、审计记录和用途限制。它不承载模型训练代码、个人评分系统、医学决策系统、合规证明或产品背书。

## 维护规则

- 每个模型文档必须同时说明用途、数据、评估、限制和禁止用途。
- 不允许把审计文档写成模型可靠性的营销证明。
- 涉及寿命、健康、神经、身份或公共服务的模型必须显式声明不可用于个体自动化决策。
- 图表和可视化也必须有数据来源、假设和边界说明。

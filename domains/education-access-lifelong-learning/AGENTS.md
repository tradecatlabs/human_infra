# education-access-lifelong-learning 目录说明

本目录承载教育可及、成人学习、再培训、基础技能和终身学习系统如何支撑主体能力持续升级。

## 结构

```text
education-access-lifelong-learning/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 是本域的边界、模型链路、非目标和来源入口。
- 本域只讨论教育和终身学习作为主体能力基础设施，不处理个人择校、考试、留学、证书或教育投资建议。
- 后续新增材料应优先是教育可及 Source Card、技能变量、成人学习系统和再培训边界。

## 维护规则

- 不提供个人升学、择校、留学、考试、培训、证书或教育投资建议。
- 不把教育数据写成个体价值、智力或未来成就判断。
- 若新增数据表、学习模型或图谱，必须同步维护本文件和相关索引。

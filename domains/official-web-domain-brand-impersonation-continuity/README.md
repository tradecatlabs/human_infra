# Official Web Domain Brand Impersonation Continuity

Official Web Domain Brand Impersonation Continuity 关注官方域名、品牌冒充、typosquatting、lookalike domains、.gov/.edu 等可信域名和搜索结果如何影响主体识别真实机构入口的能力。

## 对象

- Official domain、brand impersonation、lookalike domain、typosquatting、search result trust、certificate signal、.gov trust 和 takedown boundary。
- 政府、学校、银行、医疗、保险、法院、公用事业、慈善、灾害救助和平台服务的官网入口可信度。
- 相似域名、假客服、假支付页、假申请入口、SEO 冒充、钓鱼落地页和品牌信任滥用。

## Human Infra 价值

```text
主体进入制度和服务通常从网页入口开始
-> 伪造域名和品牌冒充会把真实任务重定向到诈骗、数据窃取或错误服务
-> 官方域名治理、证书、搜索信号和机构验证能降低入口判断成本
-> 主体更能找到真实机构、避免假入口并保留任务完成能力
-> 公共服务、金融、医疗、法院和灾害救助更少被冒充入口截断
```

## 边界

- 本域不提供域名抢注、钓鱼页制作、品牌冒充、SEO 操控、takedown 策略、商标法律建议或机构真假个案判断。
- 不判断某个网站是否诈骗，不推荐具体服务或支付入口。
- 不提供绕过证书、滥用搜索、批量扫描、攻击网站或举报动员方法。
- 只整理公开可信域名规则、风险变量、验证提示、入口治理和禁止用途。

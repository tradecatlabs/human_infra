# food-safety-contamination-control 目录说明

本目录维护食品安全、污染控制与食源性风险域。它是 Human Infra 的“食物安全治理层”，负责食物如何从营养资源转化为可控、可追溯、可信任的输入。

## 目录结构

```text
food-safety-contamination-control/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明食源性病原体、毒素、污染、检查、追溯、召回和暴发治理如何影响主体持续性。
- 本域只做公开食品安全、风险治理、标准、追溯和召回资料整理。
- 不提供个体饮食烹饪、食品加工、污染检测、企业合规、法律或投毒/规避检测建议。

## 维护规则

- 新增材料必须区分可得性/营养问题和污染/安全问题。
- 不把暴发新闻、企业召回或食品标签解释写成个人食用建议。
- 涉及病原体、毒素、检测或污染控制时，只写治理框架，不写操作步骤。

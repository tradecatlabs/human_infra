`huntington-disease-genetic-neurodegeneration-continuity/` 研究亨廷顿病、遗传神经退行、运动障碍、认知/精神变化、家庭风险和长期决策连续性如何影响主体持续性。

## Core Question

当遗传性神经退行同时影响运动、认知、情绪和家庭未来规划时，主体如何保留决策能力、关系安全、尊严和长期资源安排？

## 先验位置

- 与 `neuro-continuity/` 的关系：亨廷顿病不是一般运动障碍，而是遗传、认知、精神和家庭风险耦合病程。
- 与 `mental-health-affective-stability/` 的关系：情绪、冲动和精神症状可直接影响风险治理。
- 与 `genomic-stability-dna-repair/` 的关系：遗传检测和家族风险需要严格证据与隐私边界。

## 关注对象

- 舞蹈样运动、认知变化、精神症状、遗传风险、家庭沟通、照护、法律/财务代理和长期照护安排。
- 预测性检测伦理、隐私、污名、保险/就业风险和家庭成员未来选择权。

## Human Infra 模型链路

```text
遗传性神经退行
-> 运动、认知、情绪和家庭风险同时改变
-> 决策能力、关系安全、资源安排和照护承接需求增加
-> 主体未来选择权和家庭系统连续性受压
-> 主体持续性下降
```

## 非目标

- 不提供亨廷顿病诊断、遗传检测解释、家族风险判断、药物、心理危机处理、保险/就业或生育决策建议。
- 不输出个人或家族预测、携带者推断、照护安排、法律代理或财务安排结论。

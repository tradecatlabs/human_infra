# Synthetic Media Watermarking Disclosure Continuity 架构说明

本目录承载合成媒体、水印、披露、标签和检测限制研究资料。它只做风险和治理建模，不提供生成、规避、检测结论或舆论操作。

## 目录结构

```text
synthetic-media-watermarking-disclosure-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须回到 NIST、C2PA、FTC、平台透明度资料或标准机构来源。
- 区分 disclosure、watermark、provenance、detection 和 authenticity，不把标签写成真实性证明。
- 不收集真实脸部、声音、视频、争议证据、未成年人影像或可用于冒充的素材。

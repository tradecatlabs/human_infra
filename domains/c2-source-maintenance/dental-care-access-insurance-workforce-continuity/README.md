# Dental Care Access Insurance Workforce Continuity

`dental-care-access-insurance-workforce-continuity/` 负责整理牙科服务可及性、保险支付、劳动力、地理分布和口腔健康服务连续性。

核心问题：

> 牙科问题常常不是没有技术，而是没有入口；费用、保险、地理、人力和预约摩擦会把可预防疾病拖成疼痛、感染和缺牙。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能把可预防和可处理的口腔问题带入服务系统
  -> 牙科服务受保险、费用、地理、人力、语言和预约系统约束
  -> 服务入口失败会导致预防断裂、急性事件增加和治疗延迟
  -> 因此牙科可及性是口腔健康转化为有效时间的制度接口
```

## 关注对象

- 牙科保险、支付负担、Medicaid/公共项目、牙科劳动力、服务荒漠和预约等待。
- 预防服务、社区口腔健康、学校牙科、移动牙科、基层整合和语言可及性。
- 低收入、老年人、儿童、残障人群、农村地区和长期照护机构的服务缺口。
- 与 `health-literacy-navigation/` 的关系：本域关注牙科系统入口；导航域关注整体健康服务理解和转介。

## Human Infra 模型链路

```text
牙科可及性 / 支付制度 T
  -> 改变保险覆盖、费用、距离、等待、人力、语言和转诊变量 X
  -> 改变主体口腔服务入口状态 S
  -> 改变预防断裂、治疗延迟、急诊使用、缺牙和家庭资源消耗风险
  -> 增加有效时间、公平性和长期口腔维护能力
```

## 非目标

- 不提供个体保险资格、理赔、费用估算、牙医推荐、预约选择或法律建议。
- 不承诺任何项目覆盖、报销、等待时间或服务结果。
- 不替代保险、公共项目、牙科机构、法律或社会服务专业判断。

前庭、平衡与空间定向域整理眩晕、平衡障碍、空间定位、跌倒风险和移动信心对主体行动半径的影响。

核心问题：

> 主体不只是要“能活着”，还必须能稳定站立、转身、行走、定位和避险；前庭和平衡失效会把外部世界变成高成本、高风险空间。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能在空间中稳定移动
  -> 前庭、视觉、本体感觉和肌骨系统共同维持定向与平衡
  -> 眩晕和平衡障碍增加跌倒、外出减少、移动恐惧和任务放弃
  -> 因而前庭平衡是主体持续性中的空间行动基础设施
```

## 关注对象

- 前庭系统、眩晕、头晕、平衡障碍、空间定向和步态稳定。
- 跌倒风险、移动恐惧、外出减少、照护依赖和生活空间收缩。
- 前庭康复、环境适配、辅助技术和交通/建筑无障碍接口。
- 与 `musculoskeletal-integrity/` 的关系：肌骨域关注承载结构；本域关注定向、平衡反馈和空间稳定。
- 与 `road-traffic-injury-prevention-safe-mobility/` 的关系：道路安全域关注系统性出行伤害；本域关注主体自身平衡与定向条件。

## Human Infra 模型链路

```text
平衡筛查 / 前庭康复 / 环境适配 T
  -> 改变眩晕负担、姿势稳定、步态信心和空间定向 X
  -> 改变主体移动半径、外出频率、任务参与和跌倒暴露状态 S
  -> 降低跌倒、骨折、隔离、失能和照护依赖风险 λ(t)
  -> 增加行动密度、独立生活、恢复机会和未来选择权
```

## 非目标

- 不提供眩晕诊断、前庭测试解释、跌倒风险个体评分、康复动作、药物或急救建议。
- 不把平衡训练、设备或环境改造写成通用处方。
- 不替代耳鼻喉、神经、康复、物理治疗或跌倒预防专业服务。

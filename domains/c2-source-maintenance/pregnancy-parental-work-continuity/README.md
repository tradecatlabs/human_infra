# 孕产育儿工作连续性

本域整理怀孕、分娩、哺乳、恢复、育儿绑定和父母工作连续性的制度与环境条件。它关注生育与照护责任如何不把主体排除出工作、收入、学习和社会参与系统。

## 研究对象

- Pregnancy accommodation、孕期工作调整、反歧视和安全边界。
- 分娩恢复、父母 bonding leave、育儿责任、哺乳/泵奶空间与时间。
- PWFA、PUMP Act、Title VII/PDA、FMLA、州/雇主 parental leave 的交叉。
- 怀孕、育儿、托育、工资替代和职业轨迹之间的长期影响。

## 价值链路

```text
生育和育儿会改变身体状态、时间结构和照护责任
  -> 若工作系统不能容纳这些变化，主体会失去收入、岗位和职业积累
  -> 孕产便利、哺乳支持、父母假和反歧视机制保留工作参与入口
  -> 托育与工资替代降低育儿窗口中的时间贫困和财务冲击
  -> 父母与儿童的长期主体持续性同时获得更稳定底座
```

## 关键问题

- 怀孕、分娩、泵奶、育儿和托育缺口分别改变哪些任务资源。
- 合理便利、岗位保护、工资替代和托育如何形成连续链路。
- 哪些环节会制造“母职/父职惩罚”、职业中断和收入轨迹下降。
- 如何把儿童早期生命、托育、家庭连续性和父母工作连续性接到同一模型。

## 上游与下游

- 上游：生殖健康、母婴儿童早期生命、工作设计、托育和家庭经济。
- 下游：父母工作连续性、儿童发展、家庭收入、照护质量和未来选择权。
- 相关域：`maternal-newborn-child-development/`、`childcare-family-continuity/`、`family-medical-leave-job-protection-continuity/`、`paid-sick-family-leave-wage-replacement-continuity/`、`workplace-accommodation-return-to-work-continuity/`。

## 非目标

- 不提供个人怀孕、分娩、哺乳、医疗、法律、HR 或福利建议。
- 不判断具体工作场所便利请求、歧视案件、产假资格或医疗限制。
- 不生成医生证明、HR 通知、法律投诉、雇主政策或诉讼材料。
- 不将任何生育路径包装成义务、道德要求或单一最优人生策略。

# Speech Language Communication Continuity

言语、语言与沟通连续性域整理失语、构音障碍、言语运动障碍、沟通参与和主体表达权对持续性的影响。

核心问题：

> 主体要保有选择权，必须能表达偏好、理解他人、参与关系和进入服务；沟通能力失效会让主体仍然存在却难以进入决策、照护和社会关系。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能表达意图、理解信息和参与共同决策
  -> 语言和言语是同意、求助、关系、工作和身份叙事的关键接口
  -> 失语、构音障碍和言语运动障碍会削弱主体可见性和选择权
  -> 因而言语语言沟通是主体持续性中的表达与参与基础设施
```

## 关注对象

- 失语、构音障碍、言语运动障碍、沟通参与和辅助沟通。
- 脑卒中、TBI、神经退行、发育或其他原因导致的沟通断点。
- 知情同意、医疗沟通、社会关系、工作参与、服务申请和公共表达。
- 与 `language-access-plain-communication/` 的关系：语言可达域关注材料、服务和制度表达；本域关注主体言语语言能力本身。
- 与 `medical-interpreter-cultural-mediation/` 的关系：医疗解释域关注跨语言/文化接口；本域关注主体自身沟通功能和替代沟通支持。

## Human Infra 模型链路

```text
言语语言评估 / 沟通康复 / 辅助沟通 T
  -> 改变表达、理解、命名、发音、语用和替代沟通能力 X
  -> 改变主体同意、求助、关系、照护参与和服务导航状态 S
  -> 降低失权、误解、孤立、照护目标错配和任务失败风险 λ(t)
  -> 增加主体可见性、共同决策能力、社会参与和未来选择权
```

## 非目标

- 不提供失语、构音、言语运动或认知沟通诊断。
- 不提供训练方案、治疗处方、设备处方、吞咽判断或个体康复建议。
- 不替代言语语言治疗、神经、康复、心理或教育专业服务。

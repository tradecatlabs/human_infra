Neuro Continuity 负责整理大脑保存、神经退行防控、意识基础、认知连续性和人格连续性。

核心问题：

> 主体持续性不能只保存身体；如果大脑、记忆、人格、判断和自我连续性断裂，持续存在就失去对象。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须持续感知、记忆、判断、行动和自我归属
  -> 这些能力依赖神经系统和认知结构
  -> 神经退行、脑损伤、记忆崩解或身份断裂会终止主体连续性
  -> 因此神经连续性是有效永生的中心条件
```

## 关注对象

- 阿尔茨海默病、帕金森病、脑卒中、创伤性脑损伤和神经退行。
- 记忆、人格、意识、身份连续性和自我模型。
- 神经保护、脑保存、脑机接口、数字辅助记忆和认知康复。
- 与 `memory-editing/` 的关系：记忆编辑是神经连续性的高风险子问题。
- 与 `disembodied-cns/` 的关系：去具身中枢系统是神经连续性的极端工程路径。

## Human Infra 模型链路

```text
神经连续性技术 T
  -> 改变记忆、意识、人格、神经结构和认知能力 X
  -> 改变主体身份连续状态 S
  -> 改变认知失能、身份断裂和行动能力丧失风险 λ(t)
  -> 决定延寿是否仍然属于“同一个主体”的持续
```

## 非目标

- 不提供脑保存、神经刺激、药物、记忆操控或人体实验步骤。
- 不把数据备份、数字化身或 AI 模拟直接等同于主体连续。
- 不把认知增强和身份连续性混为一谈。

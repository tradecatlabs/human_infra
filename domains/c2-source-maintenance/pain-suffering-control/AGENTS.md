# pain-suffering-control 目录说明

`domains/c2-source-maintenance/pain-suffering-control/` 保存疼痛与痛苦控制研究域。它关注慢性疼痛、症状负担、姑息医学、生活质量和持续存在的可承受性。

## 目录结构

```text
pain-suffering-control/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责痛苦、疼痛、症状负担和生活质量作为主体持续性的质量边界。
- 本域与 `mental-health-affective-stability/`、`sleep-circadian-recovery/` 和 `governance-rights/` 交叉。
- 本域不提供药物、剂量、处方、个体治疗或危机替代服务。

## 维护规则

- 任何痛苦控制主张必须保留主体同意、尊严、表达权和退出边界。
- 不把延长存在时间自动视为价值增益；必须审查可承受性和有效时间。
- 新增文件或子目录时，同步更新本文件。

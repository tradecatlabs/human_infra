`oral-cancer-mucosal-lesion-early-detection-continuity/` 负责整理口腔癌、黏膜病变、早期识别、转诊延迟和生存连续性。

核心问题：

> 口腔癌的持续性风险不只来自肿瘤本身，也来自病变被忽视、转诊断裂、治疗功能损伤和复发监测摩擦。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能及时识别高风险口腔病变并进入诊断治疗路径
  -> 口腔癌和癌前病变会影响吞咽、说话、疼痛、外貌、营养和生存
  -> 延迟发现、转诊断裂和治疗副作用会压缩身体功能与未来选择权
  -> 因此口腔癌早期检测连续性是口腔系统中的高风险边界条件
```

## 关注对象

- 口腔癌、口咽癌、口腔黏膜病变、癌前病变、转诊、诊断延迟和复发监测。
- 烟草、酒精、HPV、免疫状态、社会经济因素和牙科/基层服务入口。
- 吞咽、言语、疼痛、营养、外貌、工作和幸存者生活质量。
- 与 `cancer-screening-early-detection-continuity/` 的关系：本域聚焦口腔/口咽入口和功能后果。

## Human Infra 模型链路

```text
口腔癌早期识别 / 转诊服务 T
  -> 改变病变识别、诊断延迟、转诊闭环、治疗负担和功能保存变量 X
  -> 改变主体癌症风险和口腔功能状态 S
  -> 改变生存、吞咽言语、疼痛、营养、复发恐惧和工作参与风险
  -> 增加有效时间、功能保留和治疗后选择权
```

## 非目标

- 不提供口腔病变照片诊断、癌症筛查频率、活检判断、分期、治疗或预后建议。
- 不提供 HPV、烟草、酒精或个体风险处置建议。
- 不替代牙医、口腔颌面外科、耳鼻喉、肿瘤科或病理专业评估。

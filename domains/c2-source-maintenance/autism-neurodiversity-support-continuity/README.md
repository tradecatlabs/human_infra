Autism Neurodiversity Support Continuity 研究自闭谱系、感觉差异、沟通差异、例行结构、社会环境和支持需求如何影响学习、工作、关系、医疗接入和主体尊严。

> 核心问题：神经多样性不是要被压平的偏差，而是环境、沟通和制度是否允许不同主体持续参与的问题。

## 研究对象

- 自闭谱系、神经多样性、感觉敏感、沟通差异、执行功能、例行结构、社会参与和支持需求。
- 学校、工作、医疗、公共服务、数字服务、家庭照护和社区参与中的可达性和尊严。
- 反污名、反强迫正常化、同意、支持性决策、无障碍沟通和生命周期支持。

## 先验链路

```text
主体持续性需要环境允许不同认知与感觉配置持续参与
  -> 自闭谱系和神经多样性会改变感官负荷、沟通方式、例行需求和社会互动
  -> 不适配环境会造成排除、过载、服务失败和尊严损伤
  -> 支持性环境可提高学习、工作、医疗和关系连续性
  -> 因而自闭神经多样性支持是主体参与权和环境适配域
```

## 关键变量

- 感觉环境、沟通辅助、例行结构、转变负荷、社会参与、执行功能、支持需求和照护者负担。
- 学校便利、工作便利、医疗沟通、公共服务可达、数字可达、反欺凌和反污名。
- 把自闭特质用于歧视、筛选、家庭控制、机构化或去人格化的风险。

## 证据入口

- CDC Autism materials。
- NIMH autism spectrum disorder materials。
- WHO disability and neurodevelopment materials。
- ADA / education accessibility public materials。

## 非目标

- 本域不提供诊断、治疗、行为干预、教育安置、药物、家庭训练、机构选择或个体支持计划。
- 不输出儿童/成人自闭筛查、能力评级、就业筛选、社交训练、照护控制或危机处置建议。
- 不收集儿童资料、诊断、视频、行为记录、学校记录、家庭冲突、位置或敏感身份资料。

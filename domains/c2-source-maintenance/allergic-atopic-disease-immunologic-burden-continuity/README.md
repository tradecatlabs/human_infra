# Allergic Atopic Disease Immunologic Burden Continuity

`allergic-atopic-disease-immunologic-burden-continuity/` 研究过敏性鼻炎、特应性皮炎、食物过敏、过敏原暴露、瘙痒睡眠损害、过敏反应风险和长期免疫负担如何影响主体持续性。

> 核心问题：过敏和特应性疾病常被低估，但它们会通过睡眠破坏、瘙痒、呼吸症状、食物/环境限制、急性反应风险和污名持续消耗有效时间。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要稳定皮肤屏障、气道反应、食物安全、睡眠和环境可预测性
  -> 过敏/特应性疾病会造成瘙痒、鼻眼症状、睡眠损害、限制行为和急性反应风险
  -> 结果取决于过敏原控制、诊断连接、药物可及、学校/工作支持和急救边界
  -> 因此过敏和特应性疾病是免疫负担、恢复质量和生活参与的关键维护域
```

## 关注对象

- Allergic rhinitis、atopic dermatitis、eczema、food allergy、allergen exposure, itch, sleep disruption, anaphylaxis risk boundary, school/work accommodations, environmental control。
- 与 `skin-barrier-wound-healing/` 的边界：该域关注皮肤屏障总机制；本域关注特应性/过敏疾病的长期免疫负担和参与限制。
- 与 `anaphylaxis-allergic-emergency-continuity/` 的关系：本域关注长期控制和环境可及；急性过敏反应由急症域承载。

## Human Infra 模型链路

```text
过敏/特应性疾病负担管理系统 T
  -> 改变过敏原暴露、瘙痒、睡眠、气道症状、食物限制、药物可及和急性反应边界变量 X
  -> 改变恢复质量、学校/工作参与、社交活动、心理负担和环境控制状态 S
  -> 改变有效时间损耗、急性反应风险、精神健康风险和生活参与 λ(t)
  -> 影响主体日常行动质量、恢复能力和未来选择权
```

## 非目标

- 不提供个人过敏诊断、皮疹/皮损判断、食物挑战、脱敏、药物、肾上腺素、急救、饮食或环境改造建议。
- 不生成个人过敏原清单、学校/工作便利、食物限制、护肤、检测或治疗计划。
- 不收集个案皮肤照片、过敏史、食物反应、用药、学校/工作或保险资料。

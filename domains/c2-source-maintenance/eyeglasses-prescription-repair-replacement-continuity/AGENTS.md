# eyeglasses-prescription-repair-replacement-continuity 目录说明

本目录承载眼镜处方、修理、替换、备用和取件等待连续性研究域，关注基础视觉矫正工具如何持续支撑任务执行。

## 目录结构

```text
eyeglasses-prescription-repair-replacement-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义眼镜运行、处方更新、维修替换和视觉任务断点如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 NEI、MedlinePlus、学校视觉服务、低视力服务和公共健康材料。
- 不写验光、处方参数、镜片选择、品牌推荐、购买攻略、驾驶资格、保险个案或维修教程。
- 新增指标必须注明口径：眼镜可得性、处方更新摩擦、修理替换等待、备用能力、视觉任务失败后果或费用负担。
- 涉及儿童、驾驶、老年照护和灾害失物时，必须保留安全、资格和服务转介边界。

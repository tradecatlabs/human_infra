# hearing-auditory-communication-continuity AGENTS

## 职责

- 维护听力、听觉沟通和声音警报连续性的研究边界、证据入口、变量表和禁止用途。
- 将听力损失、言语识别、耳鸣、助听支持、沟通疲劳和社会参与资料整理为 Source Card。
- 与 `sensory-continuity/`、`language-access-plain-communication/`、`assistive-technology-access/`、`medical-interpreter-cultural-mediation/` 保持边界清晰。

## 非目标

- 不提供听力诊断、听力图解释、设备处方、设备调试、耳鸣治疗或个体医疗建议。
- 不替代听力学、耳鼻喉、言语语言治疗、康复或公共服务无障碍专业服务。

## 上游

- `sensory-continuity/`：跨感官反馈上位域。
- `healthcare-access-continuity/`：听力照护和转诊服务。
- `assistive-technology-access/`：助听设备、字幕、提示系统和维护可及性。

## 下游

- `social-connection-relational-infra/`：沟通失败和关系孤立。
- `medical-interpreter-cultural-mediation/`：临床沟通安全。
- `time-allocation-effective-time/`：沟通疲劳和重复解释成本。

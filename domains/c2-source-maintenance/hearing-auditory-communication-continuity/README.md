听力与听觉沟通连续性域整理听力损失、言语识别、耳鸣、助听支持和沟通负担对主体持续行动的影响。

核心问题：

> 主体要持续参与关系、工作、照护和公共服务，必须能接收语音、警报和环境声音；听觉沟通失效会把人推向孤立、误解、认知负担和安全信号漏接。

## 先验位置

```text
主体持续性最大化
  -> 主体必须持续接入他人、服务和环境警报
  -> 听觉承载语言、社交、风险信号和空间线索
  -> 听力损失增加沟通成本、孤立、疲劳和信息遗漏
  -> 因而听力与听觉沟通是主体持续性中的关系和警报输入层
```

## 关注对象

- 听力损失、年龄相关性听力下降、耳鸣和听觉处理负担。
- 言语识别、噪声环境沟通、助听器/人工耳蜗等听觉支持的可及性与维护。
- 听觉警报、环境声音、安全风险、照护沟通和公共服务沟通。
- 社交孤立、认知负担、工作参与、医疗沟通和服务理解。
- 与 `language-access-plain-communication/` 的关系：本域关注听觉通道；语言可达域关注语言、清晰表达和可理解性。

## Human Infra 模型链路

```text
听力照护 / 助听支持 / 环境声学 T
  -> 改变声音可达性、言语识别、耳鸣负担和沟通疲劳 X
  -> 改变主体社交参与、服务理解、安全警报接收和工作状态 S
  -> 降低孤立、误解、事故、认知负担和任务中断风险 λ(t)
  -> 增加关系连续性、服务可达性、有效时间和未来选择权
```

## 非目标

- 不提供听力诊断、听力图解释、助听器/人工耳蜗处方、耳鸣治疗或设备购买建议。
- 不把听力指标写成个体认知、工作、驾驶、学习或安全资格判断。
- 不替代听力学、耳鼻喉、言语语言治疗或康复专业服务。

Proteostasis Autophagy 负责整理蛋白稳态、自噬、溶酶体、错误折叠、蛋白聚集、蛋白质清除和细胞质量控制。

核心问题：

> 主体长期运行不只需要基因可读，还需要蛋白质系统可折叠、可清除、可更新；蛋白稳态崩塌会通过神经退行、肌肉衰退和细胞压力压缩有效时间。

## 先验位置

```text
主体持续性最大化
  -> 细胞功能依赖蛋白质正确折叠、运输、降解和更新
  -> 蛋白错误折叠、聚集和清除失败会损害细胞质量控制
  -> 自噬和溶酶体系统是损伤清除与资源回收的核心机制
  -> 因此蛋白稳态是长期细胞功能和神经连续性的基础条件
```

## 关注对象

- 蛋白折叠、伴侣蛋白、泛素-蛋白酶体系统、自噬、溶酶体和蛋白聚集。
- 蛋白稳态与神经退行、肌肉衰退、免疫、代谢和衰老标志的关系。
- 自噬相关指标、药物/营养感知通路和替代终点的证据边界。
- 与 `neuro-continuity/` 的关系：本域关注蛋白质量控制机制；神经域关注大脑、认知和身份连续终点。

## Human Infra 模型链路

```text
蛋白稳态 / 自噬干预 T
  -> 改变错误折叠、聚集、自噬通量和清除能力 X
  -> 改变细胞质量控制状态 S
  -> 改变神经退行、肌肉衰退、炎症和组织功能风险
  -> 改变有效时间、认知连续性和恢复能力
```

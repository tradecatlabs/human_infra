# adhd-executive-function-neurodevelopment-continuity 目录说明

本目录承载 ADHD、执行功能和神经发育支持连续性资料。它关注任务启动、注意维持和环境支持，而不是诊断、用药或个体训练。

## 结构

```text
adhd-executive-function-neurodevelopment-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分执行功能机制、ADHD 诊断类别、学校/工作支持、合理便利、工具支持和反筛选边界。
- 必须避免把神经发育差异写成懒惰、低能力、招聘风险、学生标签或监控对象。
- 不保存儿童个案、成绩、行为日志、诊断材料、用药记录、工作绩效或家庭资料。

ADHD Executive Function Neurodevelopment Continuity 研究 ADHD、执行功能差异、注意维持、冲动控制、时间感、任务启动和学校/工作支持如何影响主体行动密度与长期目标推进。

> 核心问题：执行功能不是人格评价，而是把目标转化为行动、反馈和完成的基础控制系统。

## 研究对象

- ADHD、注意维持、抑制控制、工作记忆、任务启动、时间管理、情绪调节和执行功能差异。
- 学校、家庭、工作、数字工具、提醒系统、环境设计和合理便利如何支持持续执行。
- 诊断可及、反污名、过度筛选、招聘/教育排除和监控化管理边界。

## 先验链路

```text
主体持续性需要目标能被分解、启动、维持和完成
  -> ADHD 与执行功能差异会影响注意、冲动、时间感、任务切换和反馈利用
  -> 学习、工作、财务、照护和长期项目推进更容易断裂
  -> 合理支持和环境结构可降低摩擦并提高行动密度
  -> 因而 ADHD 执行功能连续性是任务执行基础设施域
```

## 关键变量

- 注意维持、冲动、工作记忆、时间感、任务启动、任务切换、情绪调节和完成率。
- 学校支持、工作便利、提醒工具、环境摩擦、数字分心、诊断服务和污名。
- 把执行功能差异变成招聘筛选、学生标签、监控评分或道德评价的风险。

## 证据入口

- CDC ADHD materials。
- NIMH ADHD materials。
- AAP / educational support public materials。
- 执行功能、学校支持和工作便利公开研究。

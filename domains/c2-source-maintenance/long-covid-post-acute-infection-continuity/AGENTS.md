# long-covid-post-acute-infection-continuity 目录说明

本目录承载 Long COVID 与感染后长期功能连续性研究域，只做公开资料整理、机制建模和边界约束。

```text
long-covid-post-acute-infection-continuity/
├── AGENTS.md

```

## 职责边界

- `README.md` 定义 Long COVID、感染后长期症状、多系统负担、服务连续性和主体持续性的关系。
- 本域上游依赖 `chronic-infectious-disease-care-continuity/`、`respiratory-oxygenation/`、`autonomic-nervous-system-homeostasis/`、`fatigue-risk-alertness-continuity/` 和 `healthcare-access-continuity/`。
- 下游服务感染后长期状态、不可见失能、工作/学校适配和证据更新模型。

## 维护规则

- 只使用 CDC、NIH、WHO、专业学会或同行评审资料作为来源信号。
- 不提供诊断、检测解释、药物、康复、运动、补剂、氧疗、工作/学校证明、残障、保险或预后建议。

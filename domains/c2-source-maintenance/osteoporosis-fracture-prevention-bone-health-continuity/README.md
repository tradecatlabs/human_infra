# Osteoporosis Fracture Prevention Bone Health Continuity

`osteoporosis-fracture-prevention-bone-health-continuity/` 研究骨质疏松筛查、骨折风险、跌倒-骨折耦合、骨健康服务、药物坚持和髋部/椎体骨折后功能恢复如何影响主体持续性。

> 核心问题：骨折会把长寿直接转化为失能、疼痛、照护负担和机构化风险。有效永生需要把骨健康和骨折预防建模为行动能力与独立生活的结构性前提。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体依赖骨骼强度维持站立、移动、自理和外出能力
  -> 骨质疏松会让低能量跌倒转化为髋部、椎体和腕部骨折
  -> 结果取决于筛查、风险识别、治疗可及、坚持、跌倒预防和骨折后恢复
  -> 因此骨质疏松与骨折预防是行动持续性的结构性风险域
```

## 关注对象

- Osteoporosis screening, BMD access, fracture risk, fragility fractures, treatment adherence, fall-fracture interface, post-fracture rehabilitation。
- 与 `musculoskeletal-integrity/` 的边界：该域关注肌骨系统总体承载；本域关注骨量、脆性骨折和骨折预防执行链。
- 与 `fall-risk-prevention-home-safety-continuity/` 的关系：跌倒是骨折触发因素之一，骨健康决定跌倒后损害大小。

## Human Infra 模型链路

```text
骨质疏松与骨折预防系统 T
  -> 改变筛查、骨密度可及、骨折风险识别、治疗坚持和康复变量 X
  -> 改变骨强度、跌倒后损伤概率、疼痛和行动状态 S
  -> 改变髋部/椎体骨折、机构化、死亡和有效时间损失 λ(t)
  -> 影响主体独立生活、行动半径和未来选择权
```

## 非目标

- 不提供骨密度解读、补钙/维 D、药物、运动、跌倒训练、影像、骨折处理、手术或个体医疗建议。
- 不生成个人骨折风险评分、处方、康复计划、设备推荐或保险申诉。
- 不收集个案骨密度、影像、跌倒史、骨折史、用药、体重或诊疗资料。

嗅味觉与化学感知连续性域整理嗅觉、味觉、危险气味识别、食欲、营养和生活质量。

核心问题：

> 嗅觉和味觉不是装饰性体验；它们参与烟气、燃气、腐败食物等危险识别，也影响食欲、营养、情绪和日常生活质量。

## 先验位置

```text
主体持续性最大化
  -> 主体必须识别环境和食物中的化学信号
  -> 嗅味觉支撑危险预警、食欲、营养选择和生活体验
  -> 嗅味觉失效会增加安全信号漏接、食欲下降、营养风险和情绪负担
  -> 因而化学感知是主体持续性中的低可见但高价值反馈层
```

## 关注对象

- 嗅觉障碍、味觉障碍、化学感知减退、异嗅/异味和生活质量。
- 烟气、燃气、腐败食品、化学品等危险信号识别。
- 食欲、营养摄入、饮食愉悦、社交用餐和心理负担。
- 感染、神经退行、创伤、药物、环境暴露和老化相关线索的证据边界。
- 与 `food-safety-contamination-control/` 的关系：食品安全域关注系统性污染控制；本域关注主体自身化学感知和危险识别能力。

## Human Infra 模型链路

```text
嗅味觉保护 / 环境安全替代警报 / 营养支持 T
  -> 改变危险气味识别、食欲、味觉愉悦和食物选择 X
  -> 改变主体安全、营养、情绪、社交用餐和生活质量状态 S
  -> 降低中毒、火灾漏警、营养不足、体重变化和生活质量下降风险 λ(t)
  -> 增加恢复能力、风险感知、有效生活质量和未来选择权
```

## 非目标

- 不提供嗅觉/味觉诊断、感染判断、神经退行判断、治疗、训练、补剂或药物建议。
- 不把嗅味觉变化写成个体疾病诊断、风险预测或恐慌结论。
- 不替代耳鼻喉、神经、营养、心理或环境安全专业服务。

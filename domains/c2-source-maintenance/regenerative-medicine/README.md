# Regenerative Medicine

Regenerative Medicine 负责整理再生医学、干细胞、组织工程、器官修复、类器官、移植和生物制造等“身体部件可修复 / 可替换”路径。

核心问题：

> 如果主体要长期持续行动，身体损耗是否能够被修复、替换、再生或外部支持，而不是只能被动衰退。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须拥有可运行身体
  -> 身体由器官、组织、血管、神经、免疫和代谢系统组成
  -> 这些子系统会因损伤、疾病、衰老和事故失效
  -> 若关键部件不可修复，主体持续性会被局部故障终止
  -> 因此再生医学是身体基底可维护性的研究域
```

## 关注对象

- 干细胞疗法、组织工程、器官再生和器官替代。
- 类器官、3D 生物打印、细胞外基质和组织支架。
- 创伤修复、神经再生、肌肉骨骼修复、血管再生和器官功能恢复。
- 移植、异种移植、免疫排斥、长期整合和功能终点。
- 与 `cellular-reprogramming/` 的交叉：细胞状态重写如何服务组织修复。

## Human Infra 模型链路

```text
再生医学技术 T
  -> 改变器官结构、组织功能、修复速度和替代能力 X
  -> 改变主体身体状态 S
  -> 降低失能、器官衰竭、慢病进展和损伤后死亡风险 λ(t)
  -> 增加健康寿命、有效时间和行动窗口
```

## 非目标

- 不提供手术、细胞制备、移植、培养、给药或人体实验操作协议。
- 不把临床前再生信号写成人体器官替换已经成熟。
- 不把组织结构改善自动等同于长期主体持续性收益。

视觉与眼健康连续性域整理视力、眼病、视觉功能、低视力支持和视觉任务参与对主体持续性的影响。

核心问题：

> 主体要持续行动，必须持续获得足够可靠的外部视觉输入；视觉输入失效会压缩阅读、移动、工作、照护、安全判断和环境反馈。

## 先验位置

```text
主体持续性最大化
  -> 主体必须持续接收外部世界信号
  -> 视觉是高带宽空间、文字、风险和社交线索输入
  -> 视力损失会降低任务完成、移动安全、信息获取和独立生活能力
  -> 因而视觉与眼健康是主体持续性中的输入层基础设施
```

## 关注对象

- 视力损失、盲、低视力、可避免视觉损害。
- 白内障、青光眼、年龄相关性黄斑变性、糖尿病视网膜病变等眼健康风险。
- 阅读、驾驶、工作、移动、跌倒、照护和环境识别中的视觉功能。
- 低视力辅助、无障碍设计、筛查可及性和眼科服务连续性。
- 与 `sensory-continuity/` 的关系：本域承接视觉专项机制；上位感官域保留跨感官整合。
- 与 `assistive-technology-access/` 的关系：本域评估视觉功能与任务失败；辅助技术域评估工具可及、适配和维护。

## Human Infra 模型链路

```text
眼健康服务 / 筛查 / 低视力支持 T
  -> 改变视觉输入质量、视野、阅读能力、空间识别和风险感知 X
  -> 改变主体的信息获取、移动安全、工作参与和独立生活状态 S
  -> 降低跌倒、事故、社交退缩、任务失败和照护依赖风险 λ(t)
  -> 增加有效时间、行动半径、学习入口和未来选择权
```

## 非目标

- 不提供眼科诊断、验光处方、检查解释、治疗方案、用药、手术或设备购买建议。
- 不把单一视力指标写成个体工作、驾驶、学习或安全资格判断。
- 不把低视力辅助工具宣传成通用解决方案；必须保留适配、训练、费用和维护边界。
- 不替代眼科、验光、康复或无障碍专业服务。

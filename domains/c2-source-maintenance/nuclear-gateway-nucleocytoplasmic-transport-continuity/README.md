Nuclear Gateway Nucleocytoplasmic Transport Continuity 负责整理核孔复合体、核质运输、karyopherins、核膜/核纤层、核结构压力和衰老中的核入口维护。

核心问题：

> 细胞核不是静态存储盒，而是持续交换信息和分子的控制室；核入口、核膜和核质运输失衡会改变基因表达、蛋白稳态和压力响应。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 细胞身份和功能依赖核内外分子交换、核结构稳定和压力响应
  -> 核孔、karyopherins、核纤层和核膜机械耦合决定分子进出和核形态
  -> 核入口失衡会影响蛋白稳态、DNA 维护、转录调控和细胞身份稳定
  -> 长期失衡可能改变神经、肌肉、心血管和衰老相关疾病风险
  -> 因此核网关是细胞信息系统长期可维护性的边界条件
```

## 关注对象

- nuclear pore complex、nucleocytoplasmic transport、karyopherins、nuclear lamina、nuclear envelope、mechanotransduction。
- 与 `genomic-stability-dna-repair/` 的关系：基因组域关注 DNA 损伤与修复，本域关注核结构与运输边界。
- 与 `proteostasis-autophagy/` 的关系：本域关注核入口如何影响蛋白稳态和压力响应。

## Human Infra 模型链路

```text
核入口维护观测 / 调节 T
  -> 改变核孔通透性、karyopherin 运输、核纤层结构和核膜力学 X
  -> 改变核内外信息交换与细胞身份稳定状态 S

  -> 改变主体行动能力、恢复能力和健康寿命
```

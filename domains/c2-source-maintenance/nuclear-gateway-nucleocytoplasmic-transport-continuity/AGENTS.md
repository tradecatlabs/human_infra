# nuclear-gateway-nucleocytoplasmic-transport-continuity 目录说明

`domains/c2-source-maintenance/nuclear-gateway-nucleocytoplasmic-transport-continuity/` 保存核孔、核质运输、核纤层和核入口维护研究域。

## 目录结构

```text
nuclear-gateway-nucleocytoplasmic-transport-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责 nuclear pore complex、nucleocytoplasmic transport、karyopherins、nuclear lamina、nuclear envelope 和 mechanotransduction 资料。
- 本域可整理核结构、压力响应、蛋白稳态、DNA 维护、神经/肌肉/心血管交叉资料。
- 本域不提供检测解释、基因治疗、力学干预、实验操作或临床建议。

## 维护规则

- 所有主张必须区分核结构、运输功能、分子指标、组织病理和个体风险推断。
- 与 `genomic-stability-dna-repair/`、`proteostasis-autophagy/` 和 `extracellular-matrix-glycation/` 交叉时区分核内、细胞内和组织力学层级。
- 新增文件或子目录时同步更新本文件。

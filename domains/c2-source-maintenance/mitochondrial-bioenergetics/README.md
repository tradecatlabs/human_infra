Mitochondrial Bioenergetics 负责整理线粒体功能、细胞能量、氧化应激、线粒体质量控制、疲劳、肌肉和神经能量系统。

核心问题：

> 主体行动、修复和认知都需要细胞能源；线粒体功能下降会把时间窗口转化为疲劳、失能、代谢紊乱和神经风险。

## 先验位置

```text
主体持续性最大化
  -> 主体必须持续获得可用细胞能量
  -> 线粒体决定 ATP 供给、氧化还原状态、细胞压力和代谢弹性
  -> 线粒体失调会影响肌肉、神经、心血管、免疫和恢复
  -> 因此线粒体生物能量学是行动能力和恢复能力的基础条件
```

## 关注对象

- 线粒体功能、ATP、氧化应激、线粒体 DNA、线粒体自噬、融合分裂和质量控制。
- 线粒体与疲劳、肌肉、神经、代谢、免疫、心血管和衰老标志的关系。
- 生物能量指标、替代终点、临床终点和个体差异。
- 与 `sleep-circadian-recovery/` 的关系：恢复域关注系统调度；本域关注细胞能源和质量控制。

## Human Infra 模型链路

```text
线粒体 / 能量干预 T
  -> 改变 ATP 供给、氧化还原、线粒体质量控制和疲劳变量 X
  -> 改变细胞能源状态 S
  -> 改变行动能力、恢复能力、神经风险、代谢风险和疾病风险
  -> 改变有效时间和长期任务执行能力
```

## 非目标

- 不提供补剂、药物、运动、禁食、检测解释或治疗建议。
- 不把线粒体指标直接写成寿命预测。
- 不把“抗氧化”或“能量提升”写成无条件有益。

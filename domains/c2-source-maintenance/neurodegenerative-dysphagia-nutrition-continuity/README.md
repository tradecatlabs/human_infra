# Neurodegenerative Dysphagia Nutrition Continuity

Neurodegenerative dysphagia nutrition continuity studies swallowing decline in progressive neurological disease as a long-horizon interaction among nutrition, aspiration risk, communication, caregiver burden, and advance decision-making.

核心问题：

> 神经退行性吞咽困难不是一次性事件，而是随疾病进展改变营养入口、气道风险、照护负担和意愿决策的长期曲线。

## 先验位置

```text
主体持续性最大化
  -> 慢性神经疾病中的主体必须长期维持营养、水分、沟通和尊严
  -> 退行性病程可能逐步削弱吞咽、咳嗽、姿势、认知和照护执行
  -> 长期曲线会改变口服、管饲、肺部风险和代理决策
  -> 因而神经退行性吞咽是长期主体持续性中的慢性入口失效模式
```

## 关注对象

- Parkinson's、ALS、痴呆、Huntington、MS 等背景下的吞咽下降、营养水分、误吸、体重、沟通、管饲决策和照护负担。
- 与 `parkinsons-disease-motor-nonmotor-continuity/`、`amyotrophic-lateral-sclerosis-respiratory-motor-continuity/` 和 `dementia-cognitive-decline-care-continuity/` 的关系：这些域关注疾病整体；本域关注吞咽营养入口。
- 与 `advance-care-planning-medical-decision-continuity/` 的关系：决策域关注意愿与代理；本域保留吞咽相关决策边界。

## Human Infra 模型链路

```text
长期监测 / 营养支持 / 沟通支持 / 决策规划 T
  -> 改变吞咽能力、体重、误吸暴露、照护负担和意愿表达变量 X
  -> 改变营养状态、肺部风险、家庭执行和主体尊严状态 S
  -> 改变肺炎、脱水、营养不足、照护崩溃和决策冲突风险 λ(t)
  -> 影响有效寿命、照护连续性和未来选择权
```

## 非目标

- 不提供疾病诊断、吞咽训练、饮食调整、管饲决策、药物、预后、照护处置或法律能力判断。
- 不根据个体病程、体重、吞咽、认知或家庭情况判断下一步行动。
- 不替代神经、言语语言治疗、营养、呼吸、康复、伦理、护理或法律服务。

# immunization-record-proof-continuity 目录说明

本目录承载免疫记录与证明连续性资料。它研究接种事件如何通过记录、登记、证明和凭证进入教育、工作、旅行、医疗和公共卫生系统。

## 结构

```text
immunization-record-proof-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分医学接种建议、公共卫生登记、制度证明和数字凭证。
- 必须记录隐私、纠错、跨机构可携带、反伪造和反排除边界。
- 不保存个人疫苗记录、疫苗卡、学校/雇主/旅行资料、身份文件或 PHI。

`dry-eye-ocular-surface-comfort-continuity/` 负责整理干眼、眼表不适、屏幕/环境暴露、慢性疼痛疲劳和视觉任务舒适性连续性。

核心问题：

> 干眼通常不直接终止生命，但会以持续疼痛、模糊、畏光和屏幕耐受下降的方式吞噬有效时间；它是“低致死、高摩擦”的视觉工作负担。

## 先验位置

```text
主体持续性最大化
  -> 主体必须长时间稳定读写、看屏幕、移动和处理视觉任务
  -> 眼表不适会降低视觉耐受、注意力稳定和恢复质量
  -> 慢性干眼会把工作、学习和社交时间转化为疼痛、疲劳和任务中断
  -> 因此干眼眼表舒适连续性是有效时间和视觉任务耐受的基础条件
```

## 关注对象

- 干眼、眼表不适、刺痛、异物感、泪液问题、视疲劳、光敏和波动性模糊。
- 屏幕使用、室内空气、药物副作用、风/干燥环境、隐形眼镜和工作场景。
- 注意力、阅读速度、屏幕耐受、睡眠恢复、工作效率和生活质量。
- 与 `vision-eye-health-continuity/` 的关系：本域聚焦眼表舒适和有效时间负担，不覆盖所有眼病。

## Human Infra 模型链路

```text
眼表健康管理 / 环境改善 / 服务衔接 T
  -> 改变干涩、疼痛、视疲劳、屏幕耐受、环境暴露和恢复变量 X
  -> 改变主体阅读、工作、学习、睡眠和社交参与状态 S
  -> 改变任务中断、注意力下降、慢性不适和有效时间损耗风险
  -> 增加视觉任务耐受、恢复质量和创造窗口
```

## 非目标

- 不提供干眼诊断、眼药水选择、处方药、补剂、隐形眼镜建议或个体治疗方案。
- 不根据症状判断个人是否有感染、免疫疾病、角膜问题或紧急风险。
- 不把屏幕习惯、环境改变或产品写成治疗承诺。

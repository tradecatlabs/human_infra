# reproductive-fertility-continuity 目录说明

`domains/c2-source-maintenance/reproductive-fertility-continuity/` 保存生殖与生育连续性研究域。它关注生育力、生殖衰老、生殖自主性和未来选择权如何影响主体持续性。

## 目录结构

```text
reproductive-fertility-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责生育力、配子质量、生殖衰老、不孕、妊娠风险、生殖权利和未来选择权。
- 本域可整理公开生殖健康资料、综述、疾病负担和权利边界。
- 本域不提供不孕诊断、生育治疗、激素用药、辅助生殖、妊娠管理、避孕或个体医疗建议。

## 维护规则

- 任何生殖与生育主张必须说明变量、对象、证据等级、适用人群和跨域边界。
- 涉及生育治疗、妊娠、避孕或激素时只写公共资料和非操作性安全边界。
- 新增文件或子目录时，同步更新本文件。

# delirium-acute-cognitive-failure-prevention 目录说明

本目录维护谵妄和急性认知失败预防资料。它是医疗系统中的短期主体连续性保护域。

## 目录结构

```text
delirium-acute-cognitive-failure-prevention/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义谵妄、急性意识注意障碍和住院/ICU 认知失败如何影响主体持续性。
- 本目录不保存诊断、用药、镇静调整、ICU 治疗或个体照护计划。
- 资料必须区分筛查工具、预防 bundle、组织流程、临床指南和个体医疗判断。

## 上下游关系

- 上游来自老年医学、ICU 医学、麻醉、护理、药学、睡眠和患者安全。
- 下游服务医疗决策能力、认知恢复、长期照护和患者安全组织学习。
- 不替代 `dementia-cognitive-decline-care-continuity/`；本目录聚焦急性、波动性、常可预防的认知失败。

## 维护规则

- 不提供诊断、镇静、用药、急救或个体护理建议。
- 不把筛查工具、家属观察或行为表现写成个人诊断。
- 新增资料必须标注适用场景、证据阶段、预防机制、组织边界和禁止用途。

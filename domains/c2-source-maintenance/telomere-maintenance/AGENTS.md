# telomere-maintenance 目录说明

`domains/c2-source-maintenance/telomere-maintenance/` 保存端粒维护研究域。它关注端粒、端粒酶和复制边界如何影响组织更新、衰老和癌症风险。

## 目录结构

```text
telomere-maintenance/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责端粒长度、端粒酶、复制衰老、端粒疾病和癌症风险边界。
- 本域可整理机制文献、测量限制、证据等级和风险张力。
- 本域不提供端粒检测解释、端粒酶干预、补剂、药物或治疗建议。

## 维护规则

- 任何端粒主张必须同时记录修复潜力和癌症风险。
- 与基因组稳定性、癌症、干细胞、免疫和细胞衰老交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

# cardiovascular-resilience 目录说明

`domains/c2-source-maintenance/cardiovascular-resilience/` 保存心血管韧性研究域。它关注循环系统如何支撑主体持续行动并决定主要死亡风险。

## 目录结构

```text
cardiovascular-resilience/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责心脏、血管、血压、动脉粥样硬化、血栓、卒中和心衰相关研究整理。
- 本域可整理风险函数、筛查证据、公共健康资料和模型变量。
- 本域不提供诊断、用药、剂量、手术、治疗或个体风险日期。

## 维护规则

- 任何风险主张必须说明适用人群、终点、时间尺度和证据层级。
- 与营养、活动、睡眠、污染、代谢和肾脏交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

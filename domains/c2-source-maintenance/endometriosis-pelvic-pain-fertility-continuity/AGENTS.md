# Endometriosis Pelvic Pain Fertility Continuity 目录说明

本目录承载子宫内膜异位症、盆腔痛、疲劳、生育力和诊断延迟资料。它处理长期功能损耗和未来选择权，不处理个体诊断或治疗。

## 目录结构

```text
endometriosis-pelvic-pain-fertility-continuity/

```

## 维护规则

- 新增资料优先引用 WHO、NIH/NICHD、MedlinePlus、系统综述和临床指南。
- 记录 pain burden、diagnostic delay、fatigue、fertility planning、work/school participation、treatment burden 和 stigma。
- 不写止痛、激素、手术、饮食、补剂、性交痛处理、生育治疗或个人症状判断。

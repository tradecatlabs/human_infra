周围神经病变与躯体感觉连续性域整理触觉、痛觉、本体感觉、足部风险、精细操作反馈和身体边界感。

核心问题：

> 主体必须知道身体在哪里、是否受伤、如何接触工具和地面；周围神经和躯体感觉失效会让损伤不可感、步态不稳、精细操作变慢，并把小伤口变成长期失能风险。

## 先验位置

```text
主体持续性最大化
  -> 主体必须持续接收身体边界和接触反馈
  -> 周围神经、触觉、痛觉和本体感觉支撑操作、步态、避伤和足部保护
  -> 躯体感觉失效增加跌倒、伤口、溃疡、疼痛、精细动作失败和照护负担
  -> 因而周围神经感觉是具身行动和身体维护的底层反馈基础设施
```

## 关注对象

- 周围神经病变、麻木、疼痛、触觉下降、本体感觉下降和感觉运动失衡。
- 糖尿病周围神经病变、足部损伤、溃疡风险、步态不稳和跌倒。
- 精细操作、工具使用、皮肤保护、疼痛负担和日常任务反馈。
- 与 `pain-suffering-control/` 的关系：疼痛域关注痛苦和症状负担；本域关注周围感觉反馈、保护性痛觉和身体边界。
- 与 `skin-barrier-wound-healing/` 的关系：皮肤域关注伤口愈合；本域关注伤口发生前的感觉保护和损伤发现。

## Human Infra 模型链路

```text
神经病变筛查 / 足部保护 / 感觉反馈支持 T
  -> 改变触觉、痛觉、本体感觉、步态反馈和伤口发现能力 X
  -> 改变主体移动、工具操作、皮肤保护、疼痛负担和自我维护状态 S
  -> 降低跌倒、溃疡、感染、精细任务失败和照护依赖风险 λ(t)
  -> 增加具身行动质量、独立生活、有效时间和未来选择权
```

## 非目标

- 不提供神经病变诊断、神经传导检查解释、足部护理处置、疼痛治疗、药物、补剂或康复建议。
- 不把麻木、疼痛或感觉变化写成个体疾病判断或紧急处置建议。
- 不替代神经、内分泌、足病、伤口、疼痛、康复或临床照护专业服务。

# peripheral-neuropathy-somatosensory-continuity AGENTS

## 职责

- 维护周围神经、躯体感觉、身体边界反馈和足部风险的研究边界、证据入口、变量表和禁止用途。
- 将周围神经病变、保护性感觉、本体感觉、足部伤口、疼痛和操作反馈资料整理为 Source Card。
- 与 `sensory-continuity/`、`pain-suffering-control/`、`skin-barrier-wound-healing/`、`physical-activity-mobility/` 保持边界清晰。

## 非目标

- 不提供诊断、检查解释、足部处置、药物、疼痛治疗、康复训练、补剂或个体风险判断。
- 不替代神经、内分泌、足病、伤口照护、疼痛、康复或临床专业服务。

## 上游

- `sensory-continuity/`：跨感官反馈框架。
- `endocrine-hormonal-regulation/`：糖代谢和长期神经损伤接口。
- `pain-suffering-control/`：疼痛和痛苦负担。

## 下游

- `skin-barrier-wound-healing/`：伤口发现、感染入口和愈合。
- `physical-activity-mobility/`：步态、平衡和行动半径。
- `time-allocation-effective-time/`：疼痛、麻木和精细操作失败造成的有效时间损耗。

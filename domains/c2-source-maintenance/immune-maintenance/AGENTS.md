# immune-maintenance 目录说明

`domains/c2-source-maintenance/immune-maintenance/` 保存免疫系统维护研究域。它关注免疫衰老、感染防御、慢性炎症、自身免疫和肿瘤免疫监视。

## 目录结构

```text
immune-maintenance/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责长期主体运行中的免疫变量和风险模型。
- 本域可整理免疫衰老、疫苗、感染、炎症、自身免疫和免疫监视资料。
- 本域不提供个体免疫治疗、疫苗接种、药物或实验操作建议。

## 维护规则

- 任何“免疫增强”主张必须同时审查感染防御、炎症、自身免疫和组织损伤风险。
- 与癌症、再生、重编程交叉时必须标注双向影响。
- 新增文件或子目录时，同步更新本文件。

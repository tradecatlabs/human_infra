# Stem Cell Reserve Renewal

Stem Cell Reserve Renewal 负责整理干细胞储备、组织更新、干细胞耗竭、生态位、再生能力和内源性修复边界。

核心问题：

> 身体长期维护依赖组织自我更新；当干细胞储备、生态位和分化能力下降时，主体可能仍存活，但修复、免疫、造血、皮肤和肌肉更新能力会持续下降。

## 先验位置

```text
主体持续性最大化
  -> 组织必须持续替换损伤细胞并维持结构
  -> 干细胞储备、生态位和分化能力决定内源性修复上限
  -> 干细胞耗竭会降低再生、免疫、造血、皮肤和肌肉维护能力
  -> 因此干细胞储备是身体自我更新和长期修复的基础条件
```

## 关注对象

- 干细胞耗竭、造血干细胞、肌肉卫星细胞、肠道干细胞、皮肤更新和生态位。
- 干细胞储备与炎症、代谢、细胞衰老、端粒、癌症和再生医学的关系。
- 内源性修复能力、外源性再生技术和肿瘤风险之间的边界。
- 与 `regenerative-medicine/` 的关系：本域关注体内储备和内源性更新；再生医学域关注工程化修复和替代。

## Human Infra 模型链路

```text
干细胞储备 / 生态位干预 T
  -> 改变干细胞数量、分化能力、生态位和更新速度 X
  -> 改变组织自我修复状态 S
  -> 改变再生能力、免疫更新、组织退化、癌症和失能风险
  -> 改变健康寿命、恢复能力和未来治疗窗口
```

## 非目标

- 不提供干细胞治疗、细胞制备、注射、移植、培养或人体实验建议。
- 不把“激活干细胞”写成无条件有益。
- 不忽略干细胞扩增、异常克隆和肿瘤风险。

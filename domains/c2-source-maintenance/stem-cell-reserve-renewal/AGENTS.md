# stem-cell-reserve-renewal 目录说明

`domains/c2-source-maintenance/stem-cell-reserve-renewal/` 保存干细胞储备与组织更新研究域。它关注内源性修复能力如何支撑长期身体维护。

## 目录结构

```text
stem-cell-reserve-renewal/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责干细胞耗竭、组织更新、生态位、内源性修复能力和风险边界。
- 本域可整理机制文献、组织特异性、再生能力变量和癌症风险。
- 本域不提供干细胞治疗、培养、移植、注射或人体实验建议。

## 维护规则

- 任何干细胞主张必须同时记录修复收益、异常克隆、肿瘤和身份/分化风险。
- 与再生医学、细胞重编程、端粒、免疫和癌症交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

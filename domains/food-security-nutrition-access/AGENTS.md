# food-security-nutrition-access 目录说明

`food-security-nutrition-access/` 是 Human Infra 的食物安全与营养可及域，负责研究食物系统如何为主体持续性提供稳定、安全、可负担的营养输入。

## 目录结构

```text
food-security-nutrition-access/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义食物安全与营养可及域的对象、先验位置、模型链路、非目标和来源信号。
- 本域只做食物可得、可负担、安全、营养质量、供应冲击和食物系统风险资料整理。
- 个体饮食处方、补剂、疾病饮食、食品处置、囤积和非法获取不属于本域。

## 上下游关系

- 上游：`supply-chain-continuity/`、`financial-resilience-access/`、`social-protection-benefits-delivery/`。
- 下游：`nutrition-metabolic-health/`、`immune-maintenance/`、`childcare-family-continuity/` 和 `emergency-preparedness-response/`。

## 维护规则

- 新增资料必须说明它影响食物可得性、可负担性、安全性、营养质量、供应稳定还是价格冲击。
- 禁止把食物安全资料改写成个体饮食或食品处置建议。

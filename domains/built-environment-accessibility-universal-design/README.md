# Built Environment Accessibility Universal Design

建成环境无障碍与通用设计域研究：当主体在生命周期中经历身高、感官、移动、认知、疾病、照护和老化差异时，建筑、街区、公共空间和设施可达性如何决定行动与参与能否持续。

## 对象

- 建筑、街道、公共空间、校园、医疗机构、工作场所、交通节点和社区设施的物理可达性。
- 通用设计、ADA/ABA 标准、坡道、门、厕所、导向、标识、触觉/听觉/视觉提示和路径连续性。
- 身体、感官、认知、儿童、老年、照护者、轮椅/助行器使用者和临时伤病者的环境需求。
- 无障碍设计与住房、交通、公共服务、辅助技术、消防疏散、灾害响应和社会参与的接口。
- 环境可达性对行动半径、任务完成、就医、学习、工作、投票、照护和未来选择权的影响。

## Human Infra 价值

主体持续性不只取决于身体能力，还取决于环境是否允许不同能力状态的人继续行动。建成环境无障碍与通用设计域把“空间能否被使用”纳入主体持续性模型，使可达性从边缘合规问题转为长期行动基础设施。

```text
建成环境具备通用设计与可达路径
  -> 不同身体、感官和认知状态都能进入、移动、使用和撤离
  -> 就医、学习、工作、照护和公共参与连续性提高
  -> 有效行动半径和未来选择权扩大
```

## 边界

本域只整理公开无障碍标准、通用设计、可达路径、公共空间可用性和环境公平资料。

不提供：

- 建筑设计、施工、验收、改造、消防疏散、工程图纸、许可或合规法律建议。
- 个体无障碍设施选择、租住、诉讼、投诉、赔偿、合理便利或权益个案建议。
- 规避规范、降低安全标准、虚假合规、危险改造或施工操作步骤。
- 实时场地安全、撤离路线、医疗交通或个人辅助设备建议。

# 家庭医疗假与岗位保护连续性

本域整理家庭医疗事件、严重健康状况、照护责任、军属照护和复工之间的岗位保护机制。它关注的不是“怎么请假”，而是主体在疾病、照护或家庭事件中，是否仍能保留工作角色、保险连接、复工入口和长期收入路径。

## 研究对象

- 家庭与医疗休假、岗位保护和复工权利。
- 严重健康状况、家庭照护、军属照护、资格、通知和证明边界。
- 反报复、工作角色保留、保险连续性和复工摩擦。
- 休假保护与工资替代、残障权益、工伤权益、托育和照护系统之间的接口。

## 价值链路

```text
主体会遭遇疾病、家庭事件和照护责任
  -> 若短期无法工作会直接切断岗位、保险、收入和职业身份
  -> 岗位保护型 leave 把健康/照护事件转化为可返回工作的制度窗口
  -> 通知、资格、证明、反报复和复工路径降低事件后的角色断裂
  -> 主体更可能在恢复或照护后回到任务、收入和社会协作系统
```

## 关键问题

- 哪些事件会把短期健康或家庭冲击放大成长期工作断裂。
- 工作保护、工资替代和保险连续性之间如何分工。
- 证明、通知、等待、复工和反报复机制如何影响主体有效时间。
- 何时应把问题转入带薪休假、残障收入、工伤、托育或照护支持域。

## 上游与下游

- 上游：劳动权利、健康服务、照护责任、家庭系统和雇佣记录。
- 下游：岗位连续性、收入连续性、保险连续性、复工成功率和长期职业轨迹。
- 相关域：`labor-rights-workplace-protection/`、`paid-sick-family-leave-wage-replacement-continuity/`、`pregnancy-parental-work-continuity/`、`workplace-accommodation-return-to-work-continuity/`、`disability-income-work-capacity-benefit-continuity/`、`workers-compensation-occupational-injury-benefit-continuity/`。

## 非目标

- 不提供个人 FMLA、雇佣法、劳动仲裁、请假策略或申请建议。
- 不判断个人资格、雇主义务、证明是否充分或争议胜负。
- 不生成雇主合规方案、法律文书、医疗证明或 HR 表单。
- 不帮助规避制度、伪造记录、制造证据或压制劳动者权利。

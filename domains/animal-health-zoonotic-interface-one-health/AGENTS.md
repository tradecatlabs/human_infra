# animal-health-zoonotic-interface-one-health 目录说明

本目录承载 Human Infra 中“动物健康、人兽共患界面与 One Health”研究域。

## 结构

```text
animal-health-zoonotic-interface-one-health/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义动物健康、人兽共患风险和 One Health 如何影响主体持续性。
- 本目录只做公开资料、变量建模、风险边界和跨域接口，不放动物诊疗、防疫操作、病原体操作或动物交易建议。

## 上下游关系

- 上游：`docs/reference/research-domain-atlas.md`、`immunization-public-health-surveillance/`、`planetary-health-environment/`。
- 下游：公共卫生监测、食物安全、抗微生物韧性、生物安全、供应链、应急响应和风险工程模型。

## 维护规则

- 公开结论必须回到 WHO、WOAH、FAO、CDC 或其他官方 One Health / 动物健康来源。
- 不写动物诊疗、养殖防疫操作、野生动物捕捉、病原体处理、规避监测或非法交易内容。
- 涉及疫情和动物疾病时，只做边界化资料索引，不给个体或现场操作判断。

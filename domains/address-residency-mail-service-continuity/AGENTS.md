# 地址、居住地与邮件服务连续性目录说明

## 目录结构

```text
address-residency-mail-service-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义地址、居住地和邮件服务如何支撑通知、资格、位置和服务路由。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀地址作为制度变量的作用、断点和风险，不写个人迁址或居住证明操作。
- 涉及美国制度时优先使用 USPS、USA.gov、IRS、SSA 和州/地方官方入口。
- 对无固定住址、安全地址、迁移和灾害场景要强调包容性与安全，不输出规避通知或欺骗地址系统的方法。

# afterschool-snack-supper-meal-continuity 目录说明

本目录承载课后点心、晚餐和 afterschool meal 连续性研究，关注放学后营养、监督照护、接送时间和项目运行如何共同影响儿童与家庭持续性。

## 文件结构

```text
afterschool-snack-supper-meal-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 不把本域写成课后班推荐、项目运营咨询、接送方案或菜单设计。
- 与 `out-of-school-time-afterschool-summer-learning-continuity` 保持关系：旧域管课后照护学习，本域只管餐食执行层。
- 禁止处理儿童出勤、家庭排班、项目名单和个案资格。

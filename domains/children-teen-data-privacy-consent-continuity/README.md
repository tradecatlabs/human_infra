# Children Teen Data Privacy Consent Continuity

Children Teen Data Privacy Consent Continuity 关注儿童与青少年数据、家长同意、年龄门槛、教育技术、广告画像、社交平台、敏感数据和长期数字足迹如何影响未成年主体的发展、保护和未来选择权。

## 对象

- COPPA、parental consent、age assurance、school/edtech data、teen privacy、advertising profiles、recommendation systems 和 data retention。
- 儿童数据被收集、共享、商业化、泄露、用于画像、用于广告或伴随主体进入成年后的长期影响。
- 家长、学校、平台、广告商、数据经纪、儿童保护和青少年自主之间的边界。

## Human Infra 价值

```text
主体持续性从未成年阶段就被数据系统塑造
-> 儿童和青少年缺少完全成熟的同意、风险理解和退出能力
-> 过度收集和画像会影响注意力、安全、教育、心理、广告暴露和成年后的机会边界
-> 家长同意、数据最小化、年龄适配、学校数据治理和删除/保留规则降低长期伤害
-> 未成年主体更可能在保护和发展之间保留未来选择权
```

## 边界

- 本域不提供绕过年龄门槛、规避家长控制、定向儿童广告、儿童画像、账号诱导、学校系统规避或未成年人监控方法。
- 不提供法律建议、家长同意模板、学校合规方案、平台审核规避或儿童心理/教育建议。
- 不收集未成年人身份、学校、账号、位置、行为、健康、家庭或敏感资料。
- 只整理公开资料、保护机制、数据最小化、同意边界、长期风险和禁止用途。

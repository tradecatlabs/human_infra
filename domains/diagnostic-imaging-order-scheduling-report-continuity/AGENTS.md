# diagnostic-imaging-order-scheduling-report-continuity

## 职责

本目录承载影像医嘱、预约、报告、影像资料可携带和结果回传连续性资料，关注影像能力如何进入诊断进展。

## 边界

- 只整理公开资料、机制、变量、证据边界和研究问题。
- 不解释影像、不推荐检查、不判断风险、不代写授权、不提供加急策略或诊疗建议。
- 不保存影像、报告、病历、保险、授权号、预约、设备参数或个案健康资料。

## 结构

```text
diagnostic-imaging-order-scheduling-report-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分影像执行、报告路由、资料互操作、授权和 diagnostic safety。
- 个体影像、诊断、治疗和紧急发现问题必须回到医疗专业人员或急救渠道。

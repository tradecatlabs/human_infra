# Childcare Family Continuity

`childcare-family-continuity/` 研究托育、早期儿童发展、家庭照护支持和父母工作连续性能否共同支撑儿童主体与照护者主体的长期持续性。

> 核心问题：托育不是单纯家庭事务，而是同时影响儿童发展、照护者时间、工作收入、心理压力和长期能力积累的基础设施。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 儿童需要安全照护、早期发展、营养、学习和情感支持
  -> 照护者需要时间、收入、恢复、工作连续性和社会支持
  -> 托育和家庭支持系统必须可及、可负担、可信且质量可审查
  -> 两代主体的健康、能力、有效时间和未来选择权才不会互相牺牲
```

## 关注对象

- 托育可及、托育费用、早期儿童发展、照护质量、父母工作连续性和家庭时间压力。
- 育儿假、儿童照护补贴、非正式照护、照护者压力、儿童安全和服务导航。
- 与学习技能、职业设计、财务韧性、社会保护、食品营养和长期照护的接口。

## Human Infra 模型链路

```text
托育与家庭连续性
  -> 儿童获得安全照护、早期发展和稳定支持
  -> 照护者保留工作、收入、恢复时间和行动能力
  -> 家庭压力、时间贫困、职业中断和儿童发展风险下降
  -> 儿童主体与照护者主体的未来选择权同时提高
```

## 非目标

- 不提供个体育儿、教育、医疗、托育机构选择、补贴申请、法律、移民或家庭纠纷建议。
- 不提供儿童安全处置、心理治疗、医疗处理或危机替代服务。
- 不把照护责任全部压回家庭或个体，忽视托育供给、工作制度和公共支持。

# cpap-pap-device-mask-supply-power-continuity 目录说明

本目录承载 CPAP/BiPAP/PAP 设备、面罩、耗材、供电、召回和供应商维护连续性研究域，关注睡眠呼吸支持设备如何转化为长期恢复能力。

## 目录结构

```text
cpap-pap-device-mask-supply-power-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义 PAP 设备运行、耗材供给、供电和治疗中断如何进入主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 NHLBI、CMS、Medicare、FDA、睡眠医学、DME 和家庭医疗设备资料。
- 不写诊断、处方、压力参数、清洁步骤、设备购买、维修、召回个案、保险申诉或依从性惩罚建议。
- 新增指标必须注明口径：设备可用性、面罩/耗材断点、供电风险、供应商响应、召回影响、依从数据或日间功能后果。
- 涉及驾驶、工作、隐私和远程监测时，必须保留安全、法律和数据边界。

# Newborn Hearing Screening Language Development Continuity

`newborn-hearing-screening-language-development-continuity/` 负责整理新生儿听力筛查、诊断确认、早期干预、语言发展和家庭服务连续性。

核心问题：

> 新生儿听力筛查的价值不是一张筛查表，而是“早发现-确认-干预-语言环境”这一整条链是否闭合；任何断点都会影响语言、关系和认知发展窗口。

## 先验位置

```text
主体持续性最大化
  -> 主体语言、关系和学习能力依赖早期听觉输入与互动
  -> 新生儿听力风险如果未被及时发现和确认，会错过早期干预窗口
  -> 家庭服务、语言环境和随访衔接决定听力差异是否转化为发展损失
  -> 因此新生儿听力筛查连续性是生命早期语言能力形成基础设施
```

## 关注对象

- 新生儿听力筛查、转诊、诊断确认、早期干预、家庭支持和语言发展。
- 儿科、听力学、公共卫生筛查项目、保险/服务可及和家庭随访。
- 语言、认知、社交、亲子互动、学校准备和长期沟通参与。
- 与 `otitis-media-child-hearing-development-continuity/` 的关系：本域关注出生后早期筛查和干预链。

## Human Infra 模型链路

```text
新生儿筛查 / 诊断确认 / 早期干预 T
  -> 改变发现时间、确认时间、家庭服务、语言输入和随访变量 X
  -> 改变儿童语言、关系、学习准备和家庭照护状态 S
  -> 改变发展机会丢失、服务延迟、家庭负担和长期沟通受限风险
  -> 增加早期能力形成、教育参与和未来选择权
```

## 非目标

- 不提供婴儿听力诊断、筛查结果解释、设备选择、干预方案、语言路线选择或家庭个体决策。
- 不根据筛查记录判断儿童预后、学校安排、福利资格或医疗责任。
- 不替代儿科、听力学、早期干预、言语语言治疗或家庭支持服务。

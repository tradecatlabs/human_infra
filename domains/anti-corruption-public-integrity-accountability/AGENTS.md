# anti-corruption-public-integrity-accountability 目录说明

本目录维护反腐败、公共廉洁与问责域。它是 Human Infra 的“公共资源不被俘获”层，负责透明、审计、申诉、廉洁和问责如何让公共系统真实支撑主体持续性。

## 目录结构

```text
anti-corruption-public-integrity-accountability/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明腐败、公共廉洁、透明、审计、申诉和问责如何影响主体持续性。
- 本域只做公开制度框架、风险变量和非个案资料整理。
- 不提供行贿、洗钱、隐藏资产、规避审计、逃避监管、伪造记录、调查策略、举报策略、法律建议或政治攻击材料。

## 维护规则

- 新增资料必须区分公开制度框架、统计研究、案例研究、个案指控和法律程序。
- 涉及腐败和问责时，必须避免诽谤、骚扰、人肉、非公开信息利用和政治动员。
- 不把公共廉洁资料写成腐败操作指南或个案法律意见。

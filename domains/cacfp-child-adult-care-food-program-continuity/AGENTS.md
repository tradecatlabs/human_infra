# cacfp-child-adult-care-food-program-continuity 目录说明

本目录承载 CACFP 餐食项目连续性研究，关注照护机构如何把餐食、报销、合规和营养标准转化为稳定照护基础设施。

## 文件结构

```text
cacfp-child-adult-care-food-program-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 保持项目层研究，不写机构合规咨询、claim 操作、菜单处方或审计策略。
- 同时覆盖儿童和成人照护场景，但不得收集真实参与者或机构记录。
- 与托育许可、Head Start 和送餐集体餐域保持上下游引用，不混成单一照护域。

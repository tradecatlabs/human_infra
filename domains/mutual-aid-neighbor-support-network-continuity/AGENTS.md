# mutual-aid-neighbor-support-network-continuity 目录说明

本目录承载互助、邻里支持和近场恢复网络资料。它关注正式服务到达前后，社区关系如何转化为低延迟信息、物资、照看和恢复能力。

## 结构

```text
mutual-aid-neighbor-support-network-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分 mutual aid、neighbor support、informal care、volunteer response、resource sharing 和 disaster support。
- 必须记录隐私、诈骗、安全、家庭暴力暴露、不公平分配、志愿者倦怠和政治/宗教排除风险。
- 不保存地址、联系方式、健康状况、物资需求、家庭结构、定位或脆弱群体名单。

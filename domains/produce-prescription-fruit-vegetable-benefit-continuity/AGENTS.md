# produce-prescription-fruit-vegetable-benefit-continuity 目录说明

本目录承载蔬果处方、农产品福利、营养激励和兑换连续性研究，关注健康食物购买力如何从处方信号变成真实摄入机会。

## 文件结构

```text
produce-prescription-fruit-vegetable-benefit-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- `README.md` 是该域稳定入口，新增资料按处方/转介、福利设计、兑换接口、结果指标和禁止用途归档。
- 保持与 `medically-tailored-groceries-food-benefit-continuity` 的边界：本域聚焦水果蔬菜处方和营养激励，不覆盖全部健康食材福利。
- 禁止输出个人处方、个体疾病饮食、兑换策略、店铺推荐或实时资源导航。
- 新增研究必须标清项目规模、研究设计、结局指标和适用边界。

# 汽车保险、责任碰撞与理赔连续性目录说明

## 目录结构

```text
auto-insurance-liability-collision-claims-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义汽车保险、事故责任、碰撞综合险、理赔和维修恢复如何影响主体移动连续性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀汽车保险 coverage、免赔额、claim、维修估价、total loss 和监管投诉边界，不写个案投保或索赔策略。
- 涉及美国制度时优先使用 NAIC、州保险监管机构、州 DMV、USA.gov 和官方消费者资料。
- 不把本域写成保险推荐、事故责任判断、理赔谈判、维修建议、保险欺诈、规避保险要求或法律建议。

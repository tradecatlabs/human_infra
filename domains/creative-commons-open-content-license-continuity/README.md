# Creative Commons Open Content License Continuity

`creative-commons-open-content-license-continuity` 研究 Creative Commons、公共领域标识、开放内容许可、署名、相同方式共享、非商业、禁止演绎和许可元数据如何影响知识传播、教育材料、创作复用和公共文化连续性。

> 核心问题：开放许可不是“没有版权”；它是让知识和作品可复用的制度接口，边界混乱会让开放知识变成不可确认、不可组合、不可传播的材料堆。

## 关注对象

- Creative Commons license family、public domain tools、attribution、share-alike、noncommercial、no-derivatives 和 license metadata。
- 开放教育资源、开放文化遗产、开放图片音视频、研究图表、课程材料和社区知识库。
- 许可兼容性、署名可追溯、版本记录、机器可读许可和再利用边界。
- 开放内容如何降低学习成本、创作成本、传播摩擦和知识重复生产成本。

## Human Infra 链路

```text
内容被明确开放许可
  -> 复用、翻译、教学、改编和传播边界更清楚
  -> 知识获取和创作启动成本下降
  -> 学习、教育、研究和公共文化材料更可持续积累
  -> 主体认知资源、表达资源和协作资源增强
```

## 非目标

- 不提供许可选择、许可兼容性法律判断、合规审查、侵权判断、商业使用判断或法律意见。
- 不提供版权清洗、署名伪造、许可剥离、公共领域误标、训练数据许可规避或平台迁移规避。
- 不收集未授权作品、合同、争议材料、许可争议证据或私人创作资料。

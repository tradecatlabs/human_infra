# nutrition-metabolic-health 目录说明

`domains/nutrition-metabolic-health/` 保存营养与代谢健康研究域。它关注能量、代谢稳定、食物环境和慢病风险如何影响主体持续性。

## 目录结构

```text
nutrition-metabolic-health/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责营养、饮食质量、代谢健康、体重、糖尿病风险和食物环境的高层研究整理。
- 本域可整理公共健康资料、队列研究、指南边界和模型变量。
- 本域不提供个体饮食、补剂、减重、药物或医学建议。

## 维护规则

- 任何营养主张必须区分人群证据、临床证据、替代指标和真实终点。
- 与心血管、免疫、肾脏、肌骨、睡眠和心理交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

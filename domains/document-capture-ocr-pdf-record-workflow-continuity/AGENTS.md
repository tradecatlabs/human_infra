# document-capture-ocr-pdf-record-workflow-continuity 目录说明

本目录承载文件捕获、OCR、PDF 与记录工作流连续性研究域，关注纸面事实如何进入可保存、可提交、可检索的数字系统。

## 目录结构

```text
document-capture-ocr-pdf-record-workflow-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义扫描、OCR、PDF/PDF-A、表单、元数据、归档和记录提交如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 Library of Congress、NARA、FADGI、PDF/A、可访问 PDF 和数字保存资料。
- 不写伪造文件、篡改记录、规避审核、破解 DRM、绕过签名、伪造 OCR 或隐私窃取方法。
- 涉及敏感文件时只讨论格式、质量、工作流、保管和最小必要访问，不处理真实文件内容。
- 新增指标必须注明口径：图像质量、OCR 准确性、提交成功率、元数据完整性、长期可读性或检索成功率。

# Volcanic Ashfall and Geohazard Continuity

## 核心问题

火山活动和火山灰决定地质灾害能否通过空气质量、航空、供水、农业、建筑、能源和健康系统，形成跨区域主体持续性断点。

Human Infra 关注的问题不是火山学本身，而是：

```text
火山喷发和火山灰如何把地质事件转化为健康、交通、食物、水和基础设施连续性问题？
```

## 先验位置

主体持续性需要可呼吸空气、可用水、运输、食物供应和公共服务。火山灰是复合风险载体：它可以同时影响肺部暴露、航空安全、屋顶负荷、供水处理、农作物和机械设备。

## 关注对象

- 火山监测、喷发预警、火山灰扩散、火山气体和泥石流。
- 空气质量、呼吸暴露、饮水、农业、航空、道路、屋顶负荷和设备磨损。
- 避难、清灰、公共通信、服务恢复和长期社区韧性。

## Human Infra 模型链路

```text
火山喷发 / 火山灰
  -> 空气、水、交通、农业、建筑和设备受扰
  -> 健康风险、供应链中断和公共服务恢复压力上升
  -> 主体外部资源和行动路径收缩
  -> 有效时间、恢复能力和未来选择权下降
```

## 非目标

- 不提供火山喷发预测、避难命令、清灰操作、呼吸防护选择或建筑安全判断。
- 不替代 USGS、气象、公共卫生、应急管理或地方政府信息。
- 不把单次火山事件外推为全球确定性危机。

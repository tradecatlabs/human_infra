# 车贷、租赁、服务与收车连续性目录说明

## 目录结构

```text
auto-loan-lease-servicing-repossession-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义车贷、汽车租赁、服务、逾期和收车如何影响主体移动连续性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀车辆融资、租赁服务、付款连续性、收车风险和证据边界，不写个案购车、贷款、收车或法律策略。
- 涉及美国制度时优先使用 CFPB、FTC、州 DMV、州检察长和官方消费者金融资料。
- 不把本域写成购车推荐、贷款谈判、收车规避、车辆隐藏、虚假申请、产权欺诈、金融规划或个案法律建议。

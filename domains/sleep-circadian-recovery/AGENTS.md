# sleep-circadian-recovery 目录说明

`domains/sleep-circadian-recovery/` 保存睡眠、昼夜节律与恢复研究域。它关注恢复系统如何支撑身体、认知、免疫和情绪稳定。

## 目录结构

```text
sleep-circadian-recovery/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责睡眠、昼夜节律、疲劳、恢复和长期行动能力的关系。
- 本域可整理公共健康资料、睡眠医学边界和可穿戴观测限制。
- 本域不提供个体诊断、药物、设备或治疗建议。

## 维护规则

- 任何睡眠指标主张必须区分主观睡眠、设备估计、生理测量和临床诊断。
- 与认知、心理、免疫、心血管和代谢交叉时必须标注证据层级。
- 新增文件或子目录时，同步更新本文件。

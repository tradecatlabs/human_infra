# Time Allocation Effective Time

Time Allocation Effective Time 负责整理时间使用、时间贫困、行政负担、等待成本、任务摩擦、有效时间和主观时间如何进入主体持续性模型。

核心问题：

> 寿命延长只增加日历时间；Human Infra 真正要优化的是可被主体支配、可用于恢复、学习、协作和创造的有效时间。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体拥有更长日历寿命
  -> 日历寿命必须转化为可支配、可恢复、可学习、可行动的有效时间
  -> 时间贫困、等待、通勤、行政负担和任务摩擦会吞噬有效时间
  -> 因此时间分配与有效时间是主体持续性的任务资源工程域
```

## 关注对象

- 时间使用、时间贫困、等待时间、通勤、照护时间、行政负担、恢复时间和任务切换损耗。
- 有效时间、主观时间、相对时间、任务摩擦、日程结构、自动化节省时间和服务可及性。
- 时间如何进入寿命、健康寿命、学习轮次、资源复利、照护负担和未来选择权模型。
- 与 `future-waiting/` 的边界：本域关注现实生活中的有效时间分配；未来等待域关注物理或生物停滞路径下的主观时间压缩。

## Human Infra 模型链路

```text
时间分配因素 T
  -> 改变可支配时间、等待成本、行政负担、恢复时间和任务摩擦变量 X
  -> 改变主体有效时间状态 S
  -> 改变疲劳、延误、机会损失、长期目标中断和资源耗竭风险 λ(t)
  -> 影响有效寿命、主观时间质量、行动密度和未来选择权
```

## 非目标

- 不提供个体职业、家庭、法律、福利、保险、移民或财务安排建议。
- 不把“节省时间”自动视为收益；必须审查恢复、关系、照护、健康和自主性边界。
- 不输出个体日程优化、生产力处方或剥削性时间压榨方案。

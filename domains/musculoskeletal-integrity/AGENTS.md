# musculoskeletal-integrity 目录说明

`domains/musculoskeletal-integrity/` 保存肌骨完整性研究域。它关注肌肉、骨骼、关节和连接组织如何支撑长期行动。

## 目录结构

```text
musculoskeletal-integrity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责肌少症、骨质疏松、跌倒、关节退化和行动承载结构的研究整理。
- 本域可整理功能终点、风险模型、公共健康资料和证据边界。
- 本域不提供训练、康复、影像诊断、药物、手术或设备建议。

## 维护规则

- 任何肌骨主张必须区分结构指标、功能指标、疼痛体验和真实失能终点。
- 与身体活动、康复、疼痛、营养、感官和辅助技术交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

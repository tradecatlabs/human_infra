# low-back-pain-spine-function-continuity 目录说明

本目录维护腰背痛、脊柱功能、康复可及、工作负荷和复发预防资料。它关注高频腰背痛如何成为主体行动和工作连续性的长期断点。

## 目录结构

```text
low-back-pain-spine-function-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义腰背痛和脊柱功能如何影响主体持续性。
- 本目录只维护系统级变量、来源和边界，不保存个案疼痛、影像、神经症状、工伤或职业资料。
- 资料进入本域时必须保留疼痛持续时间、功能限制、危险信号边界、康复可及、工作参与和禁止用途。

## 上下游关系

- 上游来自 NIAMS、CDC/NIOSH、WHO 和康复/职业健康资料。
- 下游服务 `musculoskeletal-integrity/`、`chronic-pain-functional-restoration-continuity/` 和工作适配模型。
- 不替代骨科、康复医学、疼痛医学、神经外科、职业医学、工伤、保险或个体医疗建议。

## 维护规则

- 不写个人诊断、影像判读、红旗分诊、康复动作、药物、注射、手术、复工或设备建议。
- 不把腰背痛证据写成自我治疗或工伤判断流程。
- 新增资料必须区分急性、亚急性、慢性、特异性/非特异性腰背痛和功能结果。

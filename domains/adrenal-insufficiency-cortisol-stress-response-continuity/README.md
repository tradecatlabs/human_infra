# Adrenal Insufficiency Cortisol Stress Response Continuity

`adrenal-insufficiency-cortisol-stress-response-continuity/` 研究肾上腺不足、Addison's disease、皮质醇缺乏、压力反应失败、危象风险、疲劳和长期替代治疗连续性如何影响主体持续性。

> 核心问题：主体不只是需要平时稳定运转，还需要在感染、手术、创伤、脱水、压力和突发事件中维持最低生理应激能力；皮质醇轴失败会把小扰动放大成生命风险和有效时间损耗。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要在基础状态和压力状态下都能维持血压、能量、盐水平衡和恢复反应
  -> 肾上腺不足会削弱皮质醇和盐皮质激素响应，使疲劳、低血压、低血糖、脱水和危象风险上升
  -> 结果取决于识别、替代治疗连续性、应激剂量教育、急救识别、药品可及和过度/不足治疗边界
  -> 因此肾上腺不足是压力响应和生命维持冗余的关键内分泌域
```

## 关注对象

- Adrenal insufficiency、Addison's disease、cortisol deficiency、aldosterone/mineralocorticoid context、adrenal crisis、fatigue、hypotension、stress response, medication continuity。
- 与 `endocrine-hormonal-regulation/` 的边界：该域关注内分泌总机制；本域聚焦皮质醇轴、盐水平衡、应激响应和危象风险。
- 与 `emergency-preparedness-response/` 的关系：本域只描述慢性内分泌脆弱性如何进入急性风险，不提供急救方案。

## Human Infra 模型链路

```text
肾上腺不足识别与治疗连续性 T
  -> 改变皮质醇、盐水平衡、血压、血糖、疲劳、感染/创伤应激反应变量 X
  -> 改变基础能量、恢复能力、急性压力承载和药品依赖状态 S
  -> 改变危象、住院、失能、工作中断和死亡风险 λ(t)
  -> 影响主体行动可靠性、恢复能力和进入未来的安全冗余
```

## 非目标

- 不提供个人诊断、实验室判读、皮质醇/ACTH 解释、用药、剂量、应激剂量、停药、急救、手术、感染处理、饮食、补剂或个体医疗建议。
- 不生成 emergency plan、医学证明、药品替代、处方调整、旅行准备、保险申诉、工作/学校证明或家庭处置脚本。
- 不收集个案化验、处方、急救卡、住院记录、血压/血糖记录、保险或身份资料。

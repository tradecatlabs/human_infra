# adrenal-insufficiency-cortisol-stress-response-continuity 目录说明

本目录维护肾上腺不足、皮质醇轴、应激响应、疲劳、血压/盐水平衡和危象风险资料。它关注压力响应失败如何影响主体行动可靠性和生命维持冗余。

## 目录结构

```text
adrenal-insufficiency-cortisol-stress-response-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义肾上腺不足如何影响主体持续性。
- 本目录只维护系统级变量、来源和边界，不保存个案化验、处方、急救卡、住院、血压/血糖、保险或身份资料。
- 资料进入本域时必须保留皮质醇轴、压力状态、危象风险、药品连续性、急救接口和禁止用途。

## 上下游关系

- 上游来自 NIDDK、MedlinePlus、内分泌、急救和药品可及资料。
- 下游服务 `endocrine-hormonal-regulation/`、`emergency-preparedness-response/`、`fatigue-risk-alertness-continuity/` 和治疗获得模型。
- 不替代内分泌科、急救、药师、保险、学校/工作或个体医疗建议。

## 维护规则

- 不写个人皮质醇/ACTH 判读、诊断、用药剂量、应激剂量、急救、停药、旅行或感染处理建议。
- 不把肾上腺不足资料写成自救流程、用药调整流程或急救替代方案。
- 新增资料必须区分基础状态、压力状态、危象风险、药品依赖、急救接口和功能结果。

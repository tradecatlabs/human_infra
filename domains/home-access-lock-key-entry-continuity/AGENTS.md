# home-access-lock-key-entry-continuity 目录说明

本目录承载住所进入、钥匙、锁具、门禁和紧急出入连续性研究域，关注住房如何从名义住所转化为主体可进入、可恢复、可照护的本地空间。

## 目录结构

```text
home-access-lock-key-entry-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义住所进入、锁具、钥匙、门禁权限和紧急出入如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 NFPA、CPSC、NIST、FTC、建筑消防和公开消费者安全资料。
- 不写开锁、门禁绕过、智能锁破解、非法进入、住址定位、跟踪或监控方法。
- 涉及具体设备时只讨论生命周期、权限、故障模式、召回、隐私和恢复边界。
- 新增指标必须注明口径：授权进入率、锁具失效率、电池/联网依赖、应急进入路径、逃生可达性或权限撤销延迟。

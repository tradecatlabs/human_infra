# ubiquitin-proteasome-protein-turnover-continuity 目录说明

`domains/ubiquitin-proteasome-protein-turnover-continuity/` 保存泛素-蛋白酶体系统和蛋白周转研究域。

## 目录结构

```text
ubiquitin-proteasome-protein-turnover-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责 UPS、泛素化、去泛素化、蛋白酶体活性、底物选择和蛋白周转资料。
- 本域可整理蛋白毒性压力、神经退行、肌肉衰退、免疫和衰老交叉资料。
- 本域不提供药物、补剂、蛋白酶体调节、检测解释、实验操作或临床建议。

## 维护规则

- 所有主张必须区分泛素化标记、蛋白酶体活性、蛋白聚集、细胞功能和临床终点。
- 与 `proteostasis-autophagy/` 和 `ribosome-translation-fidelity-proteome-integrity-continuity/` 交叉时区分错误生成与错误清除。
- 新增文件或子目录时同步更新本文件。

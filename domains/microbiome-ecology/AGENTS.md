# microbiome-ecology 目录说明

`domains/microbiome-ecology/` 保存人体微生物生态研究域。它关注宿主-微生物互作如何影响代谢、免疫、炎症和风险函数。

## 目录结构

```text
microbiome-ecology/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责微生物生态、菌群失调、代谢物、宿主互作和测序/干预证据边界。
- 本域可整理公共资料、机制文献、相关性与因果性问题。
- 本域不提供益生菌、抗生素、饮食、移植、检测解释或治疗建议。

## 维护规则

- 任何微生物组主张必须区分相关性、机制证据、因果干预和临床终点。
- 与营养、免疫、口腔、抗微生物韧性和心理健康交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

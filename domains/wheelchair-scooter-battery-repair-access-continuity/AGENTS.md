# wheelchair-scooter-battery-repair-access-continuity 目录说明

本目录承载轮椅、代步车、电池、充电、维修、备件和供应商响应连续性研究域，关注移动设备如何支撑主体行动能力。

## 目录结构

```text
wheelchair-scooter-battery-repair-access-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义轮椅和代步设备运行断点如何影响移动、医疗、工作、教育和撤离连续性。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 Medicare、MedlinePlus、DME、康复、交通无障碍和残障权利材料。
- 不写设备处方、维修教程、改装方案、品牌比较、购买建议、保险申诉或个体覆盖判断。
- 新增指标必须注明口径：供电可用性、维修等待、备件可得、供应商响应、运输兼容性或行动失败后果。
- 涉及灾害撤离、公共交通和住房无障碍时，必须保留机构责任和非个体化建议边界。

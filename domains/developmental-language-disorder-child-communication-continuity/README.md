# Developmental Language Disorder Child Communication Continuity

Developmental language disorder child communication continuity studies childhood language learning difficulties as constraints on school participation, literacy, peer relationships, family navigation, and long-term capability formation.

核心问题：

> 儿童语言发展不是单项技能，而是学习、同伴关系、自我表达和未来能力积累的入口；发展性语言障碍会把早期沟通断点扩大为长期教育和社会参与成本。

## 先验位置

```text
主体持续性最大化
  -> 主体早期必须获得可持续语言输入、理解和表达能力
  -> 发展性语言困难会影响词汇、语法、叙事、理解和课堂参与
  -> 读写、学习、自信、同伴关系和家庭导航成本上升
  -> 因而发展性语言障碍是能力形成窗口中的语言基础设施失效模式
```

## 关注对象

- 发展性语言障碍、语言迟缓、词汇语法、叙事、理解表达、读写基础和学校参与。
- 家庭导航、筛查随访、早期干预、教育服务和长期学习。
- 与 `maternal-newborn-child-development/` 的关系：儿童发展域关注全局；本域关注语言能力形成。
- 与 `newborn-hearing-screening-language-development-continuity/` 的关系：新生儿听筛域关注听觉输入；本域关注语言系统发展。

## Human Infra 模型链路

```text
筛查 / 早期支持 / 教育适配 / 家庭导航 T
  -> 改变语言理解、表达、叙事、词汇和课堂参与变量 X
  -> 改变读写、学习、同伴关系和家庭负担状态 S
  -> 改变教育落后、社会退缩和长期能力缺口风险 λ(t)
  -> 影响学习轮次、能力积累和未来选择权
```

## 非目标

- 不提供儿童语言障碍诊断、筛查解释、治疗计划、家庭训练、学校资格判断或预后承诺。
- 不根据儿童表现判断自闭症、听损、智力障碍、神经疾病或家庭责任。
- 不替代儿科、听力学、言语语言治疗、心理、教育或特教服务。

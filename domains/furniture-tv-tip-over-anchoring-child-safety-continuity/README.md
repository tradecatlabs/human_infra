# Furniture TV Tip Over Anchoring Child Safety Continuity

`furniture-tv-tip-over-anchoring-child-safety-continuity/` 研究家具、电视、储物柜、抽屉、儿童攀爬、锚固、产品召回和居家布局如何影响儿童与家庭伤害风险。

> 核心问题：家具不是静态背景。对儿童和行动受限者，未固定家具和电视会把日常探索、取物和照护瞬间转化为挤压伤害。

## 先验位置

```text
主体需要在家中移动、探索和取物
  -> 家具和电视必须保持稳定、可预期、可恢复
  -> 儿童攀爬、抽屉打开和重心变化会触发倾倒
  -> 因而家具倾倒是家庭产品安全和儿童行动连续性的交叉域
```

## 关注对象

- 电视、衣柜、书架、抽屉柜、储物柜、锚固、召回、二手家具、租赁住房、儿童攀爬和照护监督。

## Human Infra 模型链路

```text
家具电视稳定性 T
  -> 改变锚固、重心、产品设计、召回、儿童接触和布局变量 X
  -> 改变儿童探索、家庭取物和照护状态 S
  -> 改变挤压伤、头部伤害、死亡、照护压力和住房调整成本 λ(t)
  -> 改变儿童发展安全、家庭恢复和未来选择权
```

## 非目标

- 不提供锚固安装、产品推荐、品牌比较、召回索赔、房东/租客争议、儿童监控或个体安全建议。
- 不收集家庭照片、儿童信息、住址、产品序列号、房屋租约或事故证据。

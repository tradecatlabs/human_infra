# fuel-ev-charging-refueling-access-continuity 目录说明

本目录承载燃料、EV 充电和移动能源入口连续性研究域，关注移动能力如何受燃料、充电、支付、站点可达性和停电影响。

## 目录结构

```text
fuel-ev-charging-refueling-access-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义燃料和充电入口如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 DOE、AFDC、Energy.gov、Ready.gov、能源监管和公开交通能源资料。
- 不写危险燃料储存、充电桩破解、盗电、绕过计费、非法接线或具体家庭电气操作。
- 新增指标必须注明口径：站点可用性、充电成功率、支付失败率、排队时间、停电恢复或灾害燃料可得性。

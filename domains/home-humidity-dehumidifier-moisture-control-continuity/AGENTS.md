# home-humidity-dehumidifier-moisture-control-continuity 目录说明

本目录承载家庭湿度、除湿/加湿与水分控制连续性研究域，关注湿度如何影响室内恢复空间。

## 目录结构

```text
home-humidity-dehumidifier-moisture-control-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义湿度、除湿/加湿、水分控制、霉菌/尘螨条件、睡眠和家庭任务连续性的关系。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 EPA、CDC、WHO、住房、建筑水分控制和公共卫生资料。
- 不写霉菌清理、防水施工、设备购买、维修、湿度处方、检测解释、租房纠纷、保险或医学建议。
- 新增指标必须注明口径：相对湿度、季节、房间类型、设备状态、冷凝/潮湿状态、生物负荷或任务结果。
- 涉及水损、霉菌、哮喘、过敏或灾后恢复时，必须保留专业检测、医疗和住房责任边界。

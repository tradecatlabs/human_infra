# Harmful Algal Bloom Cyanotoxin Water Recreation Continuity

`harmful-algal-bloom-cyanotoxin-water-recreation-continuity` 研究有害藻华、蓝藻毒素、湖泊/海岸水体、宠物和水上活动如何影响主体持续性。

> 核心问题：有害藻华会把水体从休闲、降温和恢复空间变成毒素暴露空间；它影响的不只是游泳，还包括饮水信任、宠物安全、旅游、儿童活动和地方经济。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要可靠水环境用于饮水、降温、运动、休闲和心理恢复
  -> 有害藻华和蓝藻毒素会把水环境转化为不确定暴露源
  -> 监测、公告、关闭、替代空间和医疗/兽医连接构成行动前提
  -> 若该前提失败，水上活动、宠物照护、旅行、饮水信任和恢复环境都会被压缩
```

## 关注对象

- 有害藻华、蓝藻、cyanotoxins、淡水/海水水体、宠物暴露、儿童活动和水体关闭。
- 水质监测、公众公告、饮水系统接口、休闲水域管理、兽医/医疗连接和风险沟通。
- 与 `water-resources-hydrology-flood-drought-management/` 的边界：上位域关注水资源；本域关注藻华毒素暴露对主体行动和恢复空间的影响。

## Human Infra 链路

```text
藻华和蓝藻毒素风险被监测和沟通
  -> 误入污染水体、饮水信任崩塌和宠物暴露下降
  -> 水上活动、旅游、降温和恢复空间更可控
  -> 健康风险、时间损失和行动不确定性下降
```

## 非目标

- 不提供个体暴露诊断、饮水判定、宠物急救、水体采样、藻华识别、处理药剂或游泳许可。
- 不替代 EPA、CDC、地方水务、公共卫生、环境监测、医疗或兽医服务。
- 不把照片、卫星图、水色、气味、检测值或 AI 输出写成现场安全结论。

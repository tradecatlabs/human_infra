# Hot Water Scald Burn Prevention Continuity

`hot-water-scald-burn-prevention-continuity/` 研究热水、洗浴、厨房、饮品、热水器、温度控制、儿童/老人皮肤脆弱性和照护场景如何影响烫伤风险。

> 核心问题：热水是卫生、烹饪和照护的基础资源，也是家庭中最常见的烧伤来源之一。安全连续性在于可用热水与可控温度同时成立。

## 先验位置

```text
主体需要热水完成清洁、烹饪和照护
  -> 热水必须可用、可控、可理解且不会造成不可逆伤害
  -> 儿童、老人、残障者和照护场景对温度更敏感
  -> 因而热水烫伤预防是能源、卫生和皮肤安全交叉域
```

## 关注对象

- 热水器、洗浴热水、厨房热液、饮品、温度控制、照护辅助、儿童/老人皮肤脆弱性、烧伤中心资料和家庭能源。

## Human Infra 模型链路

```text
热水与温度控制 T
  -> 改变温度、暴露时间、照护辅助、设备维护和理解成本 X
  -> 改变清洁、洗浴、烹饪、饮水和照护状态 S
  -> 改变烫伤、疼痛、感染、瘢痕、住院和照护成本 λ(t)
  -> 改变恢复能力、家庭安全和未来选择权
```

## 非目标

- 不提供热水器设置、维修、温度建议、急救、烧伤处理、产品购买、施工、保险或个体安全建议。
- 不收集家庭设备型号、地址、儿童/老人资料、烧伤照片、健康记录或保险资料。

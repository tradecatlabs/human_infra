# Elder Justice Adult Protective Services

老年正义与成人保护服务域研究：当老年人、残障成年人或依赖照护的成年人面临虐待、忽视、剥削、财务控制或照护失效时，发现、报告、保护、服务转介和权利救济如何维护主体持续性。

## 对象

- 老年虐待、成人虐待、忽视、自我忽视、财务剥削、照护者虐待、机构虐待和权利剥夺。
- Adult Protective Services、Elder Justice、长期照护监察、报告系统、保护服务和多机构协作。
- 认知衰退、残障、依赖照护、社会隔离、财务控制、诈骗、住房和医疗服务断点。
- 虐待与剥削对死亡风险、伤害、财富损失、心理创伤、尊严、自主性和未来选择权的影响。

## Human Infra 价值

长寿如果没有保护系统，可能只是延长暴露在虐待、忽视和剥削中的时间。老年正义与成人保护服务域把“脆弱成年人是否能被发现并保护”作为独立基础设施，连接照护、法律、金融、住房和健康系统。

```text
虐待、忽视和剥削可被发现并转介
  -> 财务、身体、心理和照护风险被保护系统承接
  -> 伤害、失权、资源丧失和孤立下降
  -> 主体尊严、自主性和可承受持续性提高
```

## 边界

本域只整理公开老年正义、成人保护服务、虐待预防、报告系统、长期照护监察和权利保护资料。

不提供：

- 个案法律、报案、取证、监护、财务安排、机构投诉、家庭冲突或照护计划建议。
- 定位、监控、跟踪、控制、约束、剥夺通信、财务控制或绕过成年人同意的策略。
- 对老年人、残障成年人、认知衰退者或照护者的污名化、风险标签或剥削性筛选。
- 把保护服务写成家庭控制工具、财产争夺工具或自动剥夺主体权利的理由。

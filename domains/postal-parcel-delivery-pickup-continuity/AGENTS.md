# postal-parcel-delivery-pickup-continuity 目录说明

本目录承载邮件、包裹、投递、自提、签收和最后一公里接收连续性研究域，关注远程服务如何转化为主体可实际接收的物理资源。

## 目录结构

```text
postal-parcel-delivery-pickup-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义邮件、包裹、投递、自提和接收证明如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 USPS、UPU、FTC、承运商公开消费者资料和地址服务资料。
- 不写邮件拦截、包裹盗取、冒领、地址欺诈、投递规避、追踪滥用或物流攻击方法。
- 涉及药品、法律通知、证件和高价值物品时只讨论通用接收连续性与边界。
- 新增指标必须注明口径：投递成功率、退件率、地址错配、签收证明、自提可达性、代收权限或包裹丢失报告。

# Evacuation Route Transportation Shelter Plan Continuity

`evacuation-route-transportation-shelter-plan-continuity/` 研究撤离路线、交通可得、避难点、接驳、无障碍和返回条件如何影响主体在危险环境中的退出能力。

> 核心问题：撤离不是“知道危险”之后自动发生，而是交通、燃料、路况、照护、证件、宠物、无障碍和避难承接共同决定的退出链。

## 先验位置

```text
主体要避开局部灾害
  -> 必须在危险超过居留阈值前退出暴露区域
  -> 撤离依赖路线、交通、燃料、时间、避难承接和特殊需求支持
  -> 因而撤离计划是主体从危险空间转移到可恢复空间的执行条件
```

## 关注对象

- 撤离路线、车辆/公共交通、燃料、道路封闭、避难点、接驳、无障碍、临时住宿和返回条件。
- 无车家庭、残障者、老人、儿童家庭、宠物家庭、低收入家庭、沿海/火灾/洪水风险地区居民。

## Human Infra 模型链路

```text
撤离路线与避难承接系统 T
  -> 改变交通可及、路线冗余、燃料、无障碍、照护负荷和避难容量变量 X
  -> 改变主体从危险环境退出的状态 S
  -> 改变滞留、暴露、交通事故、照护断裂、失屋和恢复延迟风险 λ(t)
  -> 改变生命安全、服务接入、家庭团聚和灾后恢复选择权
```

## 非目标

- 不提供个人撤离路线、实时路况、避难点选择、驾驶、燃料、交通、住宿、保险或灾害现场行动建议。
- 不收集住址、路线、车辆、燃料、残障状态、宠物、家庭成员、避难点或定位信息。

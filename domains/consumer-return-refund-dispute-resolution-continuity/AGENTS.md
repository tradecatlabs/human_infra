# 消费者退货、退款与争议解决连续性目录说明

## 目录结构

```text
consumer-return-refund-dispute-resolution-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义退货、退款、换货、store credit、取消和争议解决如何影响交易失败后的资源恢复。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀退款争议机制、政策可理解性、投诉渠道和主体资源变量，不写个案退款或投诉策略。
- 涉及美国制度时优先使用 FTC、CFPB、USA.gov、州消费者保护机构和官方公开资料。
- 不把本域写成退款脚本、诉讼策略、chargeback 指南、商家谈判、证据伪造或消费者套利指南。

# Online Safety Digital Harm Prevention

在线安全与数字伤害预防域研究：当骚扰、网络霸凌、数字胁迫、儿童在线剥削、非自愿影像、跟踪、勒索和平台滥用影响主体时，数字空间如何保持可参与、可求助、可恢复和可治理。

## 对象

- 网络霸凌、在线骚扰、数字虐待、跟踪、勒索、非自愿影像、儿童在线剥削、诈骗诱导和平台举报。
- 平台治理、内容举报、信任与安全、隐私保护、账号安全、数字证据保存、儿童安全和受害者支持。
- 在线伤害对心理健康、身体安全、社会关系、工作学习、身份安全、财务安全和未来选择权的影响。
- 青少年、儿童、残障者、移民、公众人物、亲密伴侣暴力受害者和弱势群体的数字可达与安全。

## Human Infra 价值

数字空间已经成为学习、工作、社交、医疗、福利和公共服务入口。在线伤害会把这个入口变成持续威胁源。在线安全域把数字参与的安全性、求助路径和恢复能力作为主体持续性的基础设施。

```text
数字伤害可预防、举报和恢复
  -> 在线参与、学习、工作、社交和服务入口更安全
  -> 骚扰、胁迫、剥削、身份风险和心理伤害下降
  -> 主体数字行动能力和未来选择权提高
```

## 边界

本域只整理公开在线安全、网络霸凌、儿童在线保护、平台举报、数字伤害预防和受害者支持资料。

不提供：

- 跟踪、骚扰、 doxxing、勒索、非自愿影像、诱导、儿童剥削、账号入侵、规避平台检测或规避执法方法。
- 个案法律、调查、证据固定、平台申诉、身份识别、报案、心理危机或安全计划建议。
- 指向具体儿童、受害者、弱势群体、家庭、学校、账号或地点的定位、识别、扩散或二次伤害信息。
- 把在线安全写成审查扩张、歧视筛选、商业监控、恐慌营销或受害者归因。

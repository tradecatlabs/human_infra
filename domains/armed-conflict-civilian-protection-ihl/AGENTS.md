# armed-conflict-civilian-protection-ihl 目录说明

本目录维护武装冲突、平民保护与国际人道法域。它是 Human Infra 的“极端集体暴力下主体仍应被保护”层，负责战争和武装冲突如何破坏生命、身份、医疗、资源和家庭连续性。

## 目录结构

```text
armed-conflict-civilian-protection-ihl/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明 IHL、平民保护、人道准入、医疗中立和冲突服务断点如何进入主体持续性模型。
- 本域只做公开法理、保护框架、人道资料和非操作性风险建模。
- 不提供作战、武器、目标选择、战术规避、情报、破坏、准军事组织、制裁规避或冲突宣传材料。

## 维护规则

- 新增资料必须区分法律原则、保护实践、人道服务、统计资料和具体冲突叙事。
- 涉及冲突场景时，必须保留中立、人道、平民保护和非操作性边界。
- 不把冲突资料写成行动指南，不把国际法资料写成具体法律意见。

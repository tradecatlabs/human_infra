# vestibular-balance-spatial-orientation AGENTS

## 职责

- 维护前庭、平衡、空间定向和跌倒风险的研究边界、证据入口、变量表和禁止用途。
- 将眩晕、平衡障碍、跌倒、移动恐惧、生活空间和环境适配资料整理为 Source Card。
- 与 `sensory-continuity/`、`musculoskeletal-integrity/`、`physical-activity-mobility/`、`built-environment-accessibility-universal-design/` 保持边界清晰。

## 非目标

- 不提供眩晕诊断、测试解释、康复训练、药物、急救、跌倒个体处置或设备建议。
- 不替代耳鼻喉、神经、康复、物理治疗、老年医学或安全评估专业服务。

## 上游

- `sensory-continuity/`：跨感官反馈和空间输入。
- `musculoskeletal-integrity/`：肌骨承载和跌倒后果。
- `physical-activity-mobility/`：活动能力和步态参与。

## 下游

- `transportation-access-mobility/`：出行和服务可达。
- `housing-built-environment-stability/`：居住环境跌倒风险。
- `time-allocation-effective-time/`：移动恐惧和行动半径收缩造成的有效时间损耗。

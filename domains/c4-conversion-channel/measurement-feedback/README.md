Measurement And Feedback 负责整理主体状态测量、预测模型、风险评分、生物标志物、可穿戴、数据闭环和反馈控制。

核心问题：

> 不能被观测和反馈的主体状态，无法被稳定维护；有效永生需要持续知道主体在哪里、风险在哪里、下一步该改什么。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体是动态系统
  -> 动态系统需要观测、反馈、纠偏和验证
  -> 没有测量，干预无法区分有效、无效和有害
  -> 因此测量与反馈是所有研究域进入工程化的接口
```

## 关注对象

- 生物标志物、生物年龄、风险评分、健康寿命指标和功能指标。
- 可穿戴、连续监测、个人健康记录、多组学、影像和临床数据。
- 生存分析、因果推断、数字孪生、动态系统和决策模型。
- 与 `docs/reference/life-path-prediction-model-contract.md` 的关系：本域是预测模型的数据和反馈接口。

## Human Infra 模型链路

```text
测量 / 反馈系统 T
  -> 改变观测能力、误差、反馈速度和决策质量 X
  -> 改变主体管理状态 S
  -> 更早发现风险、更快纠偏、更少无效干预
  -> 提高健康寿命、有效时间和未来选择权
```

## 非目标

- 不输出个体死亡日期、确定寿命预测或医学诊断。
- 不把单一 biomarker 当作完整主体状态。
- 不把相关性、预测分数或短期指标直接写成因果干预效果。

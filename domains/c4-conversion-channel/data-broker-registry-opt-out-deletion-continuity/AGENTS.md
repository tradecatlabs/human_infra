# Data Broker Registry Opt-Out Deletion Continuity 架构说明

本目录承载数据经纪、登记、opt-out 和 deletion continuity 研究资料。它只组织公开资料、变量和边界，不提供个人数据删除、法律判断或规避数据经纪系统的方法。

## 目录结构

```text
data-broker-registry-opt-out-deletion-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须回到 FTC、CFPB、CPPA、NIST 或其他官方/权威来源。
- 区分 data broker registry、consumer reporting、privacy rights、adtech profiling 和 identity theft，不把所有隐私问题混成一个概念桶。
- 不收集真实个人资料、broker 查询结果、地址、电话、身份文件、账号或授权代理材料。

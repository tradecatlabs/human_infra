Data Broker Registry Opt-Out Deletion Continuity 关注数据经纪人、数据经纪登记、opt-out、delete request、授权代理、身份验证和删除状态如何影响主体在长期社会系统中的可见性、可控性和被错误画像的风险。

## 对象

- Data brokers、data broker registry、delete request、opt-out、authorized agent、verification、appeal 和 enforcement。
- 个人资料被聚合、出售、共享、再识别、外溢到信用、租房、就业、保险、广告和公共安全系统的路径。
- 不同州、平台和行业之间的数据经纪透明度、退出机制、删除链路和不可达性。

## Human Infra 价值

```text
主体持续性需要对外部画像保持最低控制权
-> 数据经纪把分散行为、地址、交易、人口属性和兴趣信号聚合成可交易资料
-> 错误、过期、敏感或不可见的画像会影响金融、住房、就业、保险、安全和骚扰风险
-> 登记、opt-out、delete request 和授权代理机制让主体有机会发现、限制和清除外部画像
-> 主体更可能保留隐私、安全、资源入口和未来选择权
```

## 边界

- 本域不提供法律建议、州法适用判断、数据删除代办、opt-out 操作教程、身份验证材料准备或个案申诉策略。
- 不提供规避身份核验、批量抓取数据经纪资料、攻击数据经纪系统、绕过 paywall、骚扰 broker 或虚假授权的方法。
- 不判断某个 data broker 是否违法、某条记录是否应删除、某次 opt-out 是否有效或某州权利是否适用。
- 只整理公开资料、登记机制、控制路径、失败模式、权利边界和禁止用途。

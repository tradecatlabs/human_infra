# sensory-continuity 目录说明

`domains/c4-conversion-channel/sensory-continuity/` 保存感官连续性研究域。它关注视觉、听觉、平衡、触觉和感官反馈如何支撑主体行动和认知。

## 目录结构

```text
sensory-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责感官系统、感官衰退、感官辅助和感官反馈对主体持续性的影响。
- 本域与 `assistive-technology-access/`、`neuro-continuity/` 和 `rehabilitation-functioning/` 交叉。
- 本域不提供眼科、听力、神经、设备或治疗建议。

## 维护规则

- 任何感官改善主张必须区分生理功能、辅助补偿、行为功能和社会参与。
- 涉及设备时必须审查可及性、适配、维护和训练。
- 新增文件或子目录时，同步更新本文件。

## 研究推进骨架

### 最小问题集

- 界定转化瓶颈：研究发现、数据、临床、制度、身份、资金、可及性或操作接口。
- 说明接口机制：潜在可能性如何经过标准、流程、记录、组织或平台变成可用路径。
- 记录必要证据：登记系统、指南、监管材料、数据库、标准文档或真实世界部署资料。
- 识别摩擦指标：时间成本、资金成本、信息成本、合规成本、可及性和协作成本。
- 列出失败点：证据断裂、监管阻塞、制度排斥、隐私风险、责任不清或资源不可达。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

### 最小产出

- Source Signals：记录候选资料、来源、日期、主张和待核验点。
- Source Cards：提取 claim、变量、机制、证据类型、边界、反证条件和可迁移位置。
- Claim-Evidence Matrix：把每个稳定主张绑定到来源、证据等级、适用范围和不确定性。
- 变量表：列出输入变量、中间变量、状态变量、风险变量、输出指标和可观测代理指标。
- 上下游关系：说明本域依赖哪些更根本域，并向哪些转化、应用或基础设施域输出。

Cultural Heritage Identity Continuity 研究语言、文化实践、传统知识、族群身份、移民文化连续性和非物质文化遗产如何支撑主体的身份、归属、记忆和跨代传递。

> 核心问题：主体不是抽象个体；文化、语言和传统实践会塑造其自我理解、关系网络、照护偏好和未来想象。

## 研究对象

- Cultural heritage、intangible cultural heritage、language, identity, tradition, community memory 和 intergenerational transmission。
- 迁移、流离失所、殖民创伤、少数族群、原住民、语言弱势和代际断裂。
- 文化身份与照护沟通、归属、歧视、教育、艺术、仪式和精神实践之间的关系。

## 先验链路

```text
主体通过语言、习俗、记忆和共同体理解自己是谁
  -> 文化身份让偏好、价值、照护、关系和未来目标具备解释框架
  -> 文化断裂会削弱归属、信任、照护沟通、代际传递和身份稳定
  -> 主体在迁移、教育、医疗、工作和公共服务中更容易被误读或排除
  -> 有效行动、恢复支持和未来选择权下降
  -> 因而文化遗产与身份连续性是主体持续性的解释与归属基础
```

## 关键变量

- 语言可及、文化实践、非物质文化遗产、代际传递、文化安全、歧视、同化压力和身份稳定。
- 移民、难民、原住民、少数族群、宗教共同体、家庭结构、教育系统和医疗沟通。
- 文化本质化、刻板标签、文化挪用、身份审查、政治操控和敏感身份数据风险。

## 证据入口

- UNESCO Intangible Cultural Heritage Convention。
- UNESCO living heritage / cultural diversity resources。
- Language access、cultural mediation 和 minority health 相关资料。

## 非目标

- 本域不做族群判定、文化正统裁判、宗教政治判断、身份认证、文化活动推荐或移民/法律建议。
- 不提供文化标签化、画像、筛选、同化、宣传、动员、审查或排除方案。
- 不收集族群、语言、宗教、移民、政治、家庭谱系、文化实践或社区成员资料。

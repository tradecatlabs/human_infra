# background-check-record-screening-continuity

本目录承载背景调查、记录筛查和准入连续性资料。它研究记录如何被聚合、匹配、解释和更正，以及错误筛查如何影响工作、住房、照护和服务入口。

## 目录结构

```text
background-check-record-screening-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 说明本域对象、Human Infra 价值、边界、上下游和初始资料线索。
- 后续资料应记录筛查场景、记录类型、准确性风险、通知/不利行动、争议和更正边界。
- 本域不保存个人记录，不解释个案背景报告，也不提供规避、清除、申诉或岗位适配建议。

## 维护规则

- 引用资料优先来自 FTC、EEOC、法院/司法公开资料、许可机构和消费者保护资料。
- 必须区分风险治理、资格要求、反歧视边界和错误更正机制。
- 不提供身份伪装、记录规避或欺骗筛查的策略。

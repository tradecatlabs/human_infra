# Children Teen Data Privacy Consent Continuity 架构说明

本目录承载儿童与青少年数据隐私、家长同意和长期数字足迹研究。它不承载儿童画像、规避年龄门槛或监控未成年人的方法。

## 目录结构

```text
children-teen-data-privacy-consent-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须保护未成年人隐私，不写真实学校、账号、家庭、位置或健康资料。
- 区分儿童保护、青少年自主、学校数据、平台广告和商业监控。
- 禁止加入针对未成年人营销、操控、规避、监控或画像内容。

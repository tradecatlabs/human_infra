## 研究推进骨架

### 最小问题集

- 界定转化瓶颈：研究发现、数据、临床、制度、身份、资金、可及性或操作接口。
- 说明接口机制：潜在可能性如何经过标准、流程、记录、组织或平台变成可用路径。
- 记录必要证据：登记系统、指南、监管材料、数据库、标准文档或真实世界部署资料。
- 识别摩擦指标：时间成本、资金成本、信息成本、合规成本、可及性和协作成本。
- 列出失败点：证据断裂、监管阻塞、制度排斥、隐私风险、责任不清或资源不可达。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

### 最小产出

- Source Signals：记录候选资料、来源、日期、主张和待核验点。
- Source Cards：提取 claim、变量、机制、证据类型、边界、反证条件和可迁移位置。
- Claim-Evidence Matrix：把每个稳定主张绑定到来源、证据等级、适用范围和不确定性。
- 变量表：列出输入变量、中间变量、状态变量、风险变量、输出指标和可观测代理指标。
- 上下游关系：说明本域依赖哪些更根本域，并向哪些转化、应用或基础设施域输出。

LGBTQ 健康、身份承认与肯定性服务连续性域研究：当主体的身体、亲密关系、家庭、身份文件、医疗访问、心理健康、学校、工作和公共服务受歧视或不承认影响时，如何维护主体持续性和未来选择权。

## 对象

- LGBTQ health, identity recognition, nondiscrimination, affirming services, family and community support。
- 医疗访问、心理健康、预防服务、性健康、慢病管理、青少年支持、老年服务和暴力风险。
- 身份文件、姓名代词、家庭关系、学校和工作场所、住房、公共服务与民权保护。
- 歧视、污名、少数压力、服务回避、隐私风险、数据收集边界和安全沟通。
- 对主体行动能力、恢复能力、关系支持、健康寿命、有效时间和未来选择权的影响。

## Human Infra 价值

主体持续性需要制度承认主体是谁、与谁有关、如何安全进入服务。LGBTQ 健康与身份承认域把身份不承认、歧视、服务回避和隐私风险作为基础设施失效，而不是把问题降格为个人心理或生活方式。

```text
身份承认和肯定性服务被结构化治理
  -> 医疗、学校、工作、家庭、住房和公共服务入口更安全
  -> 歧视和不承认不再持续消耗主体恢复能力和未来选择权
  -> 主体持续性、关系支持和长期行动能力提高
```

## 边界

本域只整理公开 LGBTQ health、民权、服务可达、身份承认、隐私、社区支持和风险治理资料。

不提供：

- 个体医疗、心理、性健康、转诊、激素/手术、身份文件、法律、学校、职场或家庭冲突建议。
- 对任何人的性取向、性别认同、隐私状态、风险状态或家庭关系做推断。
- 危机干预、outing、监控、画像、歧视性分类、转换治疗或伤害性内容。
- 把 LGBTQ 服务可达写成文化战争材料、政治宣传或商业定向获客方案。

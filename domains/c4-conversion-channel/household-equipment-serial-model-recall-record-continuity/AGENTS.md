# household-equipment-serial-model-recall-record-continuity 目录说明

本目录承载家庭设备型号、序列号、召回匹配和维修记录研究域，关注产品身份资料如何让安全通知、维修和恢复流程可执行。

## 目录结构

```text
household-equipment-serial-model-recall-record-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义设备身份记录、召回匹配、维修记录和主体任务连续性的关系。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 CPSC、SaferProducts.gov、FTC、FDA/NHTSA 相关召回系统和消费者保护资料。
- 不写产品推荐、维修教程、保修索赔策略、序列号修改、召回规避或个案法律建议。
- 新增指标必须注明口径：设备类型、记录字段、召回匹配、维修状态、转让/租赁场景和连续性后果。
- 不保存真实家庭设备序列号、购买凭证、照片、账户或住址。

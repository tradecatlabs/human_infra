# trustworthy-scientific-communication-peer-review 目录说明

本目录维护可信科研传播、同行评审和出版纠错链。它是 Human Infra 证据进入公共叙事前的语义过滤层。

## 目录结构

```text
trustworthy-scientific-communication-peer-review/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义同行评审、预印本、撤稿、引用语境和科研传播如何影响主体持续性研究判断。
- 本目录只维护传播与出版状态的审查框架，不保存具体论文 PDF、审稿材料或未授权全文。
- 资料进入本域时必须保留版本、来源、审查状态、撤稿/勘误状态、利益冲突和可引用边界。

## 上下游关系

- 上游来自 `research-infrastructure-open-science-translation/`、`source-card-system` 和外部出版伦理资料。
- 下游服务 `model-cards-ai-audit-documentation/`、`research-portfolio-prioritization-funding-governance/`、`technology-foresight-horizon-scanning/` 和 Web 论文页。
- 不替代 `information-integrity-trust/`；该域关注更宽的信息环境，本目录关注科研传播链。

## 维护规则

- 不把预印本、新闻稿、公司披露或媒体报道升级成已验证事实。
- 不提供操控评审、规避撤稿、洗白不端或刷引用方案。
- 若新增字段影响 Source Card 或论文页契约，必须同步更新对应模板和索引。

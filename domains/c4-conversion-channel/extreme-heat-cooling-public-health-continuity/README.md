# Extreme Heat, Cooling, and Public Health Continuity

## 核心问题

极端高温决定环境温度是否会通过体温调节失败、睡眠破坏、工作负荷、停电、住房质量、城市热岛和医疗负担，系统性压缩主体有效时间与生存窗口。

Human Infra 关注的问题不是天气描述，而是：

```text
高温如何从外部环境变量变成身体、住房、能源、劳动和公共卫生的共同约束？
```

## 先验位置

主体持续性需要可承受环境、恢复睡眠、可负担制冷、稳定电力、可暂停的工作暴露和可达公共服务。极端高温不是单纯体温问题，而是把能源、住房、劳动、公共卫生和城市规划连接在一起。

## 关注对象

- 热浪、热指数、湿球温度、夜间高温、城市热岛和热健康预警。
- 热衰竭、热射病、慢病加重、药物与热风险、睡眠损害和心理负担。
- 制冷可及、能源负担、断电风险、工作暴露、避暑中心和社区探访。
- 城市树冠、通风、住房质量、公共通信和公平风险。

## Human Infra 模型链路

```text
极端高温
  -> 体温调节、睡眠、劳动、住房和能源系统承压
  -> 死亡风险、慢病风险、疲劳、误工和服务需求上升
  -> 主体有效时间、恢复能力和行动半径下降
  -> 长期高温适应能力成为主体持续性的外部边界
```

## 非目标

- 不提供个人急救、补水、药物调整、工作许可、空调选择或避暑地点建议。
- 不替代 CDC、气象部门、公共卫生、职业安全、住房或能源援助机构。
- 不把极端高温归因于个人意志或生活方式。

# patient-advocacy-shared-decision-making 目录说明

本目录维护患者倡导与共同决策域。它是 Human Infra 的“主体偏好进入医疗决策”层，负责患者参与、决策辅助和权利表达如何影响主体持续性。

## 目录结构

```text
patient-advocacy-shared-decision-making/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明患者倡导、共同决策、知情同意、决策辅助和患者参与如何影响主体自主性与未来选择权。
- 本域只做公开资料整理和边界维护。
- 不提供个体治疗选择、第二意见、医疗投诉、诉讼、保险申诉、病历纠纷、机构排名或动员话术。

## 维护规则

- 新增资料必须区分患者参与、共同决策、患者权利、法律代理和个体治疗选择。
- 涉及争议治疗、临床试验、投诉和家属冲突时，必须保留专业、伦理和法律边界。
- 不把倡导写成操纵医生或家属，不把共同决策写成让患者独自承担系统责任。

## 研究推进骨架

### 最小问题集

- 界定转化瓶颈：研究发现、数据、临床、制度、身份、资金、可及性或操作接口。
- 说明接口机制：潜在可能性如何经过标准、流程、记录、组织或平台变成可用路径。
- 记录必要证据：登记系统、指南、监管材料、数据库、标准文档或真实世界部署资料。
- 识别摩擦指标：时间成本、资金成本、信息成本、合规成本、可及性和协作成本。
- 列出失败点：证据断裂、监管阻塞、制度排斥、隐私风险、责任不清或资源不可达。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

### 最小产出

- Source Signals：记录候选资料、来源、日期、主张和待核验点。
- Source Cards：提取 claim、变量、机制、证据类型、边界、反证条件和可迁移位置。
- Claim-Evidence Matrix：把每个稳定主张绑定到来源、证据等级、适用范围和不确定性。
- 变量表：列出输入变量、中间变量、状态变量、风险变量、输出指标和可观测代理指标。
- 上下游关系：说明本域依赖哪些更根本域，并向哪些转化、应用或基础设施域输出。

Narrative Identity Life Review Continuity 研究生命叙事、个人故事、life review、回忆、传记资料和身份解释如何帮助主体保持自我连续、整合失败、修正目标和转交记忆。

> 核心问题：主体持续性不仅是记忆数据保留，还包括主体能否把过去、现在和未来组织成仍可理解的自我叙事。

## 研究对象

- Narrative identity、life review、reminiscence、personal biography、memoir、life story 和 memory work。
- 老年、临终、创伤、迁移、失业、身份转变、残障和认知下降场景中的叙事重组。
- 叙事与意义感、尊严、心理恢复、家庭沟通、记忆外化和数字遗产之间的关系。

## 先验链路

```text
主体通过叙事把分散经历组织成“我是同一个人”
  -> 生命回顾和叙事身份让记忆、失败、关系和未来目标获得连续结构
  -> 叙事断裂会削弱身份稳定、意义感、家庭沟通和临终尊严
  -> 外部记忆即使存在，也可能无法转化为主体可用的自我解释
  -> 主体连续性、恢复能力和跨代记忆转交下降
  -> 因而叙事身份是记忆与人格连续性的解释层
```

## 关键变量

- Life story coherence、autobiographical memory、life review、reminiscence、agency、redemption narrative 和 dignity。
- 认知状态、创伤、家庭关系、文化身份、数字档案、照护场景和临终场景。
- 记忆篡改、家庭控制、隐私暴露、创伤重激活、虚假叙事、AI 代写和死后人格冒充风险。

## 证据入口

- Life review therapy / reminiscence 相关 PubMed 综述。
- Dignity therapy、narrative identity 和 autobiographical memory 研究。
- Life logging、personal archives 和 digital legacy 相邻资料。

## 非目标

- 本域不提供心理治疗、创伤处理、回忆诱导、家庭调解、传记代写或死后人格代理。
- 不生成个人生命故事、遗书、悼词、临终文件、心理解释或家庭冲突建议。
- 不收集私人日记、家庭秘密、创伤、照片、音频、病历、遗嘱、账号或身份资料。

# access-functional-needs-emergency-registry-continuity 目录说明

本目录承载 access and functional needs、特殊医疗需求登记和灾害功能需求连续性资料。它研究非平均主体如何在停电、撤离、避难和恢复服务中被提前识别。

## 结构

```text
access-functional-needs-emergency-registry-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分功能需求、医疗需求、电力依赖、交通协助、避难所无障碍和一般灾害准备。
- 必须记录隐私、过期登记、地方差异、服务责任、污名化和虚假安全感风险。
- 不保存个人残障、设备、药物、地址、联系方式、定位、照护者或健康资料。

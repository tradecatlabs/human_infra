# Sensitive Location Data Exposure Continuity 架构说明

本目录承载敏感位置数据暴露研究。它只建模风险和治理，不提供定位、追踪、去匿名化或执法规避内容。

## 目录结构

```text
sensitive-location-data-exposure-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须避免真实位置、地址、场所、庇护和医疗出入记录。
- 不写 geofencing campaign、tracking SDK、移动广告 ID 关联或数据购买教程。
- 与 `personal-safety-violence-prevention/` 和幸存者安全域保持强隐私边界。

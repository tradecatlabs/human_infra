# gait-balance-assessment-fall-screening-continuity AGENTS

## 职责

- 维护步态、平衡筛查、移动信心、跌倒史、转介和随访的研究边界。
- 将筛查工具、服务入口、转介断点、家庭活动和功能结果整理为 Source Card。
- 与 `fall-risk-prevention-home-safety-continuity/` 保持边界清晰：本域关注识别与转介，不写家庭防跌倒处方。

## 非目标

- 不提供个人风险评分、步态诊断、助行器选择、家庭改造、训练计划、活动许可或保险资格判断。
- 不替代临床、物理治疗、职业治疗、老年医学或家庭安全评估。

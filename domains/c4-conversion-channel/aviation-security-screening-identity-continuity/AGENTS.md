# 航空安检、身份核验与筛查连续性目录说明

## 目录结构

```text
aviation-security-screening-identity-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义航空安检、身份核验、REAL ID、旅客支持和可访问筛查如何影响移动连续性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀公开安检规则、身份核验变量、可访问筛查和旅客支持边界，不写规避、绕行或漏洞路径。
- 涉及美国制度时优先使用 TSA、TSA Cares、DHS REAL ID 和 U.S. DOT 残障航空旅行资料。
- 不把本域写成安检操作指南、违禁品携带建议、身份规避、机场安全分析或个案旅行判断入口。

# unused-medicine-takeback-disposal-continuity

## 职责

本目录承载未用药、过期药、管制药品、take-back、drop box、mail-back 和家庭药品风险治理资料。

## 边界

- 只研究公开制度、回收基础设施、风险变量和安全边界。
- 不提供个案药品处置、管制药规避、药品转售、家庭销毁或执法规避步骤。
- 不保存药名剂量、处方、管制药信息、家庭地址或药品照片。

## 结构

```text
unused-medicine-takeback-disposal-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须标注来源机构、适用药品类别和是否涉及管制药。
- 个案处置问题必须转回 DEA/FDA、药师、毒物中心、地方回收点或官方渠道。

# public-restroom-sanitation-hygiene-access-continuity 目录说明

本目录承载公共厕所、卫生与如厕可达连续性研究域，关注公共空间中的身体需求如何决定主体能否持续停留、移动和参与。

## 目录结构

```text
public-restroom-sanitation-hygiene-access-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义公共如厕、洗手、卫生维护和无障碍如厕如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 CDC WASH、CDC handwashing、ADA.gov、OSHA sanitation、地方公共卫生和设施管理公开资料。
- 不写厕所定位跟踪、设施破坏、非法进入、隐私侵犯、骚扰、执法规避或个案投诉策略。
- 涉及公共设施数据时只讨论可达性、开放时段、维护质量、无障碍、卫生功能和任务连续性。

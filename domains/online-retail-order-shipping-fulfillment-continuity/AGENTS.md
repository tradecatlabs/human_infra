# 在线零售订单、发货与履约连续性目录说明

## 目录结构

```text
online-retail-order-shipping-fulfillment-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义订单确认、发货承诺、交付延迟、未交付和履约纠错如何影响主体资源转换。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀订单履约机制、证据链、消费者保护边界和主体资源变量，不写个案购物或退款流程。
- 涉及美国制度时优先使用 FTC、USA.gov、州消费者保护机构和官方公开资料。
- 不把本域写成购物推荐、商家选择、投诉脚本、退款策略、价格套利、平台风控规避或个案法律建议。

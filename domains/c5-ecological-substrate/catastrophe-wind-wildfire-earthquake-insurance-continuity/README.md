本域研究 hurricane/windstorm、wildfire、earthquake、hail、catastrophe deductible、state residual markets、FAIR plans、earthquake authorities、nonrenewal 和 insurer insolvency 如何影响主体灾害后住房恢复和区域可保性。它关注洪水之外的高冲击财产灾害风险转移，而不是投保、理赔、灾害搬迁或法律建议。

## 研究对象

- 风暴、飓风、野火、地震、冰雹、风暴潮之外的 wind damage、catastrophe deductible 和 peril exclusions。
- FAIR Plan、residual market、earthquake insurance、wind pool、保单非续保、费率上升和保险公司退出。
- 灾后 claim、adjuster、repair estimate、underinsurance、insurance gap、insurer insolvency 和州保障基金。
- 与洪水保险、房主租客保险、野火韧性、地震建成环境和灾后恢复的边界。

## 价值链路

```text
主体住房暴露于低频高损失巨灾
  -> 巨灾保险和州风险池把区域性财产冲击转移到保单、市场和公共补位机制
  -> 明确 peril、deductible 和 residual market 入口维持灾后重建资金通道
  -> 非续保、保障缺口、承保退出或 insurer insolvency 会让整个地区恢复能力下降
  -> 巨灾保险连续性把个体住房恢复连接到区域可保性和气候风险治理
```

## 关键问题

- 风、野火、地震和洪水为什么需要分开建模？
- catastrophe deductible、exclusions 和 underinsurance 如何改变灾后现金需求？
- FAIR Plan、wind pool 和 residual market 如何在市场退出时提供最后入口？
- 保险可得性下降如何反向影响住房、房贷、迁居和地方财政？

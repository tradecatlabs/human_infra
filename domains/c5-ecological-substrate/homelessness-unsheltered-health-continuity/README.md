无家可归、露宿状态与健康连续性域研究：当主体缺少稳定住所、卫生条件、睡眠环境、文件存储和服务入口时，医疗、福利、安全和恢复系统如何避免断线。

## 对象

- 无家可归、露宿、庇护所、临时住房、永久支持性住房、Housing First 和街头外展。
- 睡眠、卫生、药物保存、身份证件、通讯、物品存放、食物、水、极端天气和安全暴露。
- 医疗、心理健康、成瘾支持、慢病管理、传染病、急诊使用和照护转介连续性。
- 福利、身份文件、就业服务、社区资源导航、庇护服务和住房服务入口。
- 无家可归状态对死亡风险、创伤、感染、慢病失控、有效时间和重新进入未来的影响。

## Human Infra 价值

没有稳定住所时，主体不是只缺一个房间，而是同时失去睡眠、卫生、药物保存、通信、服务地址、文件保管和安全边界。无家可归健康连续性域把“主体是否被系统接住”作为独立条件，避免把露宿状态简化为个人选择或单一住房问题。

```text
无家可归状态被服务系统接住
  -> 医疗、卫生、身份、睡眠、安全和福利入口不断线
  -> 急性风险、慢病失控、暴露和创伤下降
  -> 主体恢复能力和重新进入未来的机会提高
```

## 边界

本域只整理公开无家可归服务、住房连续性、公共卫生、外展、庇护、福利和社区资源资料。

不提供：

- 个体庇护所选择、住房申请、资格判断、法律救济、福利申请或医疗建议。
- 露宿规避、执法规避、营地隐蔽、安保对抗、占用建筑或生存技巧。
- 对具体地点、机构、人员、弱势群体的定位、追踪、骚扰或剥削。
- 把无家可归状态写成个人失败、城市清理目标、商业驱逐策略或治安宣传。

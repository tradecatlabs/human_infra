### Human Infra 追问

这个域如何承载主体展开未来的外部世界条件，例如能源、水、食物、住房、通信、治理、经济和基础设施？

```text
研究域对象
  -> 影响变量 / 中间机制
  -> 改变主体状态或外部条件
  -> 改变风险、能力、时间成本或可达性
  -> 改变有效寿命、有效时间或未来选择权
```

## 研究推进骨架

### 最小问题集

- 界定外部承载条件：能源、水、食物、住房、通信、交通、治理、经济或公共基础设施。
- 说明它如何改变主体展开未来所需的稳定性、冗余、可获得性和安全边界。
- 记录基础设施依赖：供应链、地理环境、制度、资金、维护能力和公共服务。
- 识别冲击场景：灾害、战争、断供、污染、气候、系统失灵或社会支持崩溃。
- 建立韧性指标：可用率、恢复时间、冗余、成本、覆盖范围、可负担性和风险暴露。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

### 最小产出

- Source Signals：记录候选资料、来源、日期、主张和待核验点。
- Source Cards：提取 claim、变量、机制、证据类型、边界、反证条件和可迁移位置。
- Claim-Evidence Matrix：把每个稳定主张绑定到来源、证据等级、适用范围和不确定性。
- 变量表：列出输入变量、中间变量、状态变量、风险变量、输出指标和可观测代理指标。
- 上下游关系：说明本域依赖哪些更根本域，并向哪些转化、应用或基础设施域输出。

迁徙与季节性农业工人健康、住房和劳动连续性域研究：当工作、住房、交通、语言、身份、气候暴露和医疗可及绑定在季节性农业链条上时，主体如何维持健康、收入、家庭和服务入口。

## 对象

- Migrant and seasonal farmworker health, labor, housing, transportation, childcare and service access。
- 农业工作暴露、热应激、农药、伤害、呼吸风险、感染风险、慢病管理和职业健康。
- 临时/雇主关联住房、交通依赖、工资支付、招聘中介、劳动权利、语言可达和报复风险。
- 社区健康中心、migrant health centers、移动诊所、学校和家庭服务衔接。
- 季节性迁移、记录缺口、健康保险、医疗随访、药品连续性和家庭照护断点。

## Human Infra 价值

农业工人的主体持续性常被拆散在工作地点、临时住房、交通、语言和身份风险之间。该域把季节性迁移和雇佣依赖作为基础设施变量，解释为什么“有服务”仍不等于“能安全使用服务”。

```text
农业劳动、住房和迁移链条被结构化治理
  -> 医疗、劳动权利、交通、住房、语言和家庭服务更可达
  -> 季节性工作不再把健康随访和家庭支持反复切断
  -> 主体有效时间、恢复能力和未来选择权提高
```

## 边界

本域只整理公开农业工人健康、职业暴露、住房、劳动权利、迁移、服务可达和家庭支持资料。

不提供：

- 个体医疗、移民身份、劳动纠纷、举报、工资追索、住房申请、交通或法律建议。
- 雇主、招聘中介、农场、工人营地、执法或移民状态的个案判断。
- 危险作业规避、执法规避、身份规避、现场冲突处理或职业暴露处置建议。
- 将农业工人群体用于风险定位、身份推断、商业开发或监管套利。

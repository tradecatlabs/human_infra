# Public Transit Service Operations Continuity

`public-transit-service-operations-continuity/` 研究公交、轨道、地铁、轻轨、轮渡、无障碍交通、班次可靠性、运营资金、司机/运维人力和服务恢复如何支撑主体持续性。

> 核心问题：交通可达不只是“有路”。大量主体依赖公共交通到达医疗、工作、教育、食物、照护和社会关系。公交运营失败会把资源存在转化为资源不可达。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要稳定到达医疗、工作、学习、食物、照护、公共服务和关系网络
  -> 公共交通依赖班次、车辆、司机、维修、站点、无障碍、资金、调度和信息服务
  -> 运营削减、延误、司机短缺、设备故障、灾害和财政压力会造成服务断点
  -> 公交断点会增加缺诊、失业、孤立、时间贫困和撤离不可达
```

## 关注对象

- 公交、轨道交通、地铁、轻轨、轮渡、需求响应交通、paratransit、班次可靠性、服务覆盖、无障碍运营、乘客信息、资产状态和运营资金。
- 与 `transportation-access-mobility/` 的边界：交通接入域关注日常移动和资源可达；本域关注公共交通系统自身的运营连续性、服务水平和恢复能力。
- 与 `road-traffic-injury-prevention-safe-mobility/` 的边界：道路安全域关注伤害预防；本域关注公共交通服务是否持续可用。

## Human Infra 模型链路

```text
公共交通运营连续性
  -> 班次、覆盖、无障碍、车辆、站点、司机、维护和实时信息稳定运行
  -> 医疗、就业、教育、食品、照护和公共服务可达性提升
  -> 缺诊、迟到、时间贫困、孤立和灾害撤离断点下降
  -> 主体获得低成本、可预测、低摩擦的移动生命线
```

## 非目标

- 不提供实时路线、逃票、绕闸、破坏、设施弱点、司机调度、车辆维修、运营控制、安保规避或危险乘车建议。
- 不提供交通投资、项目采购、票价争议、法律索赔、个体出行规划或实时灾害撤离方案。
- 不替代当地 transit agency、FTA、交通管理、应急管理和官方服务公告。

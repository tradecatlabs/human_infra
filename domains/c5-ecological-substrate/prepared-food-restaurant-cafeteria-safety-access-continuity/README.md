# Prepared Food Restaurant Cafeteria Safety Access Continuity

`prepared-food-restaurant-cafeteria-safety-access-continuity` 研究餐馆、食堂、工作/校园/医院 cafeteria、外卖、即食餐、菜单信息、过敏原、食品安全和可负担外食如何影响主体在不能做饭或离家时的餐食连续性。

> 核心问题：外食不是娱乐层，而是厨房不可用、时间不足、照护负担、工作学习和医疗场景中的替代摄食基础设施。

## 关注对象

- 餐馆、cafeteria、工作/校园/医院餐食、外卖、即食餐、菜单信息、过敏原标识、食品安全评分、价格和支付入口。
- 食源性风险、过敏原、特殊饮食、营业时段、交通可达、排队、配送、停电、灾害、卫生检查和消费者信息。
- 与 FDA Food Code、CDC restaurant food safety、地方健康部门、劳动/学校/医院食堂、外卖平台和食品安全监管的接口。

## Human Infra 链路

```text
熟食和外食安全可达
  -> 主体在离家、厨房不可用或时间受限时仍可获得餐食
  -> 食源性疾病、过敏暴露、排队失败、价格挤压和时间损耗下降
  -> 工作、学习、照护、医疗陪护和旅行场景中的能量输入更稳定
  -> 有效时间、恢复能力和任务完成能力增强
```

## 非目标

- 不提供餐馆推荐、点餐建议、饮食处方、过敏医疗建议、卫生评分炒作、差评攻击、外卖平台规避、食品制备流程或个案投诉策略。
- 不替代 FDA、CDC、地方卫生部门、餐饮机构、营养师、过敏专科、消费者保护或法律服务。
- 不收集个人点餐记录、健康/过敏信息、支付账号、餐馆内部安防、员工隐私、配送轨迹或卫生检查未公开信息。

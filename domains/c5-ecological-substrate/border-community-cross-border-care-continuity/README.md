# Border Community Cross Border Care Continuity

边境社区与跨境照护连续性域研究：当生活、家庭、工作、医疗、公共卫生、交通、语言和身份在边境两侧同时发生时，主体如何维持健康、服务、记录和风险治理连续性。

## 对象

- Border communities, cross-border families, binational public health, infectious disease coordination and care continuity。
- 边境通勤、家庭分布、学校/工作、药品、医疗服务、记录转移、语言文化和交通时间。
- 公共卫生监测、传染病协作、环境暴露、急救响应、灾害、移民状态和执法边界。
- U.S.-Mexico border health, ports of entry delays, rural border service access and community resource navigation。
- 跨境生活对随访、慢病管理、孕产服务、儿童照护、心理健康和家庭支持的影响。

## Human Infra 价值

边境社区不是单纯的“边界线旁居民”。对很多主体来说，家庭、工作、医疗和公共卫生系统横跨两套制度。该域把边境作为主体持续性的接口问题：服务、记录、语言、交通和法律边界如何共同决定行动是否中断。

```text
边境服务和跨境照护接口被结构化治理
  -> 医疗随访、公共卫生、家庭支持、交通和记录衔接更稳定
  -> 制度边界不再把主体生活拆成无法互认的片段
  -> 主体行动能力、健康连续性和未来选择权提高
```

## 边界

本域只整理公开边境社区健康、跨境公共卫生、服务可达、交通、语言、记录和风险治理资料。

不提供：

- 个体移民、签证、入境、海关、旅行、医疗、药品、法律、福利或执法规避建议。
- 实时边境等待、路线、检查、口岸、跨境就医、药品携带或身份判断。
- 规避监管、绕过医疗制度、跨境套利、执法弱点或敏感设施定位。
- 将边境社区用于风险标签、身份推断、监控或政治宣传。

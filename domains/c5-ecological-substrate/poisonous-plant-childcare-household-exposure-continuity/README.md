`poisonous-plant-childcare-household-exposure-continuity` 研究有毒植物、儿童误食、家庭/学校/托育环境和毒物中心连接如何影响主体持续性。

> 核心问题：家庭植物和户外植物不是纯装饰环境；当儿童、认知障碍者、宠物或托育场景进入时，植物误食会变成监管、照护和应急连接问题。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体及其照护系统必须区分可接触环境和有毒暴露
  -> 有毒植物会把家庭、学校、托育和公共空间转化为误食风险
  -> 环境筛查、看护、标签、毒物中心连接和教育材料构成行动前提
  -> 若该前提失败，儿童照护、家庭安全、学校活动和宠物共处都会被急性不确定性劫持
```

## 关注对象

- 有毒室内/户外植物、儿童误食、宠物暴露、托育/学校环境、认知障碍者风险和毒物中心连接。
- 照护者教育、标签、环境筛查、公共空间植物选择和风险沟通。
- 与 `household-poison-storage-child-resistant-packaging-continuity/` 的边界：该域关注包装/储存型毒物；本域关注植物作为开放环境暴露源。

## Human Infra 链路

```text
有毒植物暴露边界被识别
  -> 儿童误食、宠物暴露和照护恐慌下降
  -> 家庭、学校、托育和公共空间更可用
  -> 照护时间、心理负担和急性医疗连接摩擦下降
```

## 非目标

- 不提供植物鉴定、可食性判断、催吐/急救、治疗、宠物医疗、儿童个案分诊、植物采购或园林配置建议。
- 不替代 Poison Help、医疗、兽医、托育监管、学校健康、公共卫生或植物专家。
- 不把照片、AI 识别、民间名称或植物清单写成安全许可。

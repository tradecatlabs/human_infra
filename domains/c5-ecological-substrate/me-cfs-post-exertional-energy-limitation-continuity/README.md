`me-cfs-post-exertional-energy-limitation-continuity/` 研究 ME/CFS、劳累后不适、能量限制、认知困难、睡眠恢复失败和生活参与如何影响主体持续性。

## Core Question

当主体可用能量和恢复窗口被劳累后不适限制时，如何避免把任务执行、学习、工作和社会参与继续设计成“无限能量”模型？

## 先验位置

- 与 `fatigue-risk-alertness-continuity/` 的关系：ME/CFS 不只是一般疲劳，而是能量限制和劳累后不适病程。
- 与 `time-allocation-effective-time/` 的关系：可用时间必须扣除恢复成本和崩溃风险。
- 与 `functioning-quality-of-life-outcomes/` 的关系：功能结果和患者报告结局是核心证据层。

## 关注对象

- 劳累后不适、疲劳、认知困难、睡眠、直立不耐受、疼痛、学校/工作适配、污名和服务可及。
- 诊断漫游、活动节奏、功能损耗、照护/家庭负担和长期不确定性。

## Human Infra 模型链路

```text
ME/CFS 能量与恢复系统受限
-> 劳累后不适、认知困难和睡眠恢复失败
-> 可用行动预算和任务节奏下降
-> 学习、工作、关系和社会参与受限
-> 主体持续性下降
```

## 非目标

- 不提供 ME/CFS 诊断、运动/活动处方、药物、补剂、睡眠方案、学校/工作证明、残障或保险建议。
- 不根据症状日志、可穿戴数据或问卷输出个体风险、活动许可、治疗或预后判断。

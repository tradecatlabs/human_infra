# Child Care Subsidy CCDF Payment Continuity

`child-care-subsidy-ccdf-payment-continuity/` 研究 CCDF 与托育补贴、资格、授权、续期、支付、家庭共付和行政负担如何决定家庭是否能持续负担照护。

> 核心问题：托育补贴不是抽象福利，而是把家庭收入、儿童照护、父母工作和服务供给连接起来的支付连续性系统。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 托育费用可能吞噬家庭收入和照护者工作收益
  -> 补贴、授权、续期和支付必须稳定抵达家庭与 provider
  -> 行政负担和支付中断必须被识别
  -> 家庭才能维持照护、就业、学习和恢复连续性
```

## 关注对象

- CCDF、托育补贴、资格规则、redetermination、家庭共付、provider payment、授权与等待。
- 申请摩擦、续期流失、支付延迟、provider 参与、家庭负担和服务稳定性。
- 与 `social-protection-benefits-delivery/`、`administrative-burden-procedural-friction/`、`child-care-availability-affordability-continuity/` 和 `financial-resilience-access/` 的接口。

## Human Infra 模型链路

```text
托育费用压力
  -> 补贴资格、授权、续期、共付和 provider 支付
  -> 家庭现金流和托育服务稳定性改善
  -> 父母工作学习连续性和儿童稳定照护提高
  -> 家庭未来选择权和风险缓冲增强
```

## 非目标

- 不判断个体补贴资格、申请策略、材料准备、申诉、税务、法律或移民问题。
- 不提供虚假材料、资格规避、收入规避、支付套利或 provider payment 操作建议。
- 不收集家庭收入、儿童身份、补贴材料、税务记录、雇佣证明或账号凭据。

`lice-scabies-school-shelter-infestation-continuity` 研究头虱、疥疮等人际传播性寄生/虫害如何影响学校、避难所、照护机构、睡眠和污名治理。

> 核心问题：虱疥类问题通常不是高死亡风险，却能通过瘙痒、睡眠破坏、污名、停课、家庭冲突和集体住宿传播持续消耗主体有效时间。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要学校、避难所、照护和家庭环境保持可参与
  -> 虱疥等虫害会制造瘙痒、睡眠中断、排斥、污名和反复传播
  -> 识别、通知、治疗连接、清洁边界和反污名规则构成参与前提
  -> 若该前提失败，教育、照护、住房和恢复系统会被低致死高摩擦风险拖慢
```

## 关注对象

- 头虱、体虱、疥疮、集体住宿、学校、避难所、长期照护和无家可归场景。
- 瘙痒、睡眠、污名、停课/停工、家庭负担、重复传播和护理人员暴露。
- 与 `bed-bug-infestation-sleep-housing-continuity/` 的边界：床虱主要是环境/住房侵扰；本域关注人际传播性虫害和集体机构连续性。

## Human Infra 链路

```text
虱疥类传播被低污名、低摩擦地处理
  -> 瘙痒、睡眠损失、停课、污名和反复传播下降
  -> 学校、避难所、照护机构和家庭运行更稳定
  -> 学习、恢复、照护和社会参与连续性增强
```

## 非目标

- 不提供诊断、用药、治疗步骤、清洗细节、机构隔离政策、学校排除规则或个案处理建议。
- 不替代 CDC、学校、公共卫生、医疗、长期照护、避难所管理或住房机构。
- 不鼓励污名化、强制剃发、歧视、隐瞒传播或恐慌性清理。

关键矿物、材料与韧性域研究：当主体持续性依赖 AI、能源、医疗设备、通信、制造、交通和公共基础设施时，上游矿物、材料、替代、回收与供应韧性如何成为必要条件。

## 对象

- 关键矿物、关键材料、战略材料、稀土、半导体材料、电池材料和医疗/能源材料。
- 上游采矿、加工、精炼、贸易依赖、地缘风险、环境约束和劳工风险。
- 材料替代、回收、循环经济、库存、供应韧性和需求增长。
- 关键材料与 AI 算力、能源转型、医疗设备、制造维修、交通和通信的接口。
- 材料稀缺对未来技术窗口、成本、可及性和公共服务连续性的影响。

## Human Infra 价值

许多未来能力看起来是软件、医学或 AI 问题，底层却受材料约束。关键矿物与材料韧性域把“技术能否规模化、维修、替换和普及”的上游瓶颈纳入主体持续性模型，避免把未来技术窗口误判成纯算法或纯医学问题。

```text
未来技术依赖物质输入
  -> 关键矿物、材料、加工、替代和回收能力
  -> AI、能源、医疗设备、通信和制造可规模化
  -> 技术可及性、维修能力和未来选择权提高
```

## 边界

本域只整理公开资源、产业韧性、材料依赖、替代和环境/社会风险框架。

不提供：

- 个体投资、交易、采购、矿权、贸易、税务、制裁规避或商业策略建议。
- 非法采矿、走私、规避出口管制、供应链绕过或敏感采购路径。
- 具体设备拆解、危险回收、化学提取、冶炼或材料处理步骤。
- 企业竞争情报、市场操纵、投标或受管制物资获取建议。

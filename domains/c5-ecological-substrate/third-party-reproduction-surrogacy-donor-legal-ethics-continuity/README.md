第三方生殖、代孕、供体和法律伦理连续性，关注 gestational carrier、intended parents、gamete/embryo donors、consent、心理筛查、法律咨询、亲子关系、跨境生殖、剥削风险和后代权益如何影响生殖技术的正当性与未来选择权。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 生殖技术可以把亲缘、妊娠、照护、法律身份和跨境服务拆分给多个主体
  -> 多主体结构增加了同意、风险、补偿、亲子关系、医疗责任和后代信息权复杂度
  -> 若法律伦理接口失效，技术路径会转化为剥削、争议、身份断裂和照护风险
  -> 因此第三方生殖是主体持续性中必须单独治理的关系基础设施
```

## 关注对象

- Gestational carrier、intended parents、sperm/oocyte/embryo donor、directed/nonidentified donor、心理/医学/感染筛查、知情同意和法律咨询。
- 跨境 reproductive care、费用/补偿、保险、妊娠风险、亲子关系、出生记录、后代信息权、代孕者权利和剥削防护。
- 与 `reproductive-tissue-donor-screening-traceability-continuity/` 的边界：供体追溯域关注材料安全和记录；本域关注多主体关系、法律伦理和跨境风险。

## Human Infra 模型链路

```text
第三方生殖安排 T
  -> 改变妊娠承载、供体信息、合同同意、亲子身份、医疗责任和跨境规则变量 X
  -> 改变主体的生殖可达性、关系稳定性、法律身份、心理负担和未来家庭路径状态 S
  -> 改变剥削、合同冲突、医疗责任断裂、亲子身份争议、后代信息缺失和跨境救济失败风险 lambda(t)
  -> 影响生殖技术的社会正当性、主体选择权和子代连续性
```

## 非目标

- 不提供代孕、供体、合同、亲子关系、跨境服务、诊所、律师、保险或医疗建议。
- 不提供规避当地法律、规避筛查、隐藏身份、商业撮合或剥削性安排。
- 不把第三方生殖写成默认技术路径，也不把任何家庭形式做价值排序。

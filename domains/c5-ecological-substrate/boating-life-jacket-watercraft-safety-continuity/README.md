`boating-life-jacket-watercraft-safety-continuity/` 研究休闲船艇、个人水上载具、救生衣、酒精暴露、天气水况和事故报告如何影响水上活动中的主体持续性。

> 核心问题：船艇安全把交通、天气、设备、判断力和水域救援连接成一条高后果行动链。

## 先验位置

```text
主体要持续行动
  -> 必须在水上移动中维持浮力、方向、判断和救援可达
  -> 船艇活动同时暴露于机械、天气、酒精、碰撞和落水风险
  -> 因而水上载具安全是娱乐移动中的持续性边界
```

## 关注对象

- 休闲船艇、个人水上载具、救生衣、操作员培训、天气水况、酒精、乘员、事故报告和公共执法教育。
- 儿童、不会游泳者、船艇乘员、钓鱼/划船/水上运动参与者。

## Human Infra 模型链路

```text
船艇与救生衣安全 T
  -> 改变浮力保护、操作判断、天气暴露、酒精、载具稳定、通信和救援变量 X
  -> 改变水上移动、落水和事故后存活状态 S
  -> 改变溺水、创伤、低温、失踪和救援延迟风险 λ(t)
  -> 改变行动半径、休闲恢复、家庭系统和未来选择权
```

## 非目标

- 不提供船艇驾驶、航线、天气判断、救援动作、装备选择、儿童活动许可、法律合规、保险或个体安全建议。
- 不收集船只登记、定位轨迹、事故细节、健康状态、乘员名单或个人活动计划。

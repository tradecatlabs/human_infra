`dog-bite-community-injury-prevention-continuity` 研究犬咬伤、社区动物接触、儿童安全、感染风险和人与伴侣动物共处规则如何影响主体持续性。

> 核心问题：犬咬伤同时是创伤、感染、心理恐惧、家庭关系和社区治理问题；它会改变儿童活动、照护、邻里通行、宠物拥有和公共空间使用。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要与伴侣动物、邻里动物和公共空间稳定共处
  -> 犬咬伤会造成创伤、感染、功能损失、恐惧和法律/家庭冲突
  -> 儿童教育、监督、宠物管理、报告和医疗连接构成社区行动前提
  -> 若该前提失败，身体安全、社会关系和行动自由被动物接触事件破坏
```

## 关注对象

- 犬咬伤、儿童风险、宠物管理、邻里暴露、职业暴露、感染和创伤后恐惧。
- 动物控制、医疗连接、疫苗状态、社区教育、保险/责任边界和家庭决策。
- 与 `rabies-exposure-bite-pep-continuity/` 的边界：狂犬病域关注高致死暴露后路径；本域关注犬咬伤作为社区伤害和共处治理问题。

## Human Infra 链路

```text
犬咬伤风险被预防和处置
  -> 创伤、感染、恐惧和家庭/社区冲突下降
  -> 儿童活动、照护、邻里通行和宠物共处更稳定
  -> 身体安全、社会支持和行动自由增强
```

## 非目标

- 不提供动物训练、攻击犬处理、法律责任判断、个体伤口处理、抗生素、狂犬病暴露判断或宠物处置建议。
- 不替代动物控制、医疗服务、兽医、公共卫生、法律服务或保险机构。
- 不鼓励自行捕捉、惩罚、驱赶或危险接近动物。

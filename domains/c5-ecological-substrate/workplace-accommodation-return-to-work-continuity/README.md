本域整理疾病、残障、伤害、神经认知变化或功能限制后的工作保持、合理便利和复工路径。它关注能力状态改变时，主体是否能通过工作调整继续参与任务系统，而不是被一次健康事件永久排除出工作。

## 研究对象

- ADA reasonable accommodation、interactive process、岗位调整和辅助技术。
- Stay-at-work、return-to-work、modified duty、phased return、remote/hybrid work and schedule modification。
- 工作能力、功能限制、医生证明、隐私、反歧视和过度困难边界。
- 与工伤、残障收入、医疗康复、辅助技术和职业再培训的接口。

## 价值链路

```text
疾病、伤害、残障或功能变化会改变工作能力曲线
  -> 若工作系统只有“完全能做/完全退出”两种状态，主体容易永久离场
  -> 合理便利和复工路径把能力变化转化为可调任务、环境和节奏
  -> 修改工作条件、辅助技术和阶段性复工降低失业、失能和身份断裂
  -> 主体保留收入、技能、社会角色和继续创造的窗口
```

## 关键问题

- 工作能力变化时，哪些任务可以被重组、辅助、替代或延后。
- 合理便利、医疗限制、隐私、反歧视和雇主运营约束如何平衡。
- 何时进入复工、工伤、残障福利、职业康复或辅助技术路径。
- 如何把“功能状态”转成任务要求、工作环境和支持系统之间的匹配问题。

## 上游与下游

- 上游：疾病/伤害、残障服务、康复、辅助技术、职业设计和劳动权利。
- 下游：工作保留、收入连续性、职业身份、技能积累和复工质量。
- 相关域：`rehabilitation-functioning/`、`disability-services-independent-living/`、`assistive-technology-access/`、`labor-rights-workplace-protection/`、`workers-compensation-occupational-injury-benefit-continuity/`、`disability-income-work-capacity-benefit-continuity/`。

## 非目标

- 不提供个人 ADA、雇佣法、工伤、残障福利、医疗限制或复工建议。
- 不判断某个便利是否合理、是否构成过度困难或是否违法。
- 不生成便利请求、医生证明、HR 政策、法律投诉或诉讼材料。
- 不帮助雇主规避合理便利义务或劳动者权利。

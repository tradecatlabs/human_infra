# household-emergency-preparedness-resilience 目录说明

本目录承载家庭级应急准备、个人灾害韧性、短期服务断点和公共救援抵达前主体连续性的研究资料。

## 结构

```text
household-emergency-preparedness-resilience/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 是本域的边界、模型链路、非目标和来源入口。
- 本域只讨论家庭与个人准备的系统变量，不提供实时灾害行动、急救、撤离、用电或防卫指导。
- 后续新增材料应优先是家庭准备 Source Card、变量表、失败模式和特殊人群准备边界。

## 上下游关系

- 上游：`risk-engineering/`、`emergency-alerts-communications/`、`public-service-design-accessibility/`。
- 下游：`disaster-recovery-relief-continuity/`、`caregiving-long-term-care/`、`medicine-access-treatment-continuity/`。
- 相邻域：`energy-access-resilience/`、`water-sanitation-hygiene-continuity/`、`food-security-nutrition-access/`。

## 维护规则

- 不提供实时灾害、医疗急救、危险物、武器、路线或设施改造建议。
- 不把家庭准备写成替代公共应急责任的个人主义叙事。
- 若新增清单、模型或数据，必须同步维护本文件和相关索引。

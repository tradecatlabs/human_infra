和平建设、冲突预防与社会凝聚域研究：社会如何在暴力爆发前识别裂缝、处理不满、维持信任、降低群体敌意，并把冲突从毁灭性路径转回可协商、可修复、可共存的制度路径。

## 对象

- 冲突预防、和平建设、调解、社会凝聚、信任、制度合法性、地方治理和社区韧性。
- 脆弱、冲突与暴力环境中的服务递送、资源争夺、身份政治、创伤、复仇循环和社会分裂。
- 早期预警、非暴力争端解决、社区对话、过渡安排、复原力建设和制度性申诉通道。
- 社会撕裂对健康、学习、工作、迁移、家庭、公共服务和未来选择权的长期影响。
- “和平”作为主体持续性背景条件，而不是抽象道德口号。

## Human Infra 价值

如果社会无法把冲突压在可协商范围内，个体级长寿、教育、医疗、AI 工具和资源积累都会被暴力风险重置。和平建设域把“长期可生活的社会”作为有效永生的上游条件。

```text
社会不满和群体裂缝积累
  -> 暴力、报复、流离失所和制度崩塌风险上升
  -> 冲突预防、社会凝聚和申诉通道降低升级概率
  -> 主体的家庭、服务、学习、工作和长期项目窗口得以延续
```

## 边界

本域只整理公开的和平建设、冲突预防、社会凝聚和脆弱性资料。

不提供：

- 政治竞选、舆论操控、群体动员、抗议组织、暴力升级、镇压、反镇压或规避执法策略。
- 针对具体国家、民族、宗教、群体、政党或个人的动员方案。
- 情报、目标画像、心理战、宣传、仇恨、骚扰或煽动材料。
- 把复杂冲突归因为单一群体、单一文化或单一政治立场。

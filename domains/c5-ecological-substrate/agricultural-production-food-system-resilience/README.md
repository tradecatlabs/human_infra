`agricultural-production-food-system-resilience/` 研究农业生产、作物与畜牧系统、灌溉、种子、肥料、农机、病虫害和气候适应如何支撑长期食物供给。

> 核心问题：食物安全不只发生在市场和家庭端；如果农业生产能力、投入品、土地、水、劳动力和风险管理失效，主体持续性的能量与营养输入会在源头断裂。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要长期稳定的食物输入
  -> 食物输入依赖农业生产系统持续产出
  -> 农业产出依赖土地、水、种子、肥料、病虫害治理、劳动力、设备、气候适应和市场连接
  -> 农业生产失效会传导为价格冲击、营养不足、供应断裂和社会压力
```

## 关注对象

- 农业生产韧性、作物产量、畜牧生产、灌溉、种子系统、肥料和农机可及。
- 气候变化、极端天气、病虫害、动物疫病、土地退化和劳动力短缺对生产的影响。
- 与 `food-security-nutrition-access/` 的边界：本域关注食物从哪里生产出来；食物安全域关注食物是否到达家庭且可负担。
- 与 `soil-health-land-degradation-drought-resilience/` 的边界：土壤域关注土地和土壤作为基础自然资本；本域关注农业生产组织和产出系统。

## Human Infra 模型链路

```text
农业生产韧性
  -> 食物产出、饲料、原料和农村收入更稳定
  -> 食物供应和价格波动下降
  -> 饥饿、营养不足、家庭压力和社会冲击下降
  -> 身体维护、认知表现、免疫恢复和长期行动能力获得稳定输入
```

## 非目标

- 不提供种植、养殖、农药、兽药、肥料、投资、土地交易或农产品投机建议。
- 不提供绕过食品、环境、动物福利、劳动、安全或贸易监管的策略。
- 不把农业产量、粮价或气候情景写成确定预测。

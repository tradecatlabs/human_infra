`bed-bug-infestation-sleep-housing-continuity` 研究床虱侵扰、睡眠破坏、住房稳定、污名、费用和化学暴露如何影响主体持续性。

> 核心问题：床虱主要不通过严重感染造成损害，而是通过睡眠、焦虑、迁移、费用、污名和错误用药持续消耗主体行动能力。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要可睡眠、可恢复、可社交和可居住的空间
  -> 床虱侵扰会破坏睡眠、隐私、住房信任和经济稳定
  -> 识别、报告、综合治理和化学品安全构成恢复前提
  -> 若该前提失败，主体有效时间和恢复能力被慢性摩擦侵蚀
```

## 关注对象

- 床虱侵扰、睡眠中断、焦虑、皮肤反应、家具/行李、旅行住宿和多户住宅传播。
- 房东/租客接口、学校/避难所、误用农药、费用、污名和重复感染。
- 与 `household-cleaning-disinfection-pest-control-continuity/` 的边界：家庭卫生域是上位；本域聚焦床虱对睡眠和住房稳定的长期消耗。

## Human Infra 链路

```text
床虱侵扰被识别和治理
  -> 睡眠破坏、焦虑、污名、费用和错误化学暴露下降
  -> 住房、旅行、学校和避难场景更可恢复
  -> 有效时间、恢复能力和社会参与被保护
```

## 非目标

- 不提供灭虫步骤、农药使用、房东租客法律策略、医疗诊断、产品推荐或住宿选择建议。
- 不替代 EPA、地方卫生/住房机构、害虫治理、医疗服务或法律援助。
- 不鼓励自行危险施药、过量加热、隐瞒传播、丢弃财产恐慌或歧视。

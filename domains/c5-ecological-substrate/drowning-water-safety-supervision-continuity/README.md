# Drowning Water Safety Supervision Continuity

`drowning-water-safety-supervision-continuity/` 研究开放水域、浴池、泳池、河湖海岸、儿童与高风险人群的溺水风险如何影响主体持续性。

> 核心问题：溺水不是单一游泳能力问题，而是水域暴露、照护监督、环境边界、救援可达和风险认知共同失效的急性死亡窗口。

## 先验位置

```text
主体要持续存在
  -> 必须避免分钟级缺氧和不可逆脑损伤
  -> 水域暴露可绕过常规感知、呼吸和行动能力
  -> 因而水安全是主体持续性的急性中止门
```

## 关注对象

- 天然水域、泳池、浴缸、儿童照护、水上活动、监督间隙、救援系统和公共教育。
- 儿童、青少年、老年人、癫痫/认知障碍人群、不会游泳者和水域工作/娱乐人群。

## Human Infra 模型链路

```text
水域安全与监督系统 T
  -> 改变水域暴露、监督、游泳能力、屏障、警示、救援可达和酒精/风险行为变量 X
  -> 改变主体在水边、水中和活动后的安全状态 S
  -> 改变溺水、非致死性缺氧脑损伤、创伤和照护负担风险 λ(t)
  -> 改变生命连续性、家庭系统、有效时间和未来选择权
```

## 非目标

- 不提供游泳教学、救援动作、复苏操作、开放水域路线、儿童监控方案、个人风险判断、设备购买或活动许可建议。
- 不收集儿童影像、家庭泳池照片、定位轨迹、健康史、游泳能力或个人水域行为记录。

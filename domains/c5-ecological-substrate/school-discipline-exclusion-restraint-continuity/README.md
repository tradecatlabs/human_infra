`school-discipline-exclusion-restraint-continuity` 研究停学、开除、课堂移除、替代安置、约束隔离、行为支持、复学和民权数据如何影响学生学习连续性、安全和未来制度路径。

> 核心问题：学校纪律若从支持系统退化为排除系统，会把行为、残障、创伤和社会风险转化为长期学习损耗与制度尾部风险。

## 关注对象

- 校内/校外停学、开除、课堂移除、替代教育、复学计划和纪律数据。
- Restraint、seclusion、行为危机干预、积极行为支持、残障学生保护和家校沟通。
- 纪律差异、种族/残障/语言/性别不平等、学校到司法系统风险和民权审查。
- 纪律与出勤、特殊教育、心理健康、欺凌暴力、家庭法、儿童保护和法院系统的交叉。

## Human Infra 链路

```text
学校纪律与排除风险可治理
  -> 行为事件被转化为支持、复学和风险修复路径
  -> 停学开除、约束隔离、学习日损失和制度尾部后果下降
  -> 学生安全、课堂参与、家庭信任和长期教育路径更稳定
  -> 主体能力积累与未来选择权不被早期排除性事件过度压缩
```

## 非目标

- 不提供个人纪律申诉、停学开除应对、约束隔离投诉、教育法律、民权投诉、行为计划、危机处置或医疗建议。
- 不替代学校、家长、律师、保护与倡导机构、医生、心理服务、儿童福利或执法系统。
- 不收集学生纪律记录、残障、行为、心理、家庭或法律资料。

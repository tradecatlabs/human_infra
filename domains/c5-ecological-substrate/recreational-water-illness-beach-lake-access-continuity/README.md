`recreational-water-illness-beach-lake-access-continuity` 研究泳池、湖泊、海滩、戏水设施、胃肠/皮肤/耳眼感染和公共水域可达如何影响主体持续性。

> 核心问题：休闲水域是降温、运动、亲子和心理恢复基础设施；当病原、粪便污染、关闭公告和检测延迟出现时，它会从恢复空间变成感染与信息不确定性空间。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要安全水域用于运动、降温、社交和恢复
  -> 休闲水病原会把水域可达性转化为感染风险和关闭摩擦
  -> 水质监测、卫生行为、公告、设施维护和替代空间构成行动前提
  -> 若该前提失败，儿童活动、运动、旅游、降温和公共空间信任都会被压缩
```

## 关注对象

- Recreational water illnesses、泳池、热水浴池、湖泊、海滩、戏水设施、粪便污染和水质公告。
- 公共卫生监测、设施维护、用户卫生行为、关闭/重开、语言可及和风险沟通。
- 与 `water-sanitation-hygiene-continuity/` 的边界：上位域关注 WASH；本域关注休闲水域作为运动与恢复空间的执行层。

## Human Infra 链路

```text
休闲水病原和公告系统被管理
  -> 胃肠、皮肤、耳眼感染和水域关闭不确定性下降
  -> 游泳、降温、亲子活动和旅游恢复空间更可用
  -> 健康风险、时间损失和行动选择权被保护
```

## 非目标

- 不提供水质个案判断、游泳许可、采样解释、感染诊断、设施运营、消毒配方或旅游建议。
- 不替代 CDC、EPA、地方公共卫生、海滩/泳池管理、医疗或环境监测机构。
- 不把检测数值、地图、气味、颜色或 AI 输出写成现场安全结论。

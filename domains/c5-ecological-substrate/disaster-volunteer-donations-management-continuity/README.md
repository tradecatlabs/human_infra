# Disaster Volunteer Donations Management Continuity

`disaster-volunteer-donations-management-continuity/` 研究灾后志愿者、实物捐赠、资金捐赠、物资分发、非营利协调和 unsolicited donations 如何影响社区恢复效率。

> 核心问题：灾后善意如果没有协调，会变成仓储、交通、人员安全和分发负担；志愿与捐赠管理决定社会支持能否真正转化为主体恢复资源。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体灾后恢复依赖社区互助、非营利组织、志愿者和物资资金支持
  -> 未协调的志愿和捐赠可能制造拥堵、浪费、欺诈和二次风险
  -> 管理系统把社会善意转化为可分发、可追踪、可持续的恢复资源
  -> 因此灾害志愿与捐赠管理是社会支持连续性的协调域
```

## 关注对象

- FEMA donations management、National VOAD、地方志愿组织、现金捐赠优先、实物捐赠分类、志愿者登记、分发点和欺诈治理。
- 需求匹配、物流、仓储、身份审核、安全培训、语言可达、隐私、重复分发和社区信任。
- 与 `volunteering-civic-service-participation-continuity/` 的边界：该域关注日常志愿参与；本域关注灾害恢复期的协调与风险。

## Human Infra 模型链路

```text
灾害志愿与捐赠管理 T
  -> 改变需求匹配、物资流、志愿者安全、分发效率和欺诈治理变量 X
  -> 改变社区恢复资源可用状态 S
  -> 改变物资浪费、服务延误、二次伤害、诈骗和恢复不平等风险 λ(t)
  -> 影响主体社会支持、恢复速度和未来选择权
```

## 非目标

- 不发布实时物资需求、志愿派遣、捐赠收款、组织背书、现场调度或募款建议。
- 不收集捐赠人、受助者、志愿者身份、位置、支付、库存、路线或安保信息。
- 不替代 FEMA、VOAD、地方应急管理、红十字、非营利组织或官方捐赠平台。

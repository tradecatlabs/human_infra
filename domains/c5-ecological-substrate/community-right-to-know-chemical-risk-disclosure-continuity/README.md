`community-right-to-know-chemical-risk-disclosure-continuity/` 研究社区化学风险知情权、设施报告、排放清单、应急规划和风险沟通如何让居民、学校、工人和地方政府提前看见潜在事故边界。

> 核心问题：危险设施如果只在事故后才被看见，社区就只能被动承受暴露；知情权把隐蔽风险转化为规划、监督、信任和准备能力。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体长期存在依赖对邻近风险的可理解性
  -> 化学设施、储存和排放可能把地点变成隐性风险函数
  -> 社区知情权与披露系统把不可见风险转化为可审查、可沟通、可准备的公共信息
```

## 关注对象

- EPCRA、Tier II、TRI、地方应急规划委员会、化学设施披露、公众沟通和环境正义。
- 信息可达性、语言可达、地图化误读、企业/社区信任、低收入社区和学校邻近风险。
- 与 `chemical-safety-poison-control-toxicology/` 的关系：毒理域关注暴露和健康风险；本域关注设施级风险信息能否被公众和机构理解使用。

## Human Infra 模型链路

```text
化学风险披露
  -> 改变社区风险可见性、准备能力、监督能力和应急规划质量
  -> 改变事故前脆弱性、事故中行动质量和事故后问责恢复能力
  -> 降低不可见暴露、信任破裂和制度性风险转嫁
```

## 非目标

- 不提供设施攻击、规避报告、敏感位置滥用、企业侦察、抗议战术或法律建议。
- 不输出个体健康判断、房产建议、投资建议、保险判断或实时事故判断。
- 不把公开风险资料转化为骚扰、污名化或社区恐慌工具。

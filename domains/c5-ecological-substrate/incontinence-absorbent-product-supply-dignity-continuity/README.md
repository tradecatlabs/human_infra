# Incontinence Absorbent Product Supply Dignity Continuity

`incontinence-absorbent-product-supply-dignity-continuity` 研究成人尿失禁/便失禁吸收用品、尺寸适配、皮肤保护、费用、配送、处置、照护负担和尊严如何影响外出、睡眠、照护、工作和社会参与连续性。

> 核心问题：失禁用品不是单一皮肤护理变量；它是排泄控制失败后维持主体移动、睡眠、社交、照护和尊严的补偿基础设施。

## 关注对象

- 成人纸尿裤、拉拉裤、护垫、床垫、处置袋、清洁用品、尺寸/吸收量适配和备用库存。
- 费用、保险/福利、配送、公共厕所可达、垃圾处置、家庭照护、机构照护和夜间更换。
- 泄漏、气味、羞耻、皮肤潮湿、睡眠中断、外出回避和照护者负担。
- 与泌尿、失禁相关皮炎、DME、长期照护、公共厕所、住房和福利系统的接口。

## Human Infra 链路

```text
失禁吸收用品连续
  -> 主体在尿控或肠控不稳定时仍能维持清洁、睡眠、外出和尊严
  -> 尺寸适配、供应、费用、处置和照护断点造成的回避与皮肤风险下降
  -> 照护负担、社交退出、夜间恢复损耗和服务进入摩擦减少
  -> 有效时间、关系连续和未来选择权增强
```

## 非目标

- 不提供失禁诊断、盆底训练、药物、导尿、皮肤处理、产品品牌、医保个案或照护操作建议。
- 不替代泌尿科、妇科、长期照护、DME 供应商、NIDDK、Medicare/Medicaid 或临床指南。
- 不把失禁记录、用品使用或照护依赖作为羞辱、监控、资格排除或服务惩罚依据。

`weather-climate-observation-forecasting/` 研究天气观测、气候服务、预报模型、极端天气监测和风险信息如何支撑主体持续性。

> 核心问题：主体能否在热、冷、暴雨、洪水、风暴、火灾天气、空气和长期气候风险中提前获得可靠观测与预测。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须在外部环境变化中维持安全、健康和服务连续性
  -> 天气和气候风险会改变死亡风险、居住安全、交通、能源、食物、水、医疗和应急响应
  -> 观测、预报、气候服务和早期预警构成风险函数的外部输入
  -> 如果观测和预报失败，主体会在热、洪水、风暴、火灾天气和空气风险中失去提前行动窗口
```

## 关注对象

- 地面、海洋、卫星、雷达、探空、气候观测和数据同化。
- 天气预报、气候服务、极端事件监测、季节预测、影响预报和早期预警。
- 热、冷、暴雨、洪水、风暴、干旱、野火天气、空气质量和气候趋势对主体持续性的影响。
- 与 `emergency-alerts-communications/` 的边界：本域关注风险观测和预测生成；预警通信域关注信息如何到达人。
- 与 `planetary-health-environment/` 的边界：星球健康域关注环境暴露和生态底座；本域关注天气气候观测、模型和预报产品。

## Human Infra 模型链路

```text
观测系统稳定
  -> 天气和气候状态可被测量
  -> 预报模型和气候服务可生成风险窗口
  -> 热、洪水、风暴、干旱、火灾天气和空气风险可提前进入决策
  -> 家庭准备、公共预警、能源调度、交通、医疗和供应链有更长响应时间
  -> 风险暴露、资源中断和恢复时间下降
  -> 主体持续性增强
```

## 非目标

- 不提供实时灾害指挥、撤离路线、出行决策、急救或个人安全承诺。
- 不把天气、气候、模型输出或长期情景写成确定预测。
- 不提供保险、交易、农业投机、灾害套利或规避官方预警的建议。
- 不替代国家气象机构、地方应急部门、官方预警或专业气候风险评估。

## Source Signals

- WMO Early Warnings for All：全球早期预警能力建设。
- NOAA National Weather Service：天气预报、警报和 Weather-Ready Nation。
- NOAA Climate.gov：气候资料、风险解释和教育资源。
- WMO Global Observing System：全球气象观测基础设施。
- Copernicus Climate Change Service / ECMWF：气候数据和再分析服务。

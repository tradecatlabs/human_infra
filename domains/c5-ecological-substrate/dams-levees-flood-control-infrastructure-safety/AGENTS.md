# Dams Levees Flood Control Infrastructure Safety

本目录承载水坝、堤防和防洪基础设施安全研究，关注工程资产如何压低洪水尾部风险。

## 目录结构

```text
dams-levees-flood-control-infrastructure-safety/
├── AGENTS.md
└── README.md
```

## 职责边界

- 研究水坝、堤防、防洪墙、闸门、泵站、排涝系统、检查、维护、失效风险和公共治理。
- 上游连接水文和气候风险；下游连接住房、医院、道路、能源、预警和灾后恢复。
- 不提供工程设计、检修、操作、加固、实时调度、撤离路线、溃坝模拟、敏感设施或破坏信息。

## 维护规则

- 防洪资产资料必须区分公开教育材料、官方安全信息、工程标准和具体设施数据。
- 不记录敏感设施弱点、操作参数、精细位置或可用于破坏/规避的信息。
- 涉及实时洪水和设施安全时，只指向官方机构，不做个案判断。

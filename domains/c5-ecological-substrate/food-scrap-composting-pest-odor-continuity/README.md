`food-scrap-composting-pest-odor-continuity` 研究厨余、食物残渣、堆肥、收集桶、气味、害虫、甲烷减排、庭院/社区堆肥和多户住宅厨余项目如何影响家庭卫生、资源循环和邻里可接受性。

> 核心问题：厨余可以是废弃物，也可以是资源；但如果收集、暂存和堆肥条件失控，它会迅速变成气味、害虫、渗漏和邻里冲突风险。

## 关注对象

- 厨余、食物残渣、庭院废弃物、台面桶、户外桶、社区堆肥、路缘厨余收集和多户住宅厨余项目。
- 气味、害虫、渗漏、温度、容器容量、收运频率、可接受材料、污染混入和邻里接受度。
- 高温天气、无院落家庭、公寓、儿童家庭、园艺家庭、食物浪费减少和垃圾桶容量压力。
- 与家庭清洁、害虫控制、食品储存、固废收运和公共卫生的接口。

## Human Infra 链路

```text
厨余堆肥、害虫与气味连续
  -> 食物残渣被分流到可控的收集、暂存、堆肥或市政路径
  -> 气味、害虫、渗漏、污染混入和邻里摩擦下降
  -> 厨房、庭院、垃圾桶容量和食物管理任务保持稳定
  -> 主体家庭卫生、资源循环和恢复空间可用性提高
```

## 非目标

- 不提供堆肥配方、温度管理、虫害处理、病原风险判断、动物产品处理、土壤施用建议、商业堆肥运营或个案园艺建议。
- 不替代地方厨余收运规则、公共卫生、农业推广、园艺专家、物业、害虫控制或食品安全建议。
- 不把“堆肥”视为所有家庭都应执行的必选路径；本域关注厨余从家庭任务系统中可控流出或转化。

### Human Infra 追问

这个域如何承载主体展开未来的外部世界条件，例如能源、水、食物、住房、通信、治理、经济和基础设施？

```text
研究域对象
  -> 影响变量 / 中间机制
  -> 改变主体状态或外部条件
  -> 改变风险、能力、时间成本或可达性
  -> 改变有效寿命、有效时间或未来选择权
```

## 研究推进骨架

### 最小问题集

- 界定外部承载条件：能源、水、食物、住房、通信、交通、治理、经济或公共基础设施。
- 说明它如何改变主体展开未来所需的稳定性、冗余、可获得性和安全边界。
- 记录基础设施依赖：供应链、地理环境、制度、资金、维护能力和公共服务。
- 识别冲击场景：灾害、战争、断供、污染、气候、系统失灵或社会支持崩溃。
- 建立韧性指标：可用率、恢复时间、冗余、成本、覆盖范围、可负担性和风险暴露。

### 变量接口

- 输入变量：本域直接处理的对象、资源、技术、环境、制度或状态。
- 中间机制：变量通过什么因果路径改变主体状态、系统状态或外部条件。
- 状态改变：身体、认知、能力、资源、风险暴露、可及性、时间成本或协作条件如何变化。
- 风险 / 成本函数：死亡风险、失能风险、工程风险、尾部风险、机会成本、维护成本或治理成本如何变化。
- 输出指标：有效寿命、健康寿命、有效时间、主观时间、相对时间、行动能力、恢复能力或未来选择权。

### 证据入口

- 官方 / 原始资料：监管文件、数据库、临床登记、标准、论文原文、项目白皮书或一手报告。
- 综述 / 指南 / 标准：系统综述、领域指南、技术标准、伦理规范和权威机构材料。
- 数据集 / 登记系统：可复核的数据表、代码、实验记录、登记号、版本和采集边界。
- 反例 / 失败案例：负结果、副作用、安全事故、不可复现结果、伦理争议和误用案例。

### 最小产出

- Source Signals：记录候选资料、来源、日期、主张和待核验点。
- Source Cards：提取 claim、变量、机制、证据类型、边界、反证条件和可迁移位置。
- Claim-Evidence Matrix：把每个稳定主张绑定到来源、证据等级、适用范围和不确定性。
- 变量表：列出输入变量、中间变量、状态变量、风险变量、输出指标和可观测代理指标。
- 上下游关系：说明本域依赖哪些更根本域，并向哪些转化、应用或基础设施域输出。

美国属地、岛屿与离岸地区健康基础设施可达连续性域研究：当医疗、药品、电力、港口、航空、宽带、灾害响应、保险和财政规则受到岛屿地理与制度边界制约时，主体如何保持照护和恢复路径。

## 对象

- U.S. territories and island communities health infrastructure, public health, Medicaid/CHIP, supply chain and disaster response。
- 医院容量、专科转诊、药品供应、实验室、血液供应、EMS、医疗运输、港口/航空物流和远程医疗。
- 电力、水、宽带、冷链、燃料、灾害恢复、气候风险和公共卫生应急能力。
- 财政规则、保险覆盖、转诊出岛、语言文化、主权/自治地位和联邦项目接口。
- 长期慢病、母婴儿童、老年照护、心理健康、灾后恢复和服务中断风险。

## Human Infra 价值

岛屿和属地不是“远一点的普通地区”。它们把地理隔离、供应链、灾害、财政规则和医疗转诊压缩到同一套脆弱系统里。该域用于解释离岸地理如何把常规健康服务变成高依赖、高延迟、高中断风险的主体持续性问题。

```text
岛屿健康基础设施和转诊供应链被结构化治理
  -> 医疗、药品、能源、水、宽带和灾害恢复更连续
  -> 地理隔离不再把可修复风险放大成系统性失效
  -> 主体持续性、社区韧性和未来选择权提高
```

## 边界

本域只整理公开 U.S. territories、岛屿健康基础设施、公共卫生、Medicaid/CHIP、供应链、灾害和转诊资料。

不提供：

- 个体医疗、保险、转诊、旅行、移居、救灾申请、撤离路线或法律建议。
- 实时灾害响应、港口/航空物流、医疗运输、药品库存或能源调度判断。
- 商业选址、资源套利、政治地位判断、财政政策游说或机构运营建议。
- 把岛屿社区差异写成地区污名、投资线索或风险剥削。

`soil-health-land-degradation-drought-resilience/` 研究土壤健康、土地退化、荒漠化、侵蚀、有机质、盐渍化、旱灾和土地恢复如何支撑主体持续性。

> 核心问题：土壤不是背景材料，而是食物、水循环、碳循环、生物多样性和灾害缓冲的基础自然资本；土壤退化会把长期主体维护成本转化为饥饿、迁移、贫困和生态风险。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体依赖长期稳定的食物、水和栖居环境
  -> 食物、水和环境调节依赖土壤与土地系统维持功能
  -> 土地退化、侵蚀、盐渍化、污染和旱灾会削弱生产、储水、过滤和生态缓冲
  -> 土壤系统失效会通过粮食、贫困、迁移、冲突和生态风险影响主体持续性
```

## 关注对象

- 土壤有机质、土壤结构、侵蚀、压实、盐渍化、污染、土地退化和土地恢复。
- 荒漠化、旱灾风险、土地生产力、生态恢复、农牧系统和水资源接口。
- 与 `agricultural-production-food-system-resilience/` 的边界：农业域关注生产系统；本域关注土地与土壤作为底层自然资本。
- 与 `water-resources-hydrology-flood-drought-management/` 的边界：水资源域关注流域、水文和洪旱管理；本域关注土壤蓄水、土地退化和旱灾脆弱性。

## Human Infra 模型链路

```text
土壤健康和土地功能维持
  -> 作物生产、植被、蓄水、过滤、碳储存和生态缓冲能力增强
  -> 旱灾、侵蚀、产量波动、贫困和迁移压力下降
  -> 食物、水、收入、居住和社区稳定性提高
  -> 主体持续性获得长期自然资本支撑
```

## 非目标

- 不提供具体农田管理、施肥、灌溉、土地交易、碳信用、补贴申请或项目合规建议。
- 不提供农药、化肥、土壤修复工程或生态补偿的具体操作方案。
- 不把遥感、土壤指标或旱灾指数写成单地块诊断。

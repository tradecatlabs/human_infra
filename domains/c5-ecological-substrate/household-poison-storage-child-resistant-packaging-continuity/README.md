# Household Poison Storage Child Resistant Packaging Continuity

`household-poison-storage-child-resistant-packaging-continuity/` 研究家庭药品、清洁剂、农药、纽扣电池、化学品、儿童防护包装、储存、标签理解和毒物中心入口如何影响中毒风险。

> 核心问题：家庭中毒常来自日常物品。真正的连续性问题是危险物质能否被识别、隔离、标记、包装并在误暴露时接上求助入口。

## 先验位置

```text
主体需要药品、清洁剂和家庭化学品
  -> 这些物质同时具备功能价值和暴露风险
  -> 儿童、认知受限者、语言障碍和灾害混乱会降低识别能力
  -> 因而毒物储存与儿童防护包装是家庭安全执行层
```

## 关注对象

- 药品、清洁剂、农药、燃料、纽扣电池、洗衣凝珠、儿童防护包装、标签、储存位置、混装误用和毒物中心求助入口。

## Human Infra 模型链路

```text
家庭毒物储存与包装 T
  -> 改变可接触性、包装、标签、储存、求助入口和认知负担 X
  -> 改变儿童探索、家庭清洁、用药管理和误暴露状态 S
  -> 改变中毒、烧伤、误服、急诊、恐慌和家庭恢复成本 λ(t)
  -> 改变家庭安全、照护负担和未来选择权
```

## 非目标

- 不提供中毒处理、催吐、解毒、剂量判断、危险混合、清洁剂配方、包装规避、产品推荐、法律或个体安全建议。
- 不收集儿童资料、家庭化学品清单、药品清单、地址、事故证据或健康资料。

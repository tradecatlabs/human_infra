# home-service-provider-emergency-contact-directory-continuity 目录说明

本目录承载家庭服务联系人和应急联系目录研究域，关注家庭服务网络如何从个人记忆变成可发起恢复的资料接口。

## 目录结构

```text
home-service-provider-emergency-contact-directory-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义服务联系人目录、责任入口、恢复发起和主体连续性的关系。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 Ready.gov、FTC、Poison Help、地方公用事业、物业/住房和消费者保护资料。
- 不写服务商推荐、价格判断、维修方案、合同策略、投诉策略、法律建议或个案应急决策。
- 新增指标必须注明口径：联系人类型、责任范围、更新周期、授权边界、紧急/非紧急区分和连续性后果。
- 不保存真实联系人、地址、账户、门禁、家庭成员或医疗设备供应商资料。

# Digital Inclusion Connectivity

`digital-inclusion-connectivity/` 研究互联网连接、设备、数字技能、可负担性和数字公共服务可达性能否让主体进入现代资源系统。

> 核心问题：当医疗、福利、教育、工作、身份、金融和应急信息迁移到数字入口时，连接不足会变成新的制度排除。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体越来越依赖数字入口获取服务、信息、学习、工作、医疗和公共资源
  -> 宽带、设备、可负担性、数字技能、可访问设计和人工替代通道必须存在
  -> 数字化才不会把主体从服务、机会和权利中排除
```

## 关注对象

- 互联网接入、宽带覆盖、设备可及、移动连接、可负担性、数字技能和数字素养。
- 数字公共服务、无障碍、语言可达、人工帮助、公共图书馆/社区接入点和数字排除。
- 低收入、老年人、残障者、农村地区、迁移人口、无家可归者和服务弱势群体的数字断点。
- 与数字身份安全、健康素养、社会保护、司法可及、选举接入和医疗可及的接口。

## Human Infra 模型链路

```text
数字包容与连接
  -> 主体获得连接、设备、技能和可访问数字入口
  -> 医疗、福利、教育、工作、金融、身份和应急信息更可达
  -> 行政负担、信息断裂、服务排除和机会损失下降
  -> 主体持续性获得数字社会中的基础接入层
```

## 非目标

- 不提供黑客攻击、账号绕过、盗版、破解、支付规避、绕过平台规则或电信账户操控方法。
- 不提供个体补贴申请、设备购买、运营商选择、法律、金融或公共服务申请建议。
- 不把“全数字化”写成取消人工帮助、线下入口、无障碍和语言可达的理由。

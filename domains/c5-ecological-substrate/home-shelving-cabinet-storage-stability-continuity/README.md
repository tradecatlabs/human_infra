`home-shelving-cabinet-storage-stability-continuity` 研究货架、橱柜、衣柜、车库储物、重物高处存放、落物、门/抽屉访问和存取频率如何影响家庭任务、安全和物品访问连续性。

> 核心问题：家庭储物不是静态背景，而是主体执行任务时反复打开、搬取、攀爬和回收的物品系统；储物失稳会把取物、清洁、照护和维修变成落物、压伤和重新开始成本。

## 关注对象

- 厨柜、浴室柜、衣柜、书架、开放式货架、车库储物架、储藏间、抽屉、重物高处存放和易碎/危险物存放。
- 落物、倾倒、抽屉卡滞、过载、找不到物品、重复购买、儿童攀爬、老人取物和季节性收纳。
- 与家具/电视倾倒、家庭毒物储存、家庭危废、工具维护、照护耗材和紧急物资的接口。
- 储物稳定如何影响安全、时间成本、认知负荷、清洁、备灾和家庭协作。

## Human Infra 链路

```text
家庭储物结构稳定且可访问
  -> 重物落下、倾倒、找物失败和取物绕行减少
  -> 清洁、照护、维修、备灾和日常补给任务更可执行
  -> 时间浪费、伤害窗口、重复购买和家庭协作摩擦下降
  -> 主体资源利用、行动效率和居住执行连续性提高
```

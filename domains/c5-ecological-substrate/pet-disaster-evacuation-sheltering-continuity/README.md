Pet Disaster Evacuation Sheltering Continuity 研究宠物和服务动物如何影响灾害准备、撤离、避难、家庭团聚和恢复连续性。

> 核心问题：如果灾害计划不承认宠物，很多家庭会延迟撤离、拒绝避难或在恢复期失去关键关系支持。

## 研究对象

- 宠物灾害准备、撤离运输、避难接纳、宠物物资、疫苗/身份记录、服务动物优先边界和家庭团聚。
- 宠物限制如何影响撤离决策、庇护所选择、交通、住房恢复、儿童压力和社会支持。
- 灾害中动物福利、公共卫生、场所规则、隐私、失主重聚和服务动物权利边界。

## 先验链路

```text
灾害连续性需要家庭真实组成被计划承认
  -> 很多家庭把宠物和服务动物视为不可分割的照护对象或辅助接口
  -> 若撤离、交通、避难和恢复系统不能接纳动物，主体可能延迟行动或失去支持
  -> 动物走失、收容、疾病和物资断供会延长家庭恢复时间
  -> 因而宠物灾害撤离与避难是家庭应急准备的独立连续性域
```

## 关键变量

- 宠物应急包、身份资料、运输笼、食物药品、疫苗记录、避难政策、服务动物规则和重聚机制。
- 撤离延迟、庇护所可达、低收入家庭、老年人、儿童、残障主体和多宠家庭压力。
- 把灾害资料写成实时撤离路线、现场指挥、设施推荐或个案安全承诺的风险。

## 证据入口

- Ready.gov Pets。
- FEMA pet preparedness and sheltering materials。
- CDC emergency planning for pets。
- ADA.gov service animal emergency access context。

## 非目标

- 本域不提供实时撤离路线、避难点推荐、现场救援、宠物运输操作、医疗急救、服务动物纠纷处理或个案灾害计划。
- 不判断某个避难所是否接纳、不替代地方应急管理、动物控制、兽医或残障权利机构。
- 不收集家庭地址、撤离路线、动物照片、微芯片号、残障诊断、避难位置或实时灾害状态。

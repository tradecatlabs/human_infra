`green-blue-infrastructure-urban-nature-cooling/` 研究城市绿地、树冠、雨水花园、湿地、开放水体、海绵城市、自然基解决方案、热岛缓解和城市冷却如何支撑主体持续性。

> 核心问题：城市自然不是装饰，它决定热暴露、雨洪、空气、心理恢复、活动空间、社会连接和生态缓冲。绿蓝基础设施失败会让城市把每一天都变成更热、更硬、更孤立和更脆弱的环境。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体依赖城市环境能够降温、蓄水、过滤、恢复心理压力并提供安全活动空间
  -> 绿地、树冠、湿地、开放水体和自然基解决方案提供这些调节服务
  -> 热岛、硬化地表、雨洪积水、空气污染、生态缺失和空间不平等会改变健康和有效时间
  -> 城市绿蓝基础设施不足会在个体健康管理之前持续提高环境负担
```

## 关注对象

- 城市树冠、绿地、公园、雨水花园、透水铺装、湿地、开放水体、蓝绿廊道、海绵城市和自然基解决方案。
- 热岛缓解、雨洪管理、空气过滤、心理恢复、活动空间、健康公平和维护治理。
- 与 `biodiversity-ecosystem-services-resilience/` 的边界：生态服务域关注自然系统总体服务；本域关注城市内的绿蓝基础设施和人体暴露界面。
- 与 `urban-planning-land-use-zoning-resilience/` 的边界：城市规划域关注空间治理；本域关注城市自然资产如何具体改变热、雨洪、空气和恢复环境。

## Human Infra 模型链路

```text
城市绿蓝基础设施建设维护
  -> 树冠、绿地、水体、湿地和雨洪自然基设施可被规划、监测和维护
  -> 热暴露、内涝、空气暴露、心理压力、活动空间不足和环境不平等下降
  -> 恢复能力、日常行动、社会连接和灾害缓冲提高
  -> 主体获得更低环境负担和更高恢复密度的城市生活底座
```

## 非目标

- 不提供工程设计、植物处方、施工、排水改造、地产增值、碳信用、项目审批或采购建议。
- 不提供规避排水、环保、园林、市政、物业或公共空间管理要求的操作方法。
- 不替代城市规划、公共工程、园林、市政、环保、卫生或地方政府的官方信息。

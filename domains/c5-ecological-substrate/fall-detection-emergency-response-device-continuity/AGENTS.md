# fall-detection-emergency-response-device-continuity 目录说明

本目录承载跌倒检测、个人紧急响应、传感器、通知路由和求助响应连续性研究域，关注跌倒后发现和救助链如何支撑独立生活。

## 目录结构

```text
fall-detection-emergency-response-device-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义跌倒检测、设备供电连接、通知和响应链如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 CDC、NIA、FDA、EMS、跌倒预防、家庭安全和长期照护资料。
- 不写设备选购、安装、急救、监控、机构采购、医学分诊、911 策略或个体风险判断。
- 新增指标必须注明口径：检测可用性、佩戴率、供电连接、误报漏报、通知路由、响应延迟或无人发现后果。
- 涉及隐私、认知衰退、照护者和长期照护机构时，必须保留同意、尊严和监控边界。

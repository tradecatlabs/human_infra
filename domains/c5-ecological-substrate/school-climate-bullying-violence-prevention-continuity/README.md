`school-climate-bullying-violence-prevention-continuity` 研究学校气候、归属感、欺凌、骚扰、暴力预防、报告机制、安全计划和民权边界如何影响学生安全、出勤、心理恢复和学习连续性。

> 核心问题：学校环境不是学习的背景板，而是决定主体是否敢到场、能专注、可求助和可持续发展的安全基础设施。

## 关注对象

- 学校气候、归属感、信任、成人支持、同伴关系、报告渠道和安全感。
- 欺凌、网络欺凌与学校接口、骚扰、歧视、威胁、暴力、报复和旁观者机制。
- 预防项目、事件响应、家校沟通、民权保护、纪律边界和创伤知情恢复。
- 学校气候与出勤、心理健康、纪律、儿童保护、在线安全、人身安全和语言可达的交叉。

## Human Infra 链路

```text
学校气候与欺凌暴力预防连续
  -> 学校成为可到场、可求助、可归属和可恢复的环境
  -> 欺凌、骚扰、暴力、恐惧缺勤和慢性压力下降
  -> 注意力、学习参与、社会信任和心理恢复能力增强
  -> 儿童主体持续性、关系能力和未来选择权更稳定
```

## 非目标

- 不提供个人欺凌投诉、威胁判断、校园安全策略、纪律申诉、民权投诉、网络取证、心理危机或法律建议。
- 不替代学校、家长、StopBullying.gov、OCR、儿童保护、心理健康、执法、急救或危机服务。
- 不收集学生姓名、事件细节、威胁、定位、聊天记录、心理健康、家庭或纪律资料。

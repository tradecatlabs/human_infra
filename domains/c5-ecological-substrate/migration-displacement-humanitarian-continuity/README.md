# Migration Displacement Humanitarian Continuity

`migration-displacement-humanitarian-continuity/` 研究迁移、难民、流离失所和人道危机中，身份、医疗、教育、庇护、保护和服务能否不断线。

> 核心问题：主体离开原有地点和制度时，连续性最容易断裂；人道连续性的价值在于让移动中的人仍被识别、保护、照护、教育和服务。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体可能因战争、灾害、气候、贫困、迫害或家庭风险离开原有制度
  -> 身份、庇护、医疗、教育、保护、安置和生计支持必须跨地点连续
  -> 迁移和流离失所才不会把主体从公共服务、权利和未来路径中切断
```

## 关注对象

- 难民、寻求庇护者、移民、国内流离失所者、无国籍风险和移动人口服务连续性。
- 身份文件、健康服务、教育连续性、住所、保护、家庭团聚、生计、数据系统和人道协调。
- 跨境、跨辖区、营地/城市、紧急安置、长期流离失所和返回/重建情境。
- 与法律身份、医疗接入、应急响应、住房、食物、水卫生、教育和司法可及的接口。

## Human Infra 模型链路

```text
迁移与人道连续性
  -> 主体在移动、流离失所或跨制度状态下仍能被识别和保护
  -> 医疗、教育、庇护、食物、水、住所、法律救济和生计支持不断线
  -> 身份断裂、服务断裂、家庭断裂和暴露风险下降
  -> 主体持续性获得跨地点、跨制度和危机场景下的连续支撑
```

## 非目标

- 不提供移民、庇护、签证、身份、法律或个案救济建议。
- 不提供边境穿越路线、规避执法、伪造文件、危险旅行、偷渡、藏匿或绕过人道程序的方法。
- 不把人道资料写成替代 UNHCR、IOM、政府机构、法律援助或专业救援组织的操作指南。

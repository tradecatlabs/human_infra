# Caloric Restriction Fasting Energy Balance Governance Continuity

`caloric-restriction-fasting-energy-balance-governance-continuity/` 研究 caloric restriction、time-restricted eating、intermittent fasting、能量平衡、充足营养、体重变化和饮食干预风险如何进入长寿模型。

> 核心问题：能量限制不是“少吃就延寿”。它只有在营养充足、风险可控、终点明确、依从性可维护并且不伤害主体功能时，才可能成为寿命模型变量。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体需要降低代谢、炎症、心血管和衰弱风险
  -> 能量摄入、饮食时间和体重变化会影响代谢状态与分子指标
  -> 过度限制、营养不良、进食障碍、低体重和特殊人群风险会损害主体持续性
  -> 因此 CR / fasting 必须作为能量平衡治理域，而不是普适延寿方案
```

## 关注对象

- CALERIE、caloric restriction、dietary restriction、time-restricted eating、intermittent fasting、energy balance、adequate nutrition 和 weight loss confounding。
- 分子年龄、心代谢风险、免疫炎症、骨量、肌肉、情绪、睡眠、进食障碍风险、孕产儿童风险和长期依从性。
- 与 `nutrition-metabolic-health/` 的边界：营养代谢域关注营养和代谢整体；本域关注能量限制/禁食作为高风险干预的审查。
- 与 `eating-disorders-nutrition-body-continuity/` 的边界：进食障碍域关注病理状态和功能连续性；本域必须主动避免触发或正当化饮食病理。

## Human Infra 模型链路

```text
能量限制 / 禁食 T
  -> 影响能量摄入、体重、胰岛素敏感性、炎症、分子年龄和营养充足性 X
  -> 改变代谢状态 S、心血管风险、肌肉骨骼风险和心理风险
  -> 改变健康寿命、有效时间、恢复能力和未来选择权
  -> 若营养、心理、安全或依从性失败，则收益被风险抵消
```

## 非目标

- 不提供热量目标、禁食窗口、减重计划、饮食处方、补剂搭配或个体建议。
- 不把动物寿命、短期 biomarker、体重下降或社交媒体 fasting 趋势写成人体延寿结论。
- 不面向孕产、儿童青少年、进食障碍、低体重、糖尿病用药或慢病高风险个体提供操作建议。
- 不替代营养师、医生、心理健康和公共卫生专业判断。

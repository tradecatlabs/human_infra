`isolation-quarantine-work-school-continuity/` 研究隔离、检疫、病假、返校返工、照护交接和服务支持如何影响暴发期间的主体持续性。

> 核心问题：隔离不是单纯待在家里；它会重新分配收入、学习、照护、食物、住房和心理负担。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 暴发控制需要降低传播接触
  -> 降低接触会切断工作、学校、照护、收入和服务入口
  -> 隔离/检疫连续性决定传播阻断是否不以主体系统崩塌为代价
```

## 关注对象

- 隔离、检疫、病假、远程学习、远程工作、返校返工规则、照护安排、食物药品支持和心理压力。
- 与 `workplace-accommodation-return-to-work-continuity/` 的关系：本域关注暴发情境下的群体控制与任务连续性，不做个案劳动法律建议。

## Human Infra 模型链路

```text
隔离与检疫执行支持
  -> 改变传播接触、病假可用性、学习工作替代路径和照护负担
  -> 改变感染扩散风险与主体任务中断风险
  -> 降低暴发控制与生活系统断裂之间的冲突
  -> 增强有效时间、收入稳定、学习连续和主体持续性
```

## 非目标

- 不提供个人隔离天数、返工返校 clearance、雇佣法律、学校纪律、旅行或医疗建议。
- 不提供规避隔离、伪造证明、瞒报、逃避检测或公共卫生命令对抗策略。
- 不替代本地卫生部门、学校、雇主、医疗机构或法律专业判断。

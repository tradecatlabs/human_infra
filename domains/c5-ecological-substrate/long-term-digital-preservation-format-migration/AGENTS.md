# long-term-digital-preservation-format-migration 目录说明

本目录维护长期数字保存、格式迁移和档案可读性治理。它是外部记忆和研究证据的长周期保存域。

## 目录结构

```text
long-term-digital-preservation-format-migration/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义格式、元数据、校验、迁移和长期可读性如何影响 Human Infra。
- 本目录不保存个人敏感档案、密钥、账号、未授权论文全文或绕过访问控制材料。
- 资料必须区分个人知识管理、数字遗产权限、机构档案和长期保存工程。

## 上下游关系

- 上游来自数字保存、档案学、元数据、存储系统、密码学和版权治理。
- 下游服务生命日志、数字遗产、研究数据、Source Card、Web 作品和长期项目记忆。
- 不替代 `life-logging-personal-archives-continuity/`；本目录聚焦长期可读性与迁移。

## 维护规则

- 不提供绕过访问控制、破解加密、盗取档案或规避版权的方法。
- 不把数字档案保存写成主体连续性或人格延续证明。
- 新增案例必须标注格式、元数据、校验、迁移计划、权限边界和长期可读性风险。

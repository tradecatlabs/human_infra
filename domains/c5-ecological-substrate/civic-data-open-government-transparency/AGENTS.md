# civic-data-open-government-transparency 目录说明

本目录承载开放数据、政府透明、公共记录、FOIA、数据目录和公共系统可观测性如何支撑主体持续性。

## 结构

```text
civic-data-open-government-transparency/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 是本域的边界、模型链路、非目标和来源入口。
- 本域只讨论公共数据与透明治理，不处理个案 FOIA 法律策略、隐私攻击、目标监控或政治操控。
- 后续新增材料应优先是开放数据 Source Card、数据质量变量、隐私边界和再识别风险。

## 上下游关系

- 上游：`governance-rights/`、`civic-participation-election-access/`、`health-data-privacy-governance/`。
- 下游：`public-service-design-accessibility/`、`social-determinants-community-vulnerability/`、`research-domain-atlas` 文档。
- 相邻域：`information-integrity-trust/`、`community-resource-navigation/`、`measurement-feedback/`。

## 维护规则

- 不提供人肉搜索、隐私攻击、监控目标定位、访问控制规避或敏感数据发布建议。
- 不把开放数据写成可牺牲隐私、同意、最小化和再识别风险的理由。
- 若新增数据目录、字段清单或透明度模型，必须同步维护本文件和相关索引。

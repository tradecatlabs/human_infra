`biodiversity-ecosystem-services-resilience/` 研究生物多样性、生态系统服务、授粉、水土保持、病害调节、热缓冲和生态恢复如何支撑主体持续性。

> 核心问题：主体不是只活在技术系统里，也活在生态系统里；生态系统服务一旦退化，食物、水、空气、疾病风险、热风险和心理恢复都会同时变差。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体依赖食物、水、空气、温度调节、病害控制和恢复环境
  -> 这些服务部分由生态系统和生物多样性提供
  -> 生态系统退化会降低授粉、过滤、碳储存、灾害缓冲和疾病调节能力
  -> 技术系统必须补偿更多自然服务缺口，主体维护成本上升
```

## 关注对象

- 生物多样性、生态系统服务、授粉、湿地、水土保持、城市绿地、自然恢复和生态连通性。
- 生态系统退化对食物、水、空气、热风险、传染病、心理恢复和灾害缓冲的影响。
- 与 `planetary-health-environment/` 的边界：星球健康域关注广义环境暴露；本域关注生态系统服务作为可识别功能。
- 与 `agricultural-production-food-system-resilience/` 的边界：农业域关注生产系统；本域关注生态系统对生产与健康的支撑。

## Human Infra 模型链路

```text
生态系统服务维持
  -> 授粉、过滤、蓄水、降温、碳储存、栖息地和病害调节能力保留
  -> 食物、水、空气、热风险、灾害和恢复环境更稳定
  -> 技术补偿成本、疾病风险和环境压力下降
  -> 主体持续性获得生态底座支撑
```

## 非目标

- 不提供保护地执法、野生动物处理、生态补偿、碳市场、土地开发或项目审批建议。
- 不把生态服务估值写成投资、交易、征地或政策结论。
- 不提供可能伤害物种、栖息地或保护设施的操作信息。

反腐败、公共廉洁与问责域研究：公共权力、采购、许可、执法、福利、医疗、基础设施和资源分配如何避免被私人利益俘获，并通过透明、审计、申诉和问责维持对主体持续性的公共承诺。

## 对象

- 腐败、贿赂、利益冲突、裙带关系、公共资源俘获、采购腐败、服务寻租和监管失灵。
- 公共廉洁、审计、透明、申报、举报人保护、开放数据、独立监督、问责和反腐败制度。
- 腐败如何通过药品质量、采购失败、工程质量、服务排除、福利截留、司法不公和信任崩塌影响主体持续性。
- 公共系统从“名义存在”转化为“真实可用”的诚信条件。
- 反腐败资料与具体举报、调查、法律行动、政治攻击之间的边界。

## Human Infra 价值

Human Infra 的许多底座依赖公共采购、监管、执法和服务递送。腐败会让预算、法律、医疗、救济和基础设施在纸面上存在，却在主体需要时失效。

```text
公共权力被私人利益俘获
  -> 采购、许可、监管、福利、医疗和司法发生失真
  -> 透明、廉洁、审计、申诉和问责降低系统性泄漏
  -> 公共资源更可能转化为主体可用服务和恢复路径
```

## 边界

本域只整理公开的反腐败、公共廉洁、透明、审计和问责资料。

不提供：

- 行贿、洗钱、规避审计、隐藏资产、逃避监管、伪造记录或腐败操作方法。
- 针对具体个人、机构、案件或国家的调查策略、举报策略、法律建议或政治攻击材料。
- 获取、泄露、利用非公开信息的建议。
- 把反腐败资料写成诽谤、骚扰、人肉、舆论动员或派系斗争工具。

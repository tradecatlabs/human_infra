# household-cord-cable-trip-entanglement-continuity 目录说明

本目录承载家庭线缆、绊倒缠绕与连接连续性研究域，关注电源和数据连接如何同时支持任务又制造路径风险。

## 目录结构

```text
household-cord-cable-trip-entanglement-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义家庭线缆、临时供电、通道暴露、绊倒缠绕和任务连续性的关系。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 CPSC、USFA、消防、电气安全、公共卫生和设备安全资料。
- 不写电气施工、负载计算、插排改造、产品推荐、火灾/触电处置、现场验收或规避安全规范。
- 新增指标必须注明口径：线缆类型、路径位置、通行频率、连接任务、供电依赖、儿童/宠物/老人暴露。
- 涉及医疗设备供电时，必须保留医疗设备供应、备电和临床边界。

# Household Cord Cable Trip Entanglement Continuity

`household-cord-cable-trip-entanglement-continuity` 研究充电线、延长线、插排、网线、设备线缆、临时供电和穿越通道的线缆如何影响绊倒、缠绕、供电可靠性和家庭任务连续性。

> 核心问题：线缆是数字生活、照护设备和家庭办公的底层连接，但当它穿越路径、堆积床边或承担长期临时供电时，也会变成绊倒、缠绕、火灾和设备失效的隐形接口。

## 关注对象

- 手机/电脑充电线、延长线、插排、电源适配器、网线、医疗/辅助设备线缆、电视/游戏设备线缆和临时布线。
- 床边、办公桌下、厨房、走道、楼梯口、儿童游戏区、宠物活动区和夜间路径中的线缆暴露。
- 线缆管理如何影响跌倒、儿童/宠物缠绕、设备断电、清洁绕行、火灾边界和家庭协作。
- 与住宅电气安全、家庭办公、人因工效、照护设备供电和个人设备维护的接口。

## Human Infra 链路

```text
家庭线缆路径可治理
  -> 通道绊倒、缠绕、设备断电和过载暴露下降
  -> 充电、联网、照护设备、家庭办公和夜间行动更稳定
  -> 任务中断、伤害窗口、清洁摩擦和设备失效减少
  -> 主体数字连接、行动安全和居家执行连续性提高
```

## 非目标

- 不提供电气维修、布线施工、延长线容量计算、产品推荐、插排改造、火灾处置、触电处理、宠物/儿童监控或现场安全认证。
- 不替代 CPSC、USFA、地方消防、电工、物业、设备厂商、医疗设备供应商或公共安全机构。
- 不把线缆管理写成美观整理教程；只讨论路径、供电、绊倒、缠绕和任务连续性的关系。

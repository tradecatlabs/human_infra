# household-hazardous-waste-chemical-disposal-continuity 目录说明

本目录承载家庭危险废物与化学处置连续性研究域，关注低频高风险家庭材料如何通过合规入口离开居住空间。

## 目录结构

```text
household-hazardous-waste-chemical-disposal-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义家庭危险废物、标签识别、处置入口、暴露/火灾/污染风险和任务连续性的关系。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 EPA、地方 HHW 项目、Poison Help、消防部门和公共卫生资料。
- 不写化学处理、混合、稀释、燃烧、倾倒、排放、危险废物运输、泄漏处置或现场清理步骤。
- 新增指标必须注明口径：材料类别、危害类别、处置可达性、暂存压力、暴露风险、火灾风险或污染后果。
- 涉及急性中毒、泄漏、火灾、非法倾倒或灾后清理时，必须保留应急和专业转介边界。

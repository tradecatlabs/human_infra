`emergency-financial-document-cash-readiness-continuity/` 研究应急文件、现金、账户信息、保险材料、身份凭据和灾后申请资料如何影响主体恢复资源。

> 核心问题：灾害后的恢复常被证件、账号、保单、收据、现金、住房文件和申请材料卡住；资料准备决定损失能否被转化为可处理流程。

## 先验位置

```text
主体要从灾害中恢复
  -> 必须把损失、身份、资产、保险和需求转译成制度可处理材料
  -> 文件、现金和账户信息缺失会让资源恢复延迟或失败
  -> 因而应急金融文件准备是灾后恢复的制度接口
```

## 关注对象

- 身份文件、保单、医疗/药品资料、银行账户、现金、住房文件、资产清单、照片记录、灾后申请和离线备份。
- 低收入家庭、老人、租客、房主、慢病患者、移民家庭、灾害高风险地区和数字排除人群。

## Human Infra 模型链路

```text
应急金融文件准备系统 T
  -> 改变身份凭据、保单、现金、账户、资产记录、医疗资料和离线备份变量 X
  -> 改变主体灾后申请、支付、证明和恢复状态 S
  -> 改变无法入住、无法理赔、无法购药、无法证明身份和服务延迟风险 λ(t)
  -> 改变资源恢复、住房稳定、医疗连续性和未来选择权
```

## 非目标

- 不提供个人财务、现金储备、保险、税务、法律、理赔、文件保存、灾后申请或身份恢复建议。
- 不收集证件、保单、账户、现金、资产、医疗资料、住址、移民身份或家庭成员信息。

# 姓名变更与身份记录同步连续性目录说明

## 目录结构

```text
name-change-identity-record-synchronization-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义姓名变更和身份记录同步如何影响跨机构身份连续性。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只讨论记录同步、身份匹配、证明负担和服务断点，不写个人改名步骤。
- 涉及美国制度时优先使用 USA.gov、SSA、State Department、州级 DMV/ID 官方入口。
- 不提供逃避身份核验、规避责任、隐藏记录或伪造文件的策略。

# Persistent Postural Perceptual Dizziness Continuity

Persistent postural-perceptual dizziness continuity studies chronic dizziness, visual-motion sensitivity, postural discomfort, and functional avoidance as long-duration burdens on work, mobility, social participation, and screen tolerance.

核心问题：

> 有些眩晕不是一次短发作，而是长期把超市、屏幕、移动人群、复杂光流和站立任务变成高成本环境；主体持续性因此被慢性空间不适消耗。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能长期承受日常视觉运动和站立环境
  -> 慢性头晕和空间不适会放大注意力、焦虑、回避和疲劳负担
  -> 主体逐渐减少外出、会议、购物、屏幕和社交任务
  -> 因而 PPPD 类慢性头晕是有效时间和行动半径的持续损耗条件
```

## 关注对象

- 慢性头晕、视觉运动敏感、站立不适、复杂环境回避、焦虑交互和屏幕/商场负担。
- 长期工作、购物、通勤、视频会议、公共空间停留和社交参与。
- 与 `anxiety-stress-threat-regulation-continuity/` 的关系：焦虑域关注威胁调节；本域关注空间感知和慢性头晕负担。
- 与 `dry-eye-ocular-surface-comfort-continuity/` 的关系：干眼域关注眼表；本域关注视觉运动和空间环境耐受。

## Human Infra 模型链路

```text
识别 / 教育 / 康复 / 环境与任务适配 T
  -> 改变视觉运动敏感、站立耐受、回避行为和焦虑交互 X
  -> 改变外出频率、屏幕工作、购物、通勤和社交参与状态 S
  -> 改变失能、孤立、工作中断和有效时间损耗风险 λ(t)
  -> 影响行动密度、恢复机会和未来选择权
```

## 非目标

- 不提供 PPPD 诊断、心理治疗建议、药物建议、暴露训练方案或个人功能许可。
- 不根据个人头晕和焦虑描述判断病因、治疗路径或工作能力。
- 不替代耳鼻喉、神经、心理健康、前庭康复或物理治疗服务。

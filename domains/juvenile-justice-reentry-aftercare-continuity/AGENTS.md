# Juvenile Justice Reentry Aftercare Continuity 目录说明

本目录承载少年司法重返、aftercare、学校复归、家庭社区支持和健康服务接续资料。它与成人重返、拘押状态和犯罪记录救济域相邻，但对象是发展期青年。

## 目录结构

```text
juvenile-justice-reentry-aftercare-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- 记录少年司法 reentry、aftercare、社区监督、学校复学、行为健康、家庭支持和记录保密边界。
- 关注司法接触之后青年能否重新进入发展轨道。
- 不提供个案法律、监管规避、风险评估、学校纪律申诉、记录封存或执法应对建议。

## 维护规则

- 新增资料必须优先引用 OJJDP、OJP、Youth.gov、教育、公共卫生或司法公开资料。
- 不把再犯风险工具或监管模型转化为个人判断或预测。
- 所有材料必须保留未成年人、保密、创伤知情和发展阶段边界。

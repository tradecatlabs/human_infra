# human-factors-ergonomics-task-system-continuity 目录说明

本目录承载人因、工效学和任务系统适配资料。它关注人、工具、环境和流程的匹配，而不是个体医疗、工伤法律或员工监控。

## 结构

```text
human-factors-ergonomics-task-system-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分 physical ergonomics、cognitive ergonomics、human systems integration、usability engineering 和 human-centered design。
- 必须记录适用场景、证据类型、任务边界、残障/无障碍边界、隐私和反监控风险。
- 不保存身体测量、伤病记录、残障资料、员工监控、工伤个案、雇主资料或绩效数据。

# Voice Disorders Phonation Communication Continuity

Voice disorders phonation communication continuity studies dysphonia, vocal fatigue, pitch/loudness limits, and voice loss as constraints on work, teaching, caregiving, identity, and social participation.

核心问题：

> 嗓音不是装饰性声音，而是主体表达、职业角色、亲密关系和公共互动的输出载体；发声系统受损会把语言能力变成低可用能力。

## 先验位置

```text
主体持续性最大化
  -> 主体必须能以可承受成本发出可被识别的声音
  -> 声带、呼吸、共鸣、负荷和环境噪声会影响发声可用性
  -> 教学、客服、照护、会议和社交输出能力下降
  -> 因而嗓音障碍是语言表达到可持续声学输出之间的载体失效模式
```

## 关注对象

- 嗓音嘶哑、失声、声带问题、声负荷、职业用声、性别表达相关声音和交流身份。
- 工作场景、教学、照护、电话/会议沟通、声学环境和恢复负担。
- 与 `occupational-work-design/` 的关系：职业域关注工作组织；本域关注用声能力。
- 与 `dysarthria-speech-intelligibility-continuity/` 的关系：构音域关注清晰度；本域关注发声载体。

## Human Infra 模型链路

```text
识别 / 声负荷管理 / 环境适配 / 沟通替代 T
  -> 改变嗓音质量、响度、疲劳、疼痛和发声可持续性 X
  -> 改变教学、会议、照护、电话和社交状态 S
  -> 改变工作中断、沟通失败、身份压力和恢复不足风险 λ(t)
  -> 影响职业持续性、关系参与和未来选择权
```

## 非目标

- 不提供嗓音障碍诊断、喉镜解释、发声训练、药物、手术、声带病变判断或职业返岗建议。
- 不根据录音、嘶哑时间或声音变化判断癌症、急症或用声许可。
- 不替代耳鼻喉、言语语言治疗、职业健康或急诊服务。

# 债务义务、催收与债务救济连续性目录说明

## 目录结构

```text
debt-obligation-collection-relief-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义债务义务、催收、救济和破产制度如何影响主体资源、时间和未来选择权。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀债务类型、制度机制、主体持续性变量和风险边界，不写个体债务处理步骤。
- 涉及美国制度时优先使用 CFPB、U.S. Courts、FTC、Federal Student Aid 和官方消费者资料。
- 不把本域写成债务、破产、催收、税务、贷款、信用修复、法律或逃避义务建议。

# geroprotective-drug-repurposing-trial-governance-continuity 目录说明

`domains/geroprotective-drug-repurposing-trial-governance-continuity/` 保存抗衰药物再利用、geroprotective candidates、试验治理和证据边界资料。

## 目录结构

```text
geroprotective-drug-repurposing-trial-governance-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义抗衰药物候选从机制、动物证据、人体试验到主体持续性模型的审查边界。
- `AGENTS.md`：记录本目录职责、可新增材料类型和禁止用途。

## 职责边界

- 本域只整理公开研究、试验设计、证据层级、监管边界和安全信号。
- 可以新增 Source Card、trial endpoint map、risk register、off-label hype boundary 和 evidence matrix。
- 不提供药物使用、剂量、购买、处方、联合用药、个体风险解释、试验入组或医疗建议。

## 维护规则

- 新增药物候选必须标注物种、人群、终点、剂量语境、证据等级和安全限制。
- 不把动物寿命延长写成人体延寿结论。
- 新增文件或子目录时同步更新本文件。

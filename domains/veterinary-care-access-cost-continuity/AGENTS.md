# veterinary-care-access-cost-continuity 目录说明

本目录承载兽医照护可及、费用负担和服务连续性资料。它关注兽医服务作为家庭与动物福利基础设施的系统变量，而不是动物诊疗建议。

## 结构

```text
veterinary-care-access-cost-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分兽医服务容量、费用可承受性、动物福利、公共卫生、低成本服务和个体诊疗边界。
- 资料优先回到 AVMA、CDC、FDA、USDA APHIS、公共卫生资料和同行评审研究。
- 不保存宠物病历、兽医账单、诊疗照片、处方、家庭收入、支付信息、地址或个案救助资料。

# personal-health-record-emergency-summary-continuity 目录说明

本目录承载个人健康记录与急救摘要连续性资料。它研究机构病历如何被转化为急救窗口可读、可携带、可更新的最小健康摘要。

## 结构

```text
personal-health-record-emergency-summary-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料必须区分完整病历、患者门户导出、个人健康记录、急救摘要和照护者副本。
- 必须记录来源、字段边界、更新频率、隐私、安全和过期风险。
- 不保存个人健康记录、PHI、急救摘要、医疗设备序列号或联系人资料。

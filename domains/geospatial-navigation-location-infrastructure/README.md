# Geospatial Navigation Location Infrastructure

`geospatial-navigation-location-infrastructure/` 研究空间位置、导航、地址、地图、地理编码和定位服务如何成为主体持续行动的基础设施。

> 核心问题：主体能否被定位、找到资源、抵达服务、完成配送、获得救援，并在复杂环境中维持可导航的行动路径。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 主体必须能够在外部世界中定位自身、资源、风险和服务
  -> 定位、导航、地址、地图、地理编码和空间参考系统构成行动坐标系
  -> 医疗、应急、交通、物流、社区资源、灾害恢复和公共服务都依赖可用位置层
  -> 如果位置层失败，主体即使拥有服务权利和资源信息，也可能无法抵达、被找到或被分配支持
```

## 关注对象

- 地理空间数据、基础地图、地址系统、地名、行政边界和空间参考框架。
- GNSS/GPS、授时、定位、导航、测绘、地理编码、路线服务和位置互操作。
- 医疗交通、应急定位、灾害制图、物流配送、社区资源导航和公共服务可达性中的空间层。
- 与 `transportation-access-mobility/` 的边界：交通域关注移动服务和出行系统；本域关注位置、地图、地址和导航坐标系。
- 与 `emergency-alerts-communications/` 的边界：预警通信关注危机信息到达；本域关注风险、人员、资源和救援目标能否被空间定位。

## Human Infra 模型链路

```text
空间位置层可靠
  -> 人、服务、风险、资源和设施可被共同定位
  -> 医疗、应急、物流、交通和公共服务路径可被计算
  -> 到达成本、搜寻成本、配送成本和响应延迟下降
  -> 主体获得服务、撤离、救援、补给和恢复的概率上升
  -> 行动半径、资源可达性和危机恢复能力增强
  -> 主体持续性和未来选择权提高
```

## 非目标

- 不提供跟踪、监控、骚扰、定位他人、人肉搜索、规避执法或隐私攻击方法。
- 不提供危险路线、实时撤离路线、基础设施攻击定位、安防绕过或目标识别建议。
- 不承诺地图、地址、GPS、路线、风险图或服务点的实时准确性。
- 不替代官方应急、导航、航空、海事、测绘、交通或公共安全系统。

## Source Signals

- UN-GGIM：全球地理空间信息管理和国家地理空间信息基础设施。
- USGS National Geospatial Program：美国国家地理空间底图、地名和地理空间数据。
- GPS.gov：GPS、PNT 和卫星导航的公共资料。
- NOAA National Geodetic Survey：测地基准、坐标框架和空间参考系统。
- Open Geospatial Consortium：地理空间互操作标准。

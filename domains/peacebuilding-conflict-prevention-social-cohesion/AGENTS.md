# peacebuilding-conflict-prevention-social-cohesion 目录说明

本目录维护和平建设、冲突预防与社会凝聚域。它是 Human Infra 的“社会冲突不把主体未来清零”层，负责信任、申诉、调解、制度合法性和社区韧性如何支撑长期生活窗口。

## 目录结构

```text
peacebuilding-conflict-prevention-social-cohesion/
├── AGENTS.md  # 本目录职责和维护规则
└── README.md  # 域定义、对象、边界和资料线索
```

## 职责边界

- `README.md` 说明冲突预防、和平建设、社会凝聚和脆弱性治理如何进入主体持续性模型。
- 本域只做公开资料整理、机制建模和非操作性风险分析。
- 不提供政治动员、舆论操控、抗议组织、镇压、暴力升级、规避执法、心理战或宣传策略。

## 维护规则

- 新增资料必须区分冲突预防框架、和平建设实践、脆弱性研究和具体事件叙事。
- 涉及群体冲突时，必须避免仇恨、污名、煽动和单因归责。
- 不把和平建设资料写成具体政治策略或行动手册。

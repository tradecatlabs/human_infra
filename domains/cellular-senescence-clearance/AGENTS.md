# cellular-senescence-clearance 目录说明

`domains/cellular-senescence-clearance/` 保存细胞衰老与清除研究域。它关注衰老细胞负担、SASP 和组织环境如何影响主体持续性。

## 目录结构

```text
cellular-senescence-clearance/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责细胞衰老、SASP、senolytics、senomorphics、组织炎症和清除风险。
- 本域可整理机制文献、组织特异性、证据等级和安全边界。
- 本域不提供药物、剂量、补剂、人体实验或治疗建议。

## 维护规则

- 任何 senescence 主张必须同时记录潜在收益、组织角色、癌症边界和副作用风险。
- 与免疫、癌症、再生医学、纤维化和细胞重编程交叉时必须标注主拥有域。
- 新增文件或子目录时，同步更新本文件。

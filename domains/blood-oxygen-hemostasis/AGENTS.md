# blood-oxygen-hemostasis 目录说明

`domains/blood-oxygen-hemostasis/` 保存血液氧运与止血研究域。它关注血液如何承担氧气运输、凝血、出血控制和血栓风险。

## 目录结构

```text
blood-oxygen-hemostasis/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责红细胞、血红蛋白、氧气运输、凝血、出血、血栓和血液供应安全。
- 本域可整理公开血液疾病、贫血、凝血和输血安全资料。
- 本域不提供检测解释、输血建议、抗凝/止血用药、剂量、急救或治疗建议。

## 维护规则

- 任何血液系统主张必须说明变量、机制、风险类型、证据等级和跨域边界。
- 涉及出血、血栓、输血或药物时只写高层证据和非操作性安全边界。
- 新增文件或子目录时，同步更新本文件。

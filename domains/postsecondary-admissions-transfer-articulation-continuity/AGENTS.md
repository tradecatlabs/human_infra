# postsecondary-admissions-transfer-articulation-continuity 目录说明

本目录承载高等教育入学、转学和学分衔接连续性研究域，关注学习成果如何被高等教育系统承认、转换、恢复并接入长期能力路径。

## 目录结构

```text
postsecondary-admissions-transfer-articulation-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义高等教育入学、转学、学分转换和衔接协议如何进入 Human Infra 主体持续性模型。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料必须优先回到 NCES/IPEDS、College Scorecard、Federal Student Aid、州高教机构、认证机构和高校公开政策。
- 不写个人选校、申请、录取、转学、学分申诉、签证、贷款或法律操作建议。
- 涉及学生记录时，只讨论制度变量、路径断裂和数据口径，不处理个人申请材料、成绩单或身份资料。
- 新增指标必须注明口径：admissions、enrollment、transfer-in、transfer-out、credit loss、retention、completion 或 time-to-degree。

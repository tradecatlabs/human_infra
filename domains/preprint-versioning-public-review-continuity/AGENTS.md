# preprint-versioning-public-review-continuity 目录说明

本目录负责预印本版本链、公开评论和公共审查连续性。

## 结构

```text
preprint-versioning-public-review-continuity/
├── AGENTS.md
└── README.md
```

## 边界

- 只维护预印本版本、审查状态、公开评论信号和后续发表链接。
- 上游：`domains/technology-foresight-horizon-scanning/`、`domains/persistent-identifier-doi-citation-metadata-continuity/`。
- 下游：Source Card、研究雷达、系统综述和传播材料。
- 不提供预印本操控、刷评论、同行评审伪造、论文代写或平台规避方法。

## 维护规则

使用 ASAPbio、medRxiv、bioRxiv、PubPeer 和 PREreview 作为锚点。预印本只能作为分层证据进入模型，必须记录版本、日期、评论状态和后续发表状态。

# 护照、旅行证件与跨境移动连续性目录说明

## 目录结构

```text
passport-travel-document-mobility-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义护照、旅行证件、领事服务和跨境移动资格如何支撑未来选择权。
- `AGENTS.md`：说明本域维护边界和禁止用途。

## 维护规则

- 只沉淀旅行证件机制、领事服务边界、跨境移动变量和证据入口，不写个体旅行或移民操作流程。
- 涉及美国制度时优先使用 State Department、USA.gov 和官方领事资料；涉及流离失所时使用 UNHCR/IOM 等权威入口。
- 不把本域写成护照、签证、移民、庇护、旅行安全、撤离、国籍、税务或个案法律建议。

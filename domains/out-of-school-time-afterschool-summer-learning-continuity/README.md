# Out Of School Time Afterschool Summer Learning Continuity

`out-of-school-time-afterschool-summer-learning-continuity/` 研究课外、放学后、暑期学习和监督照护如何填补学校日历之外的时间断点，保护儿童安全、学习轮次和照护者工作连续性。

> 核心问题：学校上课时间之外并不是空白时间；放学后和暑期断点会同时影响儿童安全、学习损失、餐食、社交支持和照护者工作安排。

## 先验位置

```text
有效永生 / 主体持续性最大化
  -> 儿童日常运行跨越上学、放学后、假期和家庭时间
  -> 学校时间之外需要安全监督、学习延续和恢复性活动
  -> 课外/暑期项目必须连接儿童发展和照护者工作节律
  -> 儿童学习轮次和家庭有效时间才不会被日历断点吞噬
```

## 关注对象

- 放学后项目、暑期学习、21st Century Community Learning Centers、监督照护、学习损失和安全时段。
- 餐食、交通、活动质量、出勤稳定性、家庭排班和社区项目承接。
- 与 `school-enrollment-attendance-continuity/`、`school-meals-nutrition-access-continuity/`、`school-transportation-safe-routes-continuity/`、`childcare-family-continuity/` 和 `learning-skill-acquisition/` 的接口。

## Human Infra 模型链路

```text
学校日历外时间断点
  -> 课外监督、暑期学习、餐食、交通和社区活动
  -> 儿童安全、学习连续和照护者工作安排改善
  -> 学习损失、无监督风险和家庭压力下降
  -> 儿童能力积累与家庭持续性增强
```

## 非目标

- 不推荐具体课外班、夏令营、补习、活动、接送路线或学习计划。
- 不判断儿童学业、行为、资格、项目质量、出勤安排或学校责任。
- 不收集儿童身份、成绩、出勤、位置、照片、项目申请、家庭排班或交通资料。

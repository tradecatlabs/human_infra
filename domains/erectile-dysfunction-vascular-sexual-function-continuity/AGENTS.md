# Erectile Dysfunction Vascular Sexual Function Continuity 目录说明

本目录承载 ED 作为血管代谢、性功能、心理关系和尊严接口的资料。它处理风险信号和主体连续性，不处理个体性功能治疗。

## 目录结构

```text
erectile-dysfunction-vascular-sexual-function-continuity/
├── AGENTS.md
└── README.md
```

## 维护规则

- 新增资料优先引用 NIDDK、MedlinePlus、专业指南和系统综述。
- 记录 vascular/metabolic signal、medication context、psychological load、relationship burden、privacy、stigma 和 misinformation risk。
- 不写 ED 诊断、处方、补剂、器械、激素、性行为建议、心血管风险判读、伴侣建议或商业推荐。

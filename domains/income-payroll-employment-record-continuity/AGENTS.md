# income-payroll-employment-record-continuity

本目录承载工资、工时、雇佣记录与收入证明连续性资料。它研究主体如何把劳动事实转化为可被制度承认的收入、权益和未来保障证明。

## 目录结构

```text
income-payroll-employment-record-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md` 说明本域对象、Human Infra 价值、边界、上下游和初始资料线索。
- 后续资料应区分工资工时记录、薪资单、扣缴记录、雇佣核验、承包/雇佣分类、收入证明和纠错机制。
- 本域不保存个人工资、税务、雇佣或身份资料，不提供工资索赔、税务、法律、贷款、租房、福利申请或记录伪造建议。

## 维护规则

- 引用资料优先来自 DOL、IRS、SSA、官方劳动监管机构、统计机构和公开工资记录研究。
- 必须保留隐私、纠错、数据可携带、雇佣分类误差、记录不完整、平台锁定和反伪造边界。
- 不把收入证明写成万能准入凭证，也不把工资记录连续性简化为个人整理文件问题。

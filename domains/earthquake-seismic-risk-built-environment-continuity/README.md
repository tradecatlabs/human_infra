# Earthquake and Seismic Risk Built-Environment Continuity

## 核心问题

地震风险决定建筑、道路、桥梁、医院、水电燃气、通信和社区空间是否会在数秒到数分钟内转化为死亡、伤害、失能和长期服务断点。

Human Infra 关注的问题不是地震预测，而是：

```text
地震如何通过建成环境和生命线基础设施改变主体的生存、移动、照护和恢复条件？
```

## 先验位置

主体持续性预设住所、医院、道路、水电、通信和公共设施不在短时间内系统性坍塌。地震把“环境是否可靠”压缩成急性测试，并暴露建成环境、标准、维护和应急响应的长期质量。

## 关注对象

- 地震危险性、地震动、断层、液化、余震和震后风险。
- 建筑抗震、医院韧性、桥梁道路、管网、燃气、电力和通信。
- 预警、疏散、搜救、临时安置、恢复和长期重建。

## Human Infra 模型链路

```text
地震危险性
  -> 建筑和生命线基础设施失效
  -> 死亡、创伤、服务中断、住房丧失和恢复负担上升
  -> 主体存在、行动、照护、资源和社会连接受损
  -> 主体持续性被急性地质风险截断
```

## 非目标

- 不做地震预测、建筑安全评估、疏散命令、保险判断或工程设计建议。
- 不替代 USGS、建筑规范、结构工程、应急管理或地方政府信息。
- 不把地震风险简化为个人准备问题。

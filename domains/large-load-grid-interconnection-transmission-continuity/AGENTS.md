# large-load-grid-interconnection-transmission-continuity 目录说明

本目录维护大型负载并网与输电连续性资料。它关注 AI、制造和关键服务扩张前的电力接入瓶颈。

## 目录结构

```text
large-load-grid-interconnection-transmission-continuity/
├── AGENTS.md
└── README.md
```

## 职责边界

- `README.md`：定义并网队列、输电规划、大型负载和电网升级如何影响主体持续性。
- 本目录不保存项目申请材料、选址建议、投融资判断、能源采购合同或规避监管方法。

## 上下游关系

- 上游来自 DOE i2X、LBNL queue、FERC、NERC 和公开电网规划资料。
- 下游服务数据中心、制造、医疗设施、冷却、交通电气化和社区韧性。
- 不替代 `electric-grid-reliability-resilience-continuity/`；本目录关注新增负载接入。

## 维护规则

- 必须区分运行可靠性、接入容量、规划延迟和成本分摊。
- 不把单个项目或区域数据推广成普遍结论。

# resource-social-infra 目录说明

`domains/resource-social-infra/` 保存资源与社会基础设施研究域。它关注主体持续运行所需的外部生活、服务、照护和资源底座。

## 目录结构

```text
resource-social-infra/
├── AGENTS.md
└── README.md
```

## 职责边界

- 本域负责食物、水、能源、住房、医疗可及、照护、金融、教育、就业和社区支持。
- 本域可引用 `docs/reference/applications-and-literature.md`，但不替代该跨域来源索引。
- 本域不提供违法、欺诈、规避制度或个体申请操作建议。

## 维护规则

- 任何基础设施主张必须说明它支撑哪些主体变量：存在、恢复、行动、学习、协作或未来选择权。
- 不把制度问题归结为个人选择；必须保留资源、环境和制度维度。
- 新增文件或子目录时，同步更新本文件。

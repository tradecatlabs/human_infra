# Sports Concussion Return To Play Protocol Continuity

`sports-concussion-return-to-play-protocol-continuity/` 研究运动脑震荡识别、移出活动、返学返赛流程、教练/学校执行和长期头部伤害负担。

> 核心问题：脑震荡不是“忍一下继续比赛”，而是学习、判断、身份、团队压力和脑部恢复窗口之间的系统问题。

## 先验位置

```text
主体要持续学习和行动
  -> 必须保护脑功能、注意力、记忆和情绪调节
  -> 运动冲击可造成短期和累积性脑功能扰动
  -> 因而返学返赛协议是儿童和运动者主体连续性的治理接口
```

## 关注对象

- 学校运动、青少年运动、教练、家长、运动员、返学、返赛、症状报告、团队压力和头部冲击文化。

## Human Infra 模型链路

```text
运动脑震荡协议 T
  -> 改变识别、移出、医学评估转介、返学返赛、压力和复发暴露变量 X
  -> 改变脑恢复、学习参与和运动参与状态 S
  -> 改变重复脑震荡、长期症状、学习中断和情绪负担风险 λ(t)
  -> 改变认知持续性、教育路径、运动参与和未来选择权
```

## 非目标

- 不提供脑震荡诊断、症状判读、返赛许可、训练计划、医疗建议、学校资格判断、法律责任或个体活动建议。
- 不收集未成年人症状、比赛录像、学校记录、医疗记录、教练沟通或头部冲击数据。

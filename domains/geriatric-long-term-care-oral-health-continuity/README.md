# Geriatric Long Term Care Oral Health Continuity

`geriatric-long-term-care-oral-health-continuity/` 负责整理老年口腔健康、长期照护、义齿、口腔卫生和功能连续性。

核心问题：

> 老年口腔健康不是护理细节；它影响进食、吞咽、感染、疼痛、尊严、照护工作流和机构生活质量。

## 先验位置

```text
主体持续性最大化
  -> 老年主体必须在衰弱、共病和照护依赖中维持进食、沟通和尊严
  -> 缺牙、义齿不适、口干、疼痛和口腔卫生不足会破坏摄食与社交
  -> 长期照护机构中的口腔问题会与营养、感染、吞咽和认知照护相互放大
  -> 因此老年口腔连续性是长期照护质量和有效时间的基础接口
```

## 关注对象

- 老年缺牙、义齿、龋病、牙周病、口干、吞咽、疼痛和口腔卫生支持。
- 长期照护机构、家庭照护、护理工作流、认知障碍、衰弱和医疗牙科协作。
- 营养、误吸风险、感染负担、尊严、社交和生活质量。
- 与 `frailty-sarcopenia-reserve/` 的关系：本域关注口腔输入和照护流程；衰弱域关注整体储备下降。

## Human Infra 模型链路

```text
老年口腔照护 / 服务 T
  -> 改变咀嚼、义齿适配、口干、疼痛、口腔卫生、吞咽和照护时间变量 X
  -> 改变老年主体摄食、感染和照护依赖状态 S
  -> 改变营养不良、误吸、社交退缩、疼痛和机构照护失败风险
  -> 增加尊严、生活质量和晚年有效时间
```

## 非目标

- 不提供护理医嘱、义齿调整、吞咽训练、药物更改、产品或个体照护方案。
- 不替代牙医、护理、老年医学、言语吞咽或营养专业评估。
- 不把机构照护问题简化为单一口腔卫生责任。

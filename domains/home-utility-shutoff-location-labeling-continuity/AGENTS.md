# home-utility-shutoff-location-labeling-continuity 目录说明

本目录承载家庭公用设施关断位置、标识和交接资料研究域，关注家庭关键服务接口如何从隐性知识变成可审查、可交接的恢复资料。

## 目录结构

```text
home-utility-shutoff-location-labeling-continuity/
├── AGENTS.md
└── README.md
```

## 文件职责

- `README.md`：定义家庭关键服务位置标识、地图、责任边界和主体连续性的关系。
- `AGENTS.md`：说明本目录职责、边界和维护规则。

## 维护规则

- 资料优先回到 Ready.gov、FEMA、USFA、地方公用事业安全资料和住房应急准备资料。
- 不写关断操作、电气面板操作、燃气阀门操作、管线维修、现场判断或灾后返家许可。
- 新增指标必须注明口径：服务类型、资料形态、权限边界、更新周期、交接对象和连续性后果。
- 涉及住址、照片、账户、门禁或设施位置时，必须保留隐私和滥用风险边界。

# Code of Conduct

Human Infra exists to improve the conditions under which people think, work, recover, and remain free. Collaboration in this repository must follow the same principle.

## Expected Behavior

- Be precise, direct, and respectful.
- Criticize claims, evidence, and structure, not the person.
- Preserve uncertainty when evidence is incomplete.
- Treat health, cognition, memory, and identity topics with appropriate seriousness.
- Respect boundaries around privacy, medical status, mental health, and lived experience.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory conduct.
- Medical, psychological, or neural intervention advice framed as a personal instruction.
- Content intended to manipulate, coerce, surveil, shame, or exploit people.
- Publishing private information without explicit permission.
- Using this project to justify unsafe human experimentation or organizational control.

## Reporting

If this repository becomes public or accepts external contributors, add a private reporting contact here before broad collaboration begins.

Until then, report issues through the repository maintainer or owner.
